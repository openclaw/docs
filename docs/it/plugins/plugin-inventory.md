# Inventario dei Plugin

Questa pagina è generata da `extensions/*/package.json`, `openclaw.plugin.json`,
e dalle esclusioni `files` del pacchetto npm radice. Rigenerala con:

```bash
pnpm plugins:inventory:gen
```

## Definizioni

- **Pacchetto npm core:** integrato nel pacchetto npm `openclaw` e disponibile senza un'installazione separata del plugin.
- **Pacchetto esterno ufficiale:** plugin mantenuto da OpenClaw omesso dal pacchetto npm core e installato tramite ClawHub e/o npm.
- **Solo checkout del sorgente:** plugin locale del repo omesso dagli artefatti npm pubblicati e non pubblicizzato come pacchetto installabile.

I checkout del sorgente sono diversi dalle installazioni npm: dopo `pnpm install`, i plugin
inclusi vengono caricati da `extensions/<id>`, quindi le modifiche locali e le dipendenze dell'area di lavoro locali del pacchetto
sono disponibili.
