# Inventario dei Plugin

Questa pagina viene generata da `extensions/*/package.json`, `openclaw.plugin.json`
e dalle esclusioni `files` del pacchetto npm radice. Rigenerala con:

```bash
pnpm plugins:inventory:gen
```

## Definizioni

- **Pacchetto npm principale:** integrato nel pacchetto npm `openclaw` e disponibile senza un'installazione separata del Plugin.
- **Pacchetto esterno ufficiale:** Plugin mantenuto da OpenClaw omesso dal pacchetto npm principale e installato tramite ClawHub e/o npm.
- **Solo checkout del sorgente:** Plugin locale del repository omesso dagli artefatti npm pubblicati e non pubblicizzato come pacchetto installabile.

I checkout del sorgente sono diversi dalle installazioni npm: dopo `pnpm install`, i Plugin inclusi
vengono caricati da `extensions/<id>`, quindi le modifiche locali e le dipendenze dell'area di lavoro
locali del pacchetto sono disponibili.
