# `openclaw tasks flow`

I comandi flow sono sottocomandi di `openclaw tasks`, non un comando `flows` autonomo.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Per la documentazione completa, vedi [Task Flow](/it/automation/taskflow) e il [riferimento CLI tasks](/cli/index#tasks).
