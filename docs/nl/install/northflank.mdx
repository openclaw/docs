# Northflank

Implementeer OpenClaw op Northflank met een sjabloon met één klik en open het via de web-Control UI.
Dit is de eenvoudigste route zonder "terminal op de server": Northflank voert de Gateway voor je uit.

## Aan de slag

1. Klik op [OpenClaw implementeren](https://northflank.com/stacks/deploy-openclaw) om het sjabloon te openen.
2. Maak een [account op Northflank](https://app.northflank.com/signup) aan als je er nog geen hebt.
3. Klik op **OpenClaw nu implementeren**.
4. Stel de vereiste omgevingsvariabele in: `OPENCLAW_GATEWAY_TOKEN` (gebruik een sterke willekeurige waarde).
5. Klik op **Stack implementeren** om het OpenClaw-sjabloon te bouwen en uit te voeren.
6. Wacht tot de implementatie is voltooid en klik daarna op **Resources bekijken**.
7. Open de OpenClaw-service.
8. Open de openbare OpenClaw-URL op `/openclaw` en maak verbinding met het geconfigureerde gedeelde geheim. Dit sjabloon gebruikt standaard `OPENCLAW_GATEWAY_TOKEN`; als je dit vervangt door wachtwoordauthenticatie, gebruik dan dat wachtwoord.

## Wat je krijgt

- Gehoste OpenClaw Gateway + Control UI
- Persistente opslag via Northflank Volume (`/data`), zodat `openclaw.json`,
  per-agent `auth-profiles.json`, kanaal-/providerstatus, sessies en
  werkruimte herimplementaties overleven

## Een kanaal verbinden

Gebruik de Control UI op `/openclaw` of voer `openclaw onboard` uit via SSH voor instructies voor het instellen van kanalen:

- [Telegram](/nl/channels/telegram) (snelst — alleen een bot-token)
- [Discord](/nl/channels/discord)
- [Alle kanalen](/nl/channels)
