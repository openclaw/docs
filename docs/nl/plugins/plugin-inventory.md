# Plugin-inventaris

Deze pagina wordt gegenereerd op basis van `extensions/*/package.json`, `openclaw.plugin.json`,
en de `files`-uitsluitingen van het npm-hoofdpakket. Genereer deze opnieuw met:

```bash
pnpm plugins:inventory:gen
```

## Definities

- **Core-npm-pakket:** ingebouwd in het `openclaw`-npm-pakket en beschikbaar zonder afzonderlijke Plugin-installatie.
- **Officieel extern pakket:** door OpenClaw onderhouden Plugin die is weggelaten uit het core-npm-pakket en wordt geïnstalleerd via ClawHub en/of npm.
- **Alleen broncheckout:** repo-lokale Plugin die is weggelaten uit gepubliceerde npm-artefacten en niet wordt aangekondigd als installeerbaar pakket.

Broncheckouts verschillen van npm-installaties: na `pnpm install` worden gebundelde
Plugins geladen vanuit `extensions/<id>`, zodat lokale bewerkingen en pakket-lokale workspace-
afhankelijkheden beschikbaar zijn.
