# `openclaw tasks flow`

Flow-opdrachten zijn subopdrachten van `openclaw tasks`, geen zelfstandige `flows`-opdracht.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```
