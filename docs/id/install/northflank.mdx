Deploy OpenClaw di Northflank dengan templat sekali klik dan akses melalui UI Kontrol web. Ini adalah cara termudah "tanpa terminal di server": Northflank menjalankan Gateway untuk Anda.

## Cara memulai

1. Klik [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) untuk membuka templat.
2. Buat [akun di Northflank](https://app.northflank.com/signup) jika Anda belum memilikinya.
3. Klik **Deploy OpenClaw now**.
4. Atur variabel lingkungan yang diperlukan: `OPENCLAW_GATEWAY_TOKEN` (gunakan nilai acak yang kuat).
5. Klik **Deploy stack** untuk membangun dan menjalankan templat OpenClaw.
6. Tunggu hingga deployment selesai, lalu klik **View resources**.
7. Buka layanan OpenClaw.
8. Buka URL publik OpenClaw di `/openclaw` dan hubungkan menggunakan rahasia bersama yang telah dikonfigurasi. Templat ini menggunakan `OPENCLAW_GATEWAY_TOKEN` secara default; jika Anda menggantinya dengan autentikasi kata sandi, gunakan kata sandi tersebut sebagai gantinya.

## Yang Anda dapatkan

- Gateway OpenClaw yang dihosting + UI Kontrol
- Penyimpanan persisten melalui Volume Northflank (`/data`) sehingga `openclaw.json`, `auth-profiles.json` per agen, status kanal/penyedia, sesi, dan ruang kerja tetap tersedia setelah deployment ulang

## Hubungkan kanal

Gunakan UI Kontrol di `/openclaw`, atau jalankan `openclaw onboard` melalui SSH untuk mendapatkan petunjuk penyiapan kanal:

- [Telegram](/id/channels/telegram) (paling cepat, hanya memerlukan token bot)
- [Discord](/id/channels/discord)
- [Semua kanal](/id/channels)
