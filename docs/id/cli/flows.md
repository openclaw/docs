# `openclaw tasks flow`

Perintah flow adalah subperintah dari `openclaw tasks`, bukan perintah `flows` yang berdiri sendiri.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```
