# BOOT.md

Добавьте сюда краткие и однозначные инструкции по запуску. Встроенный хук `boot-md` запускает этот файл один раз для каждого рабочего пространства агента при каждом запуске Gateway, если файл существует и содержит символы, отличные от пробельных. Несколько агентов, использующих общее рабочее пространство, инициируют только один запуск.

По умолчанию хук отключён. Сначала включите его:

```bash
openclaw hooks enable boot-md
```

Если пункт контрольного списка отправляет сообщение, используйте инструмент для отправки сообщений, а затем ответьте точным токеном без вывода `NO_REPLY` (без учёта регистра).
