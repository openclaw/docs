# BOOT.md

Добавьте здесь краткие и однозначные инструкции по запуску. Встроенный хук `boot-md` запускает этот файл один раз для каждого рабочего пространства агента при каждом запуске Gateway, если файл существует и содержит непробельные символы. Если несколько агентов используют общее рабочее пространство, запуск выполняется только один раз.

По умолчанию хук отключён. Сначала включите его:

```bash
openclaw hooks enable boot-md
```

Если пункт контрольного списка отправляет сообщение, используйте инструмент для сообщений, а затем ответьте точным токеном молчания `NO_REPLY` (без учёта регистра).
