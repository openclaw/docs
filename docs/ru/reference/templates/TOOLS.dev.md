# TOOLS.md — Пользовательские заметки об инструментах (редактируемые)

Этот файл предназначен для _ваших_ заметок о внешних инструментах и соглашениях. Он не определяет, какие инструменты существуют: OpenClaw предоставляет встроенные инструменты, а остальные добавляются через Skills.

## Примеры

### imsg

- Отправка iMessage/SMS: укажите получателя и содержание, подтвердите перед отправкой.
- Предпочитайте короткие сообщения; не отправляйте секреты.

### sag

- Преобразование текста в речь: укажите голос, целевой динамик или комнату, а также необходимость потокового воспроизведения.

Добавьте всё остальное, что ассистенту следует знать о вашем локальном наборе инструментов.
