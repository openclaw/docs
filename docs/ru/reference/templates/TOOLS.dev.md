# TOOLS.md — Заметки пользователя об инструментах (можно редактировать)

Этот файл предназначен для _ваших_ заметок о внешних инструментах и соглашениях. Он не определяет, какие инструменты существуют; OpenClaw предоставляет встроенные инструменты, а остальные добавляются с помощью Skills.

## Примеры

### imsg

- Отправка iMessage/SMS: укажите получателя и содержание, перед отправкой запросите подтверждение.
- Предпочтительны короткие сообщения; не отправляйте секреты.

### sag

- Преобразование текста в речь: укажите голос, целевой динамик или комнату, а также необходимость потоковой передачи.

Добавьте любую другую информацию о вашей локальной инструментальной среде, которую должен знать ассистент.
