# TOOLS.md - пользовательские заметки об инструментах (редактируемые)

Этот файл предназначен для _ваших_ заметок о внешних инструментах и соглашениях.
Он не определяет, какие инструменты существуют; OpenClaw предоставляет встроенные инструменты внутри системы.

## Примеры

### imsg

- Отправить iMessage/SMS: опишите кому/что, подтвердите перед отправкой.
- Предпочитайте короткие сообщения; не отправляйте секреты.

### sag

- Преобразование текста в речь: укажите голос, целевой динамик/комнату и нужно ли вести потоковую передачу.

Добавьте все остальное, что ассистент должен знать о вашей локальной цепочке инструментов.
