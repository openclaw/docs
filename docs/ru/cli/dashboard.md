# `openclaw dashboard`

Откройте интерфейс управления, используя текущую аутентификацию.

```bash
openclaw dashboard
openclaw dashboard --no-open
openclaw dashboard --yes
```

- `--no-open`: вывести URL-адрес, но не запускать браузер.
- `--yes`: при необходимости запустить или установить Gateway без запроса подтверждения.

Примечания:

- По возможности разрешает настроенные ссылки SecretRef в `gateway.auth.token`.
- Учитывает `gateway.tls.enabled`: для шлюзов с включённым TLS выводит и открывает URL-адреса интерфейса управления с `https://` и подключается по `wss://`.
- При привязке `lan` или `custom` с подстановочным адресом запуск на том же хосте всегда использует local loopback, поскольку подстановочный адрес не может быть адресом назначения для браузера. Незашифрованные привязки `tailnet` и `custom` также используют `127.0.0.1`, чтобы обеспечить браузеру безопасный контекст; для конкретных хостов с включённым TLS сохраняется настроенный адрес, чтобы имена в сертификате совпадали.
- Перед передачей аутентифицированного URL-адреса local loopback для привязки к конкретному интерфейсу команда проверяет настроенный интерфейс и убеждается, что он и `127.0.0.1` принадлежат одному процессу Gateway. При неоднозначной принадлежности прослушивающего процесса команда безопасно завершается с указаниями по проверке состояния.
- Для токенов под управлением SecretRef — разрешённых или неразрешённых — выводимый, копируемый или открываемый URL-адрес никогда не содержит токен, поэтому внешние секреты не попадают в вывод терминала, историю буфера обмена или аргументы запуска браузера.
- Если `gateway.auth.token` управляется через SecretRef, но не разрешён, команда вместо недопустимого заполнителя токена выводит URL-адрес без токена и рекомендации по устранению проблемы.
- Если передать через буфер обмена или открыть в браузере URL-адрес с токенной аутентификацией не удаётся, команда записывает в журнал безопасную подсказку по ручной аутентификации с указанием `OPENCLAW_GATEWAY_TOKEN`, `gateway.auth.token` и ключа фрагмента URL `token`, не выводя значение токена.
