# Плагин передачи файлов

Получайте, просматривайте и записывайте файлы на сопряжённых узлах с помощью специальных команд узла. Позволяет обойти усечение стандартного вывода bash благодаря использованию base64 через node.invoke для двоичных файлов размером до 16 МБ.

## Распространение

- Пакет: `@openclaw/file-transfer`
- Способ установки: входит в состав OpenClaw
