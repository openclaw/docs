# Плагин передачи файлов

Получайте, просматривайте список и записывайте файлы на сопряжённых узлах с помощью специализированных команд узла. Обходит усечение стандартного вывода bash, используя base64 через node.invoke для двоичных файлов размером до 16 МБ.
