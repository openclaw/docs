`openclaw security audit` выводит структурированные результаты проверок, сгруппированные по `checkId`. Эта страница представляет собой справочный каталог этих идентификаторов. Высокоуровневую модель угроз и рекомендации по усилению защиты см. в разделе [Безопасность](/ru/gateway/security).

Некоторые проверки выполняются только при запуске `openclaw security audit --deep`: сканирование кода плагинов и Skills (`plugins.code_safety*`, `skills.code_safety*`) и проверки с активным зондированием Gateway (`gateway.probe_*`). Все остальные проверки из этой таблицы выполняются при обычном запуске `openclaw security audit`.

Уровень серьёзности вида `warn/critical` означает, что один и тот же `checkId` может выводиться с любым из этих уровней в зависимости от конфигурации (например, от того, доступен ли Gateway удалённо).
