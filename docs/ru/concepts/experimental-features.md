Экспериментальные функции — это предварительные возможности, включаемые явными флагами. Им требуется больше практической проверки, прежде чем они получат стабильные настройки по умолчанию или долгосрочные гарантии совместимости.

- По умолчанию отключены, если документация явно не предписывает включить функцию.
- Структура и поведение могут меняться быстрее, чем стабильная конфигурация.
- Если уже существует стабильный способ, предпочитайте его.
- Разворачивайте широко только после предварительного тестирования в небольшой среде.

## Облегчённый режим локальной модели

`agents.defaults.experimental.localModelLean: true` при каждом ходе исключает из непосредственно доступного агенту набора ресурсоёмкие необязательные инструменты: `browser`, `cron`, `message`, `image_generate`, `music_generate`, `video_generate`, `tts` и `pdf`. Явно разрешённые или необходимые для доставки инструменты остаются доступными, хотя Tool Search может добавить их в каталог вместо непосредственного предоставления. Если параметр `tools.toolSearch` ещё не задан, облегчённый режим также по умолчанию переводит каталоги плагинов, MCP и клиентов на структурированный Tool Search (`tool_search`, `tool_describe`, `tool_call`). Используйте `agents.list[].experimental.localModelLean`, чтобы применить этот режим только к одному агенту.

Если вы уже настраиваете Tool Search глобально, OpenClaw не изменяет эту конфигурацию. Задайте `tools.toolSearch: false`, чтобы отказаться от стандартного включения Tool Search в облегчённом режиме.

В структурированном режиме `tools` облегчённые запуски сохраняют `exec` непосредственно видимым рядом с элементами управления Tool Search, чтобы локальные модели, оптимизированные для программирования, по-прежнему могли выбирать привычный путь через оболочку. Меняется только видимость схемы: обычные политики инструментов, изоляция в песочнице и подтверждения выполнения `exec` продолжают действовать. Явные режимы `code` и `directory` сохраняют обычное поведение Compaction.

### Почему выбраны эти инструменты

У этих инструментов самые объёмные описания, наиболее широкие структуры параметров или самая высокая вероятность отвлечь небольшую модель от обычного процесса программирования и общения. Для бэкенда с небольшим контекстом или более строгой совместимостью с OpenAI это определяет разницу между следующими ситуациями:

- Схемы инструментов помещаются в запрос или вытесняют историю диалога.
- Модель выбирает правильный инструмент или формирует некорректные вызовы из-за слишком большого числа похожих схем.
- Адаптер Chat Completions укладывается в ограничения структурированного вывода или получает ошибку 400 из-за размера данных вызова инструмента.

Их удаление лишь сокращает список непосредственно доступных инструментов. У модели по-прежнему остаются `read`, `write`, `edit`, `exec`, `apply_patch`, распознавание изображений, веб-поиск и получение данных из сети (если настроены), память, а также инструменты сеансов и агентов. Дополнительные каталоги остаются доступны через Tool Search, если не задано `tools.toolSearch: false`; явные разрешения инструментов позволяют вернуть облегчённому агенту нужные инструменты для ограниченного рабочего процесса.

### Когда включать

Включайте облегчённый режим после того, как убедитесь, что модель может взаимодействовать с Gateway, но полноценные ходы агента работают некорректно:

1. Команда `openclaw infer model run --gateway --model <ref> --prompt "Reply with exactly: pong"` выполняется успешно.
2. Обычный ход агента завершается с ошибкой из-за некорректных вызовов инструментов, слишком больших запросов или игнорирования инструментов моделью.
3. Включение `localModelLean: true` устраняет проблему.

### Когда оставлять выключенным

Если ваш бэкенд корректно работает с полной стандартной средой выполнения, не включайте этот режим. Это обходное решение для локальных систем, которым нужен меньший набор инструментов, а не настройка по умолчанию для размещённых моделей или производительных локальных систем.

Облегчённый режим не заменяет `tools.profile`, `tools.allow`/`tools.deny` или аварийный параметр модели `compat.supportsTools: false`. Если конкретному агенту постоянно требуется более узкий набор инструментов, предпочитайте эти стабильные параметры.

### Включение

```json5
{
  agents: {
    defaults: {
      experimental: {
        localModelLean: true,
      },
    },
  },
}
```

Только для одного агента:

```json5
{
  agents: {
    list: [
      {
        id: "local",
        model: "lmstudio/gemma-4-e4b-it",
        experimental: {
          localModelLean: true,
        },
      },
    ],
  },
}
```

После изменения флага перезапустите Gateway. Фильтрация облегчённого режима удаляет `browser`, `cron`, `message`, `image_generate`, `music_generate`, `video_generate`, `tts` и `pdf`, если они явно не сохранены с помощью `tools.allow` или `tools.alsoAllow`; Tool Search может по-прежнему добавлять сохранённые инструменты в каталог вместо их непосредственного предоставления.

## Экспериментальный не означает скрытый

Экспериментальный статус функции должен быть явно указан в документации и в самом пути конфигурации, а не скрываться за параметром по умолчанию, который выглядит стабильным.
