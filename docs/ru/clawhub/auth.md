# Аутентификация

ClawHub использует GitHub для входа через веб-интерфейс. CLI использует токены API ClawHub, созданные
через эту учётную запись после входа.

## Вход через веб-интерфейс

Используйте GitHub для входа на сайте [clawhub.ai](https://clawhub.ai).

Для удалённых, заблокированных или отключённых учётных записей обычный вход в ClawHub недоступен.
Если после входа вы снова оказываетесь в состоянии без авторизации, возможно, у вашей учётной записи
есть ограничения. Если ваша учётная запись была заблокирована или отключена, используйте
[форму апелляции ClawHub](https://appeals.openclaw.ai/), если считаете, что это
ошибка.

## Вход через CLI

Стандартная процедура входа через CLI открывает браузер:

```bash
clawhub login
clawhub whoami
```

Что происходит:

1. CLI запускает временный сервер обратного вызова на `127.0.0.1`.
2. В браузере открывается страница входа в ClawHub.
3. После входа через GitHub ClawHub создаёт токен API.
4. Браузер перенаправляется обратно на локальный адрес обратного вызова.
5. CLI сохраняет токен в файле конфигурации ClawHub.

Если браузер не может обратиться к локальному адресу обратного вызова из-за правил брандмауэра, VPN или
прокси-сервера, используйте процедуру входа с помощью токена без браузера.

## Вход без браузера

Создайте токен в веб-интерфейсе ClawHub, затем передайте его в CLI:

```bash
clawhub login --token clh_...
```

Используйте эту процедуру для серверов, заданий CI и сред, доступных только через терминал.

При работе в удалённой оболочке, если браузер можно открыть на другом устройстве, выполните:

```bash
clawhub login --device
```

CLI выводит одноразовый код и ожидает, пока вы авторизуете его на странице
`https://clawhub.ai/cli/device`.

## Хранение токенов

Стандартные пути к файлу конфигурации:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` или `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

Чтобы переопределить путь, выполните:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

Чтобы вывести сохранённый токен для настройки CI, выполните:

```bash
clawhub token
```

## Отзыв

Токены API можно отозвать в веб-интерфейсе ClawHub.

При использовании отозванного, недействительного или отсутствующего токена возвращается `401 Unauthorized`. Снова выполните вход
с помощью `clawhub login` или предоставьте новый токен с помощью `clawhub login --token`.

Удалённые, заблокированные или отключённые учётные записи не могут продолжать использовать существующие токены API.
Если ваша учётная запись была заблокирована или отключена, используйте
[форму апелляции ClawHub](https://appeals.openclaw.ai/), если считаете, что это
ошибка.
