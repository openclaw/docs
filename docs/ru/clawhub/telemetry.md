# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчета агрегированного количества установок.

## Когда собирается телеметрия

Телеметрия отправляется только когда:

- Вы вошли в систему в CLI.
- Вы выполняете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в систему, ничего не передается.

## Что мы собираем

При каждом переданном `clawhub install` CLI отправляет одно событие установки по мере возможности.

Событие включает:

- `slug`: слаг установленного навыка.
- `version`: установленная версия, если известна.

### Что мы _не_ собираем

- Никакие пути к папкам или идентификаторы, производные от папок.
- Никакое содержимое файлов.
- Никакие журналы отдельных запусков, промпты или другой вывод CLI.

## Количество установок

ClawHub поддерживает агрегированные счетчики для каждого навыка:

- `installsAllTime`: уникальные пользователи, которые сообщили хотя бы об одной установке навыка через CLI.
- `installsCurrent`: уникальные пользователи, которые сообщили об установке и не удалили свою
  телеметрию.

## Прозрачность и пользовательские настройки

Все видят только **агрегированные счетчики установок**.

Удаление вашей учетной записи также удаляет ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную окружения:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Когда она задана, CLI не будет отправлять телеметрию установок.
