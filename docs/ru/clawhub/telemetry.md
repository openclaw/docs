# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчета агрегированного количества установок.

## Когда собирается телеметрия

Телеметрия отправляется только когда:

- Вы вошли в систему в CLI.
- Вы запускаете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в систему, ничего не отправляется.

## Что мы собираем

При каждом зарегистрированном `clawhub install` CLI отправляет одно событие установки по мере возможности.

Событие включает:

- `slug`: slug установленного навыка.
- `version`: установленная версия, если известна.

### Что мы _не_ собираем

- Никаких путей к папкам или идентификаторов, производных от папок.
- Никакого содержимого файлов.
- Никаких журналов отдельных запусков, prompts или другого вывода CLI.

## Количество установок

ClawHub поддерживает агрегированные счетчики для каждого навыка:

- `installsAllTime`: уникальные пользователи, которые сообщили как минимум об одной установке навыка через CLI.
- `installsCurrent`: уникальные пользователи, которые сообщили об установке и не удалили свою
  телеметрию.

## Прозрачность + пользовательские элементы управления

Все видят только **агрегированные счетчики установок**.

Удаление учетной записи также удаляет ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную среды:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Если она задана, CLI не будет отправлять телеметрию установок.
