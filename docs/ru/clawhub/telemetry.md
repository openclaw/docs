# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчета агрегированного числа установок.

## Когда собирается телеметрия

Телеметрия отправляется только когда:

- Вы вошли в систему в CLI.
- Вы запускаете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в систему, ничего не отправляется.

## Что мы собираем

При каждом отправляемом `clawhub install` CLI отправляет одно событие установки по мере возможности.

Событие включает:

- `slug`: slug установленного skill.
- `version`: установленная версия, если известна.

### Что мы _не_ собираем

- Никаких путей к папкам или идентификаторов, полученных из путей к папкам.
- Никакого содержимого файлов.
- Никаких журналов отдельных запусков, prompts или другого вывода CLI.

## Счетчики установок

ClawHub поддерживает агрегированные счетчики для каждого skill:

- `installsAllTime`: уникальные пользователи, которые сообщили как минимум об одной установке skill через CLI.
- `installsCurrent`: уникальные пользователи, которые сообщили об установке и не удалили свою
  телеметрию.

## Прозрачность и пользовательские настройки

Все видят только **агрегированные счетчики установок**.

Удаление вашей учетной записи также удаляет ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную окружения:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Когда она задана, CLI не будет отправлять телеметрию установок.
