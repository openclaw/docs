# Телеметрия

ClawHub использует минимальную телеметрию CLI для расчета агрегированных счетчиков установок.

## Когда собирается телеметрия

Телеметрия отправляется только если:

- Вы вошли в учетную запись в CLI.
- Вы запускаете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в учетную запись, ничего не отправляется.

## Что мы собираем

При каждом учтенном `clawhub install` CLI отправляет одно событие установки по принципу best-effort.

Событие включает:

- `slug`: slug установленного skill.
- `version`: установленная версия, если она известна.

### Что мы _не_ собираем

- Никаких путей к папкам или идентификаторов, полученных из путей к папкам.
- Никакого содержимого файлов.
- Никаких журналов отдельных запусков, промптов или другого вывода CLI.

## Счетчики установок

ClawHub поддерживает агрегированные счетчики для каждого skill:

- `installsAllTime`: уникальные пользователи, сообщившие хотя бы об одной установке skill через CLI.
- `installsCurrent`: уникальные пользователи, сообщившие об установке и не удалившие свои
  данные телеметрии.

## Прозрачность и пользовательские элементы управления

Все видят только **агрегированные счетчики установок**.

Удаление вашей учетной записи также удаляет ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную окружения:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Если она задана, CLI не будет отправлять телеметрию установок.
