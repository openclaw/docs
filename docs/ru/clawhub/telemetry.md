# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчёта совокупного количества установок.

## Когда собирается телеметрия

Телеметрия отправляется только в следующих случаях:

- Вы вошли в систему через CLI.
- Вы выполняете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. раздел «Как отключить» ниже).

Если вы не вошли в систему, никакие данные не передаются.

## Какие данные мы собираем

Для каждого зарегистрированного `clawhub install` CLI отправляет одно событие установки, если это возможно.

Событие содержит:

- `slug`: идентификатор установленного навыка.
- `version`: установленная версия, если она известна.

### Какие данные мы _не_ собираем

- Пути к папкам и полученные из них идентификаторы.
- Содержимое файлов.
- Журналы отдельных запусков, запросы и другие выходные данные CLI.

## Количество установок

ClawHub ведёт совокупные счётчики для каждого навыка:

- `installsAllTime`: уникальные пользователи, которые сообщили хотя бы об одной установке навыка через CLI.
- `installsCurrent`: уникальные пользователи, которые сообщили об установке и не удалили свои
  данные телеметрии.

## Прозрачность и пользовательское управление

Все видят только **совокупные счётчики установок**.

При удалении учётной записи также удаляются ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную среды:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

После этого CLI не будет отправлять телеметрию установки.
