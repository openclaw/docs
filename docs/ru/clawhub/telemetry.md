# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчета совокупного количества установок.

## Когда собирается телеметрия

Телеметрия отправляется только когда:

- Вы вошли в систему в CLI.
- Вы запускаете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в систему, ничего не отправляется.

## Что мы собираем

При каждом учитываемом `clawhub install` CLI отправляет одно событие установки по мере возможности.

Событие включает:

- `slug`: слаг установленного навыка.
- `version`: установленная версия, если она известна.

### Что мы _не_ собираем

- Никакие пути к папкам или идентификаторы, полученные из папок.
- Никакое содержимое файлов.
- Никакие журналы отдельных запусков, промпты или другой вывод CLI.

## Количество установок

ClawHub ведет совокупные счетчики по каждому навыку:

- `installsAllTime`: уникальные пользователи, сообщившие хотя бы об одной установке навыка через CLI.
- `installsCurrent`: уникальные пользователи, сообщившие об установке и не удалившие свои
  телеметрические данные.

## Прозрачность и пользовательские элементы управления

Все видят только **агрегированные счетчики установок**.

Удаление вашей учетной записи также удаляет ваши телеметрические данные.

## Как отключить телеметрию

Задайте переменную окружения:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Если она задана, CLI не будет отправлять телеметрию установок.
