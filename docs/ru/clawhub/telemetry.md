# Телеметрия

ClawHub использует минимальную телеметрию CLI для подсчета агрегированных установок.

## Когда собирается телеметрия

Телеметрия отправляется только если:

- Вы вошли в учетную запись в CLI.
- Вы запускаете `clawhub install <slug>`.
- Телеметрия **не отключена** (см. «Как отключить» ниже).

Если вы не вошли в учетную запись, ничего не отправляется.

## Что мы собираем

При каждом зарегистрированном `clawhub install` CLI отправляет одно событие установки по возможности.

Событие включает:

- `slug`: slug установленного навыка.
- `version`: установленная версия, если она известна.

### Что мы _не_ собираем

- Никаких путей к папкам или идентификаторов, полученных из папок.
- Никакого содержимого файлов.
- Никаких журналов отдельных запусков, prompts или другого вывода CLI.

## Счетчики установок

ClawHub поддерживает агрегированные счетчики по каждому навыку:

- `installsAllTime`: уникальные пользователи, которые сообщили хотя бы об одной установке навыка через CLI.
- `installsCurrent`: уникальные пользователи, которые сообщили об установке и не удалили свою
  телеметрию.

## Прозрачность + пользовательские элементы управления

Все видят только **агрегированные счетчики установок**.

Удаление вашей учетной записи также удаляет ваши данные телеметрии.

## Как отключить телеметрию

Задайте переменную окружения:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Если она задана, CLI не будет отправлять телеметрию установок.
