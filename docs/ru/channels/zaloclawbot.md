OpenClaw подключается к Zalo ClawBot через указанный в каталоге внешний
Plugin `@zalo-platforms/openclaw-zaloclawbot`. Для входа используется QR-код
Zalo Mini App.

## Совместимость

| Версия Plugin | Версия OpenClaw | npm dist-tag | Статус        |
| -------------- | ---------------- | ------------ | ------------- |
| 0.1.x          | >=2026.4.10      | `latest`     | Активно / бета |

## Предварительные требования

- Node.js **>= 22**
- [OpenClaw](https://docs.openclaw.ai/install) должен быть установлен (доступен CLI `openclaw`).
- Учетная запись Zalo на мобильном устройстве для сканирования QR-кода входа.

## Установка через onboard (рекомендуется)

Запустите мастер первоначальной настройки OpenClaw и выберите **Zalo ClawBot** в меню каналов:

```bash
openclaw onboard
```

Мастер устанавливает Plugin из официального каталога (с проверкой целостности), отображает QR-код для входа прямо в терминале и завершает настройку канала после того, как вы отсканируете его в приложении Zalo. Дополнительные команды не требуются.

## Ручная установка

Чтобы добавить канал в уже настроенный Gateway, выполните следующие шаги:

### 1. Установите Plugin

```bash
openclaw plugins install "@zalo-platforms/openclaw-zaloclawbot@0.1.4"
```

Используйте точную зафиксированную версию, указанную выше (она соответствует записи в официальном каталоге), чтобы OpenClaw при установке проверил пакет по хешу целостности из каталога.

### 2. Включите Plugin в конфигурации

```bash
openclaw config set plugins.entries.openclaw-zaloclawbot.enabled true
```

### 3. Сгенерируйте QR-код и войдите

```bash
openclaw channels login --channel openclaw-zaloclawbot
```

Отсканируйте QR-код, отображенный в терминале, с помощью мобильного приложения Zalo, примите Условия использования внутри Zalo Mini App и авторизуйте сеанс.

### 4. Перезапустите Gateway

```bash
openclaw gateway restart
```

## Как это работает

В отличие от стандартного канала Zalo для разработчиков, который требует зарегистрировать собственную Zalo Official Account (OA) и вставить статические учетные данные разработчика, Zalo ClawBot работает как **персональный помощник, привязанный к владельцу**, используя общую официальную инфраструктуру:

1. **Безопасная первоначальная настройка:** QR-код ведет в безопасное Zalo Mini App, которое привязывает новый выделенный приватный бот в рамках общей официальной OA напрямую к вашему Zalo User ID.
2. **Конфиденциальность с привязкой к владельцу:** По замыслу бот ограничен общением _только_ со своим владельцем. Сообщения от других пользователей отбрасываются на уровне платформы, что делает подключение приватным и безопасным.
3. **Официальный путь API:** Plugin использует API Zalo Bot Platform вместо
   автоматизации браузера или веб-сеанса.

## Под капотом

Plugin Zalo ClawBot взаимодействует с API Zalo через постоянный цикл получения сообщений long-polling. Для поддержания чистой и легковесной среды выполнения:

- Соединения long-poll используют конечную точку `getUpdates`.
- Webhook по умолчанию отключены для локальных запусков Gateway на рабочем столе/в терминале.
- Сообщения обрабатываются на стороне клиента и напрямую сопоставляются с вашей локальной средой выполнения агента.

Внешний Plugin управляет учетными данными бота в каталоге состояния OpenClaw.
Считайте этот каталог конфиденциальным и включайте его в ту же политику контроля доступа и резервного копирования, что и остальное состояние OpenClaw.

## Устранение неполадок

- **Тайм-аут входа по QR-коду:** Токен входа (`zbsk`) истекает через 5 минут по соображениям безопасности. Если срок действия QR-кода истечет до того, как вы его отсканируете, просто повторно выполните команду входа, чтобы сгенерировать новый.
- **Gateway не загружается:** Убедитесь, что версия хоста OpenClaw — `2026.4.10` или выше. Старые версии не поддерживают журнал установки внешних npm-Plugin.
