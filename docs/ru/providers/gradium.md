[Gradium](https://gradium.ai) — провайдер преобразования текста в речь для OpenClaw. Он создаёт стандартные аудиоответы (WAV), вывод Opus, совместимый с голосовыми сообщениями, и аудио u-law с частотой 8 кГц для телефонии.

| Свойство              | Значение                             |
| --------------------- | ------------------------------------ |
| Идентификатор провайдера | `gradium`                            |
| Аутентификация        | `GRADIUM_API_KEY` или конфигурация `apiKey` |
| Базовый URL           | `https://api.gradium.ai` (по умолчанию)   |
| Голос по умолчанию    | `Emma` (`YTpq7expH9539ERJ`)          |

## Установка плагина

Gradium — официальный внешний плагин. Установите его, затем перезапустите Gateway:

```bash
openclaw plugins install @openclaw/gradium-speech
openclaw gateway restart
```

## Настройка

Создайте API-ключ Gradium, затем укажите его с помощью переменной окружения или ключа конфигурации. Конфигурация имеет приоритет над переменной окружения.

<Tabs>
  <Tab title="Переменная окружения">
    ```bash
    export GRADIUM_API_KEY="gsk_..."
    ```
  </Tab>

  <Tab title="Ключ конфигурации">
    ```json5
    {
      messages: {
        tts: {
          auto: "always",
          provider: "gradium",
          providers: {
            gradium: {
              apiKey: "${GRADIUM_API_KEY}",
            },
          },
        },
      },
    }
    ```
  </Tab>
</Tabs>

## Конфигурация

```json5
{
  messages: {
    tts: {
      auto: "always",
      provider: "gradium",
      providers: {
        gradium: {
          speakerVoiceId: "YTpq7expH9539ERJ",
          // apiKey: "${GRADIUM_API_KEY}",
          // baseUrl: "https://api.gradium.ai",
        },
      },
    },
  },
}
```

| Ключ                                            | Тип    | Описание                                                                                               |
| ----------------------------------------------- | ------ | ------------------------------------------------------------------------------------------------------ |
| `messages.tts.providers.gradium.apiKey`         | строка | Разрешённый API-ключ. Поддерживает `${ENV}` и ссылки на секреты.                                                    |
| `messages.tts.providers.gradium.baseUrl`        | строка | URL API Gradium по HTTPS на `api.gradium.ai`. Завершающие косые черты удаляются. По умолчанию: `https://api.gradium.ai`. |
| `messages.tts.providers.gradium.speakerVoiceId` | строка | Идентификатор голоса по умолчанию, используемый при отсутствии переопределяющей директивы.                                  |

Формат вывода выбирается автоматически в зависимости от целевой поверхности (см. раздел [Вывод](#output)) и не настраивается в `openclaw.json`.

### Переопределение голоса для отдельного сообщения

Если активная политика синтеза речи разрешает переопределение голоса, переключите голос непосредственно в тексте с помощью токена директивы (все следующие варианты эквивалентны и принимают нативный идентификатор голоса провайдера):

```text
/voice:LFZvm12tW_z0xfGo
/voice_id:LFZvm12tW_z0xfGo
/voiceid:LFZvm12tW_z0xfGo
/gradium_voice:LFZvm12tW_z0xfGo
/gradiumvoice:LFZvm12tW_z0xfGo
```

Если политика синтеза речи запрещает переопределение голоса, директива обрабатывается, но игнорируется.

## Вывод

Формат вывода выбирается в зависимости от целевой поверхности; провайдер не синтезирует другие форматы.

| Цель                | Формат      | Расширение файла | Частота дискретизации | Флаг совместимости с голосовыми сообщениями |
| ------------------- | ----------- | ---------------- | --------------------- | ------------------------------------------- |
| Стандартное аудио   | `wav`       | `.wav`   | провайдер             | нет                                         |
| Голосовое сообщение | `opus`      | `.opus`  | провайдер             | да                                          |
| Телефония           | `ulaw_8000` | н/д              | 8 кГц                 | н/д                                         |

## Порядок автоматического выбора

Среди настроенных провайдеров TTS порядок автоматического выбора Gradium — `30`. О том, как OpenClaw выбирает активного провайдера, когда `messages.tts.provider` не закреплён, см. в разделе [Преобразование текста в речь](/ru/tools/tts).
