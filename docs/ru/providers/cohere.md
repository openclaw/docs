[Cohere](https://cohere.com) предоставляет совместимый с OpenAI инференс через свой Compatibility API. В переходный период вынесения во внешний пакет OpenClaw включает провайдер Cohere в комплект поставки, а также публикует его как официальный внешний Plugin.

| Свойство              | Значение                                                     |
| --------------------- | ------------------------------------------------------------ |
| Идентификатор провайдера | `cohere`                                                  |
| Plugin                | включён в комплект на переходный период; официальный внешний пакет |
| Переменная среды аутентификации | `COHERE_API_KEY`                                    |
| Флаг первоначальной настройки | `--auth-choice cohere-api-key`                        |
| Прямой флаг CLI       | `--cohere-api-key <key>`                                     |
| API                   | совместимый с OpenAI (`openai-completions`)                   |
| Базовый URL           | `https://api.cohere.ai/compatibility/v1`                      |
| Модель по умолчанию   | `cohere/command-a-plus-05-2026`                               |
| Контекстное окно      | 128 000 токенов                                               |

## Встроенный каталог

| Ссылка на модель                      | Входные данные | Контекст | Макс. вывод | Примечания                                                   |
| ------------------------------------- | -------------- | -------- | ------------ | ------------------------------------------------------------ |
| `cohere/command-a-plus-05-2026`       | текст, изображение | 128 000 | 64 000    | По умолчанию; флагманская агентная модель с поддержкой рассуждений |
| `cohere/command-a-03-2025`            | текст          | 256 000  | 8 000        | Предыдущая модель Command A                                   |
| `cohere/command-a-reasoning-08-2025`  | текст          | 256 000  | 32 000       | Агентные рассуждения и использование инструментов            |
| `cohere/command-a-vision-07-2025`     | текст, изображение | 128 000 | 8 000     | Анализ изображений и документов; без использования инструментов |
| `cohere/north-mini-code-1-0`          | текст, изображение | 256 000 | 64 000    | Агентное программирование; рассуждения; бесплатные лимиты     |

Модели Cohere с поддержкой рассуждений поддерживают два режима рассуждений Compatibility API. OpenClaw сопоставляет **выкл.** со значением `none`, а каждый включённый уровень размышлений — со значением `high`. Command A Vision не поддерживает использование инструментов, поэтому OpenClaw отключает инструменты агента для этой модели.

## Начало работы

1. Cohere входит в состав текущих пакетов OpenClaw. Если он отсутствует, установите внешний пакет и перезапустите Gateway:

```bash
openclaw plugins install @openclaw/cohere-provider
openclaw gateway restart
```

2. Создайте ключ API Cohere.
3. Запустите первоначальную настройку:

```bash
openclaw onboard --non-interactive \
  --auth-choice cohere-api-key \
  --cohere-api-key "$COHERE_API_KEY"
```

4. Убедитесь, что каталог доступен:

```bash
openclaw models list --provider cohere
```

Первоначальная настройка назначает Cohere основной моделью, только если основная модель ещё не настроена.

## Настройка только через переменные среды

Сделайте `COHERE_API_KEY` доступной процессу Gateway, затем выберите модель Cohere:

```json5
{
  agents: {
    defaults: {
      model: { primary: "cohere/command-a-plus-05-2026" },
    },
  },
}
```

<Note>
Если Gateway работает как демон или в Docker, задайте `COHERE_API_KEY` для этой службы. Экспорт переменной только в интерактивной оболочке не делает её доступной уже запущенному Gateway.
</Note>
