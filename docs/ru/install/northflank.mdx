Разверните OpenClaw на Northflank с помощью шаблона в один клик и получите доступ к нему через веб-интерфейс управления. Это самый простой способ обойтись без терминала на сервере: Northflank запускает Gateway за вас.

## Начало работы

1. Нажмите [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw), чтобы открыть шаблон.
2. Создайте [учётную запись Northflank](https://app.northflank.com/signup), если у вас её ещё нет.
3. Нажмите **Deploy OpenClaw now**.
4. Задайте обязательную переменную среды: `OPENCLAW_GATEWAY_TOKEN` (используйте надёжное случайное значение).
5. Нажмите **Deploy stack**, чтобы собрать и запустить шаблон OpenClaw.
6. Дождитесь завершения развёртывания, затем нажмите **View resources**.
7. Откройте сервис OpenClaw.
8. Откройте публичный URL-адрес OpenClaw по адресу `/openclaw` и подключитесь, используя настроенный общий секрет. По умолчанию этот шаблон использует `OPENCLAW_GATEWAY_TOKEN`; если вы замените его аутентификацией по паролю, используйте вместо него этот пароль.

## Что вы получите

- Размещённые Gateway OpenClaw и веб-интерфейс управления
- Постоянное хранилище на базе тома Northflank (`/data`), благодаря которому `openclaw.json`, отдельные для каждого агента `auth-profiles.json`, состояние каналов и провайдеров, сеансы и рабочая область сохраняются при повторных развёртываниях

## Подключение канала

Используйте веб-интерфейс управления по адресу `/openclaw` или запустите `openclaw onboard` через SSH, чтобы получить инструкции по настройке каналов:

- [Telegram](/ru/channels/telegram) (самый быстрый вариант — нужен только токен бота)
- [Discord](/ru/channels/discord)
- [Все каналы](/ru/channels)
