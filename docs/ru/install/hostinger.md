Запустите постоянный Gateway OpenClaw на [Hostinger](https://www.hostinger.com/openclaw): как управляемое развертывание **в один клик** или как самостоятельно администрируемую установку на **VPS**.

## Предварительные требования

- Учетная запись Hostinger ([регистрация](https://www.hostinger.com/openclaw))
- Около 5–10 минут

## Вариант A: OpenClaw в один клик

Hostinger берет на себя инфраструктуру, Docker и автоматические обновления. Это самый быстрый способ запустить экземпляр.

<Steps>
  <Step title="Приобретение и запуск">
    1. На [странице OpenClaw в Hostinger](https://www.hostinger.com/openclaw) выберите тариф Managed OpenClaw и завершите оформление заказа.

    <Note>
    При оформлении заказа можно выбрать предоплаченные кредиты **Ready-to-Use AI**, которые мгновенно интегрируются в OpenClaw — внешние учетные записи и API-ключи других поставщиков не требуются. Общение можно начать сразу. Также во время настройки можно указать собственный ключ Anthropic, OpenAI, Google Gemini или xAI.
    </Note>

  </Step>

  <Step title="Выбор канала обмена сообщениями">
    Выберите один или несколько каналов для подключения:

    - **WhatsApp** — отсканируйте QR-код, показанный в мастере настройки.
    - **Telegram** — вставьте токен бота, полученный у [BotFather](https://t.me/BotFather).

  </Step>

  <Step title="Завершение установки">
    Нажмите **Finish**, чтобы развернуть экземпляр. Когда он будет готов, откройте панель управления OpenClaw через **OpenClaw Overview** в hPanel.
  </Step>

</Steps>

## Вариант B: OpenClaw на VPS

Этот вариант предоставляет больше контроля над сервером. Hostinger развертывает OpenClaw через Docker на вашем VPS; управление выполняется через **Docker Manager** в hPanel.

<Steps>
  <Step title="Приобретение VPS">
    1. На [странице OpenClaw в Hostinger](https://www.hostinger.com/openclaw) выберите тариф OpenClaw on VPS и завершите оформление заказа.

    <Note>
    При оформлении заказа можно выбрать предоплаченные кредиты **Ready-to-Use AI**. Они мгновенно интегрируются в OpenClaw, поэтому общение можно начать без внешних учетных записей и API-ключей других поставщиков.
    </Note>

  </Step>

  <Step title="Настройка OpenClaw">
    После подготовки VPS заполните поля конфигурации:

    - **Gateway token** — создается автоматически; сохраните его для дальнейшего использования.
    - **WhatsApp number** — ваш номер с кодом страны (необязательно).
    - **Telegram bot token** — токен от [BotFather](https://t.me/BotFather) (необязательно).
    - **API keys** — требуются только в том случае, если при оформлении заказа вы не выбрали кредиты Ready-to-Use AI.

  </Step>

  <Step title="Запуск OpenClaw">
    Нажмите **Deploy**. После запуска откройте панель управления OpenClaw из hPanel, нажав **Open**.
  </Step>

</Steps>

Просмотр журналов, перезапуск и обновление выполняются через интерфейс Docker Manager в hPanel. Для обновления нажмите **Update** в Docker Manager, чтобы загрузить последнюю версию образа.

## Проверка настройки

Отправьте сообщение «Привет» своему ассистенту через подключенный канал. OpenClaw ответит и поможет задать первоначальные предпочтения.

## Устранение неполадок

**Панель управления не загружается** — подождите несколько минут, пока завершится подготовка контейнера, а затем проверьте журналы Docker Manager в hPanel.

**Контейнер Docker постоянно перезапускается** — откройте журналы Docker Manager и проверьте наличие ошибок конфигурации, например отсутствующих токенов или недействительных API-ключей.

**Бот Telegram не отвечает** — если требуется сопряжение для личных сообщений, неизвестный отправитель вместо ответа получает короткий код сопряжения. Подтвердите его в чате панели управления OpenClaw или выполните `openclaw pairing approve telegram <CODE>`, если у вас есть доступ к оболочке контейнера. См. раздел [«Сопряжение»](/ru/channels/pairing).
