Запустите постоянный OpenClaw Gateway на [Hostinger](https://www.hostinger.com/openclaw) через управляемое развертывание **1-Click** или установку на **VPS**.

## Предварительные требования

- Аккаунт Hostinger ([регистрация](https://www.hostinger.com/openclaw))
- Примерно 5-10 минут

## Вариант A: 1-Click OpenClaw

Самый быстрый способ начать работу. Hostinger управляет инфраструктурой, Docker и автоматическими обновлениями.

<Steps>
  <Step title="Купите и запустите">
    1. На [странице Hostinger OpenClaw](https://www.hostinger.com/openclaw) выберите управляемый тариф OpenClaw и завершите оформление заказа.

    <Note>
    При оформлении заказа вы можете выбрать предварительно купленные и мгновенно интегрированные в OpenClaw кредиты **готового к использованию ИИ** -- внешние аккаунты или API-ключи других провайдеров не нужны. Можно сразу начать общение. Либо укажите собственный ключ от Anthropic, OpenAI, Google Gemini или xAI во время настройки.
    </Note>

  </Step>

  <Step title="Выберите канал сообщений">
    Выберите один или несколько каналов для подключения:

    - **WhatsApp** -- отсканируйте QR-код, показанный в мастере настройки.
    - **Telegram** -- вставьте токен бота из [BotFather](https://t.me/BotFather).

  </Step>

  <Step title="Завершите установку">
    Нажмите **Готово**, чтобы развернуть экземпляр. Когда он будет готов, откройте панель OpenClaw из **Обзора OpenClaw** в hPanel.
  </Step>

</Steps>

## Вариант B: OpenClaw на VPS

Больше контроля над вашим сервером. Hostinger развертывает OpenClaw через Docker на вашем VPS, а вы управляете им через **Docker Manager** в hPanel.

<Steps>
  <Step title="Купите VPS">
    1. На [странице Hostinger OpenClaw](https://www.hostinger.com/openclaw) выберите тариф OpenClaw на VPS и завершите оформление заказа.

    <Note>
    При оформлении заказа можно выбрать кредиты **готового к использованию ИИ** -- они предварительно куплены и мгновенно интегрированы в OpenClaw, поэтому можно начать общение без внешних аккаунтов или API-ключей других провайдеров.
    </Note>

  </Step>

  <Step title="Настройте OpenClaw">
    Когда VPS будет подготовлен, заполните поля конфигурации:

    - **Токен Gateway** -- генерируется автоматически; сохраните его для дальнейшего использования.
    - **Номер WhatsApp** -- ваш номер с кодом страны (необязательно).
    - **Токен бота Telegram** -- из [BotFather](https://t.me/BotFather) (необязательно).
    - **API-ключи** -- нужны только если вы не выбрали кредиты готового к использованию ИИ при оформлении заказа.

  </Step>

  <Step title="Запустите OpenClaw">
    Нажмите **Развернуть**. Когда OpenClaw запустится, откройте панель OpenClaw из hPanel, нажав **Открыть**.
  </Step>

</Steps>

Журналы, перезапуски и обновления управляются напрямую из интерфейса Docker Manager в hPanel. Чтобы выполнить обновление, нажмите **Обновить** в Docker Manager, и будет загружен последний образ.

## Проверьте настройку

Отправьте "Hi" вашему ассистенту в подключенном канале. OpenClaw ответит и поможет пройти начальную настройку предпочтений.

## Устранение неполадок

**Панель не загружается** -- Подождите несколько минут, пока контейнер завершит подготовку. Проверьте журналы Docker Manager в hPanel.

**Контейнер Docker постоянно перезапускается** -- Откройте журналы Docker Manager и найдите ошибки конфигурации (отсутствующие токены, недействительные API-ключи).

**Бот Telegram не отвечает** -- Отправьте сообщение с кодом сопряжения из Telegram напрямую как сообщение внутри чата OpenClaw, чтобы завершить подключение.
