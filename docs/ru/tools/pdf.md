`pdf` анализирует один или несколько PDF-документов и возвращает текст. Для моделей Anthropic и Google он использует нативный ввод документов, а для всех остальных провайдеров применяет резервное извлечение текста и изображений.

## Доступность

Инструмент регистрируется, только если OpenClaw может определить для агента модель с поддержкой PDF. Порядок определения:

1. `agents.defaults.pdfModel` (явно заданные основная и резервные модели)
2. `agents.defaults.imageModel` (явно заданные основная и резервные модели)
3. Определённая для сеанса или используемая по умолчанию модель агента, если её провайдер поддерживает нативный ввод PDF (Anthropic, Google) или для неё уже настроена модель с поддержкой изображений
4. Автоматически обнаруженные провайдеры с поддержкой изображений и доступной аутентификацией; приоритет отдаётся провайдерам с нативной поддержкой PDF

Перед использованием для каждого кандидата на резервную модель проверяется аутентификация, поэтому настроенная модель `provider/model` учитывается, только если OpenClaw может аутентифицировать агента у этого провайдера. Если подходящую модель определить не удаётся, инструмент `pdf` не предоставляется.

## Входные параметры

<ParamField path="pdf" type="string">
Путь или URL одного PDF-файла.
</ParamField>

<ParamField path="pdfs" type="string[]">
Пути или URL нескольких PDF-файлов, всего не более 10.
</ParamField>

<ParamField path="prompt" type="string" default="Analyze this PDF document.">
Запрос для анализа.
</ParamField>

<ParamField path="pages" type="string">
Фильтр страниц, например `1-5` или `1,3,7-9`. Не поддерживается в нативном режиме провайдера.
</ParamField>

<ParamField path="password" type="string">
Пароль для зашифрованных PDF-файлов. Применяется ко всем PDF-файлам в запросе; используется только в резервном режиме извлечения.
</ParamField>

<ParamField path="model" type="string">
Необязательное переопределение модели в формате `provider/model`.
</ParamField>

<ParamField path="maxBytesMb" type="number">
Ограничение размера каждого PDF-файла в МБ. По умолчанию используется `agents.defaults.pdfMaxBytesMb`, а если параметр не задан — `10`.
</ParamField>

Примечания:

- `pdf` и `pdfs` объединяются, а дубликаты удаляются перед загрузкой; необходимо указать хотя бы один из них.
- `pages` разбирается как номера страниц, начинающиеся с 1; дубликаты удаляются, номера сортируются и ограничиваются значением `agents.defaults.pdfMaxPages` (по умолчанию `20`). Если диапазон не содержит ни одной страницы в допустимых пределах, ошибка возникает до вызова модели.

## Поддерживаемые ссылки на PDF

- Путь к локальному файлу (включая раскрытие `~`)
- URL `file://`
- URL `http://` и `https://`
- Управляемые OpenClaw ссылки на входящие файлы, например `media://inbound/<id>`

Другие схемы URI (например, `ftp://`) возвращают `details.error = "unsupported_pdf_reference"`. Удалённые URL `http(s)` отклоняются, если инструмент выполняется в песочнице. Если включена политика доступа только к файлам рабочей области, локальные пути за пределами разрешённых корневых каталогов отклоняются; управляемые ссылки на входящие файлы и воспроизводимые пути в хранилище входящих медиафайлов OpenClaw по-прежнему разрешены.

## Режимы выполнения

### Нативный режим провайдера

Используется для провайдеров `anthropic` и `google` — единственных провайдеров, которые в настоящее время заявляют нативную поддержку PDF-документов. Необработанные байты каждого PDF-файла передаются непосредственно в API провайдера как нативная часть документа или встроенного PDF.

Ограничения:

- `pages` не поддерживается; если параметр задан, инструмент выдаёт ошибку `pages is not supported with native PDF providers`.
- `password` не поддерживается; если параметр задан, инструмент выдаёт ошибку `password is not supported with native PDF providers`. Для зашифрованных PDF-файлов используйте модель без нативной поддержки.

### Резервный режим извлечения

Используется для всех остальных провайдеров.

1. Извлекает текст с выбранных страниц (не более `agents.defaults.pdfMaxPages`, по умолчанию `20`) с помощью встроенного Plugin `document-extract`, который использует пакет `clawpdf` (PDFium WebAssembly) для извлечения текста и изображений.
2. Если извлечённый текст короче `200` символов, преобразует те же страницы в изображения PNG. Общий бюджет преобразования составляет `4,000,000` пикселей для всех страниц, которым требуются изображения (он распределяется пропорционально между оставшимися страницами, а не выделяется каждой странице отдельно), поэтому страницы, с которых уже извлечено достаточно текста, вообще не преобразуются.
3. Отправляет извлечённый текст, все преобразованные изображения и запрос выбранной модели.

Подробности:

- Зашифрованные PDF-файлы открываются с помощью параметра верхнего уровня `password`.
- Если модель не поддерживает ввод изображений и извлечь текст невозможно, инструмент возвращает ошибку.
- Если преобразование в изображения завершается с ошибкой, OpenClaw отбрасывает изображения и продолжает работу с извлечённым текстом.
- Если целевая модель поддерживает только текст, а при извлечении были созданы изображения, OpenClaw отбрасывает изображения и отправляет только текст.

## Конфигурация

```json5
{
  agents: {
    defaults: {
      pdfModel: {
        primary: "anthropic/claude-opus-4-6",
        fallbacks: ["openai/gpt-5.4-mini"],
      },
      pdfMaxBytesMb: 10,
      pdfMaxPages: 20,
    },
  },
}
```

| Ключ                            | По умолчанию | Назначение                                                                                                         |
| ------------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------ |
| `agents.defaults.pdfModel`      | не задано    | Явно заданные основная и резервные модели PDF; затем используются `imageModel` и модель сеанса.                    |
| `agents.defaults.pdfMaxBytesMb` | `10`         | Ограничение размера каждого PDF-файла в МБ.                                                                        |
| `agents.defaults.pdfMaxPages`   | `20`         | Максимальное количество обрабатываемых страниц каждого PDF-файла.                                                  |

Полное описание полей см. в [справочнике по конфигурации](/ru/gateway/config-agents#agent-defaults).

## Выходные данные

Инструмент возвращает текст в `content[0].text`, а структурированные метаданные — в `details`.

Распространённые поля `details`:

- `model`: ссылка на определённую модель (`provider/model`)
- `native`: `true` для нативного режима провайдера, `false` для резервного режима
- `attempts`: неудачные попытки использования резервных моделей до успешного выполнения

Поля путей:

- Один входной PDF-файл: `details.pdf`
- Несколько входных PDF-файлов: `details.pdfs[]` с элементами `pdf`
- Метаданные перезаписи пути в песочнице (если применимо): `rewrittenFrom`

## Обработка ошибок

| Условие                            | Результат                                                        |
| ---------------------------------- | ---------------------------------------------------------------- |
| PDF-файл не указан                 | Выдаёт `pdf required: provide a path or URL to a PDF document`   |
| Более 10 PDF-файлов                | `details.error = "too_many_pdfs"`                                |
| Неподдерживаемая схема ссылки      | `details.error = "unsupported_pdf_reference"`                    |
| `pages` с нативным провайдером     | Выдаёт `pages is not supported with native PDF providers`        |
| `password` с нативным провайдером  | Выдаёт `password is not supported with native PDF providers`     |

## Примеры

Один PDF-файл:

```json
{
  "pdf": "/tmp/report.pdf",
  "prompt": "Summarize this report in 5 bullets"
}
```

Несколько PDF-файлов:

```json
{
  "pdfs": ["/tmp/q1.pdf", "/tmp/q2.pdf"],
  "prompt": "Compare risks and timeline changes across both documents"
}
```

Резервная модель с фильтрацией страниц:

```json
{
  "pdf": "https://example.com/report.pdf",
  "pages": "1-3,7",
  "model": "openai/gpt-5.4-mini",
  "prompt": "Extract only customer-impacting incidents"
}
```

Зашифрованный PDF-файл с резервным извлечением:

```json
{
  "pdf": "/tmp/locked.pdf",
  "password": "example-password",
  "model": "openai/gpt-5.4-mini",
  "prompt": "Summarize this contract"
}
```
