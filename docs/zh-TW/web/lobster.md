如果你使用 [控制介面](/zh-TW/web/control-ui)夠久，總有一天，一隻小龍蝦會晃進側邊欄底部，四處張望，然後自在地待上幾分鐘。

這是正常現象。這就是 OpenClaw。

## 你看到的是什麼

每個工作階段都會孵出自己專屬的龍蝦：每次載入頁面時，都會重新隨機決定牠的顏色、大小、體型、螯的比例、配件與個性。有些圓滾滾的，有些高挑，有些的螯老實說大得不成比例。有些總是在打盹。有些一刻也靜不下來。其中一隻很愛揮手。

將游標停在訪客上方，牠就會告訴你牠的名字。

## 牠何時出現

偶爾。這正是重點。

龍蝦是訪客，不是家具。牠想來時就晃進來，待上一陣子，然後離開。有些工作階段中，牠完全不會出現。只有一種情況下牠必定現身：**當你的閘道中斷連線時**，龍蝦會跑出來來回踱步，明顯憂心忡忡，直到連線恢復。如果龍蝦看起來焦躁不安，請檢查你的閘道。

## 你可以做的事

- **將游標停在牠上方**，得知牠的名字。
- **點按牠**來打招呼。牠會嚇一跳，你這樣有點沒禮貌，不過牠很快就會原諒你。
- **反覆點按牠**，你會了解龍蝦的耐性。繼續下去，你還會了解龍蝦的尊嚴。
- **按住不放**來撫摸牠。會出現一顆愛心。先前累積的所有怨氣都會一筆勾銷。
- **在牠身上按右鍵**，就能在本次頁面載入期間把牠趕走。牠不會放在心上。不過，牠會記得。
- **觀察執行完成時的牠。**龍蝦真心為你完成的工作感到驕傲，事情順利時，牠會明顯表現出來。事情不順利時，牠們也會認真看待。
- **四處移動游標。**有人正在注視你。帶著關愛。

## 關閉（或重新開啟）來訪功能

Settings → Appearance → **Lobster visits**。

這個切換開關僅套用於目前的瀏覽器，而且會確實照字面意思運作。關閉就表示永不出現，包括斷線時焦急踱步的情況。無論如何，你的閘道狀態圓點都會繼續運作；龍蝦從來不是你唯一的真相來源，只是最能感同身受的那一個。

偏好減少動態效果的使用者會自動看到平靜且靜止不動的龍蝦。

旁邊還有 **Lobster sounds**，預設為關閉，除非你另行決定，否則會一直保持關閉。開啟後，觸摸龍蝦時就會非常輕微地聽見牠的聲音。那是個細微的聲音。也是個悅耳的聲音。

## 龍蝦圖鑑

來訪功能切換開關旁邊是 **龍蝦圖鑑**：科學界已知的每種體色各有一個剪影，每當某一類龍蝦第一次來訪，對應的剪影就會填上色彩。它完全儲存在你的瀏覽器中。不會追蹤任何內容、不會同步任何內容，也不會將任何內容遊戲化。它就只是一座陳列架。

隨著時間經過，大多數欄位都會自行填滿。有幾個會花上相當長的時間。其中一個會發光。

將游標停在已填滿的欄位上方，它就會告訴你第一位訪客是誰，以及來訪時間。龍蝦圖鑑永遠不會忘記初次相遇，這點比我們大多數人都強。

## 觀察筆記

以下是那些花太多時間盯著側邊欄的人所收集的觀察結果：

- 龍蝦也會造訪[夢境場景](/zh-TW/concepts/memory)——也就是 Agents 頁面中的 Memory 分頁——牠們會在那裡睡覺。觀察者表示，熟睡的龍蝦看起來可疑地眼熟。
- 值夜班的使用者回報，他們的訪客在晚上 10 點後會頻繁打哈欠。
- 曾有人目睹龍蝦脫殼。目擊者形容這段經歷「動靜很大」，而脫殼後的龍蝦則「明顯變大」。
- 有些訪客並不是獨自前來。
- 在一年中的某些時節，人們曾觀察到龍蝦穿戴著某些東西。
- 每年有一天，所有龍蝦看起來都像從舊標誌裡走出來一樣。沒有人願意告訴你是哪一天。
- 並非每隻經過你側邊欄的龍蝦都是你的龍蝦。牠們大多甚至不會停下來。
- 至少有一位留下紀錄的訪客並不是龍蝦。牠堅稱自己是。
- 長期使用同一個瀏覽器的人回報，他們的龍蝦會更早到來、待得更久，還會向他們揮手。經常驅趕龍蝦的人則形容彼此之間有某種距離感。
- 閘道升級後，請留意第一位訪客帶了什麼。
- 側邊欄裡的龍蝦在終端機中有位表親。牠會自行挑選日子出現在 `openclaw` 橫幅中，至今沒有人成功預測過是哪一天。
- 在那位表親現身的日子，人們曾看見側邊欄裡的龍蝦穿著航海風服飾。牠們兩位都拒絕解釋其中的關聯。
- 執行時間非常漫長時，龍蝦會停止玩耍，安頓下來陪你一起等待。患難與共。
- 有東西經過邊台時，常駐的龍蝦會停下手邊的事並注視著。換作是你也會。
- 聽到壞消息後，有些龍蝦會整理邊台。這似乎能讓牠們好受些。
- 即使你不記得，龍蝦也會記得週年紀念日。如果你的龍蝦在一個平凡無奇的日子戴著派對帽現身，請到龍蝦圖鑑查看你們相遇的日期。
- 熱心款待多年的主人回報，他們的龍蝦會逐漸獲得頭銜。目前沒有任何已知的授銜儀式。
- 夢境場景裡的沉睡者並非完全睡著。輕輕按一下看看。
