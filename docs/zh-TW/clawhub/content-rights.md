# 內容權利請求

如果你認為 ClawHub 上發布的內容侵犯你的著作權或其他權利，請提交 [ClawHub 內容權利請求](https://forms.openclaw.ai/clawhub-content-rights)。

請包含：

- 一個或多個精確的 `https://clawhub.ai/<owner>/skills/<skill>` URL
- 你的姓名、組織和聯絡電子郵件
- 對權利疑慮的簡要說明
- 支援證據（如有）

ClawHub 工作人員會人工審查請求，並可能聯絡請求者或發布者以取得更多資訊。視情況而定，受影響的內容可能會被隱藏、恢復或保持不變。

對於不屬於內容權利疑慮的不安全市集內容，請使用[審核與帳戶安全](/clawhub/moderation)中所述的一般回報流程。
對於 ClawHub 本身的漏洞，請參閱[安全性](/zh-TW/clawhub/security)。
