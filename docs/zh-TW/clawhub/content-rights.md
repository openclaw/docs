# 內容權利申請

若你認為 ClawHub 上發布的內容侵犯你的著作權或其他權利，請提交 [ClawHub 內容權利申請](https://forms.openclaw.ai/clawhub-content-rights)。

請包含：

- 一個或多個確切的 `https://clawhub.ai/<owner>/skills/<skill>` URL
- 你的姓名、組織及聯絡電子郵件
- 權利疑慮的簡要說明
- 佐證資料（若有）

ClawHub 工作人員會人工審查申請，並可能聯絡申請人或發布者以取得更多資訊。視具體情況而定，受影響的內容可能會被隱藏、恢復或維持不變。

若市集內容不安全，但不涉及內容權利疑慮，請使用[內容審核與帳號安全](/clawhub/moderation)中所述的一般檢舉流程。若為 ClawHub 本身的漏洞，請參閱[安全性](/zh-TW/clawhub/security)。
