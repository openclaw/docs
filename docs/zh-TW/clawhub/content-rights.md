# 內容權利申請

如果您認為發布於 ClawHub 的內容侵害了您的著作權或其他權利，請提交 [ClawHub 內容權利申請](https://forms.openclaw.ai/clawhub-content-rights)。

請提供：

- 一個或多個確切的 `https://clawhub.ai/<owner>/skills/<skill>` 網址
- 您的姓名、組織及聯絡電子郵件
- 對權利疑慮的簡要說明
- 相關佐證（如有）

ClawHub 工作人員會人工審查申請，並可能聯絡申請人或發布者以取得更多資訊。視具體情況而定，受影響的內容可能會被隱藏、恢復或維持不變。

若市集內容存在安全疑慮，但不涉及內容權利問題，請使用[內容審核與帳號安全](/clawhub/moderation)中所述的一般檢舉流程。若要回報 ClawHub 本身的漏洞，請參閱[安全性](/clawhub/security)。
