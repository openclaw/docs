# 內容權利請求

如果你認為發布於 ClawHub 的內容侵犯你的著作權或其他權利，請提交 [ClawHub 內容權利請求](https://forms.openclaw.ai/clawhub-content-rights)。

請提供：

- 一個或多個確切的 `https://clawhub.ai/<owner>/skills/<skill>` URL
- 你的姓名、組織及聯絡電子郵件
- 權利疑慮的簡要說明
- 佐證資料（如有）

ClawHub 工作人員會人工審查請求，並可能聯絡請求者或發布者以取得更多資訊。視具體情況而定，受影響的內容可能會被隱藏、恢復或維持不變。

若市集中的不安全內容與內容權利無關，請使用[內容管理與帳號安全](/clawhub/moderation)中所述的一般檢舉流程。
若要回報 ClawHub 本身的漏洞，請參閱[安全性](/zh-TW/clawhub/security)。
