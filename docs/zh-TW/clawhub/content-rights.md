# 內容權利請求

如果你認為發布在 ClawHub 上的內容侵犯了你的著作權或其他權利，請提交 [ClawHub 內容權利請求](https://forms.openclaw.ai/clawhub-content-rights)。

請包含：

- 一個或多個確切的 `https://clawhub.ai/<owner>/skills/<skill>` URL
- 你的姓名、組織與聯絡電子郵件
- 權利疑慮的簡要說明
- 支持證據（如有）

ClawHub 工作人員會手動審查請求，並可能聯絡請求者或發布者以取得更多資訊。視情況而定，受影響的內容可能會被隱藏、恢復，或維持不變。

對於不屬於內容權利疑慮的不安全市集內容，請使用 [審核與帳戶安全](/clawhub/moderation) 中描述的一般回報流程。若是 ClawHub 本身的漏洞，請參閱 [安全性](/zh-TW/clawhub/security)。
