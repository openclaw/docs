# 可接受的使用方式

ClawHub 託管 OpenClaw 的 Skills、外掛、套件與市集詮釋資料。
請使用本頁判斷內容或發布行為是否適合放在
ClawHub 上。

這些規則適用於項目會執行的操作、要求使用者執行的內容、如何
呈現自身，以及發布者如何使用 ClawHub 的探索、安裝與
信任機制。如需瞭解審核狀態與帳號狀態，請參閱
[審核與帳號安全](/zh-TW/clawhub/moderation)。如需提出著作權或其他權利
主張，請參閱[內容權利請求](/zh-TW/clawhub/content-rights)。

## 允許的內容

ClawHub 歡迎實用、易於理解且秉持善意發布的
內容。

| 類別                                         | 符合以下條件時允許                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開發者生產力                           | 項目可協助使用者建置、測試、遷移、偵錯、撰寫文件或操作軟體。                                               |
| UI、資料與自動化工作流程               | 範圍明確、清楚列出必要的認證資訊，且高風險操作包含審查、試執行、預覽或確認流程。 |
| 防禦性安全、審核與濫用行為審查 | 工具明確定位為用於經授權的審查、保留證據，並清楚維持人工核准的界線。                          |
| 個人或團隊工作流程                       | 工作流程使用經同意的帳號、透明的設定程序與明確的權限。                                            |
| 持續維護的目錄                              | 每個項目皆有所區別、具實用性、描述準確，並受到合理維護。                                                |

情境至關重要。同一主題在範圍有限的防禦性或
基於同意的情境中可能可以接受，但若包裝成濫用工作流程則不可接受。

## 禁止的市集行為

ClawHub 也會審查發布者使用市集的方式。請勿使用 ClawHub
操弄探索機制、指標、信任訊號、審核系統或使用者
注意力。

禁止的市集行為包括：

- 大量發布大量粗製濫造、重複、僅作佔位或
  機器生成，且看不出具有實際使用者價值的項目
- 以近乎相同的 Skills 或外掛大量灌入搜尋或類別介面
- 發布數百個幾乎沒有使用量、維護、來源
  清晰度或實質差異的項目
- 透過自動化、自行重複安裝、虛假帳號、協同行動、
  付費互動或其他非自然行為，人為灌高安裝次數、下載次數、星號數或其他互動
  指標
- 建立或輪替帳號，以規避審核、封鎖、發布者限制或
  市集審查
- 在所有權、來源、功能、安全狀態、
  安裝需求，或與其他專案或發布者的關係方面誤導使用者
- 在未修正根本問題的情況下，重複上傳已遭隱藏、移除或封鎖的
  內容

大量發布本身不必然構成濫用。若項目之間具有實質差異、描述準確、
受到維護且有真實使用者使用，則大型目錄可以接受。當
大量發布伴隨內容空泛、重複、具誤導性、無人維護或
受到人為推廣的項目時，大型目錄就會成為信任與安全問題。

## 內容權利

如果你認為 ClawHub 上的內容侵犯你的著作權或其他權利，請使用
[內容權利請求](/zh-TW/clawhub/content-rights)。除非該項目同時不安全、
具有惡意或誤導性，否則請勿使用一般市集
檢舉提出著作權或權利主張。

## 審查與執行措施

ClawHub 可能會使用自動化檢查、濫用行為統計訊號、使用者檢舉與
工作人員審查，識別不安全內容或濫用發布行為。單一訊號
本身無法證明濫用；它可協助 ClawHub 判斷哪些內容需要審查。

我們可能會：

- 隱藏、暫停、移除、軟刪除違規項目，或在資源類型支援時
  永久刪除違規項目
- 封鎖不安全版本的下載或安裝
- 撤銷 API 權杖
- 軟刪除相關內容
- 限制發布權限
- 封鎖屢次違規或情節嚴重的違規者

對於明顯的濫用行為，我們不保證會先行警告再採取執行措施。如需瞭解
檢舉、審核暫停、隱藏項目、封鎖與帳號狀態，請參閱
[審核與帳號安全](/zh-TW/clawhub/moderation)。
