# 可接受使用

ClawHub 代管 OpenClaw 的 Skills、外掛、套件與市集詮釋資料。
使用本頁判斷內容或發布行為是否適合放在
ClawHub。

這些規則適用於清單會做什麼、要求使用者執行什麼、如何
呈現自身，以及發布者如何使用 ClawHub 的探索、安裝與
信任介面。關於審核狀態與帳號狀態，請參閱
[審核與帳號安全](/clawhub/moderation)。關於著作權或其他權利
主張，請參閱[內容權利請求](/clawhub/content-rights)。

## 允許的內容

ClawHub 歡迎有用、易懂且出於善意發布的內容。

| 類別                                         | 允許條件                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開發者生產力                           | 清單能協助使用者建置、測試、遷移、偵錯、記錄或營運軟體。                                               |
| 使用者介面、資料與自動化工作流程               | 範圍清楚、所需憑證明確，且有風險的動作包含審查、試跑、預覽或確認路徑。 |
| 防禦性安全、審核與濫用審查 | 工具定位為授權審查、保留證據，並清楚維持人工核准邊界。                          |
| 個人或團隊工作流程                       | 工作流程使用基於同意的帳號、透明的設定流程與明確的權限。                                            |
| 維護中的目錄                              | 每個清單都具區別性、有用、描述準確，並得到合理維護。                                                |

情境很重要。同一主題在狹義的防禦性或
基於同意的情境中可能可以接受，但若包裝成濫用工作流程則不可接受。

## 禁止的市集行為

ClawHub 也會審查發布者如何使用市集。請勿使用 ClawHub 來
操縱探索、指標、信任訊號、審核系統或使用者
注意力。

禁止的市集行為包括：

- 大量發布低投入、重複、佔位或
  機器生成且看不出具備真實使用者價值的清單
- 用幾乎相同的 Skills 或外掛灌滿搜尋或分類介面
- 發布數百個幾乎沒有使用量、維護、來源
  清晰度或有意義差異的清單
- 透過自動化、自我安裝循環、假帳號、協同行動、
  付費互動或其他非自然行為，人為灌高安裝、下載、星標或其他互動
  指標
- 建立或輪換帳號以規避審核、封鎖、發布者限制或
  市集審查
- 對所有權、來源、能力、安全態勢、
  安裝需求，或與其他專案或發布者的關聯誤導使用者
- 在未修正根本問題的情況下，反覆上傳已被隱藏、移除或封鎖的
  內容

高量發布不會自動構成濫用。大型目錄在清單具有有意義差異、
描述準確、受到維護，並由真實使用者使用時是可以接受的。
當數量與單薄、重複、誤導、缺乏維護或
人為推廣的清單同時出現時，大型目錄就會成為信任與安全問題。

## 內容權利

如果你認為 ClawHub 上的內容侵害你的著作權或其他權利，請使用
[內容權利請求](/clawhub/content-rights)。除非清單同時不安全、
惡意或具誤導性，否則請勿使用一般市集
回報來提出著作權或權利主張。

## 審查與執行

ClawHub 可能使用自動化檢查、統計濫用訊號、使用者回報與
工作人員審查來識別不安全內容或濫用發布行為。單一訊號
本身並不能證明濫用；它有助於 ClawHub 判斷哪些項目需要審查。

我們可能會：

- 隱藏、暫留、移除、軟刪除，或在資源類型支援時，
  硬刪除違規清單
- 封鎖不安全版本的下載或安裝
- 撤銷 API 權杖
- 軟刪除相關內容
- 限制發布存取權
- 封鎖重複或嚴重違規者

對於明顯濫用，我們不保證會先警告再執行處置。請參閱
[審核與帳號安全](/clawhub/moderation)，了解回報、審核暫留、
隱藏清單、封鎖與帳號狀態。
