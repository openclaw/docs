部分檢查僅會搭配 `openclaw security audit --deep` 執行：外掛／Skill 程式碼掃描（`plugins.code_safety*`、`skills.code_safety*`）以及即時閘道探測檢查（`gateway.probe_*`）。此表中的其他所有檢查都會在一般的 `openclaw security audit` 上執行。

像 `warn/critical` 這樣的嚴重性，表示同一個 `checkId` 可能會依設定以任一等級產生（例如，取決於閘道是否暴露於遠端）。在實際部署中最可能看到的高訊號值如下（並非完整清單）：
