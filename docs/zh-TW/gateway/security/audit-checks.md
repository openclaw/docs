| `models.small_params`                                         | 嚴重/資訊 | 小型模型 + 不安全的工具介面會提高注入風險                             | 模型選擇 + 沙箱/工具政策                                                                   | 否       |
| `summary.attack_surface`                                      | 資訊          | 驗證、頻道、工具與暴露態勢的彙總摘要                         | 多個鍵（請參閱發現項詳細資訊）                                                                   | 否       |
