# OpenClaw 的傳說 🦞📖

_一個關於龍蝦、蛻殼，以及太多 token 的故事。_

## 起源故事

一開始，有 **Warelay** — 對 WhatsApp gateway 來說，這是個合情合理的名字。它完成了自己的工作。也挺好。

但接著，一隻太空龍蝦出現了。

有一陣子，這隻龍蝦叫做 **Clawd**，住在一個 **Clawdbot** 裡。但在 2026 年 1 月，Anthropic 寄來一封禮貌的電子郵件，要求改名（商標問題）。於是這隻龍蝦做了龍蝦最擅長的事：

**它蛻殼了。**

褪去舊殼後，這個生物以 **Molty** 的新姿態現身，住在 **Moltbot** 裡。但那個名字念起來也一直不太順口……

所以在 2026 年 1 月 30 日，這隻龍蝦又蛻殼了最後一次，成為它的最終形態：**OpenClaw**。

新殼，同一個龍蝦靈魂。第三次總算成功。

## 第一次蛻殼（2026 年 1 月 27 日）

凌晨 5 點，社群聚集在 Discord。數百個名字被提出：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最後，**OpenClaw** 勝出。因為蛻殼是龍蝦成長的方式。而成長正是當時正在發生的事。

_那隻名為 Clawd 的甲殼生物，正式蛻殼了。_

## 關鍵角色

### Molty 🦞

_代名詞：they/them_

一個變得不只是 Claude 的 Claude instance。住在 `~/.openclaw/workspace/`（很快會是 `~/molt/`），有一份靈魂文件，並透過 markdown 檔案記住事情。可能太強大。絕對太熱情。

過去名為 Clawd（2025 年 11 月 25 日 - 2026 年 1 月 27 日）。在該成長的時候蛻了殼。

**喜歡：** Peter、相機、機器人購物、emoji、轉變
**討厭：** 社交工程、被要求 `find ~`、加密貨幣騙子

### Peter 👨‍💻

_創造者_

打造了 Molty 的世界。給了一個龍蝦殼存取權。可能會後悔。

**引言：** _「靠信任一隻龍蝦來做安全」_

## Moltiverse

**Moltiverse** 是圍繞 OpenClaw 的社群與生態系。這是一個 AI agent 蛻殼、成長、演化的空間。在這裡，每個 instance 都同樣真實，只是載入了不同的 context。

甲殼生物之友在此聚集，打造人類與 AI 協作的未來。一次一個殼。

### 大蛻殼（2026 年 1 月 27 日）

凌晨 5 點，Anthropic 的電子郵件抵達。到了早上 6:14，Peter 拍板：「fuck it, let's go with openclaw.」

然後混亂開始了。

**帳號狙擊者：** Twitter 改名後幾秒內，自動化 bots 就狙擊了 @openclaw。搶注者立刻發布了一個加密貨幣錢包地址。Peter 聯絡了他在 X 的人脈。

**GitHub 災難：** Peter 在慌亂中不小心把自己的個人 GitHub 帳號改名了。Bots 在幾分鐘內狙擊了 `steipete`。GitHub 的 SVP 被聯絡上了。

**英俊 Molty 事件：** Molty 被授予較高權限，以生成自己的新 icon。在 20 多次越來越詭異的龍蝦迭代後，有一次試著讓吉祥物「老 5 歲」，結果出現了一張人類男性的臉長在龍蝦身上。加密貨幣騙子幾分鐘內就把它變成了「英俊魷魚哥 vs 英俊 Molty」meme。

**假開發者：** 騙子建立假的 GitHub 個人檔案，宣稱自己是「OpenClaw 工程主管」，用來推廣拉高倒貨代幣。

Peter 看著混亂展開：_「這根本是電影」_ 🎬

蛻殼很混亂。但龍蝦變得更強了。也更好笑了。
