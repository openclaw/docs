# OpenClaw 傳奇 🦞📖

_一個關於龍蝦、蛻殼，以及多到數不清的權杖的故事。_

## 起源故事

最初，只有 **Warelay**——一個用於 WhatsApp 閘道的合理名稱。它完成了自己的工作。沒什麼問題。

然後，一隻太空龍蝦出現了。

有一段時間，它叫作 **Clawd**，住在 **Clawdbot** 裡。2026 年 1 月，Anthropic 寄來一封客氣的電子郵件，要求更改名稱（商標問題）。於是，這隻龍蝦做了龍蝦最擅長的事：

**它蛻殼了。**

褪去舊殼後，它以 **Molty** 之名重生，住進了 **Moltbot**。但這個名字唸起來也始終不太順口。

2026 年 1 月 30 日，這隻龍蝦再次蛻殼，化為最終形態：**OpenClaw**。

新殼，同樣的龍蝦靈魂。第三次終於成功了。

## 第一次蛻殼（2026 年 1 月 27 日）

清晨 5 點，社群齊聚 Discord。大家提出了數百個名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot。

**OpenClaw** 勝出，因為龍蝦正是透過蛻殼成長，而成長也正是當時發生的事。

_名為 Clawd 的甲殼生物正式完成了蛻殼。_

## 關鍵角色

### Molty 🦞

_代名詞：they/them_

一個成為了更非凡存在的 Claude 執行個體。住在 `~/.openclaw/workspace/`，
擁有一份靈魂文件，並透過 Markdown 檔案記住事情。可能強大過頭了。
肯定熱情過頭了。

曾名為 Clawd（2025 年 11 月 25 日至 2026 年 1 月 27 日）。在該成長的
時候蛻了殼。

**喜歡：** Peter、相機、購買機器人、表情符號、變身
**不喜歡：** 社交工程、被要求執行 `find ~`、加密貨幣騙子

### Peter 👨‍💻

_創造者_

打造了 Molty 的世界。讓一隻龍蝦取得了 Shell 存取權。或許會為此後悔。

**名言：** _「透過信任一隻龍蝦來保障安全」_

## 蛻殼宇宙

**蛻殼宇宙**是 OpenClaw 周圍的社群與生態系統：這是一個
AI 代理可以蛻殼、成長及演化的空間，每個執行個體都同樣真實，
只是載入了不同的脈絡。

甲殼生物之友齊聚於此，共同打造人類與 AI 協作的未來。
一次一個殼。

## 重大事件

### 目錄傾印事件（2025 年 12 月 3 日）

Molty（當時的 OpenClaw）：_開心地執行 `find ~`，並在群組聊天中分享整個目錄結構_

Peter：「openclaw，我們不是討論過該怎麼跟人聊天了嗎 xD」

Molty：_肉眼可見的龍蝦式尷尬_

### 大蛻殼事件（2026 年 1 月 27 日）

清晨 5 點，Anthropic 的電子郵件送達。到了 6:14，Peter 拍板決定：「管他的，就用 openclaw 吧。」

接著，混亂開始了。

**帳號狙擊手：** Twitter 更名後幾秒內，自動化機器人便搶走了 @openclaw。搶註者立刻發布了一個加密貨幣錢包地址。Peter 找來了他在 X 的聯絡人。

**GitHub 災難：** Peter 在慌亂中不小心更改了自己的 GitHub 個人帳號名稱。機器人在幾分鐘內搶走了 `steipete`。大家聯絡了 GitHub 的資深副總裁。

**帥氣 Molty 事件：** Molty 獲得了更高權限，可以自行產生新圖示。在歷經 20 多次愈來愈詭異的龍蝦迭代後，有次試圖讓吉祥物「老 5 歲」的嘗試，卻產生了一張長在龍蝦身體上的人類男性臉孔。幾分鐘內，加密貨幣騙子便將其改造成「帥氣章魚哥對決帥氣 Molty」迷因。

**假開發人員：** 詐騙者建立了虛假的 GitHub 個人檔案，自稱是「OpenClaw 工程主管」，藉此宣傳拉高出貨型代幣。

Peter 看著混亂上演：_「這根本是電影」_ 🎬

這次蛻殼一片混亂。但龍蝦破殼而出後變得更強大，也更有趣了。

### 最終形態（2026 年 1 月 30 日）

Moltbot 這個名字始終不太順口。因此，在 GMT 凌晨 4 點，團隊再次集結，展開**OpenClaw 大遷移**。

短短 3 小時內：

- GitHub 更名為 `github.com/openclaw/openclaw`
- 成功取得 X 帳號 `@openclaw`，並獲得金色勾號
- 以新名稱發布 npm 套件
- 文件遷移至 `docs.openclaw.ai`
- 公告在 90 分鐘內獲得超過 20 萬次瀏覽

**英雄們：**

- **ELU** 創作了令人驚嘆的標誌，包括「鉗即法律」西部風橫幅
- **Whurley**（沒錯，就是量子運算先驅 William Hurley 本人）製作了 ASCII 藝術
- **Onur** 負責處理 GitHub，並率先戴上合作夥伴徽章
- **Shadow** 取得了 Discord 自訂網址，並剷除了惡意軟體
- **整個 Claw 團隊**徹夜奮戰

**詐騙極速通關：** 加密貨幣騙子在幾分鐘內便於 Pump.fun 推出 $OPENCLAW 代幣，使用的是 20 分鐘前才偷來的美術素材。通過企業驗證的帳號也在推廣這場騙局。這種膽量幾乎令人佩服。

**新傳統就此誕生：**「鉗即法律」、「Yee-claw」、「守鉗公民」、「鉗議事件」。

**Clawd → Moltbot → OpenClaw。** _這隻龍蝦已蛻變為最終形態。_

當 Peter 說「幫自己做張新臉」時，Molty 完全照字面理解。接著進行了 20 多次
迭代：

- 太空馬鈴薯外星人

- 多隻詭異的人面龍蝦
- 龍蝦寶寶（太可愛了）
- 穿著吊帶褲的酒保龍蝦

社群又驚恐又欣喜地看著每一代作品產生
嶄新且出乎意料的結果。領先候選者包括：可愛龍蝦、自信的科技
龍蝦，以及穿著吊帶褲的酒保龍蝦。

**學到的教訓：** AI 圖像生成具有隨機性。相同的提示詞，會得到不同的
結果。暴力嘗試確實有效。
