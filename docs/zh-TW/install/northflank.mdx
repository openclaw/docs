在 Northflank 上使用一鍵範本部署 OpenClaw，並透過網頁控制 UI 存取。這是最簡單的「伺服器上不需要終端機」路徑：Northflank 會替你執行閘道。

## 如何開始使用

1. 點選 [部署 OpenClaw](https://northflank.com/stacks/deploy-openclaw) 以開啟範本。
2. 如果你還沒有帳號，請建立一個 [Northflank 帳號](https://app.northflank.com/signup)。
3. 點選 **立即部署 OpenClaw**。
4. 設定必要的環境變數：`OPENCLAW_GATEWAY_TOKEN`（使用強隨機值）。
5. 點選 **部署堆疊** 以建置並執行 OpenClaw 範本。
6. 等待部署完成，然後點選 **檢視資源**。
7. 開啟 OpenClaw 服務。
8. 在 `/openclaw` 開啟公開的 OpenClaw URL，並使用已設定的共用密鑰連線。此範本預設使用 `OPENCLAW_GATEWAY_TOKEN`；如果你改用密碼驗證，請改用該密碼。

## 你會取得的內容

- 託管的 OpenClaw 閘道 + 控制 UI
- 透過 Northflank Volume (`/data`) 提供持久儲存空間，讓 `openclaw.json`、每個 agent 的 `auth-profiles.json`、通道/供應商狀態、工作階段與工作區在重新部署後仍會保留

## 連接通道

使用 `/openclaw` 的控制 UI，或透過 SSH 執行 `openclaw onboard` 取得通道設定指示：
