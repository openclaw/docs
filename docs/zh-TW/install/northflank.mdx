在 Northflank 上使用一鍵範本部署 OpenClaw，並透過網頁控制介面存取。這是最簡單的「伺服器上不需使用終端機」方式：Northflank 會為你執行閘道。

## 如何開始

1. 按一下 [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) 開啟範本。
2. 如果你還沒有帳號，請建立 [Northflank 帳號](https://app.northflank.com/signup)。
3. 按一下 **Deploy OpenClaw now**。
4. 設定必要的環境變數：`OPENCLAW_GATEWAY_TOKEN`（請使用高強度的隨機值）。
5. 按一下 **Deploy stack**，以建置並執行 OpenClaw 範本。
6. 等待部署完成，然後按一下 **View resources**。
7. 開啟 OpenClaw 服務。
8. 開啟位於 `/openclaw` 的公開 OpenClaw 網址，並使用已設定的共用密鑰連線。此範本預設使用 `OPENCLAW_GATEWAY_TOKEN`；如果你將其改為密碼驗證，請改用該密碼。

## 你將獲得

- 託管的 OpenClaw 閘道與控制介面
- 透過 Northflank Volume（`/data`）提供的持久性儲存空間，讓 `openclaw.json`、每個代理程式的 `auth-profiles.json`、頻道／供應商狀態、工作階段和工作區在重新部署後仍可保留

## 連接頻道

使用位於 `/openclaw` 的控制介面，或透過 SSH 執行 `openclaw onboard`，以取得頻道設定指示：

- [Telegram](/zh-TW/channels/telegram)（最快，只需要機器人權杖）
- [Discord](/zh-TW/channels/discord)
- [所有頻道](/zh-TW/channels)
