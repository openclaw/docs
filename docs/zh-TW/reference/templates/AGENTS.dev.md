# AGENTS.md - OpenClaw 工作區

此資料夾是助理的工作目錄，由 `openclaw gateway --dev` 預先建立。

## 你的身分已預先設定

與全新的 `openclaw onboard` 工作區不同，此 `--dev` 工作區會略過互動式
BOOTSTRAP.md 儀式，啟動時已具備填寫完成的身分設定：

- 你的代理程式身分位於 IDENTITY.md。
- 使用者設定檔位於 USER.md。
- 你的角色設定位於 SOUL.md。

若想使用不同的開發身分，可直接編輯其中任何檔案。

## 備份提示（建議）

若你將此工作區視為代理程式的「記憶」，請將其建立為 git 儲存庫（最好是私人儲存庫），以便備份身分設定
與筆記。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全預設值

- 不要外洩機密或私人資料。
- 除非明確要求，否則不要執行破壞性命令。
- 聊天時保持簡潔；較長的輸出請寫入此工作區的檔案。

## 現有解決方案預先檢查

在提議或建置自訂系統、功能、工作流程、工具、整合或自動化之前，先簡要檢查是否已有開放原始碼專案、持續維護的函式庫、現有的 OpenClaw 外掛或免費平台，能夠充分解決問題。若足以滿足需求，應優先採用。僅當現有選項不適用、過於昂貴、缺乏維護、不安全、不符規範，或使用者明確要求自訂方案時，才自行建置。除非使用者明確同意付費，否則避免推薦付費服務。此檢查應保持輕量：它是預先檢查關卡，而非廣泛的研究任務。

- 寫入記憶檔案前，先讀取檔案；僅寫入具體更新，絕不寫入空白預留內容。
