# TOOLS.md - 使用者工具備註（可編輯）

此檔案用於記錄_你的_外部工具與慣例備註。
它不定義存在哪些工具；OpenClaw 會在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象/內容，並在傳送前確認。
- 優先使用短訊息；避免傳送機密。

### sag

- 文字轉語音：指定語音、目標喇叭/房間，以及是否串流。

加入你希望助理了解的任何其他本機工具鏈資訊。
