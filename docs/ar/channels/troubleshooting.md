## سلم الأوامر

شغّل هذه الأوامر بالترتيب أولاً:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

الخط الأساسي السليم:

- `Runtime: running`
- `Connectivity probe: ok`
- `Capability: read-only` أو `write-capable` أو `admin-capable`
- يعرض فحص القناة أن النقل متصل، وحيثما كان مدعوماً، `works` أو `audit ok`

## WhatsApp

### مؤشرات فشل WhatsApp

| العرض                               | أسرع فحص                                            | الإصلاح                                                                                                                              |
| ----------------------------------- | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| متصل لكن لا توجد ردود DM            | `openclaw pairing list whatsapp`                    | وافق على المرسل أو بدّل سياسة DM/قائمة السماح.                                                                                       |
| يتم تجاهل رسائل المجموعة            | تحقق من `requireMention` + أنماط الإشارة في التكوين | أشر إلى البوت أو خفف سياسة الإشارة لتلك المجموعة.                                                                                    |
| تنتهي مهلة تسجيل الدخول عبر QR بـ 408 | تحقق من متغيرات بيئة Gateway `HTTPS_PROXY` / `HTTP_PROXY` | اضبط وكيلاً يمكن الوصول إليه؛ استخدم `NO_PROXY` للتجاوزات فقط.                                                                       |
| حلقات قطع اتصال/إعادة تسجيل دخول عشوائية | `openclaw channels status --probe` + السجلات        | تتم الإشارة إلى عمليات إعادة الاتصال الأخيرة حتى عند الاتصال حالياً؛ راقب السجلات، وأعد تشغيل Gateway، ثم أعد الربط إذا استمر التذبذب. |
| تصل الردود متأخرة بثوانٍ/دقائق      | `openclaw doctor --fix`                             | يوقف Doctor عملاء TUI المحليين المتقادمين والمتحقق منهم عندما يضعفون حلقة أحداث Gateway.                                             |

استكشاف الأخطاء الكامل: [استكشاف أخطاء WhatsApp](/ar/channels/whatsapp#troubleshooting)

## Telegram

### مؤشرات فشل Telegram

| العرض                              | أسرع فحص                                          | الإصلاح                                                                                                                        |
| ---------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `/start` لكن لا يوجد تدفق رد قابل للاستخدام | `openclaw pairing list telegram`                  | وافق على الاقتران أو غيّر سياسة DM.                                                                                            |
| البوت متصل لكن المجموعة تبقى صامتة | تحقق من متطلب الإشارة ووضع خصوصية البوت           | عطّل وضع الخصوصية لإظهار المجموعة أو أشر إلى البوت.                                                                             |
| فشل الإرسال مع أخطاء شبكة           | افحص السجلات بحثاً عن فشل استدعاءات Telegram API | أصلح توجيه DNS/IPv6/الوكيل إلى `api.telegram.org`.                                                                             |
| يبلّغ بدء التشغيل أن `getMe returned 401` | تحقق من مصدر الرمز المميز المكوّن                 | أعد نسخ رمز BotFather المميز أو أنشئه من جديد وحدّث `botToken` أو `tokenFile` أو `TELEGRAM_BOT_TOKEN` للحساب الافتراضي.        |
| يتوقف الاستقصاء أو يعيد الاتصال ببطء | `openclaw logs --follow` لتشخيصات الاستقصاء       | قم بالترقية؛ إذا كانت عمليات إعادة التشغيل نتائج إيجابية كاذبة، فاضبط `pollingStallThresholdMs`. لا تزال التوقفات المستمرة تشير إلى الوكيل/DNS/IPv6. |
| يتم رفض `setMyCommands` عند بدء التشغيل | افحص السجلات بحثاً عن `BOT_COMMANDS_TOO_MUCH`     | قلّل أوامر Telegram الخاصة بـ Plugin/Skill/الأوامر المخصصة أو عطّل القوائم الأصلية.                                             |
| قمت بالترقية وقائمة السماح تحظرك   | `openclaw security audit` وقوائم السماح في التكوين | شغّل `openclaw doctor --fix` أو استبدل `@username` بمعرّفات مرسل رقمية.                                                         |

استكشاف الأخطاء الكامل: [استكشاف أخطاء Telegram](/ar/channels/telegram#troubleshooting)

## Discord

### مؤشرات فشل Discord

| العرض                                      | أسرع فحص                                                             | الإصلاح                                                                                                                                                                     |
| ------------------------------------------ | --------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| البوت متصل لكن لا توجد ردود في الخادم      | `openclaw channels status --probe`                                    | اسمح بالخادم/القناة وتحقق من نية محتوى الرسائل.                                                                                                                            |
| يتم تجاهل رسائل المجموعة                   | تحقق من السجلات بحثاً عن إسقاطات بوابة الإشارة                       | أشر إلى البوت أو اضبط `requireMention: false` للخادم/القناة.                                                                                                                |

| ردود DM مفقودة                             | `openclaw pairing list discord`                                       | وافق على اقتران DM أو عدّل سياسة DM.                                                                                                                                       |

## Matrix

| العرض                               | أسرع فحص                             | الإصلاح                                                                      |
| ----------------------------------- | ------------------------------------- | ---------------------------------------------------------------------------- |
| تم تسجيل الدخول لكنه يتجاهل رسائل الغرفة | `openclaw channels status --probe`    | تحقق من `groupPolicy` وقائمة سماح الغرف وبوابة الإشارة.                      |

| يبدو التوقيع المتقاطع/التمهيد غير صحيح | `openclaw matrix verify bootstrap`    | أصلح التخزين السري والتوقيع المتقاطع وحالة النسخ الاحتياطي في تمريرة واحدة.   |
