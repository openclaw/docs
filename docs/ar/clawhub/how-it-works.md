# كيف يعمل ClawHub

ClawHub هو طبقة السجل لمهارات وPlugins الخاصة بـ OpenClaw. فهو يمنح المستخدمين
مكانًا لاكتشاف الحزم، ويمنح الناشرين مكانًا لإصدار النسخ، ويمنح
OpenClaw بيانات وصفية كافية لتثبيت تلك الحزم وتحديثها بأمان.

## سجلات السجل

كل إدراج عام هو سجل سجل يحتوي على:

- مالك واسم مختصر أو اسم حزمة
- نسخة منشورة واحدة أو أكثر
- بيانات وصفية وملخص وملفات ونسبة المصدر
- سجل تغييرات ومعلومات وسوم مثل `latest`
- إشارات التنزيل والتثبيت والنجوم
- حالة فحص الأمان والإشراف

صفحة الإدراج هي المكان المرجعي الذي يتيح للمستخدمين فحص ما تدّعي المهارة أو
Plugin أنها تفعله قبل تثبيتها.

## Skills

المهارة هي حزمة نصية ذات نسخ تتمحور حول `SKILL.md`. ويمكن أن تتضمن
ملفات داعمة وأمثلة وقوالب وسكربتات.

يقرأ ClawHub واجهة `SKILL.md` الأمامية لفهم اسم المهارة
ووصفها ومتطلباتها ومتغيرات البيئة والبيانات الوصفية. دقة
البيانات الوصفية مهمة لأنها تساعد المستخدمين على تحديد ما إذا كانوا سيثبتون المهارة،
وتساعد الفحوصات الآلية على اكتشاف حالات عدم التطابق بين السلوك المعلن والسلوك الملحوظ.

راجع [تنسيق المهارة](/ar/clawhub/skill-format).

## Plugins

Plugins هي امتدادات OpenClaw معبأة. يخزن ClawHub بيانات الحزمة الوصفية
ومعلومات التوافق وروابط المصدر والقطع الأثرية وسجلات النسخ.

عندما يثبت OpenClaw Plugin من ClawHub، فإنه يتحقق من بيانات التوافق
الوصفية المعلنة قبل التثبيت. يمكن أن تتضمن سجلات الحزم توافق API،
والحد الأدنى لإصدار Gateway، وأهداف المضيف، ومتطلبات البيئة، وبصمات
القطع الأثرية.

استخدم مصدر تثبيت ClawHub صريحًا عندما تريد أن يكون السجل هو
مصدر الحقيقة:

```bash
openclaw plugins install clawhub:<package>
```

## النشر

ينشئ النشر سجل نسخة جديدًا غير قابل للتغيير. يستخدم الناشرون CLI الخاص بـ `clawhub`
لسير عمل السجل المصادق عليه:

```bash
clawhub skill publish ./my-skill
clawhub package publish <source> --family code-plugin --dry-run
clawhub package publish <source> --family code-plugin
```

استخدم التشغيلات التجريبية لمعاينة الحمولة المحلولة قبل الرفع. ثم تعرض الصفحات العامة
البيانات الوصفية المنشورة والملفات ونسبة المصدر وحالة الفحص.

## التثبيتات والتحديثات

تستخدم أوامر تثبيت OpenClaw ClawHub كمصدر حزم:

```bash
openclaw skills install @openclaw/demo
openclaw plugins install clawhub:<package>
```

يسجل OpenClaw بيانات مصدر التثبيت الوصفية حتى تتمكن التحديثات من حل حزمة
السجل نفسها لاحقًا. يدعم CLI الخاص بـ ClawHub أيضًا مهام سير تثبيت المهارات
وتحديثها مباشرة للمستخدمين الذين يريدون مجلدات مهارات مُدارة عبر السجل خارج
مساحة عمل OpenClaw كاملة.

## حالة الأمان

ClawHub مفتوح للنشر، لكن الإصدارات لا تزال خاضعة لبوابات الرفع
والفحوصات الآلية وبلاغات المستخدمين وإجراءات المشرفين.

تعرض الصفحات العامة ملخصات الفحص عند توفرها. قد يختفي المحتوى المحتجز أو المخفي
أو المحظور من البحث العام ومسارات التثبيت، مع بقائه
مرئيًا للمالك لأغراض التشخيص.

راجع [الأمان](/clawhub/security)، و[تدقيقات الأمان](/clawhub/security-audits)،
و[الإشراف وسلامة الحساب](/ar/clawhub/moderation)، و
[الاستخدام المقبول](/clawhub/acceptable-usage).

## الوصول إلى API

يوفر ClawHub واجهات API عامة للقراءة من أجل الاكتشاف والبحث وتفاصيل الحزم
والتنزيلات. يمكن للفهارس التابعة لجهات خارجية استخدام واجهات API هذه عندما تربط بالعودة إلى
إدراج ClawHub المرجعي، وتراعي حدود المعدل، وتتجنب الإيحاء بالتأييد.

راجع [API العامة](/ar/clawhub/api) و[HTTP API](/clawhub/http-api).
