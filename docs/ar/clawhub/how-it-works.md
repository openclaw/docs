# كيف يعمل ClawHub

ClawHub هو طبقة السجل لـ OpenClaw Skills وPlugins. يمنح المستخدمين
مكانًا لاكتشاف الحزم، ويمنح الناشرين مكانًا لإصدار النسخ، ويمنح
OpenClaw بيانات وصفية كافية لتثبيت تلك الحزم وتحديثها بأمان.

## سجلات السجل

كل إدراج عام هو سجل في السجل يتضمن:

- مالكًا واسمًا مختصرًا أو اسم حزمة
- نسخة منشورة واحدة أو أكثر
- بيانات وصفية وملخصًا وملفات وإسناد المصدر
- سجل التغييرات ومعلومات الوسوم مثل `latest`
- مؤشرات التنزيل والتثبيت والنجوم والتعليقات
- حالة الفحص الأمني والإشراف

صفحة الإدراج هي المكان المرجعي الذي يراجع فيه المستخدمون ما يدّعي Skill أو
Plugin أنه يفعله قبل تثبيته.

## Skills

Skill هي حزمة نصية ذات نسخ تتمحور حول `SKILL.md`. ويمكن أن تتضمن
ملفات داعمة وأمثلة وقوالب ونصوصًا برمجية.

يقرأ ClawHub بيانات الواجهة الأمامية في `SKILL.md` لفهم اسم Skill
ووصفها ومتطلباتها ومتغيرات البيئة والبيانات الوصفية. تهم دقة
البيانات الوصفية لأنها تساعد المستخدمين على تقرير ما إذا كانوا سيثبتون Skill
وتساعد الفحوصات الآلية على اكتشاف عدم التطابق بين السلوك المعلن والسلوك الملحوظ.

راجع [تنسيق Skill](/ar/clawhub/skill-format).

## Plugins

Plugins هي امتدادات OpenClaw مغلّفة. يخزن ClawHub بيانات الحزمة الوصفية
ومعلومات التوافق وروابط المصدر والقطع الأثرية وسجلات النسخ.

عندما يثبت OpenClaw Plugin من ClawHub، فإنه يتحقق من بيانات التوافق
الوصفية المعلنة قبل التثبيت. يمكن أن تتضمن سجلات الحزم توافق API
والحد الأدنى لإصدار Gateway وأهداف المضيف ومتطلبات البيئة وبصمات
القطع الأثرية.

استخدم مصدر تثبيت ClawHub صريحًا عندما تريد أن يكون السجل هو
مصدر الحقيقة:

```bash
openclaw plugins install clawhub:<package>
```

## النشر

ينشئ النشر سجل نسخة جديدًا غير قابل للتغيير. يستخدم الناشرون CLI الخاص بـ `clawhub`
لسير عمل السجل المصادق عليه:

```bash
clawhub skill publish ./my-skill
clawhub package publish <source> --family code-plugin --dry-run
clawhub package publish <source> --family code-plugin
```

استخدم التشغيلات التجريبية لمعاينة الحمولة المحلولة قبل الرفع. ثم تعرض الصفحات العامة
البيانات الوصفية المنشورة والملفات وإسناد المصدر وحالة الفحص.

## عمليات التثبيت والتحديث

تستخدم أوامر تثبيت OpenClaw‏ ClawHub كمصدر للحزم:

```bash
openclaw skills install <skill-slug>
openclaw plugins install clawhub:<package>
```

يسجل OpenClaw بيانات وصفية لمصدر التثبيت حتى تتمكن التحديثات لاحقًا من حل
حزمة السجل نفسها. كما يدعم ClawHub CLI سير عمل تثبيت Skills وتحديثها مباشرة
للمستخدمين الذين يريدون مجلدات Skills مُدارة عبر السجل خارج مساحة عمل
OpenClaw كاملة.

## حالة الأمان

ClawHub مفتوح للنشر، لكن الإصدارات تظل خاضعة لبوابات الرفع
والفحوصات الآلية وبلاغات المستخدمين وإجراءات المشرفين.

تعرض الصفحات العامة ملخصات الفحص عند توفرها. قد يختفي المحتوى المحتجز
أو المخفي أو المحظور من البحث العام وتدفقات التثبيت، مع بقائه
مرئيًا للمالك لأغراض التشخيص.

راجع [الأمان + الإشراف](/ar/clawhub/security) و
[الاستخدام المقبول](/ar/clawhub/acceptable-usage).

## الوصول إلى API

يوفر ClawHub واجهات API قراءة عامة للاكتشاف والبحث وتفاصيل الحزم
والتنزيلات. يمكن للفهارس الخارجية استخدام واجهات API هذه عندما تربط
بالإدراج المرجعي في ClawHub، وتحترم حدود المعدل، وتتجنب الإيحاء بالمصادقة أو التأييد.

راجع [API العامة](/ar/clawhub/api) و[HTTP API](/ar/clawhub/http-api).
