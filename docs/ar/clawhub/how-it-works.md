# كيف يعمل ClawHub

ClawHub هو طبقة السجل لـ OpenClaw Skills وPlugin. يمنح المستخدمين
مكانًا لاكتشاف الحزم، ويمنح الناشرين مكانًا لإصدار النسخ، ويمنح
OpenClaw بيانات وصفية كافية لتثبيت تلك الحزم وتحديثها بأمان.

## سجلات السجل

كل إدراج عام هو سجل في السجل يحتوي على:

- مالك واسم مختصر أو اسم حزمة
- نسخة منشورة واحدة أو أكثر
- بيانات وصفية، وملخص، وملفات، ونَسب إلى المصدر
- معلومات سجل التغييرات والوسوم مثل `latest`
- إشارات التنزيل والتثبيت والنجوم والتعليقات
- حالة فحص الأمان والإشراف

صفحة الإدراج هي المكان المرجعي الذي يفحص فيه المستخدمون ما تدّعي Skills أو
Plugin القيام به قبل تثبيتها.

## Skills

Skill هي حزمة نصية مُصدَّرة بإصدارات تتمحور حول `SKILL.md`. يمكن أن تتضمن
ملفات داعمة، وأمثلة، وقوالب، وسكربتات.

يقرأ ClawHub الواجهة الأمامية لملف `SKILL.md` لفهم اسم Skill،
ووصفها، ومتطلباتها، ومتغيرات البيئة، وبياناتها الوصفية. تهم البيانات
الوصفية الدقيقة لأنها تساعد المستخدمين على تحديد ما إذا كانوا سيثبتون Skill
وتساعد عمليات الفحص الآلية على اكتشاف عدم التطابق بين السلوك المعلن والسلوك
المرصود.

راجع [تنسيق Skill](/ar/clawhub/skill-format).

## Plugin

Plugin هي امتدادات OpenClaw مُغلَّفة. يخزن ClawHub بيانات الحزمة الوصفية،
ومعلومات التوافق، وروابط المصدر، والآثار، وسجلات الإصدارات.

عندما يثبت OpenClaw Plugin من ClawHub، فإنه يتحقق من بيانات التوافق
الوصفية المعلنة قبل التثبيت. يمكن أن تتضمن سجلات الحزم توافق API،
والحد الأدنى لإصدار Gateway، وأهداف المضيف، ومتطلبات البيئة، وملخصات
الآثار.

استخدم مصدر تثبيت ClawHub صريحًا عندما تريد أن يكون السجل هو مصدر الحقيقة:

```bash
openclaw plugins install clawhub:<package>
```

## النشر

ينشئ النشر سجل إصدار جديدًا غير قابل للتغيير. يستخدم الناشرون CLI `clawhub`
لسير عمل السجل الموثق:

```bash
clawhub skill publish ./my-skill
clawhub package publish <source> --family code-plugin --dry-run
clawhub package publish <source> --family code-plugin
```

استخدم التشغيلات التجريبية لمعاينة الحمولة المحلولة قبل الرفع. ثم تعرض
الصفحات العامة البيانات الوصفية المنشورة، والملفات، ونَسب المصدر، وحالة
الفحص.

## التثبيت والتحديثات

تستخدم أوامر تثبيت OpenClaw ClawHub كمصدر للحزم:

```bash
openclaw skills install <skill-slug>
openclaw plugins install clawhub:<package>
```

يسجل OpenClaw بيانات مصدر التثبيت الوصفية كي تتمكن التحديثات لاحقًا من
حل الحزمة نفسها من السجل. يدعم ClawHub CLI أيضًا سير عمل تثبيت Skills
وتحديثها مباشرةً للمستخدمين الذين يريدون مجلدات Skills مُدارة عبر السجل
خارج مساحة عمل OpenClaw كاملة.

## حالة الأمان

ClawHub مفتوح للنشر، لكن الإصدارات تظل خاضعة لبوابات الرفع،
والفحوصات الآلية، وبلاغات المستخدمين، وإجراءات المشرفين.

تعرض الصفحات العامة ملخصات الفحص عند توفرها. قد يختفي المحتوى المحتجز أو
المخفي أو المحظور من البحث العام ومسارات التثبيت، مع بقائه مرئيًا للمالك
لأغراض التشخيص.

راجع [الأمان + الإشراف](/ar/clawhub/security) و
[الاستخدام المقبول](/ar/clawhub/acceptable-usage).

## الوصول إلى API

يوفر ClawHub واجهات API قراءة عامة للاكتشاف، والبحث، وتفاصيل الحزم،
والتنزيلات. يمكن للفهارس التابعة لجهات خارجية استخدام واجهات API هذه
عندما تربط عائدًا إلى إدراج ClawHub المرجعي، وتحترم حدود المعدل، وتتجنب
الإيحاء بالتأييد.

راجع [API العام](/ar/clawhub/api) و[HTTP API](/ar/clawhub/http-api).
