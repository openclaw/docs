# كيف يعمل ClawHub

ClawHub هو طبقة السجل لـ Skills وPlugins في OpenClaw. يمنح المستخدمين
مكانًا لاكتشاف الحزم، ويمنح الناشرين مكانًا لإصدار النسخ، ويمنح
OpenClaw ما يكفي من البيانات الوصفية لتثبيت تلك الحزم وتحديثها بأمان.

## سجلات السجل

كل إدراج عام هو سجل في السجل يحتوي على:

- مالك ومُعرّف قصير أو اسم حزمة
- نسخة منشورة واحدة أو أكثر
- بيانات وصفية وملخص وملفات وإسناد المصدر
- معلومات سجل التغييرات والوسوم مثل `latest`
- مؤشرات التنزيل والتثبيت والنجوم
- حالة الفحص الأمني والإشراف

صفحة الإدراج هي المكان المرجعي الذي يراجع فيه المستخدمون ما تزعم Skill أو
Plugin أنها تفعله قبل تثبيتها.

## Skills

Skill هي حزمة نصية ذات نسخ تتمحور حول `SKILL.md`. ويمكن أن تتضمن
ملفات داعمة وأمثلة وقوالب وسكربتات.

يقرأ ClawHub واجهة `SKILL.md` الأمامية لفهم اسم Skill
ووصفها ومتطلباتها ومتغيرات البيئة والبيانات الوصفية. دقة
البيانات الوصفية مهمة لأنها تساعد المستخدمين على تحديد ما إذا كانوا سيثبتون Skill،
وتساعد عمليات الفحص الآلية على اكتشاف حالات عدم التطابق بين السلوك المعلن والمرصود.

راجع [تنسيق Skill](/ar/clawhub/skill-format).

## Plugins

Plugins هي إضافات OpenClaw مُغلّفة. يخزن ClawHub البيانات الوصفية للحزمة،
ومعلومات التوافق، وروابط المصدر، والقطع الأثرية، وسجلات النسخ.

عندما يثبت OpenClaw Plugin من ClawHub، فإنه يتحقق من بيانات التوافق
الوصفية المعلنة قبل التثبيت. يمكن أن تتضمن سجلات الحزم توافق API،
والحد الأدنى لإصدار Gateway، وأهداف المضيف، ومتطلبات البيئة، وبصمات
القطع الأثرية.

استخدم مصدر تثبيت ClawHub صريحًا عندما تريد أن يكون السجل هو
مصدر الحقيقة:

```bash
openclaw plugins install clawhub:<package>
```

## النشر

ينشئ النشر سجل نسخة جديدًا غير قابل للتغيير. يستخدم الناشرون CLI الخاص بـ `clawhub`
لسير عمل السجل الموثق:

```bash
clawhub skill publish ./my-skill
clawhub package publish <source> --family code-plugin --dry-run
clawhub package publish <source> --family code-plugin
```

استخدم التشغيلات الجافة لمعاينة الحمولة المحلولة قبل الرفع. ثم تعرض الصفحات العامة
البيانات الوصفية المنشورة والملفات وإسناد المصدر وحالة الفحص.

## التثبيتات والتحديثات

تستخدم أوامر تثبيت OpenClaw ClawHub كمصدر حزم:

```bash
openclaw skills install @openclaw/demo
openclaw plugins install clawhub:<package>
```

يسجل OpenClaw بيانات وصفية لمصدر التثبيت حتى تتمكن التحديثات من حل
حزمة السجل نفسها لاحقًا. يدعم CLI الخاص بـ ClawHub أيضًا مسارات عمل تثبيت
وتحديث مباشرة لـ Skill للمستخدمين الذين يريدون مجلدات Skill مُدارة بالسجل خارج
مساحة عمل OpenClaw كاملة.

## الحالة الأمنية

ClawHub مفتوح للنشر، لكن الإصدارات تظل خاضعة لبوابات الرفع،
والفحوصات الآلية، وبلاغات المستخدمين، وإجراءات المشرفين.

تعرض الصفحات العامة ملخصات الفحص عند توفرها. قد يختفي المحتوى المحتجز أو المخفي
أو المحظور من البحث العام ومسارات التثبيت، مع بقائه مرئيًا للمالك لأغراض التشخيص.

راجع [الأمان](/clawhub/security)، و[التدقيقات الأمنية](/clawhub/security-audits)،
و[الإشراف وسلامة الحساب](/ar/clawhub/moderation)، و
[الاستخدام المقبول](/clawhub/acceptable-usage).

## الوصول إلى API

يوفر ClawHub واجهات API عامة للقراءة من أجل الاكتشاف والبحث وتفاصيل الحزم
والتنزيلات. يمكن للكتالوجات التابعة لجهات خارجية استخدام واجهات API هذه عندما تعود
بالرابط إلى إدراج ClawHub المرجعي، وتحترم حدود المعدل، وتتجنب الإيحاء بالتأييد.

راجع [API العامة](/clawhub/api) و[HTTP API](/clawhub/http-api).
