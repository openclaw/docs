# كيفية عمل ClawHub

ClawHub هي طبقة السجل لمهارات OpenClaw وPluginاته. فهي تمنح المستخدمين
مكانًا لاكتشاف الحزم، وتمنح الناشرين مكانًا لإصدار النسخ، وتمنح
OpenClaw بيانات وصفية كافية لتثبيت تلك الحزم وتحديثها بأمان.

## سجلات السجل

كل إدراج عام هو سجل في السجل يتضمن:

- مالكًا واسمًا مختصرًا أو اسم حزمة
- نسخة منشورة واحدة أو أكثر
- بيانات وصفية وملخصًا وملفات ونسبة إلى المصدر
- سجل تغييرات ومعلومات وسوم مثل `latest`
- إشارات التنزيل والتثبيت والنجوم
- حالة الفحص الأمني والإشراف

صفحة الإدراج هي المكان المعتمد للمستخدمين لفحص ما تزعم المهارة أو
الـPlugin أنها تفعله قبل تثبيتها.

## Skills

المهارة هي حزمة نصية ذات نسخ تتمحور حول `SKILL.md`. ويمكن أن تتضمن
ملفات داعمة وأمثلة وقوالب وسكربتات.

يقرأ ClawHub بيانات الواجهة الأمامية في `SKILL.md` لفهم اسم المهارة
ووصفها ومتطلباتها ومتغيرات البيئة وبياناتها الوصفية. تهم دقة البيانات
الوصفية لأنها تساعد المستخدمين على تحديد ما إذا كانوا سيثبتون المهارة،
وتساعد الفحوصات الآلية على اكتشاف أوجه عدم التطابق بين السلوك المعلن
والسلوك المرصود.

انظر [تنسيق المهارة](/ar/clawhub/skill-format).

## Plugins

الـPlugins هي امتدادات OpenClaw معبأة. يخزن ClawHub بيانات الحزمة الوصفية
ومعلومات التوافق وروابط المصدر والقطع الأثرية وسجلات النسخ.

عندما يثبت OpenClaw Plugin من ClawHub، فإنه يتحقق من بيانات التوافق
الوصفية المعلنة قبل التثبيت. يمكن أن تتضمن سجلات الحزم توافق API،
والحد الأدنى لإصدار Gateway، وأهداف المضيف، ومتطلبات البيئة، وبصمات
القطع الأثرية.

استخدم مصدر تثبيت ClawHub صريحًا عندما تريد أن يكون السجل هو
مصدر الحقيقة:

```bash
openclaw plugins install clawhub:<package>
```

## النشر

ينشئ النشر سجل نسخة جديدًا غير قابل للتغيير. يستخدم الناشرون CLI الخاص
بـ`clawhub` لسير عمل السجل المصادق عليه:

```bash
clawhub skill publish ./my-skill
clawhub package publish <source> --family code-plugin --dry-run
clawhub package publish <source> --family code-plugin
```

استخدم التشغيلات التجريبية لمعاينة الحمولة المحلولة قبل الرفع. بعد ذلك
تعرض الصفحات العامة البيانات الوصفية المنشورة والملفات ونسبة المصدر
وحالة الفحص.

## التثبيتات والتحديثات

تستخدم أوامر تثبيت OpenClaw ClawHub كمصدر للحزم:

```bash
openclaw skills install @openclaw/demo
openclaw plugins install clawhub:<package>
```

يسجل OpenClaw بيانات وصفية لمصدر التثبيت حتى تتمكن التحديثات من حل
حزمة السجل نفسها لاحقًا. يدعم CLI الخاص بـClawHub أيضًا سير عمل تثبيت
المهارات وتحديثها مباشرة للمستخدمين الذين يريدون مجلدات مهارات مُدارة
من السجل خارج مساحة عمل OpenClaw كاملة.

## الحالة الأمنية

ClawHub مفتوح للنشر، لكن الإصدارات تظل خاضعة لبوابات الرفع،
والفحوصات الآلية، وبلاغات المستخدمين، وإجراءات المشرفين.

تعرض الصفحات العامة ملخصات الفحص عند توفرها. قد يختفي المحتوى المحتجز
أو المخفي أو المحظور من البحث العام وتدفقات التثبيت مع بقائه مرئيًا
للمالك لأغراض التشخيص.

انظر [الأمان](/clawhub/security)، و[تدقيقات الأمان](/clawhub/security-audits)،
و[الإشراف وسلامة الحساب](/ar/clawhub/moderation)، و
[الاستخدام المقبول](/clawhub/acceptable-usage).

## الوصول إلى API

يوفر ClawHub واجهات API قراءة عامة للاكتشاف والبحث وتفاصيل الحزم
والتنزيلات. يمكن للفهارس الخارجية استخدام واجهات API هذه عندما تربط
بالإدراج المعتمد في ClawHub، وتحترم حدود المعدل، وتتجنب الإيحاء بالتأييد.

انظر [API العامة](/ar/clawhub/api) و[HTTP API](/clawhub/http-api).
