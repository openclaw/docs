# الاستخدام المقبول

يستضيف ClawHub مهارات Skills وPluginات وحزمًا وبيانات وصفية للسوق من أجل OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). وللمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                            | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                              |
| سير عمل الواجهة والبيانات والأتمتة              | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيلًا تجريبيًا أو معاينة أو تأكيدًا. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُعرض الأداة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                                         |
| سير العمل الشخصي أو الجماعي                     | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                                     |
| الفهارس التي تتم صيانتها                        | يكون كل إدراج مميزًا ومفيدًا وموصوفًا بدقة وتتم صيانته على نحو معقول.                                                          |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في سياق دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُعبأ كسير عمل لإساءة الاستخدام.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر الجماعي لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات Skills أو Pluginات شبه متطابقة
- نشر مئات الإدراجات مع استخدام قليل أو معدوم، أو صيانة قليلة، أو وضوح مصدر
  ضعيف، أو تمييز معنوي محدود
- تضخيم عمليات التثبيت أو التنزيل أو النجوم أو مقاييس التفاعل الأخرى اصطناعيًا
  عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق أو
  التفاعل المدفوع أو سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها للتهرب من الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى مرارًا كان قد أُخفي أو أُزيل أو حُظر بالفعل
  دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة على نحو ذي معنى، وموصوفة بدقة، وتتم صيانتها،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات ضحلة أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج غير آمن أو ضارًا
أو مضللًا أيضًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية وإشارات إحصائية لإساءة الاستخدام وبلاغات المستخدمين
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مبدئيًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مبدئيًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن إنفاذًا يبدأ بالتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات وتعليقات الإشراف
والإدراجات المخفية والحظر ووضع الحساب.
