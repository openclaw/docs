# الاستخدام المقبول

يستضيف ClawHub ‏Skills وPlugin والحزم وبيانات وصف سوق OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثّل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub.
بالنسبة إلى حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). بالنسبة إلى مطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## المحتوى المسموح

يرحّب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                             | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                 | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                              |
| واجهة المستخدم والبيانات وسير عمل الأتمتة         | يكون النطاق واضحًا، وتكون بيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام   | تُقدَّم الأداة بوصفها مخصصة للمراجعة المصرح بها، وتحفظ الأدلة، وتبقي حدود الموافقة البشرية واضحة.                              |
| سير العمل الشخصي أو الخاص بالفريق                | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                                     |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُحزَّم كسير عمل لإساءة الاستخدام.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub للتلاعب
بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر الجماعي لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات Skills أو Plugin شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل
  مصطنع من خلال الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها لتجنب الإشراف أو الحظر أو حدود الناشر أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى سبق إخفاؤه أو إزالته أو حظره مرارًا
  من دون إصلاح المشكلة الأساسية

النشر كبير الحجم ليس إساءة استخدام تلقائيًا. تكون الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة على نحو ذي معنى، وموصوفة بدقة، وتتم صيانتها،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مصانة أو
مروّج لها بشكل مصطنع.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق، ما لم يكن الإدراج أيضًا غير آمن
أو خبيثًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات
المستخدمين، ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ لكنها تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مبدئيًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مبدئيًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) لمعرفة البلاغات وتعليقات الإشراف
والإدراجات المخفية والحظر ووضع الحساب.
