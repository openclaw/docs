# الاستخدام المقبول

يستضيف ClawHub عناصر Skills، وعناصر Plugin، والحزم، وبيانات السوق الوصفية الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). لمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن
نية.

| الفئة                                         | مسموح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تكون الأداة مؤطرة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الخاص بالفرق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادا شفافا، وأذونات صريحة.                                            |
| الفهارس التي تتم صيانتها                              | يكون كل إدراج مميزا ومفيدا وموصوفا بدقة وتتم صيانته بشكل معقول.                                                |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولا في إعداد دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يعبأ كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

| إساءة استخدام المنصات وتجاوز الحظر                              | الحسابات المتخفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو التفاعل المزيف، أو أتمتة الحسابات المتعددة، أو النشر الجماعي، أو روبوتات الرسائل المزعجة، أو الأتمتة المصممة لتجنب الاكتشاف.                                                                                                                                          |

| الإثراء أو المراقبة المنتهكان للخصوصية                 | جمع جهات الاتصال للرسائل المزعجة، أو كشف المعلومات الشخصية، أو الملاحقة، أو استخراج العملاء المحتملين المقترن بتواصل غير مرغوب فيه، أو المراقبة السرية، أو المطابقة البيومترية غير التوافقية، أو استخدام بيانات مسربة أو تفريغات اختراق.                                                                                                                  |

## سلوك السوق غير المسموح به

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به:

- نشر أعداد كبيرة بالجملة من الإدراجات منخفضة الجهد أو المكررة أو المؤقتة أو
  المولدة آليا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بعناصر Skills أو Plugin شبه متطابقة
- نشر مئات الإدراجات مع استخدام قليل أو معدوم، أو صيانة قليلة، أو وضوح ضعيف للمصدر،
  أو تمايز ذي معنى محدود
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيا عبر الأتمتة، أو حلقات التثبيت الذاتي، أو الحسابات المزيفة، أو النشاط
  المنسق، أو التفاعل المدفوع، أو أي سلوك آخر غير عضوي
- إنشاء حسابات أو تدويرها لتجاوز الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارا سبق إخفاؤه أو إزالته أو حظره
  من دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيا. الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة بوضوح، وموصوفة بدقة، وتتم صيانتها،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مصانة أو
مروجة اصطناعيا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم تقارير السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج أيضا غير آمن
أو ضارا أو مضللا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وتقارير المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ لكنها تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفا ناعما، أو، حيث يكون ذلك مدعوما لنوع المورد،
  حذفها نهائيا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفا ناعما
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ المسبوق بتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للتقارير، وعمليات تعليق الإشراف،
والإدراجات المخفية، والحظر، ووضع الحساب.
