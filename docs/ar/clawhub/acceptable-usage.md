# الاستخدام المقبول

يستضيف ClawHub المهارات وPlugins والحِزم وبيانات تعريف السوق الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub.
للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطور                           | يساعد الإدراج المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات المحفوفة بالمخاطر مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة بوصفها مخصصة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الجماعي                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الكتالوجات المُصانة                              | يكون كل إدراج مميزًا ومفيدًا وموصوفًا بدقة ومُصانًا بدرجة معقولة.                                                |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُغلَّف كسير عمل لإساءة الاستخدام.

لا يستضيف ClawHub محتوى يكون غرضه الرئيسي إساءة الاستخدام أو الخداع أو التنفيذ
غير الآمن أو انتهاك الحقوق.

| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| الوصول غير المصرح به أو تجاوز الأمن                      | تجاوز المصادقة، أو الاستيلاء على الحسابات، أو إساءة استخدام حدود المعدل، أو الاستيلاء على مكالمة مباشرة أو وكيل، أو سرقة جلسات قابلة لإعادة الاستخدام، أو الموافقة التلقائية على تدفقات الاقتران لمستخدمين غير معتمدين.                                                                                                                                                   |
| إساءة استخدام المنصات وتجنب الحظر                              | حسابات خفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو تفاعل مزيف، أو أتمتة متعددة الحسابات، أو نشر جماعي، أو روبوتات رسائل مزعجة، أو أتمتة مصممة لتجنب الاكتشاف.                                                                                                                                          |
| الاحتيال وعمليات النصب وسير العمل المالي الخادع             | شهادات أو فواتير مزيفة، أو تدفقات دفع خادعة، أو تواصل احتيالي، أو إثبات اجتماعي مزيف، أو سير عمل هويات اصطناعية للاحتيال، أو أدوات إنفاق/تحصيل دون موافقة بشرية واضحة.                                                                                                                    |
| الإثراء أو المراقبة المتعدية على الخصوصية                 | استخراج جهات الاتصال للرسائل المزعجة، أو التشهير بالمعلومات الشخصية، أو الملاحقة، أو استخراج العملاء المحتملين مقترنًا بتواصل غير مرغوب فيه، أو المراقبة السرية، أو المطابقة البيومترية غير التوافقية، أو استخدام بيانات مسربة أو تفريغات اختراق.                                                                                                                  |

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub للتلاعب
بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر الجماعي لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو المؤقتة أو
  المُنشأة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل
  مصطنع عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها لتجنب الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر بكميات كبيرة ليس إساءة استخدام تلقائيًا. تكون الكتالوجات الكبيرة مقبولة
عندما تكون الإدراجات مختلفة بشكل ذي معنى، وموصوفة بدقة، ومُصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الكتالوجات الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مُصانة أو
مروَّج لها بشكل مصطنع.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج غير آمن أو ضارًا
أو مضللًا أيضًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا، أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو ذوي المخالفات الجسيمة

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وحالات التعليق
للإشراف، والإدراجات المخفية، والحظر، ووضع الحساب.
