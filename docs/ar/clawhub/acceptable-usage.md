# الاستخدام المقبول

يستضيف ClawHub مهارات وPlugins وحزمًا وبيانات وصفية للسوق تخص OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر يندرج ضمن
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيفية
تمثيلها لنفسها، وكيفية استخدام الناشرين لواجهات الاكتشاف والتثبيت
والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). وللمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن
نية.

| الفئة                                         | يُسمح بها عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | تساعد القائمة المستخدمين على إنشاء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة بوصفها مخصصة للمراجعة المصرح بها، وتحافظ على الأدلة، وتُبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الجماعي                       | يستخدم سير العمل حسابات قائمة على الموافقة وإعدادًا شفافًا وأذونات صريحة.                                            |
| الفهارس الخاضعة للصيانة                              | تكون كل قائمة متميزة ومفيدة وموصوفة بدقة وخاضعة لصيانة معقولة.                                                |

السياق مهم. فقد يكون الموضوع نفسه مقبولًا ضمن سياق دفاعي محدود أو
قائم على الموافقة، وغير مقبول عند تقديمه في صورة سير عمل لإساءة الاستخدام.

غير الآمن أو انتهاك الحقوق.

## سلوك السوق المحظور

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق المحظور ما يلي:

- النشر الجماعي لأعداد كبيرة من القوائم منخفضة الجهد أو المتكررة أو النائبة أو
  المُنشأة آليًا التي لا يبدو أنها تقدم قيمة حقيقية للمستخدمين
- إغراق واجهات البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح للمصدر
  أو تمايز ذي معنى ضئيل أو معدوم
- تضخيم أعداد التثبيتات أو التنزيلات أو النجوم أو غيرها من مقاييس
  التفاعل بصورة مصطنعة من خلال الأتمتة، أو حلقات التثبيت الذاتي، أو الحسابات المزيفة، أو النشاط
  المنسق، أو التفاعل المدفوع، أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى سبق إخفاؤه أو إزالته أو حظره مرارًا
  دون إصلاح المشكلة الأساسية

لا يُعد النشر بكميات كبيرة إساءة استخدام تلقائيًا. تُقبل الفهارس الكبيرة
عندما تكون القوائم مختلفة بصورة جوهرية وموصوفة بدقة وخاضعة للصيانة
ويستخدمها مستخدمون حقيقيون. وتصبح الفهارس الكبيرة مشكلة تتعلق بالثقة والسلامة عندما
يقترن الحجم بقوائم سطحية أو متكررة أو مضللة أو غير خاضعة للصيانة أو
مروَّج لها بصورة مصطنعة.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
للمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى، إلا إذا كانت القائمة أيضًا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub عمليات تحقق آلية وإشارات إحصائية لإساءة الاستخدام وبلاغات المستخدمين
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. ولا تثبت الإشارة
بمفردها وقوع إساءة استخدام؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

يجوز لنا:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا قابلًا للاستعادة، أو، حيثما يكون ذلك مدعومًا لنوع المورد،
  حذفها نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا قابلًا للاستعادة
- تقييد صلاحية النشر
- حظر مرتكبي المخالفات المتكررة أو الجسيمة

لا نضمن توجيه تحذير قبل الإنفاذ في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات وعمليات التعليق لأغراض الإشراف
والقوائم المخفية وعمليات الحظر ووضع الحساب.
