# الاستخدام المقبول

يستضيف ClawHub Skills وPlugins والحزم وبيانات السوق الوصفية الخاصة بـ OpenClaw.  
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). لمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن نية.

| الفئة                                           | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                | تساعد القائمة المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهة المستخدم والبيانات وسير عمل الأتمتة       | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُعرض الأداة على أنها للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الجماعي                     | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الفهارس المُصانة                                | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة ومصانة بدرجة معقولة.                                                |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولًا في إعداد دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يُغلّف كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر بالجملة لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو المؤقتة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدم
- إغراق أسطح البحث أو الفئات بـ Skills أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا من خلال الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها لتجنب الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى سبق إخفاؤه أو إزالته أو حظره مرارًا
  دون إصلاح المشكلة الأساسية

النشر كبير الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون القوائم مختلفة بشكل ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم سطحية أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة أيضًا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصًا آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا ناعمًا أو، حيث يدعم نوع المورد ذلك،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا ناعمًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات وتعليقات الإشراف
والقوائم المخفية والحظر ووضع الحساب.
