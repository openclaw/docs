# الاستخدام المقبول

يستضيف ClawHub مهارات Skills وPlugins والحزم وبيانات السوق الوصفية الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر مناسبًا على
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | تساعد القائمة المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهة المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات عالية المخاطر مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تكون الأداة مصاغة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصية أو الخاصة بالفريق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الفهارس المصانة                              | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة ومصانة على نحو معقول.                                                |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُحزَّم كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيف يستخدم الناشرون السوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- نشر أعداد كبيرة بالجملة من قوائم قليلة الجهد أو مكررة أو مؤقتة أو
  مولدة آليًا ولا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات Skills أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل مصطنع
  من خلال الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى بشكل متكرر بعد إخفائه أو إزالته أو حظره بالفعل
  دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون القوائم مختلفة على نحو ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم سطحية أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة أيضًا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة فريق العمل لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub في تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا، أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد إمكانية النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
