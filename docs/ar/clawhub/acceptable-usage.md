# الاستخدام المقبول

يستضيف ClawHub ‏Skills والإضافات والحزم وبيانات السوق الوصفية الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub.
للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن نية.

## سلوك السوق غير المسموح

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub للتلاعب
بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه المستخدمين.

يشمل سلوك السوق غير المسموح:

- نشر أعداد كبيرة بالجملة من قوائم منخفضة الجهد أو مكررة أو بديلة أو
  مولدة آليا ولا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بـ Skills أو إضافات شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو تمييز ذي معنى قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى اصطناعيا
  عبر الأتمتة، أو حلقات التثبيت الذاتي، أو الحسابات المزيفة، أو النشاط المنسق،
  أو التفاعل المدفوع، أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها لتجاوز الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر بكميات كبيرة ليس إساءة استخدام تلقائيا. تكون الفهارس الكبيرة مقبولة
عندما تكون القوائم مختلفة اختلافا ذا معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم ضعيفة أو مكررة أو مضللة أو غير مصانة أو
مروجة اصطناعيا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم تقارير السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة غير آمنة أيضا
أو خبيثة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصا آلية، وإشارات إحصائية لإساءة الاستخدام، وتقارير المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت
الإشارة إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفا مبدئيا أو، حيث يدعم نوع المورد ذلك،
  حذفها حذفا نهائيا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفا مبدئيا
- تقييد صلاحية النشر
- حظر المخالفين المتكررين أو الشديدين

لا نضمن الإنفاذ بالتحذير أولا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation) للتقارير، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
