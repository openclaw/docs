# الاستخدام المقبول

يستضيف ClawHub مهارات وملحقات وحزمًا وبيانات وصفية لسوق OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تعرّف عن نفسها، وكيف يستخدم الناشرون واجهات الاكتشاف والتثبيت
والثقة في ClawHub. لمعرفة حالات الإشراف ووضع الحساب، راجع
[الإشراف وأمان الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور
بحسن نية.

| الفئة                                         | يُسمح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | تساعد القائمة المستخدمين على إنشاء البرمجيات أو اختبارها أو ترحيلها أو تصحيح أخطائها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات ومسارات عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة مصرحًا بها، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة باعتبارها مخصصة للمراجعة المصرح بها، وتحافظ على الأدلة، وتُبقي حدود الموافقة البشرية واضحة.                          |
| مسارات العمل الشخصية أو الجماعية                       | يستخدم مسار العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الفهارس الخاضعة للصيانة                              | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة وخاضعة لصيانة معقولة.                                                |

السياق مهم. فقد يكون الموضوع نفسه مقبولًا ضمن سياق دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يُقدَّم كمسار عمل لإساءة الاستخدام.

## سلوك السوق المحظور

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق المحظور ما يلي:

- النشر الجماعي لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو النائبة أو
  المُنشأة آليًا، والتي لا يبدو أنها تقدم قيمة حقيقية للمستخدمين
- إغراق واجهات البحث أو الفئات بمهارات أو ملحقات شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح للمصدر أو
  تمييز ذي معنى ضئيل أو معدوم
- تضخيم أعداد عمليات التثبيت أو التنزيلات أو النجوم أو غيرها من مقاييس
  التفاعل بصورة مصطنعة، عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو الإمكانات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- الرفع المتكرر لمحتوى سبق إخفاؤه أو حذفه أو حظره
  من دون إصلاح المشكلة الأساسية

لا يُعد النشر بكميات كبيرة إساءة استخدام تلقائيًا. تكون الفهارس الكبيرة مقبولة
عندما تكون القوائم مختلفة اختلافًا ذا معنى، وموصوفة بدقة، وخاضعة للصيانة،
ويستخدمها مستخدمون حقيقيون. وتصبح الفهارس الكبيرة مشكلة تتعلق بالثقة والأمان عندما
يقترن الحجم بقوائم سطحية أو مكررة أو مضللة أو غير خاضعة للصيانة أو
مروَّج لها بصورة مصطنعة.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق
العادية لمطالبات حقوق النشر أو الحقوق الأخرى، إلا إذا كانت القائمة أيضًا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub عمليات تحقق آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بمفردها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

يجوز لنا:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مبدئيًا، أو، عندما يكون ذلك مدعومًا لنوع المورد،
  حذفها نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مبدئيًا
- تقييد صلاحية النشر
- حظر المخالفين المتكررين أو مرتكبي المخالفات الجسيمة

لا نضمن إصدار تحذير قبل الإنفاذ في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وأمان الحساب](/clawhub/moderation) لمعرفة معلومات عن البلاغات وتعليقات الإشراف
والقوائم المخفية والحظر ووضع الحساب.
