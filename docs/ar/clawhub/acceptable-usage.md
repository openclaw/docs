# الاستخدام المقبول

يستضيف ClawHub Skills وPlugin والحزم وبيانات السوق الوصفية الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). بالنسبة إلى مطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن
نية.

| الفئة | مسموح به عندما |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين | تساعد القائمة المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها. |
| واجهات المستخدم والبيانات وسير عمل الأتمتة | يكون النطاق واضحا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة الإساءة | يتم تقديم الأداة على أنها للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة. |
| سير العمل الشخصية أو الخاصة بالفريق | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادا شفافا، وأذونات صريحة. |
| الكتالوجات المصانة | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة ومصانة بشكل معقول. |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولا في إطار دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يكون معبأ كسير عمل إساءة.

أو انتهاك الحقوق.

| الفئة | غير مسموح به |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## سلوك السوق المحظور

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق المحظور ما يلي:

- نشر أعداد كبيرة بالجملة من قوائم منخفضة الجهد أو مكررة أو نائبة أو
  منشأة آليا لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو Plugin شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو تمييز ذي معنى
  قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل
  مصطنع عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء حسابات أو تدويرها لتجاوز الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى سبق إخفاؤه أو إزالته أو حظره بشكل متكرر
  من دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة تلقائيا. الكتالوجات الكبيرة مقبولة
عندما تكون القوائم مختلفة بشكل ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الكتالوجات الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم ضعيفة أو مكررة أو مضللة أو غير مصانة أو
مروّج لها مصطنعا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم تقارير السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة أيضا غير آمنة
أو خبيثة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصا آلية، وإشارات إحصائية للإساءة، وبلاغات المستخدمين، ومراجعة
الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
الإساءة بذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفا مرنا أو، حيث يكون ذلك مدعوما لنوع المورد،
  حذفها حذفا نهائيا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفا مرنا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ المسبوق بتحذير في حالات الإساءة الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
