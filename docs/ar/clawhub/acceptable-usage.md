# الاستخدام المقبول

يستضيف ClawHub Skills وPlugins والحزم وبيانات بيانات السوق الوصفية لـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثّل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف وحالة الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولطلبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحّب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | يساعد الإدراج المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات عالية المخاطر مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة بوصفها مخصصة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصية أو الخاصة بالفِرق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الفهارس الخاضعة للصيانة                              | يكون كل إدراج مميزًا ومفيدًا وموصوفًا بدقة وخاضعًا لصيانة معقولة.                                                |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يُعبّأ كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| الوصول غير المصرح به أو تجاوز الأمان                      | تجاوز المصادقة، أو الاستيلاء على الحسابات، أو إساءة استخدام حدود المعدل، أو الاستيلاء على مكالمة مباشرة أو وكيل مباشر، أو سرقة الجلسات القابلة لإعادة الاستخدام، أو الموافقة التلقائية على تدفقات الاقتران لمستخدمين غير معتمدين.                                                                                                                                                   |
| إساءة استخدام المنصات والتحايل على الحظر                              | الحسابات الخفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو التفاعل الزائف، أو أتمتة الحسابات المتعددة، أو النشر الجماعي، أو روبوتات الرسائل المزعجة، أو الأتمتة المصممة لتجنب الاكتشاف.                                                                                                                                          |

| الإثراء أو المراقبة المتعديان على الخصوصية                 | جمع جهات الاتصال للرسائل المزعجة، أو كشف المعلومات الشخصية، أو المطاردة، أو استخراج العملاء المحتملين مع تواصل غير مرغوب فيه، أو المراقبة السرية، أو المطابقة البيومترية غير التوافقية، أو استخدام بيانات مسرّبة أو تفريغات اختراق.                                                                                                                  |

## سلوك السوق غير المسموح به

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به:

- النشر الجماعي لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدم
- إغراق أسطح البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى
  قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  من دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. تكون الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة بصورة ذات معنى، وموصوفة بدقة، وخاضعة للصيانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير خاضعة للصيانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج غير آمن أيضًا
أو ضارًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين، و
مراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا ناعمًا أو، عندما يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفًا ناعمًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ المسبوق بتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للبلاغات وتعليقات الإشراف
والإدراجات المخفية والحظر وحالة الحساب.
