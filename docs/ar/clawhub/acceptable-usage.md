# الاستخدام المقبول

يستضيف ClawHub الـ Skills وPlugin والحزم وبيانات تعريف السوق الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                            | مسموح به عندما                                                                                                                    |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيح أخطائها أو توثيقها أو تشغيلها.                         |
| واجهة المستخدم والبيانات وسير عمل الأتمتة        | يكون النطاق واضحًا، وتكون بيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات عالية المخاطر مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُعرض الأداة على أنها للمراجعة المصرح بها، وتحافظ على الأدلة، وتُبقي حدود الموافقة البشرية واضحة.                                |
| سير العمل الشخصي أو الخاص بالفريق               | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                                       |
| الفهارس المصانة                                 | يكون كل إدراج مميزًا ومفيدًا وموصوفًا بدقة ومُصانًا بدرجة معقولة.                                                                |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُغلّف كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

| إساءة استخدام المنصات وتفادي الحظر                         | الحسابات الخفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو التفاعل المزيف، أو أتمتة الحسابات المتعددة، أو النشر الجماعي، أو روبوتات الرسائل المزعجة، أو الأتمتة المصممة لتجنب الاكتشاف.                                                                                                                |

| الإثراء أو المراقبة المنتهكة للخصوصية                      | استخراج جهات الاتصال للرسائل المزعجة، أو كشف المعلومات الشخصية، أو الملاحقة، أو استخراج العملاء المحتملين المقترن بتواصل غير مرغوب فيه، أو المراقبة السرية، أو المطابقة البيومترية غير الرضائية، أو استخدام بيانات مسربة أو تفريغات اختراق.                                                                |

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر بالجملة لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو النائبة أو
  المنشأة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدم
- إغراق أسطح البحث أو الفئات بـ Skills أو Plugin شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها لتفادي الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني
  أو متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر بكميات كبيرة ليس إساءة استخدام تلقائيًا. تكون الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة على نحو ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج أيضًا غير آمن
أو خبيثًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصًا آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بمفردها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا أو، حيث يدعم نوع المورد ذلك،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) لمعرفة البلاغات، وعمليات التعليق للمراجعة،
والإدراجات المخفية، والحظر، ووضع الحساب.
