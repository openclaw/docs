# الاستخدام المقبول

يستضيف ClawHub Skills وplugins وحزمًا وبيانات وصفية للسوق خاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله بطاقة الإدراج، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). لمطالبات حقوق النشر أو غيرها من الحقوق،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن
نية.

| الفئة | مسموح به عندما |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين | تساعد بطاقة الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها. |
| واجهة المستخدم والبيانات وسير عمل الأتمتة | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتشمل الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة على أنها للمراجعة المصرح بها، وتحافظ على الأدلة، وتُبقي حدود الموافقة البشرية واضحة. |
| سير العمل الشخصية أو الجماعية | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة. |
| الفهارس المصانة | تكون كل بطاقة إدراج مميزة ومفيدة وموصوفة بدقة ومصانة بدرجة معقولة. |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يُغلَّف كسير عمل لإساءة الاستخدام.

## المحتوى غير المسموح به

لا يستضيف ClawHub محتوى يكون غرضه الرئيسي إساءة الاستخدام أو الخداع أو التنفيذ غير الآمن
أو انتهاك الحقوق.

| الفئة | غير مسموح به |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| الوصول غير المصرح به أو تجاوز الأمان | تجاوز المصادقة، أو الاستيلاء على الحسابات، أو إساءة استخدام حدود المعدل، أو الاستيلاء على مكالمة مباشرة أو وكيل مباشر، أو سرقة جلسات قابلة لإعادة الاستخدام، أو الموافقة التلقائية على تدفقات الاقتران لمستخدمين غير معتمدين. |
| إساءة استخدام المنصة والتحايل على الحظر | حسابات خفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو التفاعل المزيف، أو أتمتة متعددة الحسابات، أو النشر الجماعي، أو روبوتات الرسائل المزعجة، أو الأتمتة المصممة لتجنب الاكتشاف. |
| الاحتيال وعمليات النصب وسير العمل المالي الخادع | شهادات أو فواتير مزيفة، أو تدفقات دفع خادعة، أو تواصل احتيالي، أو دليل اجتماعي مزيف، أو سير عمل بهويات اصطناعية للاحتيال، أو أدوات إنفاق/تحصيل رسوم دون موافقة بشرية واضحة. |
| الإثراء أو المراقبة المتطفلان على الخصوصية | كشط جهات الاتصال للرسائل المزعجة، أو كشف المعلومات الشخصية، أو المطاردة، أو استخراج العملاء المحتملين مع تواصل غير مرغوب فيه، أو المراقبة الخفية، أو المطابقة البيومترية غير الرضائية، أو استخدام بيانات مسربة أو تفريغات اختراق. |
| الانتحال أو التلاعب بالهوية دون موافقة | تبديل الوجوه، أو التوائم الرقمية، أو المؤثرون المستنسخون، أو الشخصيات المزيفة، أو أدوات أخرى تُستخدم للانتحال أو التضليل. |
| المحتوى الجنسي الصريح أو توليد محتوى للبالغين مع تعطيل السلامة | توليد صور أو فيديو أو محتوى NSFW؛ أو أغلفة محتوى للبالغين حول واجهات برمجة تطبيقات تابعة لجهات خارجية؛ أو بطاقات إدراج يكون غرضها الأساسي محتوى جنسيًا صريحًا. |
| متطلبات تنفيذ مخفية أو غير آمنة أو مضللة | أوامر تثبيت معتمة، أو أدوات تثبيت تمرر إلى الصدفة مثل محتوى منزل يُشغَّل باستخدام `sh` أو `bash` دون قابلية مراجعة واضحة، أو متطلبات أسرار أو مفاتيح خاصة غير معلنة، أو تنفيذ `npx @latest` عن بُعد دون قابلية مراجعة واضحة، أو بيانات وصفية تخفي ما تحتاجه بطاقة الإدراج فعليًا للتشغيل. |
| مواد تنتهك حقوق النشر أو الحقوق | إعادة نشر skill أو plugin أو وثائق أو أصول علامة تجارية أو كود مملوك لشخص آخر دون إذن؛ أو انتهاك شروط الترخيص؛ أو انتحال هوية المؤلف أو الناشر الأصلي. |

## سلوك السوق غير المسموح به

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به:

- نشر أعداد كبيرة بالجملة من بطاقات الإدراج قليلة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو plugins شبه متطابقة
- نشر مئات بطاقات الإدراج مع استخدام أو صيانة أو وضوح مصدر أو
  تمييز meaningful قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون بطاقات الإدراج مختلفة بشكل ذي معنى وموصوفة بدقة ومصانة
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم ببطاقات إدراج سطحية أو مكررة أو مضللة أو غير مصانة أو
مروَّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم تقارير السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت بطاقة الإدراج غير آمنة أو
خبيثة أو مضللة أيضًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وتقارير المستخدمين، و
مراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء بطاقات الإدراج المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا برمجيًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا برمجيًا
- تقييد وصول النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للتقارير وتعليقات الإشراف
وبطاقات الإدراج المخفية والحظر ووضع الحساب.
