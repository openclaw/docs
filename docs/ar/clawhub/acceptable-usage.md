# الاستخدام المقبول

يستضيف ClawHub مهارات Skills وPlugins والحزم وبيانات وصفية لسوق OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع
على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن نية.

| الفئة                                             | مسموح عندما                                                                                                                       |
| ------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                  | يساعد الإدراج المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                             |
| واجهة المستخدم والبيانات وسير عمل الأتمتة          | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام    | تكون الأداة مؤطرة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                                      |
| سير العمل الشخصية أو الخاصة بالفريق               | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                                       |
| الكتالوجات التي تتم صيانتها                       | يكون كل إدراج مميزًا ومفيدًا وموصوفًا بدقة وتتم صيانته بشكل معقول.                                                               |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على الموافقة،
وغير مقبول عندما يُعبأ كسير عمل لإساءة الاستخدام.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub للتلاعب
بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه المستخدمين.

يتضمن سلوك السوق غير المسموح ما يلي:

- النشر بالجملة لأعداد كبيرة من الإدراجات قليلة الجهد أو المكررة أو المؤقتة أو
  المنشأة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات Skills أو Plugins شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل مصطنع
  عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق أو
  التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء حسابات أو تدويرها للتهرب من الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. تُعد الكتالوجات الكبيرة مقبولة
عندما تكون الإدراجات مختلفة اختلافًا ذا معنى، وموصوفة بدقة، وتتم صيانتها،
ويستخدمها مستخدمون حقيقيون. تصبح الكتالوجات الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مُصانة أو مروّج لها
اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج أيضًا غير آمن أو ضارًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ لكنها تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا ناعمًا، أو حيثما
  كان ذلك مدعومًا لنوع المورد، حذفها حذفًا نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفًا ناعمًا
- تقييد صلاحية النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وحالات التعليق للإشراف،
والإدراجات المخفية، والحظر، ووضع الحساب.
