# الاستخدام المقبول

يستضيف ClawHub Skills وPlugins والحزم والبيانات الوصفية للسوق الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف وحالة الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). للمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن
نية.

| الفئة                                             | مسموح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                 | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                              |
| واجهة المستخدم والبيانات وسير عمل الأتمتة        | يكون النطاق واضحًا، وتكون بيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام  | تُعرض الأداة على أنها مخصصة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الجماعي                      | يستخدم سير العمل حسابات قائمة على الموافقة وإعدادًا شفافًا وأذونات صريحة.                                                        |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُعبأ كسير عمل لإساءة الاستخدام.

## سلوك السوق غير المسموح به

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به:

- النشر الجماعي لأعداد كبيرة من الإدراجات منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بـ Skills أو Plugins شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمييز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها للتحايل على الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الانتماء إلى مشروع أو ناشر آخر
- رفع محتوى بشكل متكرر سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر عالي الحجم لا يُعد إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة اختلافًا ذا معنى، وموصوفة بدقة، ومُصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات ضعيفة أو مكررة أو مضللة أو غير مُصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر أو حقوقًا أخرى تخصك، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم تقارير السوق العادية
لمطالبات حقوق النشر أو الحقوق ما لم يكن الإدراج أيضًا غير آمن
أو ضارًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية وإشارات إحصائية لإساءة الاستخدام وتقارير المستخدمين
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الشديدين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للتقارير وتعليقات الإشراف
والإدراجات المخفية والحظر وحالة الحساب.
