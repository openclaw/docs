# الاستخدام المقبول

يستضيف ClawHub ‏Skills وPlugins والحزم وبيانات السوق الوصفية لـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثّل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub.
للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## سلوك السوق المحظور

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق المحظور ما يلي:

- النشر بالجملة لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو المؤقتة أو
  المولدة آليًا والتي لا يبدو أن لها قيمة حقيقية للمستخدم
- إغراق أسطح البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام قليل أو معدوم، أو صيانة ضعيفة، أو وضوح مصدر
  محدود، أو تمايز ذي معنى
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى اصطناعيًا
  عبر الأتمتة، أو حلقات التثبيت الذاتي، أو الحسابات المزيفة، أو النشاط المنسق،
  أو التفاعل المدفوع، أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها لتجاوز الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر بكميات كبيرة ليس إساءة استخدام تلقائيًا. الكتالوجات الكبيرة مقبولة
عندما تكون القوائم مختلفة بشكل ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الكتالوجات الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم سطحية أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة أيضًا غير آمنة أو
خبيثة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub الفحوصات الآلية، وإشارات إساءة الاستخدام الإحصائية، وبلاغات
المستخدمين، ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء.
لا تثبت الإشارة إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا أو، عندما يكون ذلك مدعومًا لنوع المورد،
  حذفها نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد صلاحية النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد تحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
