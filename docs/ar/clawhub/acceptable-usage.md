# الاستخدام المقبول

يستضيف ClawHub‏ Skills وPlugin والحزم وبيانات وصف السوق الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). للمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | تساعد القائمة المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحا، وبيانات الاعتماد المطلوبة صريحة، وتشمل الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تكون الأداة مصاغة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الخاص بالفريق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادا شفافا، وأذونات صريحة.                                            |
| الفهارس المصانة                              | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة ومصانة على نحو معقول.                                                |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولا في إطار دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يتم تغليفه كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

| إساءة استخدام المنصات والتهرب من الحظر                              | الحسابات الخفية بعد الحظر، أو تهيئة الحسابات أو زراعتها، أو التفاعل المزيف، أو أتمتة الحسابات المتعددة، أو النشر الجماعي، أو روبوتات الرسائل المزعجة، أو الأتمتة المصممة لتجنب الاكتشاف.                                                                                                                                          |

| الإثراء أو المراقبة المتطفلة على الخصوصية                 | جمع جهات الاتصال للرسائل المزعجة، أو كشف معلومات الأشخاص، أو الملاحقة، أو استخراج العملاء المحتملين مقرونا بتواصل غير مرغوب فيه، أو المراقبة السرية، أو المطابقة البيومترية دون موافقة، أو استخدام بيانات مسربة أو تفريغات اختراق.                                                                                                                  |

## سلوك السوق غير المسموح به

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به ما يلي:

- النشر الجماعي لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو Plugin شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو
  تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيا عبر الأتمتة، أو حلقات التثبيت الذاتي، أو الحسابات المزيفة، أو النشاط
  المنسق، أو التفاعل المدفوع، أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتهرب من الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى بشكل متكرر سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر بكثافة عالية ليس إساءة استخدام تلقائيا. تكون الفهارس الكبيرة مقبولة
عندما تكون القوائم مختلفة على نحو ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بقوائم هزيلة أو مكررة أو مضللة أو غير مصانة أو
مروّج لها اصطناعيا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
للمطالبات المتعلقة بحقوق النشر أو الحقوق، إلا إذا كانت القائمة أيضا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصا آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بذاتها؛ لكنها تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفا مرنا، أو، حيث يكون ذلك مدعوما لنوع المورد،
  حذفها حذفا نهائيا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفا مرنا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
