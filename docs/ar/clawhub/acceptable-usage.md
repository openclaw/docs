# الاستخدام المقبول

يستضيف ClawHub Skills وplugins وحزمًا وبيانات وصفية للسوق من أجل OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله بطاقة الإدراج، وما تطلب من المستخدمين تشغيله، وكيف
تقدم نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation). بالنسبة إلى مطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح به

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن
نية.

| الفئة                                         | مسموح به عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | تساعد بطاقة الإدراج المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيح أخطائها أو توثيقها أو تشغيلها.                                               |
| واجهة المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتشمل الإجراءات الخطرة مسارات للمراجعة أو التشغيل التجريبي أو المعاينة أو التأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُعرض الأداة على أنها للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو الجماعي                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |
| الفهارس المُصانة                              | تكون كل بطاقة إدراج مميزة ومفيدة وموصوفة بدقة ومُصانة بشكل معقول.                                                |

السياق مهم. قد يكون الموضوع نفسه مقبولًا في إطار دفاعي ضيق أو
قائم على الموافقة، وغير مقبول عندما يُغلّف كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

## سلوك السوق غير المسموح به

يراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح به:

- النشر الجماعي لأعداد كبيرة من بطاقات الإدراج منخفضة الجهد أو المكررة أو النائبة أو
  المُولّدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو plugins شبه متطابقة
- نشر مئات بطاقات الإدراج مع استخدام أو صيانة أو وضوح مصدر أو
  تمايز ذي معنى قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط
  المنسق أو التفاعل المدفوع أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها للتهرب من الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  من دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة
عندما تكون بطاقات الإدراج مختلفة بشكل ذي معنى، وموصوفة بدقة، ومُصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم ببطاقات إدراج ضحلة أو مكررة أو مضللة أو غير مُصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كانت بطاقة الإدراج أيضًا غير آمنة
أو ضارة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية وإشارات إساءة استخدام إحصائية وبلاغات المستخدمين و
مراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء بطاقات الإدراج المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا لينًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا لينًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن إنفاذًا يبدأ بالتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation) للاطلاع على البلاغات وتعليقات الإشراف
وبطاقات الإدراج المخفية والحظر ووضع الحساب.
