# الاستخدام المقبول

يستضيف ClawHub Skills وPlugins والحزم وبيانات السوق الوصفية الخاصة بـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تعرّف عن نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub.
للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والواضح والمنشور بحسن نية.

| الفئة                                             | مسموح عندما                                                                                                                        |
| ------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                                  | تساعد القائمة المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة        | يكون النطاق واضحا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام   | تكون الأداة مؤطرة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                                      |
| سير العمل الشخصية أو الخاصة بالفريق               | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادا شفافا، وأذونات صريحة.                                                        |
| الفهارس المصانة                                   | تكون كل قائمة مميزة ومفيدة وموصوفة بدقة ومصانة بشكل معقول.                                                                        |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولا في إطار دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يعبأ كسير عمل لإساءة الاستخدام.

غير الآمن أو انتهاك الحقوق.

| الإثراء أو المراقبة المنتهكة للخصوصية                      | كشط جهات الاتصال للرسائل المزعجة، أو التشهير بنشر المعلومات الشخصية، أو الملاحقة، أو استخراج العملاء المحتملين مع تواصل غير مرغوب، أو المراقبة السرية، أو المطابقة الحيوية من دون موافقة، أو استخدام البيانات المسربة أو تفريغات الاختراق.                                                           |

## سلوك السوق غير المسموح

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub للتلاعب
بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه المستخدمين.

يشمل سلوك السوق غير المسموح:

- النشر بالجملة لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليا والتي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام قليل أو معدوم، أو صيانة قليلة أو معدومة، أو وضوح
  ضعيف للمصدر، أو تمييز ذي معنى محدود
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى بشكل
  مصطنع عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات الزائفة أو النشاط
  المنسق أو التفاعل المدفوع أو غير ذلك من السلوك غير العضوي
- إنشاء الحسابات أو تدويرها للتهرب من الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى مرارا سبق إخفاؤه أو إزالته أو حظره
  من دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيا. تكون الفهارس الكبيرة مقبولة عندما
تكون القوائم مختلفة بشكل ذي معنى، وموصوفة بدقة، ومصانة، ويستخدمها مستخدمون
حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما يقترن الحجم بقوائم
سطحية أو مكررة أو مضللة أو غير مصانة أو مروجة اصطناعيا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى،
فاستخدم [طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق
العادية لمطالبات حقوق النشر أو الحقوق إلا إذا كانت القائمة أيضا غير آمنة أو
خبيثة أو مضللة.

## المراجعة والإنفاذ

قد يستخدم ClawHub الفحوصات الآلية، وإشارات إساءة الاستخدام الإحصائية، وبلاغات
المستخدمين، ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء.
لا تثبت الإشارة إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج
إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفا ناعما أو، عندما
  يكون ذلك مدعوما لنوع المورد، حذفها حذفا نهائيا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إلغاء رموز API
- حذف المحتوى المرتبط حذفا ناعما
- تقييد إمكانية النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن إنفاذا يبدأ بالتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation) للاطلاع على البلاغات وتعليقات الإشراف
والقوائم المخفية والحظر ووضع الحساب.
