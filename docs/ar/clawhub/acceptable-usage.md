# الاستخدام المقبول

يستضيف ClawHub ‏Skills وPlugins والحزم وبيانات سوق OpenClaw الوصفية.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. للاطلاع على حالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). ولمطالبات حقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطور                           | يساعد الإدراج المستخدمين على بناء البرمجيات أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وتدفقات عمل الأتمتة               | يكون النطاق واضحًا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تُقدَّم الأداة في إطار المراجعة المصرح بها، وتحافظ على الأدلة، وتُبقي حدود الموافقة البشرية واضحة.                          |
| تدفقات العمل الشخصية أو الخاصة بالفرق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادًا شفافًا، وأذونات صريحة.                                            |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولًا في إعداد دفاعي ضيق أو قائم على
الموافقة، وغير مقبول عندما يُعبأ كسير عمل لإساءة الاستخدام.

## سلوك السوق غير المسموح

يراجع ClawHub أيضًا كيف يستخدم الناشرون السوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- نشر أعداد كبيرة دفعة واحدة من إدراجات منخفضة الجهد أو مكررة أو بديلة أو
  مولدة آليًا لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بـ Skills أو Plugins شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز معنوي قليل أو معدوم
- تضخيم عمليات التثبيت أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى
  اصطناعيًا عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها لتفادي الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- تحميل محتوى كان قد أُخفي أو أُزيل أو حُظر مرارًا
  من دون إصلاح المشكلة الأساسية

النشر بكثافة عالية ليس إساءة استخدام تلقائيًا. تكون الفهارس الكبيرة مقبولة
عندما تكون الإدراجات مختلفة على نحو ذي معنى، وموصوفة بدقة، ومُصانة،
ومستخدمة من مستخدمين حقيقيين. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مُصانة أو
مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر أو حقوقًا أخرى تخصك، فاستخدم
[طلبات حقوق المحتوى](/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق ما لم يكن الإدراج أيضًا غير آمن أو
ضارًا أو مضللًا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصًا آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين،
ومراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بحد ذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا ناعمًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها نهائيًا
- حظر التنزيلات أو عمليات التثبيت للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا ناعمًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الشديدين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للبلاغات، وتعليقات الإشراف،
والإدراجات المخفية، والحظر، ووضع الحساب.
