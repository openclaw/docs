# الاستخدام المقبول

تستضيف ClawHub‏ Skills وPlugins والحزم وبيانات السوق الوصفية لـ OpenClaw.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما تفعله القائمة، وما تطلب من المستخدمين تشغيله، وكيف
تمثل نفسها، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation). لمطالبات حقوق النشر أو الحقوق الأخرى، راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## سلوك السوق المحظور

تراجع ClawHub أيضًا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق المحظور:

- النشر بالجملة لأعداد كبيرة من القوائم منخفضة الجهد أو المكررة أو النائبة أو
  المولدة آليًا التي لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات أو Plugins شبه متطابقة
- نشر مئات القوائم مع استخدام أو صيانة أو وضوح مصدر أو تمييز ذي معنى قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى اصطناعيًا
  عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك آخر غير عضوي
- إنشاء الحسابات أو تدويرها لتجاوز الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني أو
  متطلبات التثبيت أو الارتباط بمشروع أو ناشر آخر
- رفع محتوى مرارًا سبق إخفاؤه أو إزالته أو حظره
  دون إصلاح المشكلة الأساسية

النشر عالي الحجم ليس إساءة استخدام تلقائيًا. الفهارس الكبيرة مقبولة عندما
تكون القوائم مختلفة بشكل ذي معنى، وموصوفة بدقة، ومُصانة، ويستخدمها مستخدمون
حقيقيون. تصبح الفهارس الكبيرة مشكلة ثقة وسلامة عندما يقترن الحجم بقوائم
سطحية أو مكررة أو مضللة أو غير مُصانة أو مروّج لها اصطناعيًا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقًا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق، ما لم تكن القائمة أيضًا غير آمنة أو خبيثة أو
مضللة.

## المراجعة والإنفاذ

قد تستخدم ClawHub فحوصًا آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين، و
مراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. لا تثبت الإشارة
إساءة الاستخدام بذاتها؛ لكنها تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء القوائم المخالفة أو تعليقها أو إزالتها أو حذفها حذفًا مرنًا أو، حيث يكون ذلك مدعومًا لنوع المورد،
  حذفها حذفًا نهائيًا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- حذف المحتوى المرتبط حذفًا مرنًا
- تقييد الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ بعد التحذير أولًا في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/ar/clawhub/moderation) للبلاغات، وتعليقات الإشراف،
والقوائم المخفية، والحظر، ووضع الحساب.
