# الاستخدام المقبول

يستضيف ClawHub مهارات Skills والإضافات والحزم وبيانات سوق OpenClaw الوصفية.
استخدم هذه الصفحة لتحديد ما إذا كان المحتوى أو سلوك النشر ينتمي إلى
ClawHub.

تنطبق هذه القواعد على ما يفعله الإدراج، وما يطلب من المستخدمين تشغيله، وكيف
يمثل نفسه، وكيف يستخدم الناشرون أسطح الاكتشاف والتثبيت والثقة في ClawHub. لحالات الإشراف ووضع الحساب، راجع
[الإشراف وسلامة الحساب](/clawhub/moderation). للمطالبات المتعلقة بحقوق النشر أو الحقوق الأخرى،
راجع [طلبات حقوق المحتوى](/ar/clawhub/content-rights).

## المحتوى المسموح

يرحب ClawHub بالمحتوى المفيد والمفهوم والمنشور بحسن نية.

| الفئة                                         | مسموح عندما                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| إنتاجية المطورين                           | يساعد الإدراج المستخدمين على بناء البرامج أو اختبارها أو ترحيلها أو تصحيحها أو توثيقها أو تشغيلها.                                               |
| واجهات المستخدم والبيانات وسير عمل الأتمتة               | يكون النطاق واضحا، وبيانات الاعتماد المطلوبة صريحة، وتتضمن الإجراءات الخطرة مسارات مراجعة أو تشغيل تجريبي أو معاينة أو تأكيد. |
| الأمن الدفاعي والإشراف ومراجعة إساءة الاستخدام | تكون الأداة مؤطرة للمراجعة المصرح بها، وتحافظ على الأدلة، وتبقي حدود الموافقة البشرية واضحة.                          |
| سير العمل الشخصي أو سير عمل الفريق                       | يستخدم سير العمل حسابات قائمة على الموافقة، وإعدادا شفافا، وأذونات صريحة.                                            |
| الكتالوجات المصانة                              | يكون كل إدراج مميزا ومفيدا وموصوفا بدقة ومصانا على نحو معقول.                                                |

السياق مهم. يمكن أن يكون الموضوع نفسه مقبولا في إعداد دفاعي ضيق أو
قائم على الموافقة وغير مقبول عندما يعبأ كسير عمل لإساءة الاستخدام.

أو انتهاك الحقوق.

## سلوك السوق غير المسموح

يراجع ClawHub أيضا كيفية استخدام الناشرين للسوق. لا تستخدم ClawHub
للتلاعب بالاكتشاف أو المقاييس أو إشارات الثقة أو أنظمة الإشراف أو انتباه
المستخدمين.

يشمل سلوك السوق غير المسموح:

- نشر أعداد كبيرة بالجملة من إدراجات منخفضة الجهد أو مكررة أو نائبة أو
  مولدة آليا لا يبدو أن لها قيمة حقيقية للمستخدمين
- إغراق أسطح البحث أو الفئات بمهارات Skills أو إضافات شبه متطابقة
- نشر مئات الإدراجات مع استخدام أو صيانة أو وضوح مصدر أو تمايز ذي معنى
  قليل أو معدوم
- تضخيم التثبيتات أو التنزيلات أو النجوم أو مقاييس التفاعل الأخرى اصطناعيا
  عبر الأتمتة أو حلقات التثبيت الذاتي أو الحسابات المزيفة أو النشاط المنسق
  أو التفاعل المدفوع أو أي سلوك غير عضوي آخر
- إنشاء الحسابات أو تدويرها لتجنب الإشراف أو الحظر أو حدود الناشرين أو
  مراجعة السوق
- تضليل المستخدمين بشأن الملكية أو المصدر أو القدرات أو الوضع الأمني
  أو متطلبات التثبيت أو الانتماء إلى مشروع أو ناشر آخر
- رفع محتوى مرارا كان قد أخفي أو أزيل أو حظر بالفعل
  دون إصلاح المشكلة الأساسية

النشر كبير الحجم ليس إساءة استخدام تلقائيا. الكتالوجات الكبيرة مقبولة
عندما تكون الإدراجات مختلفة على نحو ذي معنى، وموصوفة بدقة، ومصانة،
ويستخدمها مستخدمون حقيقيون. تصبح الكتالوجات الكبيرة مشكلة ثقة وسلامة عندما
يقترن الحجم بإدراجات سطحية أو مكررة أو مضللة أو غير مصانة أو
مروجة اصطناعيا.

## حقوق المحتوى

إذا كنت تعتقد أن محتوى على ClawHub ينتهك حقوق النشر الخاصة بك أو حقوقا أخرى، فاستخدم
[طلبات حقوق المحتوى](/ar/clawhub/content-rights). لا تستخدم بلاغات السوق العادية
لمطالبات حقوق النشر أو الحقوق إلا إذا كان الإدراج أيضا غير آمن أو
خبيثا أو مضللا.

## المراجعة والإنفاذ

قد يستخدم ClawHub فحوصات آلية، وإشارات إحصائية لإساءة الاستخدام، وبلاغات المستخدمين، و
مراجعة الموظفين لتحديد المحتوى غير الآمن أو سلوك النشر المسيء. الإشارة
لا تثبت إساءة الاستخدام بذاتها؛ بل تساعد ClawHub على تحديد ما يحتاج إلى مراجعة.

قد نقوم بما يلي:

- إخفاء الإدراجات المخالفة أو تعليقها أو إزالتها أو حذفها حذفا لينا أو، عندما يكون ذلك مدعوما لنوع المورد،
  حذفها حذفا نهائيا
- حظر التنزيلات أو التثبيتات للإصدارات غير الآمنة
- إبطال رموز API
- الحذف اللين للمحتوى المرتبط
- تقييد حق الوصول إلى النشر
- حظر المخالفين المتكررين أو الجسيمين

لا نضمن الإنفاذ المسبوق بتحذير في حالات إساءة الاستخدام الواضحة. راجع
[الإشراف وسلامة الحساب](/clawhub/moderation) للاطلاع على البلاغات وتعليقات الإشراف
والإدراجات المخفية والحظر ووضع الحساب.
