# القياسات عن بُعد

يستخدم ClawHub قياسات CLI عن بُعد بالحد الأدنى لحساب إجمالي عدد عمليات التثبيت.

## متى تُجمَع القياسات عن بُعد

لا تُرسَل القياسات عن بُعد إلا عندما:

- تكون قد سجّلت الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- تكون القياسات عن بُعد **غير معطّلة** (راجع "كيفية التعطيل" أدناه).

إذا لم تكن مسجّل الدخول، فلن يُبلَّغ عن أي شيء.

## ما نجمعه

عند كل عملية `clawhub install` يُبلَّغ عنها، يرسل CLI حدث تثبيت واحدًا بأفضل جهد.

يتضمن الحدث:

- `slug`: slug الخاص بالمهارة المثبّتة.
- `version`: الإصدار المثبّت، عندما يكون معروفًا.

### ما _لا_ نجمعه

- لا مسارات مجلدات ولا معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، أو مطالبات، أو أي مخرجات CLI أخرى.

## أعداد التثبيت

يحافظ ClawHub على عدّادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين أبلغوا عن تثبيت CLI واحد على الأقل للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين أبلغوا عن تثبيت ولم يحذفوا بيانات
  القياسات عن بُعد الخاصة بهم.

## الشفافية + عناصر تحكم المستخدم

لا يرى الجميع سوى **عدادات التثبيت المجمّعة**.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياسات عن بُعد الخاصة بك.

## كيفية تعطيل القياسات عن بُعد

عيّن متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

مع تعيين هذا، لن يرسل CLI قياسات التثبيت عن بُعد.
