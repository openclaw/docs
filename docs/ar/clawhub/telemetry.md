# القياسات عن بُعد

يستخدم ClawHub قياسات CLI عن بُعد بالحد الأدنى لحساب إجمالي أعداد التثبيتات.

## متى تُجمع القياسات عن بُعد

لا تُرسل القياسات عن بُعد إلا عندما:

- تكون مسجل الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- تكون القياسات عن بُعد **غير معطلة** (راجع "كيفية التعطيل" أدناه).

إذا لم تكن مسجل الدخول، فلن يُبلَّغ عن أي شيء.

## ما نجمعه

عند كل `clawhub install` مُبلَّغ عنه، يرسل CLI حدث تثبيت واحدًا بأفضل جهد.

يتضمن الحدث:

- `slug`: slug المهارة المثبتة.
- `version`: الإصدار المثبت، عند معرفته.

### ما _لا_ نجمعه

- لا مسارات مجلدات ولا معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، أو مطالبات، أو مخرجات CLI أخرى.

## أعداد التثبيت

يحافظ ClawHub على عدادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين أبلغوا عن تثبيت CLI واحد على الأقل للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين أبلغوا عن تثبيت ولم يحذفوا
  قياساتهم عن بُعد.

## الشفافية + عناصر تحكم المستخدم

يرى الجميع **عدادات تثبيت مجمعة** فقط.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياسات عن بُعد الخاصة بك.

## كيفية تعطيل القياسات عن بُعد

عيّن متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

عند تعيينه، لن يرسل CLI قياسات التثبيت عن بُعد.
