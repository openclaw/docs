# القياسات عن بُعد

يستخدم ClawHub قياسات CLI عن بُعد بالحد الأدنى لحساب إجمالي أعداد التثبيت.

## متى تُجمع القياسات عن بُعد

لا تُرسل القياسات عن بُعد إلا عندما:

- تكون مسجّل الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- تكون القياسات عن بُعد **غير معطّلة** (راجع "كيفية التعطيل" أدناه).

إذا لم تكن مسجّل الدخول، فلن يتم الإبلاغ عن أي شيء.

## ما نجمعه

عند كل عملية `clawhub install` يتم الإبلاغ عنها، يرسل CLI حدث تثبيت واحدًا وفق أفضل جهد.

يتضمن الحدث:

- `slug`: معرّف slug للمهارة المثبّتة.
- `version`: الإصدار المثبّت، عندما يكون معروفًا.

### ما _لا_ نجمعه

- لا مسارات مجلدات ولا معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، أو مطالبات، أو أي مخرجات CLI أخرى.

## أعداد التثبيت

يحافظ ClawHub على عدّادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين أبلغوا عن تثبيت CLI واحد على الأقل للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين أبلغوا عن تثبيت ولم يحذفوا
  قياساتهم عن بُعد.

## الشفافية + عناصر تحكم المستخدم

لا يرى الجميع سوى **عدّادات التثبيت الإجمالية**.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياسات عن بُعد الخاصة بك.

## كيفية تعطيل القياسات عن بُعد

عيّن متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

عند تعيين هذا، لن يرسل CLI قياسات التثبيت عن بُعد.
