# القياسات عن بُعد

يستخدم ClawHub قياسات CLI عن بُعد محدودة لحساب إجمالي عدد عمليات التثبيت.

## متى تُجمع القياسات عن بُعد

لا تُرسل القياسات عن بُعد إلا عندما:

- تكون مسجّل الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- تكون القياسات عن بُعد **غير معطّلة** (راجع "كيفية التعطيل" أدناه).

إذا لم تكن مسجّل الدخول، فلن يُبلّغ عن أي شيء.

## ما الذي نجمعه

عند كل عملية `clawhub install` يُبلّغ عنها، يرسل CLI حدث تثبيت واحدًا وفق أفضل جهد.

يتضمن الحدث:

- `slug`: slug الخاص بالمهارة المثبّتة.
- `version`: الإصدار المثبّت، عندما يكون معروفًا.

### ما الذي _لا_ نجمعه

- لا مسارات مجلدات ولا معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، ولا مطالبات، ولا أي مخرجات CLI أخرى.

## أعداد التثبيت

يحتفظ ClawHub بعدّادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين أبلغوا عن عملية تثبيت CLI واحدة على الأقل للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين أبلغوا عن عملية تثبيت ولم يحذفوا
  قياساتهم عن بُعد.

## الشفافية + عناصر تحكم المستخدم

لا يرى الجميع إلا **عدّادات التثبيت الإجمالية**.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياسات عن بُعد الخاصة بك.

## كيفية تعطيل القياسات عن بُعد

عيّن متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

عند تعيينه، لن يرسل CLI قياسات التثبيت عن بُعد.
