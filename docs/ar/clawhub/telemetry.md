# القياسات عن بُعد

يستخدم ClawHub حدًا أدنى من قياسات CLI عن بُعد لحساب إجمالي أعداد التثبيت.

## متى تُجمع القياسات عن بُعد

لا تُرسل القياسات عن بُعد إلا عندما:

- تكون مسجل الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- لا تكون القياسات عن بُعد **معطلة** (راجع “كيفية التعطيل” أدناه).

إذا لم تكن مسجل الدخول، فلن يُبلّغ عن أي شيء.

## ما نجمعه

عند كل `clawhub install` يُبلّغ عنه، يرسل CLI حدث تثبيت واحدًا وفق أفضل جهد.

يتضمن الحدث:

- `slug`: معرّف slug للمهارة المثبتة.
- `version`: الإصدار المثبت، عند معرفته.

### ما _لا_ نجمعه

- لا مسارات مجلدات أو معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، أو مطالبات، أو أي مخرجات CLI أخرى.

## أعداد التثبيت

يحتفظ ClawHub بعدّادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين بلّغوا عن تثبيت CLI واحد على الأقل للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين بلّغوا عن تثبيت ولم يحذفوا
  قياساتهم عن بُعد.

## الشفافية + عناصر تحكم المستخدم

لا يرى الجميع إلا **عدّادات التثبيت المجمّعة**.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياسات عن بُعد الخاصة بك.

## كيفية تعطيل القياسات عن بُعد

اضبط متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

عند ضبط هذا المتغير، لن يرسل CLI قياسات التثبيت عن بُعد.
