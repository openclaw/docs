# القياس عن بُعد

يستخدم ClawHub حدًا أدنى من قياس استخدام CLI لحساب إجمالي عدد عمليات التثبيت.

## متى يُجمع القياس عن بُعد

لا يُرسل القياس عن بُعد إلا عندما:

- تكون مسجل الدخول في CLI.
- تشغّل `clawhub install <slug>`.
- لا يكون القياس عن بُعد **معطلًا** (انظر "كيفية التعطيل" أدناه).

إذا لم تكن مسجل الدخول، فلن يُبلّغ عن أي شيء.

## ما نجمعه

عند كل عملية `clawhub install` مُبلّغ عنها، يرسل CLI حدث تثبيت واحدًا بأفضل جهد.

يتضمن الحدث:

- `slug`: معرّف المهارة المثبتة.
- `version`: الإصدار المثبت، عندما يكون معروفًا.

### ما _لا_ نجمعه

- لا مسارات مجلدات ولا معرّفات مشتقة من المجلدات.
- لا محتويات ملفات.
- لا سجلات لكل تشغيل، ولا مطالبات، ولا أي مخرجات CLI أخرى.

## أعداد التثبيت

يحتفظ ClawHub بعدّادات إجمالية لكل مهارة:

- `installsAllTime`: المستخدمون الفريدون الذين أبلغوا عن تثبيت واحد على الأقل عبر CLI للمهارة.
- `installsCurrent`: المستخدمون الفريدون الذين أبلغوا عن تثبيت ولم يحذفوا
  بيانات القياس عن بُعد الخاصة بهم.

## الشفافية + عناصر تحكم المستخدم

يرى الجميع **عدّادات تثبيت مجمّعة** فقط.

يؤدي حذف حسابك أيضًا إلى حذف بيانات القياس عن بُعد الخاصة بك.

## كيفية تعطيل القياس عن بُعد

عيّن متغير البيئة:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

عند تعيين ذلك، لن يرسل CLI قياس تثبيت عن بُعد.
