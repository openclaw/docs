## الأهداف

- إعادة المحاولة لكل طلب HTTP، وليس لكل تدفق متعدد الخطوات.
- الحفاظ على الترتيب عبر إعادة محاولة الخطوة الحالية فقط.
- تجنب تكرار العمليات غير idempotent.

## القيم الافتراضية

- المحاولات: 3
- الحد الأقصى للتأخير: 30000 ms
- jitter: 0.1 ‏(10 بالمئة)
- القيم الافتراضية لكل موفر:
  - الحد الأدنى للتأخير في Telegram: ‏400 ms
  - الحد الأدنى للتأخير في Discord: ‏500 ms

## السلوك

### موفرو النماذج

- يترك OpenClaw لحزم SDK الخاصة بالموفر التعامل مع عمليات إعادة المحاولة القصيرة العادية.
- بالنسبة إلى حزم SDK المبنية على Stainless مثل Anthropic وOpenAI، يمكن أن تتضمن الاستجابات
  القابلة لإعادة المحاولة (`408` و`409` و`429` و`5xx`) القيمتين `retry-after-ms` أو
  `retry-after`. وعندما تكون مدة الانتظار هذه أطول من 60 ثانية، يحقن OpenClaw
  `x-should-retry: false` حتى تُظهر SDK الخطأ فورًا ويستطيع
  failover النموذج التبديل إلى ملف مصادقة آخر أو نموذج احتياطي.
- تجاوز الحد الأقصى باستخدام `OPENCLAW_SDK_RETRY_MAX_WAIT_SECONDS=<seconds>`.
  اضبطه على `0` أو `false` أو `off` أو `none` أو `disabled` للسماح لـ SDKs باحترام
  فترات السكون الطويلة `Retry-After` داخليًا.

### Discord

- يعيد المحاولة فقط عند أخطاء حدود المعدل (HTTP 429).
- يستخدم `retry_after` الخاص بـ Discord عند توفره، وإلا يستخدم exponential backoff.

### Telegram

- يعيد المحاولة عند الأخطاء العابرة (429، والمهلة، والاتصال/إعادة الضبط/الإغلاق، وعدم التوفر المؤقت).
- يستخدم `retry_after` عند توفره، وإلا يستخدم exponential backoff.
- لا تتم إعادة محاولة أخطاء تحليل Markdown؛ بل ترجع إلى نص عادي.

## الإعداد

اضبط سياسة إعادة المحاولة لكل موفر في `~/.openclaw/openclaw.json`:

```json5
{
  channels: {
    telegram: {
      retry: {
        attempts: 3,
        minDelayMs: 400,
        maxDelayMs: 30000,
        jitter: 0.1,
      },
    },
    discord: {
      retry: {
        attempts: 3,
        minDelayMs: 500,
        maxDelayMs: 30000,
        jitter: 0.1,
      },
    },
  },
}
```

## ملاحظات

- تنطبق عمليات إعادة المحاولة على كل طلب على حدة (إرسال رسالة، ورفع وسائط، وتفاعل، واستطلاع، وملصق).
- لا تعيد التدفقات المركبة محاولة الخطوات المكتملة.
