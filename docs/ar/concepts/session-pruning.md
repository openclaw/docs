يقوم تشذيب الجلسة بقص **نتائج الأدوات القديمة** من السياق قبل كل استدعاء إلى LLM. وهو يقلل تضخم السياق الناتج عن تراكم مخرجات الأدوات (نتائج exec، وقراءات الملفات، ونتائج البحث) من دون إعادة كتابة نص المحادثة العادي.

<Info>
يتم التشذيب داخل الذاكرة فقط -- ولا يعدّل transcript الجلسة المخزن على القرص.
ويظل السجل الكامل محفوظًا دائمًا.
</Info>

## لماذا يهم ذلك

تتراكم مخرجات الأدوات في الجلسات الطويلة، مما يؤدي إلى تضخيم نافذة السياق. وهذا
يزيد التكلفة وقد يفرض [Compaction](/ar/concepts/compaction) في وقت أبكر من
اللازم.

يكون التشذيب ذا قيمة خاصة من أجل **التخزين المؤقت للمطالبة في Anthropic**. بعد انتهاء
صلاحية TTL الخاصة بالتخزين المؤقت، يعيد الطلب التالي تخزين المطالبة الكاملة مؤقتًا. ويقلل التشذيب
حجم كتابة التخزين المؤقت، مما يخفض التكلفة مباشرة.

## كيف يعمل

1. الانتظار حتى تنتهي صلاحية TTL الخاصة بالتخزين المؤقت (الافتراضي 5 دقائق).
2. العثور على نتائج الأدوات القديمة من أجل التشذيب العادي (ويُترك نص المحادثة كما هو).
3. **تشذيب مرن** للنتائج كبيرة الحجم -- الاحتفاظ بالبداية والنهاية، وإدراج `...`.
4. **إزالة صارمة** للباقي -- استبداله بعنصر نائب.
5. إعادة تعيين TTL بحيث تعيد الطلبات اللاحقة استخدام التخزين المؤقت الجديد.

## تنظيف الصور القديم

يشغّل OpenClaw أيضًا عملية تنظيف منفصلة قابلة للتكرار للجلسات القديمة
التي كانت تحفظ كتل الصور الخام في السجل.

- يحافظ على **آخر 3 أدوار مكتملة** حرفيًا كما هي بحيث تبقى
  بادئات التخزين المؤقت للمطالبة مستقرة في المتابعات الحديثة.
- يمكن استبدال كتل الصور الأقدم التي تمت معالجتها بالفعل في سجل `user` أو `toolResult`
  بـ `[image data removed - already processed by model]`.
- هذا منفصل عن التشذيب العادي المرتبط بـ cache-TTL. وهو موجود لإيقاف
  حمولات الصور المتكررة من إفساد التخزين المؤقت للمطالبة في الأدوار اللاحقة.

## القيم الافتراضية الذكية

يقوم OpenClaw تلقائيًا بتفعيل التشذيب لملفات Anthropic الشخصية:

| نوع الملف الشخصي                                         | التشذيب مفعّل | Heartbeat |
| -------------------------------------------------------- | ------------- | --------- |
| مصادقة Anthropic عبر OAuth/الرمز (بما في ذلك إعادة استخدام Claude CLI) | نعم           | ساعة واحدة |
| مفتاح API                                                | نعم           | 30 دقيقة  |

إذا قمت بضبط قيم صريحة، فلن يتجاوزها OpenClaw.

## التفعيل أو التعطيل

يكون التشذيب معطّلًا افتراضيًا لمزوّدي non-Anthropic. للتفعيل:

```json5
{
  agents: {
    defaults: {
      contextPruning: { mode: "cache-ttl", ttl: "5m" },
    },
  },
}
```

للتعطيل: اضبط `mode: "off"`.

## التشذيب مقابل Compaction

|            | التشذيب            | Compaction              |
| ---------- | ------------------ | ----------------------- |
| **ما هو**  | يقص نتائج الأدوات  | يلخص المحادثة           |
| **هل يُحفَظ؟** | لا (لكل طلب)       | نعم (في transcript)     |
| **النطاق** | نتائج الأدوات فقط   | المحادثة كاملة          |

يكمل أحدهما الآخر -- فالتشذيب يحافظ على خفة مخرجات الأدوات بين
دورات Compaction.
