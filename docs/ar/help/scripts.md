يحتوي دليل `scripts/` على نصوص برمجية مساعدة لسير العمل المحلي ومهام العمليات.
استخدمها عندما تكون المهمة مرتبطة بوضوح بنص برمجي؛ وإلا ففضّل CLI.

## الاصطلاحات

- النصوص البرمجية **اختيارية** ما لم تُذكر في الوثائق أو قوائم تحقق الإصدار.
- فضّل واجهات CLI عندما تكون موجودة (مثال: تستخدم مراقبة المصادقة `openclaw models status --check`).
- افترض أن النصوص البرمجية خاصة بالمضيف؛ اقرأها قبل تشغيلها على جهاز جديد.

## نصوص مراقبة المصادقة البرمجية

تُغطى مراقبة المصادقة في [المصادقة](/ar/gateway/authentication). النصوص البرمجية ضمن `scripts/` هي إضافات اختيارية لسير عمل هواتف systemd/Termux.

## مساعد القراءة من GitHub

استخدم `scripts/gh-read` عندما تريد أن يستخدم `gh` رمز تثبيت GitHub App لاستدعاءات القراءة المقيدة بالمستودع، مع إبقاء `gh` العادي على تسجيل دخولك الشخصي لإجراءات الكتابة.

متغيرات البيئة المطلوبة:

- `OPENCLAW_GH_READ_APP_ID`
- `OPENCLAW_GH_READ_PRIVATE_KEY_FILE`

متغيرات البيئة الاختيارية:

- `OPENCLAW_GH_READ_INSTALLATION_ID` عندما تريد تخطي البحث عن التثبيت بناءً على المستودع
- `OPENCLAW_GH_READ_PERMISSIONS` كتجاوز مفصول بفواصل لمجموعة أذونات القراءة الفرعية المطلوب طلبها

ترتيب تحديد المستودع:

- `gh ... -R owner/repo`
- `GH_REPO`
- `git remote origin`

أمثلة:

- `scripts/gh-read pr view 123`
- `scripts/gh-read run list -R openclaw/openclaw`
- `scripts/gh-read api repos/openclaw/openclaw/pulls/123`
