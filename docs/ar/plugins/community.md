حزم المجتمع الإضافية هي حزم تابعة لجهات خارجية توسّع OpenClaw بإضافة
قنوات أو أدوات أو موفّرين أو خطافات أو إمكانات أخرى. استخدم
[ClawHub](/clawhub) بوصفه الواجهة الأساسية لاكتشاف حزم المجتمع الإضافية
العامة.

## العثور على الحزم الإضافية

ابحث في ClawHub من خلال CLI:

```bash
openclaw plugins search "calendar"
```

ثبّت حزمة إضافية من ClawHub باستخدام بادئة مصدر صريحة:

```bash
openclaw plugins install clawhub:<package-name>
```

يظل npm مسارًا مدعومًا للتثبيت المباشر خلال فترة الانتقال عند الإطلاق:

```bash
openclaw plugins install npm:<package-name>
```

استخدم [إدارة الحزم الإضافية](/ar/plugins/manage-plugins) للاطلاع على أمثلة شائعة
للتثبيت والتحديث والفحص وإلغاء التثبيت. واستخدم
[`openclaw plugins`](/ar/cli/plugins) للاطلاع على مرجع الأوامر الكامل وقواعد
اختيار المصدر.

## نشر الحزم الإضافية

انشر حزم المجتمع الإضافية العامة على ClawHub كي يتمكن مستخدمو OpenClaw من
اكتشافها وتثبيتها. يدير ClawHub قائمة الحزم المباشرة وسجل الإصدارات وحالة
الفحص وإرشادات التثبيت؛ ولا تحتفظ الوثائق بفهرس ثابت للحزم الإضافية التابعة
لجهات خارجية.

```bash
clawhub package publish your-org/your-plugin --dry-run
clawhub package publish your-org/your-plugin
```

قبل النشر، تأكد من أن الحزمة الإضافية تتضمن بيانات وصفية للحزمة وبيانًا
للحزمة الإضافية ووثائق إعداد ومالك صيانة محددًا بوضوح. يتحقق ClawHub من نطاق
المالك واسم الحزمة وإصدارها وحدود الملفات والبيانات الوصفية للمصدر قبل إنشاء
إصدار، ثم يُبقي الإصدارات الجديدة مخفية عن واجهات التثبيت والتنزيل المعتادة
حتى اكتمال المراجعة والتحقق.

قائمة التحقق قبل النشر:

| المتطلب                 | السبب                                                    |
| ----------------------- | -------------------------------------------------------- |
| منشورة على ClawHub      | يحتاج المستخدمون إلى عمل إرشادات `openclaw plugins install` |
| مستودع GitHub عام       | مراجعة المصدر وتتبع المشكلات والشفافية                   |
| وثائق الإعداد والاستخدام | يحتاج المستخدمون إلى معرفة كيفية تهيئتها                 |
| صيانة نشطة              | تحديثات حديثة أو معالجة سريعة الاستجابة للمشكلات         |
