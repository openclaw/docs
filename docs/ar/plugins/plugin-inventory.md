# جرد Plugin

تُولَّد هذه الصفحة من `extensions/*/package.json` و`openclaw.plugin.json`
واستثناءات `files` في حزمة npm الجذرية. أعد توليدها باستخدام:

```bash
pnpm plugins:inventory:gen
```

## التعريفات

- **حزمة npm الأساسية:** مضمّنة في حزمة npm‏ `openclaw` ومتاحة من دون تثبيت Plugin منفصل.
- **الحزمة الخارجية الرسمية:** Plugin تتم صيانته بواسطة OpenClaw ومحذوف من حزمة npm الأساسية، ويُثبَّت عبر ClawHub و/أو npm.
- **نسخة المصدر المحلية فقط:** Plugin محلي في المستودع ومحذوف من عناصر npm المنشورة ولا يُعلَن عنه كحزمة قابلة للتثبيت.

تختلف نسخ المصدر المحلية عن تثبيتات npm: بعد `pnpm install`، تُحمَّل
Plugin المضمّنة من `extensions/<id>` بحيث تكون التعديلات المحلية واعتماديات
مساحة العمل المحلية للحزمة متاحة.
