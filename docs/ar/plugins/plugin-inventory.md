# مخزون Plugin

تُولَّد هذه الصفحة من `extensions/*/package.json` و`openclaw.plugin.json`،
واستثناءات `files` في حزمة npm الجذرية. أعد توليدها باستخدام:

```bash
pnpm plugins:inventory:gen
```

## التعريفات

- **حزمة npm الأساسية:** مضمّنة في حزمة npm باسم `openclaw` ومتاحة دون تثبيت Plugin منفصل.
- **حزمة خارجية رسمية:** Plugin تتم صيانته بواسطة OpenClaw ومحذوف من حزمة npm الأساسية، ويُحفظ في هذا المخزون الرسمي، ويُثبَّت عند الطلب عبر ClawHub و/أو npm.
- **نسخة المصدر المحلية فقط:** Plugin محلي في المستودع ومحذوف من عناصر npm المنشورة ولا يُعلن عنه كحزمة قابلة للتثبيت.

تختلف نسخ المصدر المحلية عن تثبيتات npm: بعد `pnpm install`، تُحمَّل
Plugins المضمّنة من `extensions/<id>` بحيث تكون التعديلات المحلية وتبعيات مساحة العمل المحلية للحزمة
متاحة.

## تثبيت Plugin

استخدم عمود **التوزيع** لتحديد ما إذا كان التثبيت مطلوبًا. Plugins التي
تقول `included in OpenClaw` موجودة بالفعل في الحزمة الأساسية. تحتاج
الحزم الخارجية الرسمية إلى تثبيت واحد، ثم إعادة تشغيل Gateway.

على سبيل المثال، Discord حزمة خارجية رسمية:

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

تحاول مواصفات الحزم المجردة استخدام ClawHub أولًا، ثم الرجوع إلى npm. لفرض مصدر، استخدم
`clawhub:@openclaw/discord` أو `npm:@openclaw/discord`. بعد التثبيت، اتبع
مستند إعداد Plugin، مثل [Discord](/ar/channels/discord)، لإضافة بيانات الاعتماد
وتكوين القناة. راجع [إدارة Plugins](/ar/plugins/manage-plugins) لأوامر التحديث،
وإلغاء التثبيت، والنشر.
