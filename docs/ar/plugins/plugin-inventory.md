# جرد Plugin

هذه الصفحة مولّدة من `extensions/*/package.json`، و`openclaw.plugin.json`،
واستثناءات `files` في حزمة npm الجذرية. أعد توليدها باستخدام:

```bash
pnpm plugins:inventory:gen
```

## التعريفات

- **حزمة npm الأساسية:** مدمجة في حزمة npm `openclaw` ومتاحة بدون تثبيت Plugin منفصل.
- **حزمة خارجية رسمية:** Plugin تتم صيانته بواسطة OpenClaw ومحذوف من حزمة npm الأساسية ومثبّت عبر ClawHub و/أو npm.
- **نسخة مصدرية فقط:** Plugin محلي في المستودع محذوف من عناصر npm المنشورة ولا يُعلن عنه كحزمة قابلة للتثبيت.

تختلف نسخ المصدر عن تثبيتات npm: بعد `pnpm install`، يتم تحميل Plugins
المضمّنة من `extensions/<id>` بحيث تكون التعديلات المحلية واعتماديات مساحة العمل
المحلية للحزمة متاحة.
