# جرد Plugin

تُنشأ هذه الصفحة من `extensions/*/package.json` و`openclaw.plugin.json`،
واستثناءات `files` في حزمة npm الجذرية. أعد إنشاءها باستخدام:

```bash
pnpm plugins:inventory:gen
```

## التعريفات

- **حزمة npm الأساسية:** مدمجة في حزمة npm `openclaw` ومتاحة دون تثبيت Plugin منفصل.
- **حزمة خارجية رسمية:** Plugin تصونه OpenClaw ومحذوف من حزمة npm الأساسية، ومحفوظ في هذا الجرد الرسمي، ويُثبَّت عند الطلب عبر ClawHub و/أو npm.
- **للنسخة المصدرية المحلية فقط:** Plugin محلي في المستودع محذوف من ملفات npm المنشورة ولا يُعلَن عنه كحزمة قابلة للتثبيت.

تختلف النسخ المصدرية المحلية عن تثبيتات npm: بعد `pnpm install`، تُحمَّل
Plugins المضمّنة من `extensions/<id>` بحيث تكون التعديلات المحلية واعتماديات
مساحة العمل المحلية للحزمة متاحة.

## تثبيت Plugin

استخدم عمود **التوزيع** لتحديد ما إذا كان التثبيت مطلوبًا. Plugins التي
تقول `included in OpenClaw` موجودة مسبقًا في الحزمة الأساسية. تحتاج
الحزم الخارجية الرسمية إلى تثبيت واحد، ثم إعادة تشغيل Gateway.

على سبيل المثال، Discord حزمة خارجية رسمية:

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

تحاول مواصفات الحزم المجردة استخدام ClawHub أولًا، ثم الرجوع إلى npm. لفرض مصدر، استخدم
`clawhub:@openclaw/discord` أو `npm:@openclaw/discord`. بعد التثبيت، اتبع
مستند إعداد Plugin، مثل [Discord](/ar/channels/discord)، لإضافة بيانات الاعتماد
وتكوين القناة. راجع [إدارة Plugins](/ar/plugins/manage-plugins) للاطلاع على أوامر التحديث
وإلغاء التثبيت والنشر.
