يمكن أن يحمل خرج المساعد مجموعة صغيرة من توجيهات التسليم/العرض:

- `MEDIA:` لتسليم المرفقات
- `[[audio_as_voice]]` لتلميحات عرض الصوت
- `[[reply_to_current]]` / `[[reply_to:<id>]]` لبيانات الرد الوصفية
- `[embed ...]` للعرض الغني في واجهة Control UI

يجب أن تكون مرفقات `MEDIA:` البعيدة عناوين URL عامة ببروتوكول `https:`. يتم تجاهل `http:` العادي،
وloopback، وlink-local، والعناوين الخاصة، وأسماء المضيفين الداخلية كتوجيهات مرفقات؛
ومع ذلك تظل أدوات جلب الوسائط من جهة الخادم تفرض حراس الشبكة الخاصة بها.

تبقى صيغة صور Markdown العادية نصًا افتراضيًا. القنوات التي تعمد إلى
ربط ردود صور Markdown بمرفقات وسائط تشترك في ذلك من خلال
المهايئ الصادر الخاص بها؛ يفعل Telegram ذلك حتى يظل `![alt](url)` قادرًا على أن يصبح رد وسائط.

هذه التوجيهات منفصلة. تظل `MEDIA:` ووسوم الرد/الصوت بيانات وصفية للتسليم؛ أما `[embed ...]` فهو مسار العرض الغني المخصص للويب فقط.
تستخدم وسائط نتائج الأدوات الموثوقة محلل `MEDIA:` / `[[audio_as_voice]]` نفسه قبل التسليم، لذلك يمكن لمخرجات الأدوات النصية أن تظل قادرة على تعليم مرفق صوتي كملاحظة صوتية.

عند تفعيل البث الكتلي، تظل `MEDIA:` بيانات وصفية لتسليم واحد ضمن
الدور. إذا أُرسل عنوان URL الوسيط نفسه في كتلة مبثوثة وتكرر في حمولة
المساعد النهائية، يسلّم OpenClaw المرفق مرة واحدة ويزيل النسخة المكررة
من الحمولة النهائية.

## `[embed ...]`

`[embed ...]` هي صيغة العرض الغني الوحيدة المواجهة للوكيل في واجهة Control UI.

مثال ذاتي الإغلاق:

```text
[embed ref="cv_123" title="Status" /]
```

القواعد:

- لم تعد `[view ...]` صالحة للمخرجات الجديدة.
- تُعرض الرموز المختصرة للتضمين في سطح رسالة المساعد فقط.
- لا تُعرض إلا التضمينات المدعومة بعنوان URL. استخدم `ref="..."` أو `url="..."`.
- لا تُعرض الرموز المختصرة للتضمين بتنسيق HTML المضمن على هيئة كتلة.
- تزيل واجهة الويب الرمز المختصر من النص المرئي وتعرض التضمين ضمن السطر.
- `MEDIA:` ليست اسمًا بديلًا للتضمين ولا ينبغي استخدامها لعرض التضمين الغني.

## شكل العرض المخزّن

كتلة محتوى المساعد المعيارية/المخزنة هي عنصر `canvas` منظم:

```json
{
  "type": "canvas",
  "preview": {
    "kind": "canvas",
    "surface": "assistant_message",
    "render": "url",
    "viewId": "cv_123",
    "url": "/__openclaw__/canvas/documents/cv_123/index.html",
    "title": "Status",
    "preferredHeight": 320
  }
}
```

تستخدم الكتل الغنية المخزنة/المعروضة شكل `canvas` هذا مباشرة. لا يتم التعرف على `present_view`.
