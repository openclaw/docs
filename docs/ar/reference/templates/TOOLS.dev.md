# TOOLS.md - ملاحظات المستخدم حول الأدوات (قابل للتحرير)

هذا الملف مخصص _لملاحظاتك أنت_ حول الأدوات الخارجية والاتفاقيات.
ولا يعرّف الأدوات الموجودة؛ إذ يوفّر OpenClaw الأدوات المضمنة داخليًا.

## أمثلة

### imsg

- إرسال iMessage/SMS: صف إلى من/ماذا، وأكّد قبل الإرسال.
- فضّل الرسائل القصيرة؛ وتجنب إرسال الأسرار.

### sag

- تحويل النص إلى كلام: حدّد الصوت، والسماعة/الغرفة المستهدفة، وما إذا كان يجب البث.

أضف أي شيء آخر تريد أن يعرفه المساعد عن سلسلة الأدوات المحلية الخاصة بك.
