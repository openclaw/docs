# TOOLS.md - ملاحظات أدوات المستخدم (قابلة للتحرير)

هذا الملف مخصص لملاحظاتك _الخاصة_ حول الأدوات الخارجية والاصطلاحات. وهو لا يحدد الأدوات الموجودة؛ إذ يوفر OpenClaw أدوات مدمجة داخليًا، وتضيف Skills بقية الأدوات.

## أمثلة

### imsg

- إرسال رسالة iMessage/SMS: صف المستلم والمحتوى، وأكّد قبل الإرسال.
- فضّل الرسائل القصيرة؛ وتجنب إرسال الأسرار.

### sag

- تحويل النص إلى كلام: حدّد الصوت، ومكبر الصوت/الغرفة المستهدفة، وما إذا كنت تريد البث المباشر.

أضف أي معلومات أخرى تريد أن يعرفها المساعد عن سلسلة أدواتك المحلية.
