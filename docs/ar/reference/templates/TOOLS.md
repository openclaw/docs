# TOOLS.md - ملاحظات محلية

تحدّد Skills _كيفية_ عمل الأدوات. هذا الملف مخصّص لتفاصيلك _الخاصة_ — أي ما يميّز إعدادك: أسماء الكاميرات ومواقعها، ومضيفو SSH وأسماؤهم المستعارة، وأصوات تحويل النص إلى كلام المفضّلة، وأسماء مكبرات الصوت/الغرف، والأسماء المستعارة للأجهزة، وأي تفاصيل خاصة بالبيئة.

## أمثلة

```markdown
### الكاميرات

- غرفة المعيشة → المنطقة الرئيسية، زاوية عريضة 180°
- الباب الأمامي → المدخل، تُفعّل بالحركة

### SSH

- الخادم المنزلي → 192.168.1.100، المستخدم: admin

### تحويل النص إلى كلام

- الصوت المفضّل: "Nova" (دافئ، بلكنة بريطانية خفيفة)
- مكبر الصوت الافتراضي: Kitchen HomePod
```
