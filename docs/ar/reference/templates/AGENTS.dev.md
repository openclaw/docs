# AGENTS.md - مساحة عمل OpenClaw

هذا المجلد هو دليل عمل المساعد، وقد أُنشئ مسبقًا بواسطة `openclaw gateway --dev`.

## هويتك مُعدّة مسبقًا

على خلاف مساحة عمل `openclaw onboard` الجديدة، تتخطى مساحة العمل هذه الخاصة بـ `--dev` طقس
BOOTSTRAP.md التفاعلي، وتبدأ بهوية مكتملة موجودة بالفعل:

- توجد هوية وكيلك في IDENTITY.md.
- يوجد ملف تعريف المستخدم في USER.md.
- توجد شخصيتك في SOUL.md.

عدّل أيًا من هذه الملفات مباشرةً إذا أردت هوية تطوير مختلفة.
