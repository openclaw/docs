# `openclaw dashboard`

افتح واجهة التحكم باستخدام مصادقتك الحالية.

```bash
openclaw dashboard
openclaw dashboard --no-open
openclaw dashboard --json
openclaw dashboard --yes
```

- `--no-open`: اطبع عنوان URL، لكن لا تشغّل متصفحًا.
- `--json`: اطبع كائن اتصال واحدًا قابلًا للقراءة آليًا من دون فتح متصفح أو استخدام الحافظة أو عرض مطالبة أو بدء Gateway.
- `--yes`: ابدأ/ثبّت Gateway من دون مطالبة عند الحاجة.

## مخرجات قابلة للقراءة آليًا

استخدم `--json` لتكاملات سطح المكتب والبرامج النصية التي تحتاج إلى عنوان URL المحسوم لواجهة التحكم:

```bash
openclaw dashboard --json
```

تتضمن الاستجابة `url` و`httpUrl` و`wsUrl` و`port` و`tokenIncluded`. إذا لم يكن Gateway جاهزًا، يُرجع الأمر `{"ok":false,"reason":"..."}` وينتهي برمز خروج غير صفري. لا تُضمَّن الرموز المميزة المُدارة بواسطة SecretRef مطلقًا في `url`.

ملاحظات:

- يحل مراجع SecretRef الخاصة بـ `gateway.auth.token` المُعدّة عند الإمكان.
- يتبع `gateway.tls.enabled`: تطبع بوابات Gateway المُمكَّن فيها TLS عناوين URL لواجهة التحكم `https://` أو تفتحها، وتتصل عبر `wss://`.
- بالنسبة إلى ربط `lan` أو ربط `custom` ذي حرف بدل، تستخدم عمليات التشغيل على المضيف نفسه دائمًا عنوان الاسترجاع لأن حرف البدل ليس وجهة للمتصفح. تستخدم أيضًا روابط النص الصريح `tailnet` و`custom`‏ `127.0.0.1` لكي يتوفر للمتصفح سياق آمن؛ وتحتفظ المضيفات المحددة المُمكَّن فيها TLS بالعنوان المُعدّ حتى تتطابق أسماء الشهادات.
- قبل تسليم عنوان URL مُصادق عليه للاسترجاع لربط بواجهة محددة، يفحص الأمر الواجهة المُعدّة ويتحقق من أنها و`127.0.0.1` مملوكتان لعملية Gateway نفسها. عند غموض ملكية المستمع، يفشل الأمر بصورة آمنة مع إرشادات للحالة.
- بالنسبة إلى الرموز المميزة المُدارة بواسطة SecretRef (سواء حُلّت أم لم تُحل)، لا يتضمن عنوان URL المطبوع أو المنسوخ أو المفتوح الرمز المميز مطلقًا، لذا لا تتسرب الأسرار الخارجية إلى مخرجات الطرفية أو سجل الحافظة أو وسائط تشغيل المتصفح.
- إذا كان `gateway.auth.token` مُدارًا بواسطة SecretRef ولكنه غير محلول، يطبع الأمر عنوان URL من دون رمز مميز وإرشادات للمعالجة بدلًا من عنصر نائب غير صالح للرمز المميز.
- إذا فشل التسليم عبر الحافظة/المتصفح لعنوان URL مُصادق عليه برمز مميز، يسجّل الأمر تلميحًا آمنًا للمصادقة اليدوية يذكر `OPENCLAW_GATEWAY_TOKEN` و`gateway.auth.token` ومفتاح جزء عنوان URL‏ `token`، من دون طباعة قيمة الرمز المميز.
