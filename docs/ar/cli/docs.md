# `openclaw docs`

ابحث في فهرس وثائق OpenClaw المباشر من الطرفية.

## الاستخدام

```bash
openclaw docs                       # طباعة نقطة دخول الوثائق ومثال للبحث
openclaw docs <query...>            # البحث في فهرس الوثائق المباشر
```

| الوسيط       | الوصف                                                                                 |
| ------------ | ------------------------------------------------------------------------------------- |
| `[query...]` | استعلام بحث حرّ الصياغة. تُضم الاستعلامات متعددة الكلمات بمسافات وتُرسل كاستعلام واحد. |

عند عدم تحديد استعلام، يطبع `openclaw docs` عنوان URL لنقطة دخول الوثائق وأمر بحث نموذجيًا بدلًا من إجراء بحث.

## أمثلة

```bash
openclaw docs browser existing-session
openclaw docs sandbox allowHostControl
openclaw docs gateway token secretref
```

## آلية العمل

يستدعي `openclaw docs` العنوان `https://docs.openclaw.ai/api/search` ويعرض نتائج JSON. يستخدم طلب البحث مهلة ثابتة قدرها 30 ثانية.

## المخرجات

في طرفية غنية (TTY)، تُعرض النتائج كعنوان تتبعه قائمة نقطية: عنوان الصفحة، وعنوان URL مرتبط للوثائق، ومقتطف قصير في السطر التالي. تطبع النتائج الفارغة "لا توجد نتائج.".

في المخرجات غير الغنية (عبر الأنابيب، أو `--no-color`، أو السكربتات)، تُعرض البيانات نفسها بصيغة Markdown:

```markdown
# البحث في الوثائق: <query>

- [العنوان](https://docs.openclaw.ai/...) - مقتطف
- [العنوان](https://docs.openclaw.ai/...) - مقتطف
```

## رموز الخروج

| الرمز | المعنى                                                                       |
| ----- | ---------------------------------------------------------------------------- |
| `0`   | نجح البحث، بما في ذلك الاستجابات التي لا تتضمن أي نتائج.                     |
| `1`   | فشل استدعاء واجهة API المستضافة للبحث في الوثائق؛ تطبع stderr رسالة الخطأ.   |
