# 텔레메트리

ClawHub는 **최소한의 텔레메트리**를 사용해 **설치 수**(실제로 사용 중인 항목)를 계산하고 더 나은 정렬/필터링을 지원합니다.
이는 CLI `clawhub sync` 명령을 기반으로 합니다.

## 텔레메트리가 수집되는 경우

텔레메트리는 다음 경우에만 전송됩니다.

- CLI에서 **로그인한** 상태입니다(동기화/게시 흐름에는 이미 인증이 필요합니다).
- `clawhub sync`를 실행합니다.
- 텔레메트리가 **비활성화되어 있지 않습니다**(아래 “비활성화 방법” 참조).

로그인하지 않은 경우 아무것도 보고되지 않습니다.

## 수집하는 항목

각 `clawhub sync`에서 CLI는 발견한 항목의 **전체 스냅샷**을 스캔 루트(“폴더/루트”)별로 그룹화해 보고합니다.

각 루트에 대해 저장하는 항목은 다음과 같습니다.

- `rootId`: 표준 루트 경로의 **SHA-256 해시**입니다(서버는 원시 경로를 절대 보지 않습니다).
- `label`: 마지막 두 경로 세그먼트에서 파생된 사람이 읽을 수 있는 레이블입니다(홈 경로는 `~`로 표시됩니다).
- `firstSeenAt`, `lastSeenAt`, 선택적 `expiredAt`.

루트 아래에서 발견된 각 Skills에 대해 저장하는 항목은 다음과 같습니다.

- `skillId`(슬러그로 확인됨. 레지스트리에 존재하는 Skills만 추적됩니다).
- `firstSeenAt`, `lastSeenAt`.
- `lastVersion`(최선의 노력으로 기록됨. 현재는 알려진 경우 레지스트리와 일치하는 버전).
- 이전에 보고된 설치가 루트에서 사라질 때 선택적 `removedAt`.

### 수집하지 _않는_ 항목

- 원시 절대 폴더 경로는 수집하지 않습니다(해시된 `rootId` + 짧은 표시 레이블만).
- 파일 내용은 수집하지 않습니다.
- 실행별 로그, 프롬프트 또는 기타 CLI 출력은 수집하지 않습니다.
- 레지스트리에 업로드되지 않은 Skills는 추적하지 않습니다(알 수 없는 슬러그는 무시됩니다).

## 설치 수

Skills마다 두 개의 카운터를 유지합니다.

- `installsCurrent`: 현재 하나 이상의 활성 루트에 해당 Skills를 설치한 고유 사용자 수입니다.
- `installsAllTime`: 지금까지 해당 Skills가 설치되었다고 보고한 적이 있는 고유 사용자 수입니다.

### 여러 루트

여러 폴더에서 동기화하는 경우 각 스캔 루트를 독립적으로 처리합니다. Skills가 **어느 하나의** 활성 루트에 존재하면 “현재 설치됨”으로 간주됩니다.

### 제거 감지

`sync`가 루트별 전체 집합을 보고하기 때문에:

- 다음 동기화에서 Skills가 루트에서 사라지면 해당 루트에서 제거된 것으로 표시합니다.
- 해당 Skills가 사용자의 모든 루트에서 제거되면 더 이상 `installsCurrent`에 집계되지 않습니다.
- 텔레메트리를 삭제하지 않는 한 `installsAllTime`은 절대 감소하지 않습니다(아래 참조).

### 오래됨(120일)

**120일** 동안 텔레메트리를 보고하지 않는 루트는 오래된 것으로 표시되며, 해당 설치는 `installsCurrent` 집계에서 제외됩니다.
이는 백그라운드 작업을 피하기 위해 지연 평가됩니다(다음 텔레메트리 보고 시).

## 투명성 + 사용자 제어

ClawHub는 사용자 본인 프로필에 비공개 “설치됨” 탭을 제공합니다.

- 저장된 정확한 루트 + 설치된 Skills를 표시합니다.
- **JSON 내보내기** 보기를 포함합니다.
- 계정에 저장된 모든 텔레메트리를 제거하는 **텔레메트리 삭제** 작업을 포함합니다.

다른 모든 사람은 **집계된 설치 카운터**만 볼 수 있으며, 다른 누구도 사용자의 루트/폴더를 볼 수 없습니다.

계정을 삭제하면 텔레메트리 데이터도 함께 삭제됩니다.

## 텔레메트리 비활성화 방법

환경 변수를 설정합니다.

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

이를 설정하면 CLI는 `clawhub sync` 중 텔레메트리를 전송하지 않습니다.
