# Plugin 참조

이 페이지는 `extensions/*/package.json` 및
`openclaw.plugin.json`에서 생성됩니다. 다음 명령으로 다시 생성하세요.

```bash
pnpm plugins:inventory:gen
```

[Plugin 인벤토리](/ko/plugins/plugin-inventory)를 사용하여 배포 방식, 패키지 및 설명별로 생성된 130개의 Plugin 참조 페이지를 탐색하세요.
