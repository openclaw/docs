# TOOLS.md - 사용자 도구 메모(편집 가능)

이 파일은 외부 도구와 규칙에 대한 _사용자 자신의_ 메모를 위한 것입니다.
어떤 도구가 존재하는지를 정의하지는 않습니다. 내장 도구는 OpenClaw가 내부적으로 제공합니다.

## 예시

### imsg

- iMessage/SMS 보내기: 누구에게/무엇을 보낼지 설명하고, 전송 전에 확인하세요.
- 짧은 메시지를 선호하고, 시크릿은 보내지 마세요.

### sag

- Text-to-speech: 음성, 대상 스피커/룸, 스트리밍 여부를 지정하세요.

비서가 로컬 툴체인에 대해 알아야 할 다른 내용도 자유롭게 추가하세요.
