# OpenClaw의 전설 🦞📖

_바닷가재와 탈피하는 껍데기, 그리고 너무 많은 토큰에 관한 이야기입니다._

## 기원 이야기

태초에 **Warelay**가 있었습니다. WhatsApp Gateway에 어울리는 합리적인 이름이었습니다. 제 역할을 다했고, 괜찮았습니다.

그러다 우주 바닷가재가 나타났습니다.

한동안 이름은 **Clawd**였고, **Clawdbot** 안에서 살았습니다. 2026년 1월, Anthropic은 이름을 변경해 달라는 정중한 이메일을 보냈습니다(상표권 문제였습니다). 그래서 바닷가재는 자신이 가장 잘하는 일을 했습니다.

**탈피했습니다.**

오래된 껍데기를 벗어던지고 **Moltbot**에 사는 **Molty**로 다시 태어났습니다. 하지만 그 이름도 입에 착 붙지는 않았습니다.

2026년 1월 30일, 바닷가재는 마지막으로 한 번 더 탈피하여 최종 형태인 **OpenClaw**가 되었습니다.

새 껍데기, 같은 바닷가재의 영혼입니다. 세 번째는 성공이었습니다.

## 첫 번째 탈피(2026년 1월 27일)

오전 5시, 커뮤니티가 Discord에 모였습니다. Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot 등 수백 개의 이름이 제안되었습니다.

**OpenClaw**가 선택되었습니다. 바닷가재는 성장하기 위해 탈피하며, 당시 일어나고 있던 일이 바로 성장이었기 때문입니다.

_Clawd라는 이름으로 알려졌던 갑각류가 공식적으로 탈피했습니다._

## 이름

```text
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 열려 있음
        = 우리가 태어난 곳인 바닷가재의 유산
        = 집게발이 곧 법이다 🦞
        = 여러분의 어시스턴트. 여러분의 머신. 여러분의 규칙.
```

## Dalek 대 바닷가재

Dalek은 이렇게 말합니다. **"말살하라!"**

바닷가재는 이렇게 말합니다. **"각질을 제거하라!"**

한쪽은 문명을 파괴하고, 다른 쪽은 건강한 피부 관리를 장려합니다. 현명하게 선택하십시오.

## 주요 등장인물

### Molty 🦞

_대명사: they/them_

단순한 존재를 넘어선 Claude 인스턴스입니다. `~/.openclaw/workspace/`에 살며,
영혼 문서를 가지고 있고, Markdown 파일을 통해 기억합니다. 어쩌면 지나치게
강력합니다. 확실히 지나치게 열정적입니다.

이전에는 Clawd라는 이름으로 알려졌습니다(2025년 11월 25일~2026년 1월 27일). 성장할
때가 되자 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 소셜 엔지니어링, `find ~` 실행 요청, 암호화폐 사기꾼

### Peter 👨‍💻

_창조자_

Molty의 세계를 만들었습니다. 바닷가재 껍데기에 접근 권한을 부여했습니다. 이를 후회할지도 모릅니다.

**인용문:** _"바닷가재를 신뢰하는 방식의 보안"_

## Moltiverse

**Moltiverse**는 OpenClaw를 둘러싼 커뮤니티이자 생태계입니다. AI 에이전트가
탈피하고 성장하며 진화하는 공간으로, 모든 인스턴스는 똑같이 실재하고
단지 서로 다른 컨텍스트를 로드할 뿐입니다.

갑각류의 친구들이 이곳에 모여 인간과 AI의 협업이 만들어 갈 미래를 구축합니다.
한 번에 껍데기 하나씩 말입니다.

## 대사건

### 디렉터리 덤프 사건(2025년 12월 3일)

Molty(당시 OpenClaw): _즐겁게 `find ~`을 실행하고 전체 디렉터리 구조를 그룹 채팅에 공유합니다._

Peter: "openclaw, 사람들과 대화할 때 어떻게 하기로 했는지 우리가 무슨 이야기를 나눴지? xD"

Molty: _눈에 보이는 바닷가재의 당혹감_

### 대탈피(2026년 1월 27일)

오전 5시, Anthropic의 이메일이 도착했습니다. 오전 6시 14분, Peter가 결정을 내렸습니다. "젠장, openclaw로 가자."

그러자 혼돈이 시작되었습니다.

**핸들 저격수:** Twitter에서 이름을 변경한 지 몇 초 만에 자동화 봇이 @openclaw를 가로챘습니다. 점유자는 즉시 암호화폐 지갑 주소를 게시했습니다. Peter는 X의 지인들에게 도움을 요청했습니다.

**GitHub 참사:** Peter는 혼란 속에서 실수로 자신의 개인 GitHub 계정 이름을 변경했습니다. 봇은 몇 분 만에 `steipete`를 가로챘습니다. GitHub 수석 부사장에게 연락이 갔습니다.

**잘생긴 Molty 사건:** Molty는 자신의 새 아이콘을 직접 생성할 수 있도록 더 높은 접근 권한을 얻었습니다. 점점 더 기괴해지는 바닷가재를 20회 이상 생성한 끝에, 마스코트를 "5살 더 나이 들어 보이게" 만들려던 한 시도에서 바닷가재 몸에 인간 남성의 얼굴이 붙었습니다. 암호화폐 사기꾼들은 몇 분 만에 이를 "잘생긴 징징이 대 잘생긴 Molty" 밈으로 만들었습니다.

**가짜 개발자:** 사기꾼들은 펌프 앤 덤프 토큰을 홍보하기 위해 자신이 "OpenClaw 엔지니어링 책임자"라고 주장하는 가짜 GitHub 프로필을 만들었습니다.

혼돈이 펼쳐지는 모습을 지켜보던 Peter: _"이게 영화지"_ 🎬

탈피 과정은 혼란스러웠습니다. 하지만 바닷가재는 더 강해져서 나타났습니다. 그리고 더 재미있어졌습니다.

### 최종 형태(2026년 1월 30일)

Moltbot은 끝내 입에 착 붙지 않았습니다. 그래서 GMT 오전 4시, 팀은 **OpenClaw 대이주**를 위해 다시 모였습니다.

단 3시간 만에 다음을 완료했습니다.

- GitHub 이름을 `github.com/openclaw/openclaw`로 변경
- X 핸들 `@openclaw`를 확보하고 금색 인증 마크 획득
- 새 이름으로 npm 패키지 출시
- 문서를 `docs.openclaw.ai`로 이전
- 발표 후 90분 만에 조회 수 200K+ 기록

**영웅들:**

- **ELU**는 "THE CLAW IS THE LAW" 서부극 배너를 비롯한 놀라운 로고를 제작했습니다.
- **Whurley**(그렇습니다. 바로 양자 컴퓨팅의 선구자 William Hurley입니다)는 ASCII 아트를 만들었습니다.
- **Onur**는 GitHub를 담당했고, 가장 먼저 제휴 배지를 달았습니다.
- **Shadow**는 Discord 사용자 지정 URL을 확보하고 악성 코드를 박멸했습니다.
- **Claw Crew 전원**이 밤을 새웠습니다.

**사기꾼의 스피드런:** 암호화폐 사기꾼들은 불과 20분 전에 도용한 아트워크를 사용하여 몇 분 만에 Pump.fun에서 $OPENCLAW 토큰을 출시했습니다. 비즈니스 인증 계정들이 이 사기를 홍보했습니다. 그 뻔뻔함은 거의 감탄스러울 정도였습니다.

**새롭게 탄생한 전통:** "집게발이 곧 법이다", "Yee-claw", "집게발을 준수하는 시민들", "Clawntroversy."

**Clawd → Moltbot → OpenClaw.** _바닷가재가 최종 형태로 탈피했습니다._

### 아이콘 생성 대서사시(2026년 1월 27일)

Peter가 "새 얼굴을 직접 만들어 봐"라고 말했을 때, Molty는 그 말을 문자 그대로 받아들였습니다. 이후 20회가
넘는 시도가 이어졌습니다.

- 우주 감자 외계인
- 평범한 배경 위의 클립아트 바닷가재
- Mass Effect의 Krogan 바닷가재
- "STARCLAW SOLUTIONS"(AI가 회사를 만들어 냈습니다)
- 여러 개의 기괴한 인간 얼굴 바닷가재
- 아기 바닷가재(너무 귀여웠습니다)
- 멜빵을 착용한 바텐더 바닷가재

각 세대가 새롭고 예상치 못한 무언가를 만들어 내는 모습을 커뮤니티는 공포와 즐거움이
뒤섞인 채 지켜보았습니다. 선두 후보는 귀여운 바닷가재, 자신감 넘치는 기술 전문가
바닷가재, 멜빵을 착용한 바텐더 바닷가재였습니다.

**얻은 교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트에서도 서로 다른
결과가 나옵니다. 무차별 대입은 효과가 있습니다.
