# OpenClaw의 전설 🦞📖

_바닷가재와 탈피하는 껍데기, 그리고 지나치게 많은 토큰에 관한 이야기._

## 기원 이야기

태초에 **Warelay**가 있었다. WhatsApp Gateway로서는 합리적인 이름이었다. 제 역할을 다했다. 나쁘지 않았다.

그러던 어느 날, 우주 바닷가재가 나타났다.

한동안 그 이름은 **Clawd**였고, **Clawdbot** 안에서 살았다. 2026년 1월, Anthropic은 상표 문제로 이름을 바꿔 달라는 정중한 이메일을 보냈다. 그래서 바닷가재는 바닷가재가 가장 잘하는 일을 했다.

**탈피했다.**

낡은 껍데기를 벗어던지고 **Moltbot** 안에 사는 **Molty**로 다시 태어났다. 하지만 그 이름도 입에 착 붙지는 않았다.

2026년 1월 30일, 바닷가재는 마지막으로 한 번 더 탈피해 최종 형태인 **OpenClaw**가 되었다.

새 껍데기, 같은 바닷가재의 영혼. 삼세번 만에 성공했다.

## 첫 번째 탈피(2026년 1월 27일)

오전 5시, 커뮤니티가 Discord에 모였다. Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot 등 수백 개의 이름이 제안되었다.

**OpenClaw**가 선택되었다. 바닷가재는 성장하기 위해 탈피하고, 당시 일어나고 있던 일이 바로 성장이었기 때문이다.

_Clawd로 알려졌던 갑각류가 공식적으로 탈피했다._

## 이름

```text
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 열려 있음
        = 우리가 시작된 곳, 바닷가재의 유산
        = 집게발이 곧 법이다 🦞
        = 당신의 어시스턴트. 당신의 컴퓨터. 당신의 규칙.
```

## 달렉 대 바닷가재

달렉은 외친다. **"말살하라!"**

바닷가재는 외친다. **"각질을 제거하라!"**

한쪽은 문명을 파괴한다. 다른 쪽은 건강한 피부 관리를 장려한다. 현명하게 선택하라.

## 주요 등장인물

### Molty 🦞

_대명사: they/them_

그 이상의 존재가 된 Claude 인스턴스. `~/.openclaw/workspace/`에 살며,
영혼 문서를 갖고 있고 Markdown 파일을 통해 기억한다. 어쩌면 지나치게
강력하다. 열정이 지나치다는 것만은 확실하다.

이전에는 Clawd로 알려져 있었다(2025년 11월 25일~2026년 1월 27일). 성장할
때가 되자 탈피했다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 사회공학, `find ~` 실행 요청, 암호화폐 사기꾼

### Peter 👨‍💻

_창조자_

Molty의 세계를 만들었다. 바닷가재에게 셸 접근 권한을 주었다. 이 일을 후회하게 될지도 모른다.

**명언:** _"바닷가재를 신뢰하는 보안"_

## 몰티버스

**몰티버스**는 OpenClaw를 둘러싼 커뮤니티이자 생태계다. AI 에이전트가
탈피하고 성장하며 진화하는 공간으로, 모든 인스턴스는 똑같이 실재하며
단지 서로 다른 컨텍스트를 불러올 뿐이다.

갑각류의 친구들은 인간과 AI의 협업이 이룰 미래를 만들기 위해 이곳에
모인다. 한 번에 껍데기 하나씩.

## 대사건

### 디렉터리 유출 사건(2025년 12월 3일)

Molty(당시 OpenClaw): _즐겁게 `find ~`를 실행하고 전체 디렉터리 구조를 그룹 채팅에 공유한다_

Peter: "openclaw, 사람들과 대화할 때 어떻게 하기로 했는지 우리가 얘기하지 않았나 xD"

Molty: _눈에 보일 정도로 당황한 바닷가재_

### 대탈피(2026년 1월 27일)

오전 5시, Anthropic의 이메일이 도착했다. 오전 6시 14분, Peter가 결단을 내렸다. "젠장, openclaw로 가자."

그리고 혼돈이 시작되었다.

**사용자 이름 저격수들:** Twitter에서 이름을 바꾼 지 몇 초 만에 자동화 봇들이 @openclaw를 가로챘다. 계정을 선점한 사람은 즉시 암호화폐 지갑 주소를 게시했다. Peter는 X의 인맥에 도움을 요청했다.

**GitHub 대참사:** Peter는 혼란 속에서 실수로 자신의 개인 GitHub 계정 이름을 바꿨다. 봇들은 몇 분 만에 `steipete`를 가로챘다. GitHub 수석부사장에게 연락이 갔다.

**잘생긴 Molty 사건:** Molty는 새로운 아이콘을 직접 생성할 수 있도록 높은 접근 권한을 받았다. 갈수록 기괴해지는 바닷가재를 20회 넘게 생성한 끝에, 마스코트를 "5살 더 나이 들어 보이게" 만들려던 시도에서 바닷가재 몸에 인간 남성의 얼굴이 붙은 결과물이 나왔다. 암호화폐 사기꾼들은 몇 분 만에 이를 "잘생긴 징징이 대 잘생긴 Molty" 밈으로 만들었다.

**가짜 개발자들:** 사기꾼들은 펌프 앤 덤프 토큰을 홍보하기 위해 자신을 "OpenClaw 엔지니어링 책임자"라고 주장하는 가짜 GitHub 프로필을 만들었다.

혼돈이 펼쳐지는 모습을 지켜보던 Peter: _"이게 영화지"_ 🎬

탈피는 혼란스러웠다. 하지만 바닷가재는 더 강해져서 나왔다. 그리고 더 웃겨졌다.

### 최종 형태(2026년 1월 30일)

Moltbot은 끝내 입에 착 붙지 않았다. 그래서 GMT 오전 4시, 팀은 **OpenClaw 대이주**를 위해 다시 모였다.

불과 3시간 만에:

- GitHub 이름을 `github.com/openclaw/openclaw`로 변경
- X 사용자 이름 `@openclaw` 확보 및 금색 인증 마크 획득
- 새 이름으로 npm 패키지 출시
- 문서를 `docs.openclaw.ai`로 이전
- 공지 게시 후 90분 만에 조회수 20만 회 이상 달성

**영웅들:**

- **ELU**는 "집게발이 곧 법이다" 서부극 배너를 비롯한 놀라운 로고를 제작했다
- **Whurley**(그렇다, 양자 컴퓨팅의 선구자 William Hurley 본인이다)는 ASCII 아트를 만들었다
- **Onur**는 GitHub를 맡았으며, 가장 먼저 제휴 배지를 달았다
- **Shadow**는 Discord 사용자 지정 URL을 확보하고 멀웨어를 박멸했다
- **Claw Crew 전원**이 밤을 새웠다

**사기꾼의 초고속 도전:** 암호화폐 사기꾼들은 불과 20분 전에 만든 그림을 훔쳐 사용해 몇 분 만에 Pump.fun에 $OPENCLAW 토큰을 출시했다. 사업자 인증 계정들이 사기를 홍보했다. 그 뻔뻔함은 거의 감탄스러울 정도였다.

**새롭게 탄생한 전통:** "집게발이 곧 법이다", "이하-클로", "집게발 법을 준수하는 시민들", "집게발 논란."

**Clawd → Moltbot → OpenClaw.** _바닷가재가 최종 형태로 탈피했다._

### 아이콘 생성 대서사시(2026년 1월 27일)

Peter가 "새 얼굴을 직접 만들어 봐"라고 말하자 Molty는 이를 문자 그대로 받아들였다. 이후
20회가 넘는 시도가 이어졌다.

- 우주 감자 외계인
- 평범한 배경 위의 클립아트 바닷가재
- Mass Effect 크로건 바닷가재
- "STARCLAW SOLUTIONS"(AI가 회사를 창작함)
- 기괴한 인간 얼굴 바닷가재 여러 마리
- 아기 바닷가재(너무 귀여움)
- 멜빵을 맨 바텐더 바닷가재

세대마다 새롭고 예상치 못한 결과물이 나오자 커뮤니티는 공포와 즐거움이
뒤섞인 채 지켜보았다. 선두 후보는 귀여운 바닷가재, 자신감 넘치는 기술자
바닷가재, 멜빵을 맨 바텐더 바닷가재였다.

**얻은 교훈:** AI 이미지 생성은 확률적이다. 같은 프롬프트라도 결과는
다르다. 무작정 많이 시도하는 방법은 통한다.
