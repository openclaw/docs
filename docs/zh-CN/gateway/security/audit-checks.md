`openclaw security audit` 会生成以 `checkId` 为键的结构化发现。本页是这些 ID 的参考目录。有关高级威胁模型和加固指南，请参阅[安全](/zh-CN/gateway/security)。

某些检查仅在使用 `openclaw security audit --deep` 时运行：插件/Skills 代码扫描（`plugins.code_safety*`、`skills.code_safety*`）和实时 Gateway 网关探测检查（`gateway.probe_*`）。此表中的其他所有检查都会在普通的 `openclaw security audit` 中运行。
