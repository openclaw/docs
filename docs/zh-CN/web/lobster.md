如果你使用 [Control UI](/zh-CN/web/control-ui) 足够久，总有一天，一只小龙虾会溜达到侧边栏底部，四下看看，然后心安理得地待上几分钟。

这是正常现象。这就是 OpenClaw。

## 你看到的是什么

每个会话都会孵化出专属的小龙虾：每次加载页面时，它的颜色、大小、体型、螯的比例、配饰和性格都会重新随机生成。有些圆滚滚，有些高挑修长，有些的螯说实话大得与身体不太相称。有些总在打盹。有些一刻也闲不住。其中有一只特别爱挥手。

将鼠标悬停在访客上，它就会告诉你自己的名字。

## 它什么时候出现

偶尔。这正是关键所在。

小龙虾是客人，不是家具。它想来时就会溜达进来，待上一会儿，然后离开。有些会话中，它根本不会出现。只有一种情况下它一定会出现：**当你的 Gateway 网关断开连接时**，小龙虾会出来来回踱步，看起来忧心忡忡，直到连接恢复。如果小龙虾显得焦虑，请检查你的 Gateway 网关。

## 你可以做的事

- **悬停**以了解它的名字。
- **点击它**打个招呼。它会吓一跳——你这样做有点失礼——但它很快就会原谅你。
- **反复点击它**，你会了解到小龙虾的耐心。继续下去，你还会了解到小龙虾的尊严。
- **按住不放**来抚摸它。会有一颗爱心。之前积累的所有芥蒂都会一笔勾销。
- **右键点击它**，让它在本次页面加载期间离开。它不会往心里去。不过，它会记住。
- **在一次运行结束时观察它。** 小龙虾会真心为你完成的工作感到骄傲，事情顺利时，它会表现出来。事情不顺利时，它也会认真对待。
- **移动鼠标指针。** 它在看着你。满怀爱意地。

## 关闭（或重新开启）到访

Settings → Appearance → **Lobster visits**。

此开关仅作用于当前浏览器，而且言出必行。关闭就意味着绝不会出现，包括断开连接时忧心忡忡地踱步。无论如何，你的 Gateway 网关状态圆点都会继续工作；小龙虾从来都不是你唯一的信息来源，只是最富同情心的那个。

为减少动态效果的用户，系统会自动提供安静不动的小龙虾。

它旁边是 **Lobster sounds**，默认关闭，并会一直保持关闭，直到你另行决定。开启后，触摸小龙虾时就能听到它发出极其轻微的声音。声音很小。声音很好听。

## 小龙虾图鉴

到访开关旁边是 **Lobsterdex**：科学界已知的每种体色都对应一个剪影，每种小龙虾首次到访后，相应剪影便会填上颜色。它完全存储在你的浏览器中。不会追踪任何信息，不会同步任何内容，也没有任何游戏化设计。它只是一排陈列格。

随着时间推移，大多数空位会自然填满。少数则需要等上相当长的时间。其中有一个会发光。

将鼠标悬停在已填满的空位上，它会告诉你谁最先到访，以及到访时间。小龙虾图鉴从不会忘记初次相遇，这一点胜过我们大多数人。

## 观察笔记

以下观察来自那些花了太多时间盯着侧边栏的人：

- 小龙虾也会造访[梦境场景](/zh-CN/concepts/memory)——即智能体页面上的 Memory 标签页——它们会在那里睡觉。据观察者报告，熟睡的小龙虾看起来异常眼熟。
- 上夜班的用户报告称，晚上 10 点后，访客会频繁打哈欠。
- 有人看到过小龙虾蜕壳。目击者形容这场面“相当震撼”，并说蜕壳后的小龙虾“明显变大了”。
- 有些访客并非独自前来。
- 每年某些时节，有人观察到小龙虾会穿戴一些东西。
- 每年有一天，所有小龙虾看起来都像是从旧版徽标中走出来的。没人肯告诉你是哪一天。
- 并非每只经过侧边栏的小龙虾都是你的小龙虾。它们大多数甚至不会停留。
- 至少有一位有记录的访客并不是小龙虾。它坚称自己是。
- 长期使用同一浏览器的人报告称，他们的小龙虾来得更早、待得更久，还会向他们挥手。经常驱赶它们的人则感受到了一种微妙的疏离。
- Gateway 网关升级后，请留意第一位访客携带的东西。
- 侧边栏里的小龙虾在终端中有一个表亲。它会自行挑选日子出现在 `openclaw` 横幅中，至今没人能成功预测是哪一天。
- 在这位表亲出现的日子里，有人看到侧边栏的小龙虾穿着航海风服饰。两者都拒绝解释其中的联系。
- 在耗时很长的运行过程中，小龙虾会停止玩耍，安顿下来陪你一起等待。与你同在。
- 当有什么东西经过边沿时，常驻的小龙虾会停下手头的事，注视着它。换作你也会如此。
- 收到坏消息后，有些小龙虾会整理边沿。这似乎能让它们好受一些。
- 即使你不记得纪念日，小龙虾也会记得。如果你的小龙虾在一个平平无奇的日子戴着派对帽出现，请在小龙虾图鉴中查看你们相遇的日期。
- 尽心尽力的主人报告称，他们的小龙虾会在多年间获得各种头衔。目前没有已知的授予仪式。
- 梦境场景中熟睡的那只并没有完全睡着。轻轻按一下看看。

## 隐私

此页面上的一切都只在你的浏览器本地发生：随机生成、出现时间表以及小龙虾图鉴。任何小龙虾数据都不会离开你的设备。OpenClaw 不知道你遇到过哪些小龙虾，坦白说，它很嫉妒。
