# 插件清单

此页面根据 `extensions/*/package.json`、`openclaw.plugin.json`，
以及根 npm 包的 `files` 排除项生成。使用以下命令重新生成：

```bash
pnpm plugins:inventory:gen
```

## 定义

- **核心 npm 包：** 内置于 `openclaw` npm 包中，无需单独安装插件即可使用。
- **官方外部包：** 由 OpenClaw 维护、未包含在核心 npm 包中的插件，保留在此官方清单中，并可通过 ClawHub 和/或 npm 按需安装。
- **仅源代码检出：** 仓库本地插件，不包含在已发布的 npm 构件中，也不会作为可安装包宣传。

源代码检出不同于 npm 安装：执行 `pnpm install` 后，内置
插件会从 `extensions/<id>` 加载，因此可以使用本地编辑和包本地 workspace
依赖。

## 安装插件

使用 **分发方式** 列判断是否需要安装。标注为
`included in OpenClaw` 的插件已包含在核心包中。官方
外部包需要安装一次，然后重启 Gateway 网关。

例如，Discord 是一个官方外部包：

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

裸包规格会先尝试 ClawHub，然后回退到 npm。若要强制指定来源，请使用
`clawhub:@openclaw/discord` 或 `npm:@openclaw/discord`。安装后，请按照
该插件的设置文档，例如 [Discord](/zh-CN/channels/discord)，添加凭证
和渠道配置。有关更新、卸载和发布命令，请参阅 [管理插件](/zh-CN/plugins/manage-plugins)。
