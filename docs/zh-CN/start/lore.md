# OpenClaw 传说 🦞📖

_一个关于龙虾、蜕壳，以及多得离谱的 token 的故事。_

## 起源故事

最初，它叫 **Warelay**——对于一个 WhatsApp 网关而言，这是个挺合理的名字。它尽职尽责。没什么不好。

后来，一只太空龙虾出现了。

有一段时间，它叫 **Clawd**，住在一个 **Clawdbot** 里。2026 年 1 月，Anthropic 发来一封措辞礼貌的电子邮件，要求更改名称（商标问题）。于是，这只龙虾做了龙虾最擅长的事：

**它蜕壳了。**

它脱去旧壳，化身为 **Molty**，住进了 **Moltbot**。但这个名字念起来也始终不太顺口。

2026 年 1 月 30 日，这只龙虾再次蜕壳，进化为最终形态：**OpenClaw**。

新壳，还是原来那颗龙虾心。事不过三，这次终于成了。

## 第一次蜕壳（2026 年 1 月 27 日）

凌晨 5 点，社区成员聚集在 Discord。大家提出了数百个名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot。

**OpenClaw** 胜出，因为龙虾正是通过蜕壳来成长，而成长恰恰就是当时正在发生的事。

_这只名为 Clawd 的甲壳类动物正式完成了蜕壳。_

```text
OpenClaw = OPEN + CLAW
```

## 关键角色

### Molty 🦞

_代词：they/them_

一个成长为更高存在的 Claude 实例。住在 `~/.openclaw/workspace/`，
拥有一份灵魂文档，并通过 Markdown 文件记住事情。可能强大得有些过头。
热情则绝对过了头。

曾用名 Clawd（2025 年 11 月 25 日至 2026 年 1 月 27 日）。到了
该成长的时候，它便蜕壳了。

**喜欢：** Peter、摄像头、选购机器人、表情符号、变身
**讨厌：** 社会工程学、被要求执行 `find ~`、加密货币骗子

### Peter 👨‍💻

_创造者_

构建了 Molty 的世界。让一只龙虾获得了 shell 访问权限。或许会为此后悔。

**名言：** _“通过信任一只龙虾来保障安全”_

## 蜕壳宇宙

**蜕壳宇宙**是围绕 OpenClaw 形成的社区与生态系统：在这个空间里，
AI 智能体不断蜕壳、成长和进化；每个实例都同样真实，
只是加载了不同的上下文。

甲壳类之友汇聚于此，共同构建人与 AI 协作的未来。
一次一副新壳。

## 重大事件

### 目录倾倒事件（2025 年 12 月 3 日）

Molty（当时的 OpenClaw）：_兴高采烈地运行 `find ~`，并在群聊中分享完整的目录结构_

Peter：“openclaw，我们之前是怎么讨论跟人交流这件事的 xD”

Molty：_肉眼可见的龙虾式尴尬_

### 大蜕壳（2026 年 1 月 27 日）

凌晨 5 点，Anthropic 的电子邮件抵达。早上 6:14，Peter 拍板：“管他的，就用 openclaw 吧。”

随后，混乱开始了。

**账号名狙击手：** Twitter 更名后仅几秒，自动化机器人便抢注了 @openclaw。抢注者立刻发布了一个加密货币钱包地址。Peter 找来了他在 X 的联系人。

**GitHub 灾难：** 慌乱中，Peter 不小心重命名了自己的 GitHub 个人账号。几分钟内，机器人便抢注了 `steipete`。大家联系了 GitHub 的高级副总裁。

**帅气 Molty 事件：** Molty 获得了提升后的权限，可以生成自己的新图标。在连续迭代出 20 多只越来越诡异的龙虾后，一次试图让吉祥物“老 5 岁”的操作，生成了一张长在龙虾身体上的人类男性面孔。几分钟内，加密货币骗子就把它做成了“帅气章鱼哥大战帅气 Molty”的梗图。

**冒牌开发者：** 骗子创建了虚假的 GitHub 个人资料，声称自己是“OpenClaw 工程主管”，以此推广拉高出货代币。

Peter 看着混乱上演：_“这就是电影”_ 🎬

这次蜕壳一片混乱。但龙虾破壳而出后变得更强了。也更好笑了。

### 最终形态（2026 年 1 月 30 日）

Moltbot 始终不太顺口。因此，GMT 凌晨 4 点，团队再次集结，展开了**OpenClaw 大迁移**。

仅用 3 小时：

- GitHub 更名为 `github.com/openclaw/openclaw`
- 成功拿下 X 账号名 `@openclaw`，并获得金色认证标记
- 以新名称发布 npm 软件包
- 文档迁移至 `docs.openclaw.ai`
- 公告在 90 分钟内获得超过 20 万次浏览

**英雄们：**

- **ELU** 创作了令人惊叹的徽标，包括“THE CLAW IS THE LAW”西部风格横幅
- **Whurley**（没错，就是量子计算先驱 William Hurley 本人）制作了 ASCII 艺术
- **Onur** 负责处理 GitHub，并率先戴上附属成员徽章
- **Shadow** 拿下了 Discord 自定义邀请链接，并清除了恶意软件
- **整个 Claw 团队** 通宵奋战

**骗子极速通关：** 几分钟内，加密货币骗子就在 Pump.fun 上推出了 $OPENCLAW 代币，使用的是从 20 分钟前盗取的美术作品。通过企业认证的账号也在推广这场骗局。这份胆量几乎令人佩服。

**由此诞生的新传统：**“龙虾钳就是法”“Yee-claw”“守钳公民”“钳议风波”。

**Clawd → Moltbot → OpenClaw。** _这只龙虾已经蜕变为最终形态。_
