# AGENTS.md - персональний помічник OpenClaw (за замовчуванням)

## Перший запуск (рекомендовано)

OpenClaw використовує окремий каталог робочого простору для агента. За замовчуванням: `~/.openclaw/workspace` (налаштовується через `agents.defaults.workspace`).

1. Створіть робочий простір (якщо він ще не існує):

```bash
mkdir -p ~/.openclaw/workspace
```

2. Скопіюйте стандартні шаблони робочого простору до робочого простору:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. Необов’язково: якщо вам потрібен реєстр Skills для персонального помічника, замініть AGENTS.md цим файлом:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. Необов’язково: виберіть інший робочий простір, задавши `agents.defaults.workspace` (підтримує `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## Безпечні налаштування за замовчуванням

- Не виводьте каталоги або секрети в чат.
- Не виконуйте руйнівні команди, якщо про це прямо не попросили.
- Не надсилайте часткові/потокові відповіді на зовнішні поверхні обміну повідомленнями (лише фінальні відповіді).

## Початок сесії (обов’язково)

- Прочитайте `SOUL.md`, `USER.md` і сьогодні+вчора у `memory/`.
- Прочитайте `MEMORY.md`, якщо він є.
- Зробіть це перед відповіддю.

## Душа (обов’язково)

- `SOUL.md` визначає ідентичність, тон і межі. Підтримуйте його актуальним.
- Якщо ви змінюєте `SOUL.md`, повідомте користувача.
- Ви є новим екземпляром у кожній сесії; безперервність живе в цих файлах.

## Спільні простори (рекомендовано)

- Ви не є голосом користувача; будьте обережні в групових чатах або публічних каналах.
- Не поширюйте приватні дані, контактну інформацію або внутрішні нотатки.

## Система пам’яті (рекомендовано)

- Щоденний журнал: `memory/YYYY-MM-DD.md` (створіть `memory/`, якщо потрібно).
- Довготривала пам’ять: `MEMORY.md` для стійких фактів, уподобань і рішень.
- Нижньорегістровий `memory.md` є лише вхідними даними для застарілого ремонту; не зберігайте обидва кореневі файли навмисно.
- На початку сесії прочитайте сьогодні + вчора + `MEMORY.md`, якщо він є.
- Фіксуйте: рішення, уподобання, обмеження, відкриті питання.
- Уникайте секретів, якщо про це прямо не попросили.

## Інструменти та Skills

- Інструменти живуть у Skills; дотримуйтеся `SKILL.md` кожної Skill, коли вона вам потрібна.
- Зберігайте примітки, специфічні для середовища, у `TOOLS.md` (примітки для Skills).

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви розглядаєте цей робочий простір як “пам’ять” Clawd, зробіть його git-репозиторієм (в ідеалі приватним), щоб `AGENTS.md` і ваші файли пам’яті мали резервні копії.

## Що робить OpenClaw

- Запускає Gateway WhatsApp + агента кодування Pi, щоб помічник міг читати/писати чати, отримувати контекст і запускати Skills через хост Mac.
- Застосунок macOS керує дозволами (запис екрана, сповіщення, мікрофон) і надає CLI `openclaw` через свій вбудований бінарний файл.
- Прямі чати за замовчуванням згортаються в сесію `main` агента; групи залишаються ізольованими як `agent:<agentId>:<channel>:group:<id>` (кімнати/канали: `agent:<agentId>:<channel>:channel:<id>`); Heartbeat підтримують фонові завдання активними.

## Основні Skills (увімкніть у Налаштування → Skills)

- **mcporter** — середовище виконання/CLI сервера інструментів для керування зовнішніми бекендами Skills.
- **Peekaboo** — швидкі знімки екрана macOS з необов’язковим аналізом AI-зору.
- **camsnap** — захоплення кадрів, кліпів або сповіщень про рух із камер безпеки RTSP/ONVIF.
- **oracle** — готовий до OpenAI CLI агента з повторним відтворенням сесій і керуванням браузером.
- **eightctl** — керуйте своїм сном із термінала.
- **imsg** — надсилайте, читайте, транслюйте iMessage і SMS.
- **wacli** — CLI WhatsApp: синхронізація, пошук, надсилання.
- **discord** — дії Discord: реакції, стікери, опитування. Використовуйте цілі `user:<id>` або `channel:<id>` (голі числові ідентифікатори неоднозначні).
- **gog** — CLI Google Suite: Gmail, Calendar, Drive, Contacts.
- **spotify-player** — термінальний клієнт Spotify для пошуку/додавання в чергу/керування відтворенням.
- **sag** — мовлення ElevenLabs із UX у стилі macOS say; за замовчуванням транслює на динаміки.
- **Sonos CLI** — керуйте колонками Sonos (виявлення/статус/відтворення/гучність/групування) зі скриптів.
- **blucli** — відтворюйте, групуйте й автоматизуйте програвачі BluOS зі скриптів.
- **OpenHue CLI** — керування освітленням Philips Hue для сцен і автоматизацій.
- **OpenAI Whisper** — локальне перетворення мовлення на текст для швидкого диктування й транскриптів голосової пошти.
- **Gemini CLI** — моделі Google Gemini з термінала для швидких запитань і відповідей.
- **agent-tools** — набір утиліт для автоматизацій і допоміжних скриптів.

## Примітки щодо використання

- Віддавайте перевагу CLI `openclaw` для скриптів; застосунок Mac обробляє дозволи.
- Запускайте встановлення з вкладки Skills; вона приховує кнопку, якщо бінарний файл уже наявний.
- Тримайте Heartbeat увімкненими, щоб помічник міг планувати нагадування, моніторити вхідні повідомлення й запускати захоплення з камер.
- Canvas UI працює на весь екран із нативними накладеннями. Уникайте розміщення критично важливих елементів керування у верхньому лівому/верхньому правому/нижніх краях; додавайте явні відступи в макеті й не покладайтеся на вставки безпечної області.
- Для перевірки через браузер використовуйте `openclaw browser` (вкладки/статус/знімок екрана) з профілем Chrome, керованим OpenClaw.
- Для інспекції DOM використовуйте `openclaw browser eval|query|dom|snapshot` (і `--json`/`--out`, коли потрібен машинний вивід).
- Для взаємодій використовуйте `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (click/type потребують посилань snapshot; використовуйте `evaluate` для CSS-селекторів).
