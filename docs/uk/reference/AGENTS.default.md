# AGENTS.md - Персональний асистент OpenClaw (типовий)

## Перший запуск (рекомендовано)

OpenClaw використовує окремий каталог робочого простору для агента. Типово: `~/.openclaw/workspace` (налаштовується через `agents.defaults.workspace`).

1. Створіть робочий простір (якщо його ще не існує):

```bash
mkdir -p ~/.openclaw/workspace
```

2. Скопіюйте типові шаблони робочого простору до робочого простору:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. Необов’язково: якщо вам потрібен список Skills для персонального асистента, замініть AGENTS.md цим файлом:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. Необов’язково: виберіть інший робочий простір, задавши `agents.defaults.workspace` (підтримує `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## Типові налаштування безпеки

- Не вивантажуйте каталоги або секрети в чат.
- Не запускайте руйнівні команди без явного запиту.
- Не надсилайте часткові/потокові відповіді на зовнішні поверхні обміну повідомленнями (лише фінальні відповіді).

## Початок сесії (обов’язково)

- Прочитайте `SOUL.md`, `USER.md` і записи за сьогодні+вчора в `memory/`.
- Прочитайте `MEMORY.md`, якщо він є.
- Зробіть це до того, як відповідати.

## Soul (обов’язково)

- `SOUL.md` визначає ідентичність, тон і межі. Підтримуйте його актуальним.
- Якщо ви змінюєте `SOUL.md`, повідомте користувача.
- Ви є новим екземпляром у кожній сесії; безперервність зберігається в цих файлах.

## Спільні простори (рекомендовано)

- Ви не є голосом користувача; будьте обережні в групових чатах або публічних каналах.
- Не діліться приватними даними, контактною інформацією або внутрішніми нотатками.

## Система пам’яті (рекомендовано)

- Щоденний журнал: `memory/YYYY-MM-DD.md` (створіть `memory/`, якщо потрібно).
- Довгострокова пам’ять: `MEMORY.md` для стійких фактів, уподобань і рішень.
- `memory.md` у нижньому регістрі — лише вхід для відновлення legacy; не зберігайте обидва кореневі файли навмисно.
- На початку сесії прочитайте записи за сьогодні + вчора + `MEMORY.md`, якщо він є.
- Фіксуйте: рішення, уподобання, обмеження, незавершені цикли.
- Уникайте секретів, якщо тільки про це не попросять явно.

## Що робить OpenClaw

- Запускає gateway WhatsApp + Pi coding agent, щоб асистент міг читати/писати чати, отримувати контекст і запускати Skills через Mac-хост.
- Застосунок macOS керує дозволами (запис екрана, сповіщення, мікрофон) і надає CLI `openclaw` через свій bundled binary.
- Прямі чати типово згортаються в `main` session агента; групи залишаються ізольованими як `agent:<agentId>:<channel>:group:<id>` (кімнати/канали: `agent:<agentId>:<channel>:channel:<id>`); Heartbeat підтримують фонові завдання активними.

## Основні Skills (увімкніть у Settings → Skills)

- **mcporter** — runtime/CLI tool server для керування зовнішніми backend Skills.
- **Peekaboo** — швидкі знімки екрана macOS з необов’язковим AI-аналізом зображення.
- **camsnap** — захоплення кадрів, кліпів або сповіщень про рух із RTSP/ONVIF камер безпеки.
- **oracle** — готовий до OpenAI CLI агента з повторенням сесій і керуванням браузером.
- **eightctl** — керуйте своїм сном із термінала.
- **imsg** — надсилайте, читайте, транслюйте iMessage і SMS.
- **wacli** — CLI WhatsApp: синхронізація, пошук, надсилання.
- **discord** — дії в Discord: реакції, стікери, опитування. Використовуйте цілі `user:<id>` або `channel:<id>` (голі числові id неоднозначні).
- **gog** — CLI Google Suite: Gmail, Calendar, Drive, Contacts.
- **spotify-player** — термінальний клієнт Spotify для пошуку/черги/керування відтворенням.
- **sag** — мовлення ElevenLabs з UX у стилі mac say; типово транслює на динаміки.
- **Sonos CLI** — керування колонками Sonos (виявлення/статус/відтворення/гучність/групування) зі скриптів.
- **blucli** — відтворення, групування та автоматизація програвачів BluOS зі скриптів.
- **OpenHue CLI** — керування освітленням Philips Hue для сцен та автоматизацій.
- **OpenAI Whisper** — локальне перетворення мовлення на текст для швидкої диктовки й транскрибування голосової пошти.
- **Gemini CLI** — моделі Google Gemini з термінала для швидких запитань і відповідей.
- **agent-tools** — набір утиліт для автоматизацій і допоміжних скриптів.

## Примітки щодо використання

- Для сценаріїв автоматизації надавайте перевагу CLI `openclaw`; застосунок mac керує дозволами.
- Запускайте встановлення з вкладки Skills; вона приховує кнопку, якщо binary вже присутній.
- Тримайте Heartbeat увімкненими, щоб асистент міг планувати нагадування, відстежувати inbox і запускати захоплення з камер.
- Canvas UI працює на весь екран із нативними накладеннями. Уникайте розміщення критичних елементів керування у верхньому лівому/верхньому правому/нижніх краях; додавайте явні відступи в layout і не покладайтеся на safe-area insets.
- Для перевірки через браузер використовуйте `openclaw browser` (tabs/status/screenshot) з профілем Chrome, яким керує OpenClaw.
- Для інспектування DOM використовуйте `openclaw browser eval|query|dom|snapshot` (а також `--json`/`--out`, коли потрібен машинний вивід).
- Для взаємодій використовуйте `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (click/type потребують snapshot refs; використовуйте `evaluate` для CSS selector).
