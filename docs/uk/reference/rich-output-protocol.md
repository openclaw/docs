Вивід асистента може містити невеликий набір директив доставки/рендерингу:

- `MEDIA:` для доставки вкладень
- `[[audio_as_voice]]` для підказок представлення аудіо
- `[[reply_to_current]]` / `[[reply_to:<id>]]` для метаданих відповіді
- `[embed ...]` для розширеного рендерингу в інтерфейсі керування

Ці директиви є окремими. `MEDIA:` і теги відповіді/голосу залишаються метаданими доставки; `[embed ...]` — це вебшлях розширеного рендерингу.

Довірені медіадані з результатів інструментів використовують той самий парсер `MEDIA:` / `[[audio_as_voice]]` перед доставкою, тож застарілі виводи інструментів, як і раніше, можуть позначати аудіовкладення як голосове повідомлення.

Коли ввімкнено потокову передачу блоками, `MEDIA:` залишається метаданими одноразової доставки для ходу. Якщо ту саму URL-адресу медіа надіслано в потоковому блоці й повторено у фінальному корисному навантаженні асистента, OpenClaw доставляє вкладення один раз і видаляє дублікат із фінального корисного навантаження.

## `[embed ...]`

`[embed ...]` — це єдиний агент-орієнтований синтаксис розширеного рендерингу для інтерфейсу керування.

Приклад самозакривного запису:

```text
[embed ref="cv_123" title="Статус" /]
```

Правила:

- `[view ...]` більше не є чинним для нового виводу.
- Шорткоди embed рендеряться лише на поверхні повідомлень асистента.
- Рендеряться лише embed-елементи з URL-адресою. Використовуйте `ref="..."` або `url="..."`.
- Шорткоди embed у формі блоку з вбудованим HTML не рендеряться.
- Вебінтерфейс видаляє шорткод із видимого тексту та рендерить embed вбудовано.
- `MEDIA:` не є псевдонімом embed і не повинен використовуватися для розширеного рендерингу embed.

## Збережена форма рендерингу

Нормалізований/збережений блок вмісту асистента — це структурований елемент `canvas`:

```json
{
  "type": "canvas",
  "preview": {
    "kind": "canvas",
    "surface": "assistant_message",
    "render": "url",
    "viewId": "cv_123",
    "url": "/__openclaw__/canvas/documents/cv_123/index.html",
    "title": "Status",
    "preferredHeight": 320
  }
}
```

Збережені/відрендерені розширені блоки безпосередньо використовують цю форму `canvas`. `present_view` не розпізнається.
