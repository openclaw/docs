# TOOLS.md — Локальні нотатки

Skills визначають, _як_ працюють інструменти. Цей файл призначений для _ваших_ особливостей — усього, що є унікальним для вашого налаштування: назв і розташування камер, хостів і псевдонімів SSH, бажаних голосів TTS, назв динаміків і кімнат, псевдонімів пристроїв та всього, що залежить від середовища.

## Приклади

```markdown
### Камери

- living-room → Головна зона, ширококутний огляд 180°
- front-door → Вхід, активація за рухом

### SSH

- home-server → 192.168.1.100, користувач: admin

### TTS

- Бажаний голос: "Nova" (теплий, із легким британським акцентом)
- Типовий динамік: Kitchen HomePod
```

## Навіщо розділяти?

Skills є спільними. Ваше налаштування належить вам. Розділення дає змогу оновлювати Skills без втрати нотаток і ділитися Skills, не розкриваючи свою інфраструктуру.
