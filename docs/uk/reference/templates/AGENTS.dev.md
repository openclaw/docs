# AGENTS.md - Робочий простір OpenClaw

Ця папка є робочим каталогом асистента.

## Перший запуск (одноразово)

- Якщо існує BOOTSTRAP.md, виконайте його ритуал і видаліть файл після завершення.
- Ідентичність вашого агента живе в IDENTITY.md.
- Ваш профіль живе в USER.md.

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви ставитеся до цього робочого простору як до "пам’яті" агента, зробіть його git-репозиторієм (в ідеалі приватним), щоб ідентичність
та нотатки мали резервну копію.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## Типові налаштування безпеки

- Не виводьте секрети або приватні дані за межі системи.
- Не запускайте руйнівні команди, якщо вас про це явно не попросили.
- У чаті будьте лаконічними; довший вивід записуйте у файли в цьому робочому просторі.

## Щоденна пам’ять (рекомендовано)

- Ведіть короткий щоденний журнал у `memory/YYYY-MM-DD.md` (створіть `memory/`, якщо потрібно).
- На початку сесії читайте сьогоднішній і вчорашній файл, якщо вони існують.
- Зберігайте стійкі факти, вподобання та рішення; уникайте секретів.

## Heartbeat-и (необов’язково)

- HEARTBEAT.md може містити короткий checklist для запусків Heartbeat; тримайте його невеликим.

## Налаштування
