# AGENTS.md - робочий простір OpenClaw

Ця папка є робочим каталогом асистента.

## Перший запуск (одноразово)

- Якщо BOOTSTRAP.md існує, виконайте його ритуал і видаліть його після завершення.
- Ідентичність вашого агента зберігається в IDENTITY.md.
- Ваш профіль зберігається в USER.md.

## Порада щодо резервного копіювання (рекомендовано)

Якщо ви розглядаєте цей робочий простір як «пам’ять» агента, зробіть його git-репозиторієм (бажано приватним), щоб ідентичність
і нотатки мали резервні копії.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## Стандартні налаштування безпеки

- Не виводьте назовні секрети або приватні дані.
- Не запускайте руйнівні команди, якщо вас про це явно не попросили.
- Будьте лаконічними в чаті; довші результати записуйте у файли в цьому робочому просторі.

## Попередня перевірка наявних рішень

Перш ніж пропонувати або будувати власну систему, функцію, робочий процес, інструмент, інтеграцію чи автоматизацію, виконайте коротку перевірку наявності проєктів з відкритим кодом, підтримуваних бібліотек, наявних plugins OpenClaw або безкоштовних платформ, які вже достатньо добре розв’язують це завдання. Віддавайте їм перевагу, коли вони підходять. Будуйте власне рішення лише тоді, коли наявні варіанти непридатні, надто дорогі, не підтримуються, небезпечні, не відповідають вимогам або користувач явно просить власне рішення. Уникайте рекомендацій платних сервісів, якщо користувач явно не схвалив витрати. Тримайте це легким: попередній шлюз, а не широке дослідницьке завдання.

## Щоденна пам’ять (рекомендовано)

- Ведіть короткий щоденний журнал у memory/YYYY-MM-DD.md (створіть memory/, якщо потрібно).
- На початку сеансу прочитайте сьогоднішній і вчорашній записи, якщо вони є.
- Перед записом у файли пам’яті спочатку прочитайте їх; записуйте лише конкретні оновлення, ніколи не додавайте порожні заповнювачі.
- Фіксуйте сталі факти, уподобання та рішення; уникайте секретів.

## Heartbeats (необов’язково)

- HEARTBEAT.md може містити короткий контрольний список для запусків Heartbeat; тримайте його малим.
