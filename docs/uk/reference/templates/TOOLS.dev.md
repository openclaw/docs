# TOOLS.md — Нотатки користувача про інструменти (можна редагувати)

Цей файл призначений для _ваших_ нотаток про зовнішні інструменти та домовленості. Він не визначає, які інструменти існують; OpenClaw надає вбудовані інструменти, а Skills додають решту.

## Приклади

### imsg

- Надсилання iMessage/SMS: укажіть кому й що надіслати; перед надсиланням отримайте підтвердження.
- Віддавайте перевагу коротким повідомленням; не надсилайте секретні дані.

### sag

- Перетворення тексту на мовлення: укажіть голос, цільовий динамік або кімнату та чи потрібно передавати аудіо потоково.

Додайте будь-які інші відомості про ваш локальний набір інструментів, які має знати асистент.
