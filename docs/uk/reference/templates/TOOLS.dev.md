# TOOLS.md — Нотатки користувача про інструменти (можна редагувати)

Цей файл призначений для _ваших_ нотаток про зовнішні інструменти й домовленості.
Він не визначає, які інструменти існують; OpenClaw надає вбудовані інструменти внутрішньо.

## Приклади

### imsg

- Надіслати iMessage/SMS: опишіть кому/що, підтвердьте перед надсиланням.
- Надавайте перевагу коротким повідомленням; уникайте надсилання секретів.

### sag

- Text-to-speech: вкажіть голос, цільового мовця/кімнату і чи потрібно виконувати streaming.

Додайте все інше, що ви хочете, щоб помічник знав про ваш локальний toolchain.
