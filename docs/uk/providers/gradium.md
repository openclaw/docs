[Gradium](https://gradium.ai) — це постачальник перетворення тексту на мовлення для OpenClaw. Він створює стандартні аудіовідповіді (WAV), сумісний із голосовими повідомленнями вихідний аудіопотік Opus і 8 кГц аудіо у форматі u-law для телефонних інтерфейсів.

| Властивість          | Значення                                      |
| -------------------- | --------------------------------------------- |
| Ідентифікатор постачальника | `gradium`                     |
| Автентифікація       | `GRADIUM_API_KEY` або конфігурація `apiKey` |
| Базова URL-адреса    | `https://api.gradium.ai` (типово)                   |
| Типовий голос        | `Emma` (`YTpq7expH9539ERJ`)       |

## Установлення плагіна

Gradium — офіційний зовнішній плагін. Установіть його, а потім перезапустіть Gateway:

```bash
openclaw plugins install @openclaw/gradium-speech
openclaw gateway restart
```

## Налаштування

Створіть ключ API Gradium, а потім надайте його через змінну середовища або ключ конфігурації. Конфігурація має пріоритет над змінною середовища.

<Tabs>
  <Tab title="Змінна середовища">
    ```bash
    export GRADIUM_API_KEY="gsk_..."
    ```
  </Tab>

  <Tab title="Ключ конфігурації">
    ```json5
    {
      messages: {
        tts: {
          auto: "always",
          provider: "gradium",
          providers: {
            gradium: {
              apiKey: "${GRADIUM_API_KEY}",
            },
          },
        },
      },
    }
    ```
  </Tab>
</Tabs>

## Конфігурація

```json5
{
  messages: {
    tts: {
      auto: "always",
      provider: "gradium",
      providers: {
        gradium: {
          speakerVoiceId: "YTpq7expH9539ERJ",
          // apiKey: "${GRADIUM_API_KEY}",
          // baseUrl: "https://api.gradium.ai",
        },
      },
    },
  },
}
```

| Ключ                                            | Тип    | Опис                                                                                                            |
| ----------------------------------------------- | ------ | --------------------------------------------------------------------------------------------------------------- |
| `messages.tts.providers.gradium.apiKey`                              | рядок  | Визначений ключ API. Підтримує `${ENV}` і посилання на секрети.                                      |
| `messages.tts.providers.gradium.baseUrl`                              | рядок  | HTTPS URL-адреса API Gradium на `api.gradium.ai`. Кінцеві скісні риски видаляються. Типове значення: `https://api.gradium.ai`. |
| `messages.tts.providers.gradium.speakerVoiceId`                              | рядок  | Ідентифікатор типового голосу, який використовується за відсутності перевизначення директивою.                  |

Формат виведення вибирається автоматично залежно від цільового інтерфейсу (див. [Виведення](#output)) і не налаштовується в `openclaw.json`.

### Перевизначення голосу для окремого повідомлення

Коли активна політика мовлення дозволяє перевизначення голосу, перемикайте голоси безпосередньо в тексті за допомогою токена директиви (усі наведені варіанти рівнозначні й приймають ідентифікатор голосу у власному форматі постачальника):

```text
/voice:LFZvm12tW_z0xfGo
/voice_id:LFZvm12tW_z0xfGo
/voiceid:LFZvm12tW_z0xfGo
/gradium_voice:LFZvm12tW_z0xfGo
/gradiumvoice:LFZvm12tW_z0xfGo
```

Якщо політика мовлення забороняє перевизначення голосу, директива обробляється, але ігнорується.

## Виведення

Формат виведення вибирається залежно від цільового інтерфейсу; постачальник не синтезує інші формати.

| Цільовий інтерфейс | Формат      | Розширення файла | Частота дискретизації | Ознака сумісності з голосовими повідомленнями |
| ------------------ | ----------- | ---------------- | --------------------- | --------------------------------------------- |
| Стандартне аудіо   | `wav` | `.wav` | постачальник       | ні                                            |
| Голосове повідомлення | `opus` | `.opus` | постачальник    | так                                           |
| Телефонія          | `ulaw_8000` | немає           | 8 кГц                 | немає                                         |

## Порядок автоматичного вибору

Серед налаштованих постачальників TTS порядок автоматичного вибору Gradium — `30`. Докладніше про те, як OpenClaw вибирає активного постачальника, коли `messages.tts.provider` не зафіксовано, див. у розділі [Перетворення тексту на мовлення](/uk/tools/tts).
