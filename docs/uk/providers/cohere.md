[Cohere](https://cohere.com) надає сумісний з OpenAI інференс через свій Compatibility API. OpenClaw включає провайдер Cohere під час переходу до його винесення в зовнішній пакет, а також публікує його як офіційний зовнішній Plugin.

| Властивість                  | Значення                                                     |
| ---------------------------- | ------------------------------------------------------------ |
| Ідентифікатор провайдера     | `cohere`                                                     |
| Plugin                       | вбудований на час переходу; офіційний зовнішній пакет        |
| Змінна середовища авторизації | `COHERE_API_KEY`                                             |
| Прапорець початкового налаштування | `--auth-choice cohere-api-key`                          |
| Прямий прапорець CLI         | `--cohere-api-key <key>`                                     |
| API                          | сумісний з OpenAI (`openai-completions`)                     |
| Базова URL-адреса            | `https://api.cohere.ai/compatibility/v1`                     |
| Модель за замовчуванням      | `cohere/command-a-plus-05-2026`                              |
| Контекстне вікно             | 128 000 токенів                                              |

## Вбудований каталог

| Посилання на модель                   | Вхідні дані       | Контекст | Максимальний вихід | Примітки                                                        |
| ------------------------------------- | ----------------- | -------- | ------------------ | --------------------------------------------------------------- |
| `cohere/command-a-plus-05-2026`       | текст, зображення | 128 000  | 64 000             | Модель за замовчуванням; флагманська агентна модель для міркувань |
| `cohere/command-a-03-2025`            | текст             | 256 000  | 8 000              | Попередня модель Command A                                      |
| `cohere/command-a-reasoning-08-2025`  | текст             | 256 000  | 32 000             | Агентні міркування та використання інструментів                 |
| `cohere/command-a-vision-07-2025`     | текст, зображення | 128 000  | 8 000              | Аналіз зображень і документів; без використання інструментів    |
| `cohere/north-mini-code-1-0`          | текст, зображення | 256 000  | 64 000             | Агентне програмування; міркування; безкоштовні ліміти           |

Моделі Cohere із підтримкою міркувань підтримують два режими міркувань Compatibility API. OpenClaw зіставляє **вимкнено** з `none`, а кожен увімкнений рівень мислення — з `high`. Command A Vision не підтримує використання інструментів, тому OpenClaw залишає агентні інструменти вимкненими для цієї моделі.

## Початок роботи

1. Cohere постачається з поточними пакетами OpenClaw. Якщо його немає, установіть зовнішній пакет і перезапустіть Gateway:

```bash
openclaw plugins install @openclaw/cohere-provider
openclaw gateway restart
```

2. Створіть ключ API Cohere.
3. Запустіть початкове налаштування:

```bash
openclaw onboard --non-interactive \
  --auth-choice cohere-api-key \
  --cohere-api-key "$COHERE_API_KEY"
```

4. Переконайтеся, що каталог доступний:

```bash
openclaw models list --provider cohere
```

Початкове налаштування призначає Cohere основною моделлю, лише якщо основну модель ще не налаштовано.

## Налаштування лише через середовище

Зробіть `COHERE_API_KEY` доступною для процесу Gateway, а потім виберіть модель Cohere:

```json5
{
  agents: {
    defaults: {
      model: { primary: "cohere/command-a-plus-05-2026" },
    },
  },
}
```

<Note>
Якщо Gateway працює як демон або в Docker, задайте `COHERE_API_KEY` для цієї служби. Експорт лише в інтерактивній оболонці не робить її доступною для вже запущеного Gateway.
</Note>
