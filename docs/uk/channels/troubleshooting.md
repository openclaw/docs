Використовуйте цю сторінку, коли канал підключається, але поведінка неправильна.

## Драбина команд

Спочатку виконайте їх по черзі:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

Справна базова лінія:

- `Runtime: running`
- `Connectivity probe: ok`
- `Capability: read-only`, `write-capable`, або `admin-capable`
- Перевірка каналу показує, що транспорт підключено і, де підтримується, `works` або `audit ok`

## WhatsApp

### Ознаки збоїв WhatsApp

| Симптом                             | Найшвидша перевірка                                       | Виправлення                                                                                                                              |
| ----------------------------------- | --------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| Підключено, але немає відповідей в особистих повідомленнях         | `openclaw pairing list whatsapp`                    | Схваліть відправника або змініть політику/список дозволених для особистих повідомлень.                                                                                    |
| Групові повідомлення ігноруються              | Перевірте `requireMention` + шаблони згадок у конфігурації | Згадайте бота або послабте політику згадок для цієї групи.                                                                          |
| Вхід через QR завершується тайм-аутом із 408         | Перевірте змінні середовища Gateway `HTTPS_PROXY` / `HTTP_PROXY`      | Налаштуйте доступний проксі; використовуйте `NO_PROXY` лише для обходів.                                                                         |
| Випадкові цикли відключення/повторного входу     | `openclaw channels status --probe` + журнали           | Нещодавні повторні підключення позначаються навіть тоді, коли зараз підключення активне; перегляньте журнали, перезапустіть Gateway, потім повторно прив’яжіть, якщо нестабільність триває. |
| Відповіді надходять із затримкою в секунди/хвилини | `openclaw doctor --fix`                             | Doctor зупиняє перевірені застарілі локальні клієнти TUI, коли вони погіршують роботу циклу подій Gateway.                                    |

Повне усунення несправностей: [Усунення несправностей WhatsApp](/uk/channels/whatsapp#troubleshooting)

## Telegram

### Ознаки збоїв Telegram

| Симптом                              | Найшвидша перевірка                                    | Виправлення                                                                                                                        |
| ------------------------------------ | ------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------- |
| `/start`, але немає придатного потоку відповідей    | `openclaw pairing list telegram`                 | Схваліть прив’язування або змініть політику особистих повідомлень.                                                                                       |
| Бот онлайн, але група мовчить    | Перевірте вимогу згадки та режим приватності бота  | Вимкніть режим приватності для видимості в групі або згадайте бота.                                                                  |
| Помилки надсилання через мережеві помилки    | Перегляньте журнали щодо помилок викликів Telegram API      | Виправте маршрутизацію DNS/IPv6/проксі до `api.telegram.org`.                                                                          |
| Під час запуску повідомляється `getMe returned 401` | Перевірте налаштоване джерело токена                    | Скопіюйте заново або згенеруйте повторно токен BotFather і оновіть `botToken`, `tokenFile` або `TELEGRAM_BOT_TOKEN` облікового запису за замовчуванням.     |
| Опитування зависає або повторно підключається повільно  | `openclaw logs --follow` для діагностики опитування | Оновіть; якщо перезапуски є хибними спрацюваннями, налаштуйте `pollingStallThresholdMs`. Постійні зависання все ще вказують на проксі/DNS/IPv6. |
| `setMyCommands` відхилено під час запуску  | Перегляньте журнали щодо `BOT_COMMANDS_TOO_MUCH`         | Зменште кількість команд Telegram для plugins/skills/користувацьких команд або вимкніть нативні меню.                                                      |
| Після оновлення список дозволених блокує вас    | `openclaw security audit` і списки дозволених у конфігурації  | Запустіть `openclaw doctor --fix` або замініть `@username` числовими ID відправників.                                                |

Повне усунення несправностей: [Усунення несправностей Telegram](/uk/channels/telegram#troubleshooting)

## Matrix

### Ознаки збоїв Matrix

| Симптом                             | Найшвидша перевірка                          | Виправлення                                                                       |
| ----------------------------------- | -------------------------------------- | ------------------------------------------------------------------------- |
| Увійшли в систему, але повідомлення кімнати ігноруються | `openclaw channels status --probe`     | Перевірте `groupPolicy`, список дозволених кімнат і вимогу згадки.                  |
| Особисті повідомлення не обробляються                  | `openclaw pairing list matrix`         | Схваліть відправника або змініть політику особистих повідомлень.                                       |
| Зашифровані кімнати не працюють                | `openclaw matrix verify status`        | Повторно перевірте пристрій, потім перевірте `openclaw matrix verify backup status`.  |
| Відновлення резервної копії очікує або зламане    | `openclaw matrix verify backup status` | Запустіть `openclaw matrix verify backup restore` або повторіть із ключем відновлення. |
| Крос-підписування/початкове налаштування виглядає неправильно | `openclaw matrix verify bootstrap`     | Виправте сховище секретів, крос-підписування та стан резервної копії за один прохід.       |
