# Автентифікація

ClawHub використовує GitHub для входу у вебверсії. CLI використовує API-токени ClawHub, створені
через цей обліковий запис після входу.

## Вхід у вебверсії

Скористайтеся GitHub, щоб увійти на [clawhub.ai](https://clawhub.ai).

Видалені, заблоковані або вимкнені облікові записи не можуть завершити звичайний вхід у ClawHub.
Якщо після входу ви повертаєтеся до стану без авторизації, ваш обліковий запис може не мати належного
статусу. Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.

## Вхід через CLI

Стандартний потік входу через CLI відкриває ваш браузер:

```bash
clawhub login
clawhub whoami
```

Що відбувається:

1. CLI запускає тимчасовий callback-сервер на `127.0.0.1`.
2. Ваш браузер відкриває сторінку входу ClawHub.
3. Після входу через GitHub ClawHub створює API-токен.
4. Браузер перенаправляє назад до локального callback.
5. CLI зберігає токен у вашому конфігураційному файлі ClawHub.

Якщо ваш браузер не може дістатися локального callback через правила firewall, VPN або
proxy, використайте потік токена без графічного інтерфейсу.

## Вхід без графічного інтерфейсу

Створіть токен у вебінтерфейсі ClawHub, а потім передайте його до CLI:

```bash
clawhub login --token clh_...
```

Використовуйте цей потік для серверів, CI-завдань або середовищ лише з терміналом.

Для віддалених shell, де ви можете відкрити браузер в іншому місці, виконайте:

```bash
clawhub login --device
```

CLI виводить одноразовий код і чекає, поки ви авторизуєте його на
`https://clawhub.ai/cli/device`.

## Зберігання токенів

Стандартні шляхи конфігурації:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` або `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

Перевизначте шлях за допомогою:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

Виведіть збережений токен для налаштування CI за допомогою:

```bash
clawhub token
```

## Відкликання

Ви можете відкликати API-токени у вебінтерфейсі ClawHub.

Відкликані, недійсні або відсутні токени повертають `401 Unauthorized`. Увійдіть знову
за допомогою `clawhub login` або надайте новий токен через `clawhub login --token`.

Видалені, заблоковані або вимкнені облікові записи не можуть продовжувати використовувати наявні API-токени.
Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.
