# Автентифікація

ClawHub використовує GitHub для входу через веб. CLI використовує API-токени ClawHub, створені
через цей обліковий запис після входу.

## Вхід через веб

Використовуйте GitHub, щоб увійти на [clawhub.ai](https://clawhub.ai).

Видалені, заблоковані або вимкнені облікові записи не можуть виконати звичайний вхід у ClawHub.
Якщо після входу вас повертає у стан без входу, ваш обліковий запис може бути не в належному
стані. Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.

## Вхід у CLI

Типовий процес входу в CLI відкриває ваш браузер:

```bash
clawhub login
clawhub whoami
```

Що відбувається:

1. CLI запускає тимчасовий сервер зворотного виклику на `127.0.0.1`.
2. Ваш браузер відкриває сторінку входу ClawHub.
3. Після входу через GitHub ClawHub створює API-токен.
4. Браузер переспрямовує назад до локального зворотного виклику.
5. CLI зберігає токен у вашому конфігураційному файлі ClawHub.

Якщо ваш браузер не може дістатися локального зворотного виклику через правила фаєрвола, VPN або
проксі, використовуйте безголовий процес із токеном.

## Безголовий вхід

Створіть токен у веб-інтерфейсі ClawHub, а потім передайте його в CLI:

```bash
clawhub login --token clh_...
```

Використовуйте цей процес для серверів, завдань CI або середовищ лише з терміналом.

Для віддалених оболонок, де ви можете відкрити браузер в іншому місці, виконайте:

```bash
clawhub login --device
```

CLI виводить одноразовий код і очікує, поки ви авторизуєте його на
`https://clawhub.ai/cli/device`.

## Зберігання токена

Типові шляхи конфігурації:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` або `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

Перевизначте шлях за допомогою:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

Виведіть збережений токен для налаштування CI за допомогою:

```bash
clawhub token
```

## Відкликання

Ви можете відкликати API-токени у веб-інтерфейсі ClawHub.

Відкликані, недійсні або відсутні токени повертають `401 Unauthorized`. Увійдіть знову
за допомогою `clawhub login` або надайте новий токен через `clawhub login --token`.

Видалені, заблоковані або вимкнені облікові записи не можуть продовжувати використовувати наявні API-токени.
Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.
