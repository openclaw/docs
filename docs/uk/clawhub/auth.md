# Автентифікація

ClawHub використовує GitHub для входу у вебверсії. CLI використовує токени API ClawHub, створені
через цей обліковий запис, у який виконано вхід.

## Вхід у вебверсії

Використовуйте GitHub, щоб увійти на [clawhub.ai](https://clawhub.ai).

Видалені, заблоковані або вимкнені облікові записи не можуть завершити звичайний вхід у ClawHub.
Якщо після входу ви знову опиняєтеся в стані без входу в обліковий запис, ваш обліковий запис може бути не в належному
стані. Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.

## Вхід через CLI

Стандартний потік входу через CLI відкриває ваш браузер:

```bash
clawhub login
clawhub whoami
```

Що відбувається:

1. CLI запускає тимчасовий сервер зворотного виклику на `127.0.0.1`.
2. Ваш браузер відкриває сторінку входу ClawHub.
3. Після входу через GitHub ClawHub створює токен API.
4. Браузер перенаправляє назад до локального зворотного виклику.
5. CLI зберігає токен у вашому конфігураційному файлі ClawHub.

Якщо ваш браузер не може дістатися локального зворотного виклику через правила брандмауера, VPN або
проксі, використовуйте потік токена без браузера.

## Вхід без браузера

Створіть токен у вебінтерфейсі ClawHub, а потім передайте його CLI:

```bash
clawhub login --token clh_...
```

Використовуйте цей потік для серверів, завдань CI або середовищ лише з терміналом.

Для віддалених оболонок, де ви можете відкрити браузер в іншому місці, виконайте:

```bash
clawhub login --device
```

CLI виведе одноразовий код і чекатиме, поки ви авторизуєте його на
`https://clawhub.ai/cli/device`.

## Зберігання токенів

Стандартні шляхи конфігурації:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` або `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

Перевизначте шлях за допомогою:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

Виведіть збережений токен для налаштування CI за допомогою:

```bash
clawhub token
```

## Відкликання

Ви можете відкликати токени API у вебінтерфейсі ClawHub.

Відкликані, недійсні або відсутні токени повертають `401 Unauthorized`. Увійдіть знову
за допомогою `clawhub login` або надайте новий токен за допомогою `clawhub login --token`.

Видалені, заблоковані або вимкнені облікові записи не можуть продовжувати використовувати наявні токени API.
Якщо ваш обліковий запис було заблоковано або вимкнено, скористайтеся
[формою апеляції ClawHub](https://appeals.openclaw.ai/), якщо вважаєте, що це
помилка.
