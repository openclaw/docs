# Автентифікація

ClawHub використовує GitHub для входу у вебі. CLI використовує API-токени ClawHub, створені
через цей обліковий запис після входу.

## Вхід у вебі

Використовуйте GitHub, щоб увійти на [clawhub.ai](https://clawhub.ai).

Видалені, заблоковані або вимкнені облікові записи не можуть завершити звичайний вхід у ClawHub.
Якщо після входу ви повертаєтеся до стану без входу, ваш обліковий запис може бути не в належному
стані.

## Вхід через CLI

Стандартний потік входу через CLI відкриває ваш браузер:

```bash
clawhub login
clawhub whoami
```

Що відбувається:

1. CLI запускає тимчасовий сервер зворотного виклику на `127.0.0.1`.
2. Ваш браузер відкриває сторінку входу ClawHub.
3. Після входу через GitHub ClawHub створює API-токен.
4. Браузер перенаправляє назад до локального зворотного виклику.
5. CLI зберігає токен у вашому файлі конфігурації ClawHub.

Якщо ваш браузер не може дістатися локального зворотного виклику через правила firewall, VPN або
proxy, використовуйте потік входу без браузера за токеном.

## Вхід без браузера

Створіть токен у вебінтерфейсі ClawHub, а потім передайте його CLI:

```bash
clawhub login --token clh_...
```

Використовуйте цей потік для серверів, завдань CI або середовищ лише з терміналом.

Для віддалених shell, де можна відкрити браузер в іншому місці, виконайте:

```bash
clawhub login --device
```

CLI виводить одноразовий код і очікує, поки ви авторизуєте його на
`https://clawhub.ai/cli/device`.

## Зберігання токенів

Стандартні шляхи конфігурації:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` або `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

Перевизначте шлях за допомогою:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

## Відкликання

Ви можете відкликати API-токени у вебінтерфейсі ClawHub.

Відкликані, недійсні або відсутні токени повертають `401 Unauthorized`. Увійдіть знову
за допомогою `clawhub login` або надайте новий токен через `clawhub login --token`.

Видалені, заблоковані або вимкнені облікові записи не можуть і далі використовувати наявні API-токени.
