# Формат душі

## На диску

Душа — це один файл:

- `SOUL.md` (або `soul.md`)

Наразі onlycrabs.ai відхиляє будь-які додаткові файли.

## `SOUL.md`

- Markdown з необов’язковим YAML frontmatter.
- Сервер витягує метадані з frontmatter під час публікації.
- `description` використовується як короткий опис душі в UI/пошуку.

## Обмеження

- Загальний розмір пакета: 50 МБ.
- Текст для вбудовування містить лише `SOUL.md`.

## Слаги

- За замовчуванням утворюються з назви папки.
- Мають бути в нижньому регістрі та безпечними для URL: `^[a-z0-9][a-z0-9-]*$`.

## Версіонування + теги

- Кожна публікація створює нову версію (semver).
- Теги — це рядкові вказівники на версію; зазвичай використовується `latest`.
