# Формат душі

## На диску

Душа — це один файл:

- `SOUL.md` (або `soul.md`)

Наразі onlycrabs.ai відхиляє будь-які додаткові файли.

## `SOUL.md`

- Markdown з необов’язковим YAML-фронтматером.
- Сервер витягує метадані з фронтматера під час публікації.
- `description` використовується як короткий опис душі в інтерфейсі/пошуку.

## Обмеження

- Загальний розмір пакета: 50MB.
- Текст для ембедингів включає лише `SOUL.md`.

## Слаги

- За замовчуванням визначаються з назви папки.
- Мають бути в нижньому регістрі та безпечними для URL: `^[a-z0-9][a-z0-9-]*$`.

## Версіювання + теги

- Кожна публікація створює нову версію (semver).
- Теги — це рядкові вказівники на версію; `latest` використовується часто.
