# Формат soul

## На диску

Soul — це один файл:

- `SOUL.md` (або `soul.md`)

Наразі onlycrabs.ai відхиляє будь-які додаткові файли.

## `SOUL.md`

- Markdown з необов’язковим YAML frontmatter.
- Сервер витягує метадані з frontmatter під час публікації.
- `description` використовується як короткий опис soul в інтерфейсі користувача/пошуку.

## Обмеження

- Загальний розмір пакета: 50 МБ.
- Текст для вбудовувань включає лише `SOUL.md`.

## Slug

- За замовчуванням формується з назви папки.
- Має бути в нижньому регістрі та безпечним для URL: `^[a-z0-9][a-z0-9-]*$`.

## Версіонування + теги

- Кожна публікація створює нову версію (semver).
- Теги — це рядкові вказівники на версію; зазвичай використовується `latest`.
