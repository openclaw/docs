# Запити щодо прав на контент

Якщо ви вважаєте, що контент, опублікований на ClawHub, порушує ваші авторські
чи інші права, надішліть [запит щодо прав на контент ClawHub](https://forms.openclaw.ai/clawhub-content-rights).

Додайте:

- одну або кілька точних URL-адрес `https://clawhub.ai/<owner>/skills/<skill>`
- ваше ім’я, організацію та контактну адресу електронної пошти
- коротке пояснення проблеми з правами
- підтвердні докази, якщо вони доступні

Співробітники ClawHub розглядають запити вручну й можуть звернутися до заявника або
видавця по додаткову інформацію. Залежно від обставин, відповідний
контент може бути приховано, відновлено або залишено без змін.

Для небезпечного контенту маркетплейсу, який не стосується прав на контент, використовуйте
звичайний процес повідомлення, описаний у розділі [Модерація та безпека облікового запису](/clawhub/moderation).
Для вразливостей у самому ClawHub див. [Безпека](/uk/clawhub/security).
