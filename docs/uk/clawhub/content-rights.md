# Запити щодо прав на вміст

Якщо ви вважаєте, що вміст, опублікований на ClawHub, порушує ваші авторські чи інші
права, надішліть [запит щодо прав на вміст ClawHub](https://forms.openclaw.ai/clawhub-content-rights).

Додайте:

- одну або кілька точних URL-адрес `https://clawhub.ai/<owner>/skills/<skill>`
- ваше ім’я, організацію та контактну електронну адресу
- коротке пояснення занепокоєння щодо прав
- підтвердні докази, якщо доступні

Команда ClawHub перевіряє запити вручну й може зв’язатися із заявником або
видавцем для отримання додаткової інформації. Залежно від обставин, відповідний
вміст може бути приховано, відновлено або залишено без змін.

Для небезпечного вмісту маркетплейсу, який не стосується прав на вміст, використовуйте
звичайний процес повідомлення, описаний у розділі [Модерація та безпека облікового запису](/clawhub/moderation).
Щодо вразливостей у самому ClawHub див. [Безпека](/uk/clawhub/security).
