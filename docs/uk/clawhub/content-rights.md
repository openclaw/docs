# Запити щодо прав на вміст

Якщо ви вважаєте, що вміст, опублікований на ClawHub, порушує ваше авторське право або інші
права, надішліть [запит щодо прав на вміст ClawHub](https://forms.openclaw.ai/clawhub-content-rights).

Додайте:

- одну або кілька точних URL-адрес `https://clawhub.ai/<owner>/skills/<skill>`
- ваше ім’я, організацію та контактну адресу електронної пошти
- коротке пояснення занепокоєння щодо прав
- підтверджувальні докази, якщо є

Співробітники ClawHub розглядають запити вручну та можуть зв’язатися із заявником або
публікатором для отримання додаткової інформації. Залежно від обставин, відповідний
вміст може бути приховано, відновлено або залишено без змін.

Для небезпечного вмісту маркетплейсу, який не стосується прав на вміст, скористайтеся
звичайною процедурою повідомлення, описаною в розділі [Модерація та безпека облікового запису](/clawhub/moderation).
Щодо вразливостей у самому ClawHub див. [Безпека](/uk/clawhub/security).
