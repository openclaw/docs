# Телеметрія

ClawHub використовує мінімальну телеметрію CLI, щоб обчислювати агреговану кількість установлень.

## Коли збирається телеметрія

Телеметрія надсилається лише тоді, коли:

- Ви ввійшли в CLI.
- Ви виконуєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. «Як вимкнути» нижче).

Якщо ви не ввійшли, нічого не передається.

## Що ми збираємо

Під час кожного зареєстрованого `clawhub install` CLI надсилає одну подію встановлення за принципом best-effort.

Подія містить:

- `slug`: slug установленого skill.
- `version`: установлену версію, якщо вона відома.

### Чого ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, отриманих із папок.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, промптів або іншого виводу CLI.

## Кількість установлень

ClawHub підтримує агреговані лічильники для кожного skill:

- `installsAllTime`: унікальні користувачі, які повідомили про принаймні одне встановлення skill через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення й не видалили свою
  телеметрію.

## Прозорість + користувацькі налаштування

Усі бачать лише **агреговані лічильники встановлень**.

Видалення вашого облікового запису також видаляє ваші телеметричні дані.

## Як вимкнути телеметрію

Установіть змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Якщо її встановлено, CLI не надсилатиме телеметрію встановлень.
