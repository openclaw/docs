# Телеметрія

ClawHub використовує мінімальну телеметрію CLI для обчислення сукупної кількості встановлень.

## Коли збирається телеметрія

Телеметрія надсилається лише за таких умов:

- Ви ввійшли в обліковий запис у CLI.
- Ви запускаєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. розділ «Як вимкнути» нижче).

Якщо ви не ввійшли в обліковий запис, жодні дані не передаються.

## Що ми збираємо

Під час кожного переданого `clawhub install` CLI одноразово намагається надіслати подію встановлення.

Подія містить:

- `slug`: ідентифікатор установленої навички.
- `version`: установлену версію, якщо вона відома.

### Чого ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, утворених на їх основі.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, запитів чи іншого виводу CLI.

## Кількість встановлень

ClawHub веде сукупні лічильники для кожної навички:

- `installsAllTime`: унікальні користувачі, які повідомили принаймні про одне встановлення навички через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення й не видалили свої
  телеметричні дані.

## Прозорість і керування для користувачів

Усі бачать лише **сукупні лічильники встановлень**.

У разі видалення облікового запису також видаляються ваші телеметричні дані.

## Як вимкнути телеметрію

Установіть змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Коли її встановлено, CLI не надсилатиме телеметрію встановлень.
