# Телеметрія

ClawHub використовує мінімальну телеметрію CLI для обчислення сукупної кількості встановлень.

## Коли збирається телеметрія

Телеметрія надсилається лише тоді, коли:

- Ви ввійшли в CLI.
- Ви запускаєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. «Як вимкнути» нижче).

Якщо ви не ввійшли в систему, нічого не передається.

## Що ми збираємо

Під час кожного зареєстрованого `clawhub install` CLI надсилає одну подію встановлення за принципом best-effort.

Подія містить:

- `slug`: slug встановленого skill.
- `version`: встановлена версія, якщо відома.

### Що ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, похідних від папок.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, запитів або іншого виводу CLI.

## Кількість встановлень

ClawHub підтримує сукупні лічильники для кожного skill:

- `installsAllTime`: унікальні користувачі, які повідомили принаймні про одне встановлення skill через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення й не видалили свою
  телеметрію.

## Прозорість + користувацькі налаштування

Усі бачать лише **сукупні лічильники встановлень**.

Видалення вашого облікового запису також видаляє ваші телеметричні дані.

## Як вимкнути телеметрію

Задайте змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Коли її задано, CLI не надсилатиме телеметрію встановлень.
