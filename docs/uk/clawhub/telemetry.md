# Телеметрія

ClawHub використовує мінімальну телеметрію CLI для обчислення сукупної кількості встановлень.

## Коли збирається телеметрія

Телеметрія надсилається лише тоді, коли:

- Ви ввійшли в систему в CLI.
- Ви запускаєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. «Як вимкнути» нижче).

Якщо ви не ввійшли в систему, нічого не надсилається.

## Що ми збираємо

Для кожного зареєстрованого `clawhub install` CLI надсилає одну подію встановлення в режимі найкращого зусилля.

Подія містить:

- `slug`: slug встановленої навички.
- `version`: встановлена версія, якщо відома.

### Чого ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, похідних від папок.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, промптів або іншого виводу CLI.

## Кількість встановлень

ClawHub підтримує сукупні лічильники для кожної навички:

- `installsAllTime`: унікальні користувачі, які повідомили принаймні про одне встановлення навички через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення і не видалили свою
  телеметрію.

## Прозорість + елементи керування для користувача

Усі бачать лише **агреговані лічильники встановлень**.

Видалення вашого облікового запису також видаляє ваші дані телеметрії.

## Як вимкнути телеметрію

Установіть змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Коли її встановлено, CLI не надсилатиме телеметрію встановлень.
