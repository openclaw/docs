# Телеметрія

ClawHub використовує мінімальну телеметрію CLI, щоб обчислювати агреговані лічильники встановлень.

## Коли збирається телеметрія

Телеметрія надсилається лише коли:

- Ви ввійшли в CLI.
- Ви запускаєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. «Як вимкнути» нижче).

Якщо ви не ввійшли, нічого не надсилається.

## Що ми збираємо

Під час кожного зареєстрованого `clawhub install` CLI надсилає одну подію встановлення за принципом найкращої спроби.

Подія містить:

- `slug`: slug встановленої skill.
- `version`: встановлена версія, якщо відома.

### Чого ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, похідних від папок.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, промптів або іншого виводу CLI.

## Лічильники встановлень

ClawHub підтримує агреговані лічильники для кожної skill:

- `installsAllTime`: унікальні користувачі, які повідомили про принаймні одне встановлення skill через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення й не видалили свою
  телеметрію.

## Прозорість + керування для користувача

Усі бачать лише **агреговані лічильники встановлень**.

Видалення вашого облікового запису також видаляє ваші телеметричні дані.

## Як вимкнути телеметрію

Установіть змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Коли її встановлено, CLI не надсилатиме телеметрію встановлень.
