# Телеметрія

ClawHub використовує мінімальну телеметрію CLI, щоб обчислювати агреговану кількість встановлень.

## Коли збирається телеметрія

Телеметрія надсилається лише тоді, коли:

- Ви ввійшли в CLI.
- Ви запускаєте `clawhub install <slug>`.
- Телеметрію **не вимкнено** (див. «Як вимкнути» нижче).

Якщо ви не ввійшли, нічого не передається.

## Що ми збираємо

Під час кожного зареєстрованого `clawhub install` CLI надсилає одну подію встановлення за принципом найкращої спроби.

Подія містить:

- `slug`: slug установленого вміння.
- `version`: установлена версія, якщо вона відома.

### Чого ми _не_ збираємо

- Жодних шляхів до папок або ідентифікаторів, отриманих із папок.
- Жодного вмісту файлів.
- Жодних журналів окремих запусків, запитів або іншого виводу CLI.

## Кількість встановлень

ClawHub підтримує агреговані лічильники для кожного вміння:

- `installsAllTime`: унікальні користувачі, які повідомили принаймні про одне встановлення вміння через CLI.
- `installsCurrent`: унікальні користувачі, які повідомили про встановлення й не видалили свою
  телеметрію.

## Прозорість + засоби керування для користувача

Усі бачать лише **агреговані лічильники встановлень**.

Видалення вашого облікового запису також видаляє ваші дані телеметрії.

## Як вимкнути телеметрію

Установіть змінну середовища:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

Коли її встановлено, CLI не надсилатиме телеметрію встановлень.
