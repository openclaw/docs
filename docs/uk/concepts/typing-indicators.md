Індикатори набору тексту надсилаються до каналу чату, поки виконання активне. Використовуйте `agents.defaults.typingMode`, щоб керувати тим, **коли** починається набір тексту, і `typingIntervalSeconds`, щоб керувати тим, **як часто** індикатор оновлюється (інтервал підтримання активності, типово 6 секунд).

## Типові значення

Коли `agents.defaults.typingMode` **не задано**:

- **Прямі чати**: набір тексту починається відразу після запуску циклу моделі.
- **Групові чати зі згадкою**: набір тексту починається відразу.
- **Групові чати без згадки**: набір тексту починається, коли допущене виконання має видиму для користувача активність, наприклад активність виконання середовища або текст повідомлення.
- **Виконання Heartbeat**: набір тексту починається із запуском виконання Heartbeat, якщо визначеною ціллю Heartbeat є чат із підтримкою індикатора набору тексту й цей індикатор не вимкнено.

## Режими

Установіть для `agents.defaults.typingMode` одне з таких значень:

- `never` — індикатор набору тексту ніколи не відображається.
- `instant` — починати набір тексту **відразу після запуску циклу моделі**, навіть якщо згодом виконання поверне лише токен безмовної відповіді.
- `thinking` — починати набір тексту з **першого фрагмента міркування** або під час активного виконання середовища після прийняття ходу.
- `message` — починати набір тексту з **першої видимої для користувача активності відповіді**, наприклад активного виконання середовища або небезмовного текстового фрагмента. Токени безмовної відповіді, як-от `NO_REPLY`, не вважаються текстовою активністю.

Порядок за часом спрацювання, від найранішого: `never` -> `message`/`thinking` -> `instant`.

## Конфігурація

Задайте типове значення на рівні агента:

```json5
{
  agents: {
    defaults: {
      typingMode: "thinking",
      typingIntervalSeconds: 6,
    },
  },
}
```

Перевизначте режим або інтервал для окремого сеансу:

```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4,
  },
}
```

## Примітки

- Режим `message` не запускається від токенів безмовної відповіді, але активне виконання все одно може показувати індикатор набору тексту ще до появи тексту асистента.
- `thinking` усе ще реагує на потокове передавання міркувань (`reasoningLevel: "stream"`), а також може запускатися від активного виконання до надходження фрагментів міркування.
- Індикатор набору тексту Heartbeat є сигналом активності для визначеної цілі доставки. Він запускається на початку виконання Heartbeat, а не відповідно до часу потоку `message` або `thinking`. Установіть `typingMode: "never"`, щоб вимкнути його.
- Heartbeat не показує індикатор набору тексту, якщо ціллю Heartbeat є `"none"`, якщо ціль неможливо визначити, якщо доставку в чат вимкнено для Heartbeat або якщо канал не підтримує індикатор набору тексту.
- `typingIntervalSeconds` керує **інтервалом оновлення**, а не часом запуску. Типове значення: 6 секунд.
