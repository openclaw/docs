Експериментальні функції — це попередні можливості, які вмикаються добровільно за допомогою явних прапорців. Перш ніж отримати стабільне типове налаштування або довгостроковий контракт, вони мають пройти ширше випробування в реальних умовах.

- Типово вимкнені, якщо документація прямо не вказує ввімкнути певну функцію.
- Їхня структура та поведінка можуть змінюватися швидше, ніж стабільна конфігурація.
- Якщо стабільний спосіб уже існує, віддавайте перевагу йому.
- Широко розгортайте лише після попереднього тестування в меншому середовищі.

## Полегшений режим локальної моделі

`agents.defaults.experimental.localModelLean: true` під час кожного кроку прибирає з безпосередньо доступного агенту набору ресурсомісткі необов’язкові інструменти: `browser`, `cron`, `message`, `image_generate`, `music_generate`, `video_generate`, `tts` і `pdf`. Явно дозволені або необхідні для доставлення інструменти залишаються доступними, хоча Пошук інструментів може додавати їх до каталогу замість безпосереднього показу. Якщо `tools.toolSearch` ще не задано, полегшений режим також типово переводить каталоги плагінів, MCP і клієнтів на структурований Пошук інструментів (`tool_search`, `tool_describe`, `tool_call`). Щоб застосувати це лише до одного агента, використовуйте `agents.list[].experimental.localModelLean`.

Якщо ви вже налаштовуєте Пошук інструментів глобально, OpenClaw не змінює цю конфігурацію. Задайте `tools.toolSearch: false`, щоб відмовитися від типового Пошуку інструментів полегшеного режиму.

У структурованому режимі `tools` полегшені запуски залишають `exec` безпосередньо видимим поряд з елементами керування Пошуком інструментів, щоб локальні моделі, налаштовані для програмування, і надалі могли вибирати звичний шлях через оболонку. Це змінює лише видимість схеми: звичайна політика інструментів, ізоляція в пісочниці та схвалення виконання й надалі застосовуються. Явні режими `code` і `directory` зберігають свою звичайну поведінку Compaction.

### Чому саме ці інструменти

Ці інструменти мають найдовші описи, найширші структури параметрів або найвищу ймовірність відволікти малу модель від звичайного програмування та спілкування. Для серверної системи з малим контекстом або суворішою сумісністю з OpenAI це визначає різницю між такими ситуаціями:

- Схеми інструментів уміщуються в запит або витісняють історію розмови.
- Модель вибирає правильний інструмент або створює некоректні виклики через надмірну кількість подібних схем.
- Адаптер Chat Completions залишається в межах структурованого виведення або повертає помилку 400 через завеликий розмір даних виклику інструмента.

Їх видалення лише скорочує список безпосередньо доступних інструментів. Модель і надалі має `read`, `write`, `edit`, `exec`, `apply_patch`, розпізнавання зображень, пошук і отримання даних з інтернету (якщо налаштовано), пам’ять, а також інструменти сеансів та агентів. Додаткові каталоги залишаються доступними через Пошук інструментів, якщо не задано `tools.toolSearch: false`; явні дозволи інструментів можуть знову надати полегшеному агенту доступ до скороченого робочого процесу.

### Коли вмикати

Увімкніть полегшений режим після того, як переконаєтеся, що модель може взаємодіяти з Gateway, але звичайні кроки агента працюють неправильно:

1. Команда `openclaw infer model run --gateway --model <ref> --prompt "Reply with exactly: pong"` виконується успішно.
2. Звичайний крок агента завершується невдало через некоректні виклики інструментів, завеликі запити або ігнорування інструментів моделлю.
3. Установлення `localModelLean: true` усуває помилку.

### Коли залишати вимкненим

Якщо ваша серверна система належним чином працює з повним типовим середовищем виконання, залиште цей режим вимкненим. Це обхідне рішення для локальних стеків, яким потрібен менший набір інструментів, а не типовий режим для хостингових моделей або потужних локальних систем.

Полегшений режим не замінює `tools.profile`, `tools.allow`/`tools.deny` або аварійний механізм моделі `compat.supportsTools: false`. Щоб постійно звузити набір інструментів для певного агента, віддавайте перевагу цим стабільним параметрам.

### Увімкнення

```json5
{
  agents: {
    defaults: {
      experimental: {
        localModelLean: true,
      },
    },
  },
}
```

Лише для одного агента:

```json5
{
  agents: {
    list: [
      {
        id: "local",
        model: "lmstudio/gemma-4-e4b-it",
        experimental: {
          localModelLean: true,
        },
      },
    ],
  },
}
```

Після зміни прапорця перезапустіть Gateway. Полегшена фільтрація видаляє `browser`, `cron`, `message`, `image_generate`, `music_generate`, `video_generate`, `tts` і `pdf`, якщо ви явно не збережете їх за допомогою `tools.allow` або `tools.alsoAllow`; Пошук інструментів усе одно може додавати збережені інструменти до каталогу замість їх безпосереднього показу.

## Експериментальне не означає приховане

Експериментальну функцію потрібно чітко позначати як таку в документації та безпосередньо в шляху конфігурації, а не приховувати за параметром, що виглядає стабільним і типовим.
