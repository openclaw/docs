# `openclaw dashboard`

Відкрийте інтерфейс керування, використовуючи поточні дані автентифікації.

```bash
openclaw dashboard
openclaw dashboard --no-open
openclaw dashboard --yes
```

- `--no-open`: вивести URL-адресу, але не запускати браузер.
- `--yes`: за потреби запустити або встановити Gateway без запиту підтвердження.

Примітки:

- За можливості розпізнає налаштовані SecretRef у `gateway.auth.token`.
- Дотримується параметра `gateway.tls.enabled`: Gateway із увімкненим TLS виводять і відкривають URL-адреси інтерфейсу керування з `https://` та підключаються через `wss://`.
- Для прив’язки `lan` або `custom` із шаблонною адресою запуск на тому самому хості завжди використовує local loopback, оскільки шаблонна адреса не є адресою призначення для браузера. Незашифровані прив’язки `tailnet` і `custom` також використовують `127.0.0.1`, щоб браузер мав безпечний контекст; конкретні хости з увімкненим TLS зберігають налаштовану адресу, щоб імена сертифікатів збігалися.
- Перш ніж надати автентифіковану URL-адресу local loopback для прив’язки до конкретного інтерфейсу, команда перевіряє налаштований інтерфейс і підтверджує, що він та `127.0.0.1` належать одному процесу Gateway. Якщо власника прослуховувача неможливо визначити однозначно, команда безпечно завершується з рекомендаціями щодо перевірки стану.
- Для токенів, керованих через SecretRef (як розпізнаних, так і нерозпізнаних), виведена, скопійована або відкрита URL-адреса ніколи не містить токена, тому зовнішні секрети не потрапляють у вивід термінала, історію буфера обміну або аргументи запуску браузера.
- Якщо `gateway.auth.token` керується через SecretRef, але його не вдалося розпізнати, команда замість недійсного заповнювача токена виводить URL-адресу без токена та рекомендації щодо усунення проблеми.
- Якщо передати автентифіковану токеном URL-адресу через буфер обміну або браузер не вдається, команда записує в журнал безпечну підказку щодо ручної автентифікації із зазначенням `OPENCLAW_GATEWAY_TOKEN`, `gateway.auth.token` і ключа фрагмента URL `token`, не виводячи значення токена.
