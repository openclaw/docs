# Інвентаризація Plugin

Цю сторінку згенеровано з `extensions/*/package.json`, `openclaw.plugin.json`
та вилучень `files` у кореневому пакеті npm. Повторно згенеруйте її за допомогою:

```bash
pnpm plugins:inventory:gen
```

## Визначення

- **Основний пакет npm:** вбудований у пакет npm `openclaw` і доступний без окремого встановлення Plugin.
- **Офіційний зовнішній пакет:** Plugin, що підтримується OpenClaw, вилучений з основного пакета npm і встановлюється через ClawHub та/або npm.
- **Лише вихідний checkout:** локальний для репозиторію Plugin, вилучений з опублікованих артефактів npm і не рекламований як установлюваний пакет.

Вихідні checkout відрізняються від встановлень npm: після `pnpm install` вбудовані
Plugin завантажуються з `extensions/<id>`, тому локальні зміни та локальні для пакета залежності workspace доступні.
