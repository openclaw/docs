# Plugin передавання файлів

Отримуйте, переглядайте та записуйте файли на спарених вузлах за допомогою спеціальних команд вузла. Дає змогу уникнути обрізання стандартного виведення bash завдяки використанню base64 через node.invoke для двійкових файлів розміром до 16 МБ.

## Розповсюдження

- Пакунок: `@openclaw/file-transfer`
- Спосіб установлення: входить до складу OpenClaw
