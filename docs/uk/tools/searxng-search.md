OpenClaw підтримує [SearXNG](https://docs.searxng.org/) як **самостійно розміщуваного
постачальника без ключа** для `web_search`. SearXNG — це метапошукова система з відкритим вихідним кодом,
яка об’єднує результати з Google, Bing, DuckDuckGo та інших джерел.

Переваги:

- **Безкоштовно й без обмежень** — не потрібні API-ключ або комерційна підписка
- **Конфіденційність / ізоляція від зовнішньої мережі** — запити ніколи не залишають вашу мережу
- **Працює будь-де** — немає регіональних обмежень комерційних пошукових API

## Налаштування

<Steps>
  <Step title="Установіть Plugin">
    ```bash
    openclaw plugins install @openclaw/searxng-plugin
    ```
  </Step>
  <Step title="Запустіть екземпляр SearXNG">
    ```bash
    docker run -d -p 8888:8080 searxng/searxng
    ```

    Або скористайтеся будь-яким наявним розгортанням SearXNG, до якого маєте доступ. Відомості про
    налаштування для робочого середовища див. в [документації SearXNG](https://docs.searxng.org/).

  </Step>
  <Step title="Налаштуйте">
    ```bash
    openclaw configure --section web
    # Виберіть "searxng" як постачальника
    ```

    Або задайте змінну середовища й дозвольте автоматичному виявленню знайти її:

    ```bash
    export SEARXNG_BASE_URL="http://localhost:8888"
    ```

  </Step>
</Steps>

## Конфігурація

```json5
{
  tools: {
    web: {
      search: {
        provider: "searxng",
      },
    },
  },
}
```

Налаштування рівня Plugin для екземпляра SearXNG:

```json5
{
  plugins: {
    entries: {
      searxng: {
        config: {
          webSearch: {
            baseUrl: "http://localhost:8888",
            categories: "general,news", // необов’язково
            language: "en", // необов’язково
          },
        },
      },
    },
  },
}
```

`baseUrl` також приймає об’єкт SecretRef (наприклад, `{ source: "env", id: "SEARXNG_BASE_URL" }`).

## Змінна середовища

Як альтернативу конфігурації задайте `SEARXNG_BASE_URL`:

```bash
export SEARXNG_BASE_URL="http://localhost:8888"
```

Порядок визначення: налаштований рядок `baseUrl`, потім вбудований SecretRef середовища в
`baseUrl`, а потім `SEARXNG_BASE_URL`. Якщо жоден зі шляхів конфігурації не задано,
наявна змінна `SEARXNG_BASE_URL` і постачальника не вибрано явно, автоматичне виявлення
вибирає SearXNG.

## Довідка з конфігурації Plugin

| Поле         | Опис                                                               |
| ------------ | ------------------------------------------------------------------ |
| `baseUrl`    | Базова URL-адреса вашого екземпляра SearXNG (обов’язково)          |
| `categories` | Розділені комами категорії, як-от `general`, `news` або `science`   |
| `language`   | Код мови результатів, як-от `en`, `de` або `fr`                    |

Виклик інструмента `web_search` також приймає `count` (1–10 результатів), `categories`
і `language` як перевизначення для окремого виклику.

## Примітки

- **JSON API** — використовує нативну кінцеву точку SearXNG `format=json`, а не збирання даних із HTML
- **URL-адреси результатів із зображеннями** — результати категорії зображень містять `img_src`, коли SearXNG
  повертає пряму URL-адресу зображення
- **Без API-ключа** — одразу працює з будь-яким екземпляром SearXNG
- **Перевірка базової URL-адреси** — `baseUrl` має бути дійсною URL-адресою `http://` або `https://`
- **Мережевий захист** — базові URL-адреси `http://` мають указувати на довірений приватний хост або
  хост local loopback (публічні хости мають використовувати `https://`); базові URL-адреси `https://`, які
  визначаються як приватні або внутрішні адреси, отримують такий самий дозвіл для самостійно розміщуваних ресурсів,
  тоді як для базових URL-адрес `https://`, які визначаються як публічні, зберігається суворий захист від SSRF
- **Порядок автоматичного виявлення** — SearXNG потребує налаштованого `baseUrl` (порядок
  200 серед постачальників, які вже мають необхідні облікові дані). Постачальники без ключа,
  як-от DuckDuckGo або Ollama Web Search, ніколи не вибираються автоматично неявно;
  вони активуються лише після явного вибору `provider`
- **Самостійне розміщення** — ви контролюєте екземпляр, запити та зовнішні пошукові системи
- **Категорії** за замовчуванням мають значення `general`, якщо їх не налаштовано
- **Резервна категорія** — якщо запит із категорією, відмінною від `general`, виконується успішно, але
  повертає нуль результатів, OpenClaw один раз повторює той самий запит із `general`,
  перш ніж повернути порожній набір результатів
- **Кешування результатів** — ідентичні запити (однакові запит, кількість, категорії,
  мова та базова URL-адреса) кешуються в межах процесу протягом короткого TTL
- **Вимога до версії** — Plugin оголошує `minHostVersion: >=2026.6.9`

<Tip>
  Щоб JSON API SearXNG працював, переконайтеся, що у вашому екземплярі SearXNG формат `json`
  увімкнено у файлі `settings.yml` у розділі `search.formats`.
</Tip>
