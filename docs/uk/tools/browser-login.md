## Ручний вхід (рекомендовано)

Коли сайт вимагає входу, **увійдіть вручну** в профілі **хостового** браузера (браузері openclaw).

**Не** передавайте моделі свої облікові дані. Автоматизовані входи часто вмикають захист від ботів і можуть заблокувати обліковий запис.

Повернутися до основної документації браузера: [Браузер](/uk/tools/browser).

## Який профіль Chrome використовується?

OpenClaw керує **виділеним профілем Chrome** (з назвою `openclaw`, інтерфейс із помаранчевим відтінком). Він окремий від вашого щоденного профілю браузера.

Для викликів браузерного інструмента агентом:

- Вибір за замовчуванням: агент має використовувати свій ізольований браузер `openclaw`.
- Використовуйте `profile="user"` лише тоді, коли важливі наявні сеанси з входом, а користувач перебуває за комп’ютером, щоб натиснути/схвалити будь-який запит на під’єднання.
- Якщо у вас кілька профілів користувацького браузера, укажіть профіль явно замість здогадування.

Два прості способи отримати до нього доступ:

1. **Попросіть агента відкрити браузер**, а потім увійдіть самостійно.
2. **Відкрийте його через CLI**:

```bash
openclaw browser start
openclaw browser open https://x.com
```

Якщо у вас кілька профілів, передайте `--browser-profile <name>` (за замовчуванням використовується `openclaw`).

## X/Twitter: рекомендований потік

- **Читання/пошук/треди:** використовуйте **хостовий** браузер (ручний вхід).
- **Публікація оновлень:** використовуйте **хостовий** браузер (ручний вхід).

## Пісочниця + доступ до хостового браузера

Сеанси браузера в пісочниці **з більшою ймовірністю** вмикають виявлення ботів. Для X/Twitter (та інших суворих сайтів) віддавайте перевагу **хостовому** браузеру.

Якщо агент працює в пісочниці, браузерний інструмент за замовчуванням використовує пісочницю. Щоб дозволити керування хостом:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        browser: {
          allowHostControl: true,
        },
      },
    },
  },
}
```

Потім націльтеся на хостовий браузер:

```bash
openclaw browser open https://x.com --browser-profile openclaw --target host
```

Або вимкніть пісочницю для агента, який публікує оновлення.
