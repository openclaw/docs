# Northflank

ワンクリックテンプレートで Northflank に OpenClaw をデプロイし、Web の Control UI からアクセスします。
これは「サーバー上でターミナル不要」の最も簡単な方法です。Gateway は Northflank が実行します。

## はじめ方

1. [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) をクリックしてテンプレートを開きます。
2. まだ持っていない場合は [Northflank のアカウント](https://app.northflank.com/signup) を作成します。
3. **Deploy OpenClaw now** をクリックします。
4. 必須の環境変数 `OPENCLAW_GATEWAY_TOKEN` を設定します（強力でランダムな値を使ってください）。
5. **Deploy stack** をクリックして OpenClaw テンプレートをビルドし、実行します。
6. デプロイ完了を待ってから **View resources** をクリックします。
7. OpenClaw サービスを開きます。
8. 公開 OpenClaw URL の `/openclaw` を開き、設定した共有 secret で接続します。このテンプレートはデフォルトで `OPENCLAW_GATEWAY_TOKEN` を使用します。これをパスワード認証に置き換えた場合は、代わりにそのパスワードを使ってください。

## 得られるもの

- ホストされた OpenClaw Gateway + Control UI
- Northflank Volume（`/data`）による永続ストレージ。これにより `openclaw.json`、エージェントごとの `auth-profiles.json`、チャネル/provider 状態、セッション、ワークスペースが再デプロイ後も保持されます

## チャネルを接続する
