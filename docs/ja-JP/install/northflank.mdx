ワンクリックテンプレートを使用して Northflank に OpenClaw をデプロイし、Web Control UI からアクセスします。これは「サーバー上でターミナルを使わない」最も簡単な方法です。Northflank が Gateway を実行します。

## はじめに

1. [OpenClaw をデプロイ](https://northflank.com/stacks/deploy-openclaw)をクリックしてテンプレートを開きます。
2. Northflank のアカウントをまだお持ちでない場合は、[Northflank でアカウントを作成](https://app.northflank.com/signup)します。
3. **Deploy OpenClaw now** をクリックします。
4. 必須の環境変数 `OPENCLAW_GATEWAY_TOKEN` を設定します（強力なランダム値を使用してください）。
5. **Deploy stack** をクリックして、OpenClaw テンプレートをビルドして実行します。
6. デプロイが完了するまで待ってから、**View resources** をクリックします。
7. OpenClaw サービスを開きます。
8. `/openclaw` にある OpenClaw の公開 URL を開き、設定した共有シークレットを使用して接続します。このテンプレートでは、デフォルトで `OPENCLAW_GATEWAY_TOKEN` を使用します。パスワード認証に置き換えた場合は、代わりにそのパスワードを使用します。

## 利用できるもの

- ホストされた OpenClaw Gateway と Control UI
- Northflank Volume（`/data`）による永続ストレージ。これにより、`openclaw.json`、エージェントごとの `auth-profiles.json`、チャネル／プロバイダーの状態、セッション、ワークスペースが再デプロイ後も保持されます

## チャネルを接続する
