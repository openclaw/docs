ワンクリックテンプレートで Northflank に OpenClaw をデプロイし、Web Control UI からアクセスします。これは最も簡単な「サーバー上でターミナルを使わない」方法です。Northflank が Gateway を実行します。

## はじめに

1. [OpenClaw をデプロイ](https://northflank.com/stacks/deploy-openclaw)をクリックしてテンプレートを開きます。
2. まだ持っていない場合は、[Northflank のアカウント](https://app.northflank.com/signup)を作成します。
3. **今すぐ OpenClaw をデプロイ**をクリックします。
4. 必須の環境変数を設定します: `OPENCLAW_GATEWAY_TOKEN`（強力なランダム値を使用してください）。
5. **スタックをデプロイ**をクリックして OpenClaw テンプレートをビルドして実行します。
6. デプロイが完了するまで待ち、**リソースを表示**をクリックします。
7. OpenClaw サービスを開きます。
8. `/openclaw` の公開 OpenClaw URL を開き、設定済みの共有シークレットで接続します。このテンプレートはデフォルトで `OPENCLAW_GATEWAY_TOKEN` を使用します。パスワード認証に置き換えた場合は、代わりにそのパスワードを使用します。

## 得られるもの

- ホストされた OpenClaw Gateway + Control UI
- Northflank Volume（`/data`）による永続ストレージ。これにより、`openclaw.json`、エージェントごとの `auth-profiles.json`、チャネル/プロバイダーの状態、セッション、ワークスペースが再デプロイ後も保持されます
