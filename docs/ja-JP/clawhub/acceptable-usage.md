# 許容される利用

ClawHub は OpenClaw 向けのスキル、プラグイン、パッケージ、マーケットプレイスメタデータをホストします。
コンテンツまたは公開動作が ClawHub に属するかどうかを判断するために、
このページを使用してください。

これらのルールは、リスティングが何をするか、ユーザーに何の実行を求めるか、どのように
自身を表現するか、公開者が ClawHub の発見、インストール、信頼サーフェスを
どのように使うかに適用されます。モデレーション状態とアカウントの状態については、
[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。著作権またはその他の権利
の申し立てについては、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を参照してください。

## 許可されるコンテンツ

ClawHub は、有用で理解しやすく、誠実に公開されたコンテンツを歓迎します。

| カテゴリ                                         | 許可される条件                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開発者の生産性                           | リスティングが、ユーザーのソフトウェアのビルド、テスト、移行、デバッグ、文書化、運用に役立つ。                                               |
| UI、データ、自動化ワークフロー               | スコープが明確で、必要な認証情報が明示され、リスクのある操作にレビュー、ドライラン、プレビュー、または確認の経路が含まれる。 |
| 防御的セキュリティ、モデレーション、不正利用レビュー | ツールが承認済みレビュー向けとして位置付けられ、証拠を保持し、人間による承認境界を明確に保つ。                          |
| 個人またはチームのワークフロー                       | ワークフローが、同意に基づくアカウント、透明なセットアップ、明示的な権限を使用する。                                            |
| 保守されているカタログ                              | 各リスティングが明確に区別され、有用で、正確に説明され、適切に保守されている。                                                |

文脈が重要です。同じトピックでも、狭い防御的または
同意に基づく設定では許容される場合があり、不正利用ワークフローとしてパッケージ化されている場合は許容されません。

## 禁止されるコンテンツ

ClawHub は、主な目的が不正利用、欺瞞、安全でない
実行、または権利侵害であるコンテンツをホストしません。

| カテゴリ                                                    | 許可されない内容                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 不正アクセスまたはセキュリティ回避                      | 認証回避、アカウント乗っ取り、レート制限の不正利用、ライブ通話またはエージェントの乗っ取り、再利用可能なセッションの窃取、または承認されていないユーザー向けのペアリングフローの自動承認。                                                                                                                                                   |
| プラットフォームの不正利用とBAN回避                              | BAN後のステルスアカウント、アカウントのウォーミングまたはファーミング、偽のエンゲージメント、複数アカウントの自動化、大量投稿、スパムボット、または検出回避を目的に構築された自動化。                                                                                                                                          |
| 詐欺、スキャム、欺瞞的な金融ワークフロー             | 偽の証明書または請求書、欺瞞的な支払いフロー、スキャムのアウトリーチ、偽の社会的証明、詐欺向けの合成IDワークフロー、または明確な人間の承認なしに支出や請求を行うツール。                                                                                                                    |
| プライバシー侵害的なエンリッチメントまたは監視                 | スパム目的の連絡先スクレイピング、ドキシング、ストーキング、未承諾アウトリーチと組み合わせたリード抽出、秘密裏の監視、同意のない生体照合、または漏えいデータや侵害ダンプの使用。                                                                                                                  |
| 同意のないなりすましまたはID操作       | フェイススワップ、デジタルツイン、クローン化されたインフルエンサー、偽のペルソナ、またはなりすましや誤誘導に使用されるその他のツール。                                                                                                                                                                                                 |
| 露骨な性的コンテンツまたは安全制御を無効化した成人向け生成 | NSFW画像、動画、またはコンテンツ生成、サードパーティAPIを包む成人向けコンテンツラッパー、または主目的が露骨な性的コンテンツであるリスティング。                                                                                                                                                       |
| 隠された、安全でない、または誤解を招く実行要件        | 難読化されたインストールコマンド、明確なレビュー可能性なしに `sh` または `bash` で実行されるダウンロードコンテンツなどのパイプtoシェル型インストーラー、宣言されていないシークレットまたは秘密鍵要件、明確なレビュー可能性なしのリモート `npx @latest` 実行、またはリスティングが実行に本当に必要とするものを隠すメタデータ。 |
| 著作権侵害または権利侵害にあたる素材           | 許可なく他者のスキル、プラグイン、ドキュメント、ブランドアセット、またはプロプライエタリコードを再公開すること、ライセンス条件に違反すること、または元の作者や公開者になりすますこと。                                                                                                                            |

## 禁止されるマーケットプレイス動作

ClawHub は、公開者がマーケットプレイスをどのように使うかもレビューします。発見、指標、信頼シグナル、モデレーションシステム、またはユーザーの
注意を操作するために ClawHub を使用しないでください。

禁止されるマーケットプレイス動作には次が含まれます。

- 実際のユーザー価値があるように見えない、低労力、重複、プレースホルダー、または
  機械生成のリスティングを大量に一括公開すること
- ほぼ同一のスキルまたはプラグインで検索またはカテゴリサーフェスを氾濫させること
- 使用、保守、ソースの明確さ、または意味のある差別化がほとんどない、またはまったくない
  数百件のリスティングを公開すること
- 自動化、自己インストールループ、偽アカウント、協調的
  活動、有料エンゲージメント、またはその他の非オーガニックな行動を通じて、インストール数、ダウンロード数、スター数、またはその他のエンゲージメント
  指標を人為的につり上げること
- モデレーション、BAN、公開者制限、または
  マーケットプレイスレビューを回避するためにアカウントを作成またはローテーションすること
- 所有権、ソース、機能、セキュリティ態勢、
  インストール要件、または別のプロジェクトや公開者との提携についてユーザーに誤解を与えること
- 根本的な問題を修正せずに、すでに非表示、削除、またはブロックされた
  コンテンツを繰り返しアップロードすること

大量公開は自動的に不正利用になるわけではありません。リスティングが意味のある形で異なり、正確に説明され、保守され、
実際のユーザーに使用されている場合、大規模なカタログは許容されます。大量公開は、
量が薄い内容、重複、誤解を招く内容、未保守、または
人為的に宣伝されたリスティングと組み合わさると、信頼と安全性の問題になります。

## コンテンツ権利

ClawHub 上のコンテンツがあなたの著作権またはその他の権利を侵害していると思う場合は、
[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を使用してください。リスティングが安全でない、
悪意がある、または誤解を招く場合を除き、著作権または権利の申し立てに通常のマーケットプレイス
報告を使用しないでください。

## レビューと執行

ClawHub は、安全でないコンテンツまたは不正な公開動作を特定するために、自動チェック、統計的な不正利用シグナル、ユーザー報告、
スタッフレビューを使用する場合があります。シグナル単独で不正利用が証明されるわけではありません。ClawHub がレビューの必要な対象を判断する助けになります。

当社は次の措置を取る場合があります。

- 違反しているリスティングを非表示、保留、削除、ソフトデリート、またはリソースタイプでサポートされている場合は
  ハードデリートする
- 安全でないリリースのダウンロードまたはインストールをブロックする
- APIトークンを取り消す
- 関連コンテンツをソフトデリートする
- 公開アクセスを制限する
- 反復的または重大な違反者をBANする

明白な不正利用に対して、警告を先行する執行を保証するものではありません。報告、モデレーション保留、
非表示リスティング、BAN、アカウントの状態については、
[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。
