# 許容される利用

ClawHub は OpenClaw 向けの Skills、Plugin、パッケージ、マーケットプレイスメタデータをホストします。
コンテンツや公開時の振る舞いが ClawHub に属するかどうかを判断するために、このページを使用してください。

これらのルールは、リスティングが何をするか、ユーザーに何の実行を求めるか、自身をどのように表現するか、公開者が ClawHub の発見、インストール、信頼の各サーフェスをどのように使用するかに適用されます。モデレーション状態とアカウントの状態については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。著作権またはその他の権利の申し立てについては、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を参照してください。

## 許可されるコンテンツ

ClawHub は、有用で、理解しやすく、善意で公開されたコンテンツを歓迎します。

| カテゴリ                                         | 許可される条件                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開発者の生産性                           | リスティングが、ユーザーによるソフトウェアのビルド、テスト、移行、デバッグ、ドキュメント作成、または運用を支援する。                                               |
| UI、データ、自動化ワークフロー               | スコープが明確で、必要な認証情報が明示され、リスクのある操作にレビュー、ドライラン、プレビュー、または確認の経路が含まれている。 |
| 防御的セキュリティ、モデレーション、不正利用レビュー | ツールが許可されたレビュー向けとして位置づけられ、証拠を保持し、人間の承認境界を明確に保つ。                          |
| 個人またはチームのワークフロー                       | ワークフローが、同意に基づくアカウント、透明なセットアップ、明示的な権限を使用する。                                            |
| 保守されているカタログ                              | 各リスティングが明確に区別され、有用で、正確に説明され、妥当に保守されている。                                                |

文脈が重要です。同じトピックでも、限定的な防御的用途や同意に基づく設定では許容される一方、不正利用ワークフローとしてパッケージ化されている場合は許容されません。

## 禁止されるコンテンツ

ClawHub は、不正利用、欺瞞、安全でない実行、または権利侵害を主目的とするコンテンツをホストしません。

| カテゴリ                                                    | 許可されないもの                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 無許可アクセスまたはセキュリティ回避                      | 認証回避、アカウント乗っ取り、レート制限の悪用、ライブ通話またはエージェントの乗っ取り、再利用可能なセッションの窃取、または未承認ユーザー向けのペアリングフローの自動承認。                                                                                                                                                   |
| プラットフォームの不正利用と禁止回避                              | 禁止後のステルスアカウント、アカウントのウォーミングまたはファーミング、偽のエンゲージメント、複数アカウントの自動化、大量投稿、スパムボット、または検出回避を目的に構築された自動化。                                                                                                                                          |
| 詐欺、スキャン、欺瞞的な金融ワークフロー             | 偽の証明書や請求書、欺瞞的な支払いフロー、詐欺的なアウトリーチ、偽のソーシャルプルーフ、詐欺向けの合成アイデンティティワークフロー、または明確な人間の承認なしに支出や課金を行うツール。                                                                                                                    |
| プライバシー侵害的なエンリッチメントまたは監視                 | スパム目的の連絡先スクレイピング、ドキシング、ストーキング、未承諾アウトリーチと組み合わせたリード抽出、秘密裏の監視、同意のない生体照合、または漏えいデータや侵害ダンプの使用。                                                                                                                  |
| 同意のないなりすましまたはアイデンティティ操作       | フェイススワップ、デジタルツイン、クローン化されたインフルエンサー、偽のペルソナ、またはなりすましや誤解を招くために使用されるその他のツール。                                                                                                                                                                                                 |
| 明示的な性的コンテンツまたは安全機能が無効化された成人向け生成 | NSFW 画像、動画、またはコンテンツ生成、サードパーティ API をラップする成人向けコンテンツ、または明示的な性的コンテンツを主目的とするリスティング。                                                                                                                                                       |
| 隠された、安全でない、または誤解を招く実行要件        | 難読化されたインストールコマンド、明確なレビュー可能性なしにダウンロードしたコンテンツを `sh` または `bash` で実行するような pipe-to-shell インストーラー、宣言されていないシークレットまたは秘密鍵の要件、明確なレビュー可能性なしのリモート `npx @latest` 実行、またはリスティングが実行に本当に必要とするものを隠すメタデータ。 |
| 著作権侵害または権利侵害のある素材           | 他者の skill、Plugin、ドキュメント、ブランドアセット、またはプロプライエタリコードを許可なく再公開すること、ライセンス条項に違反すること、または元の作者や公開者になりすますこと。                                                                                                                            |

## 禁止されるマーケットプレイス上の振る舞い

ClawHub は、公開者がマーケットプレイスをどのように使用するかもレビューします。発見、指標、信頼シグナル、モデレーションシステム、またはユーザーの注目を操作するために ClawHub を使用しないでください。

禁止されるマーケットプレイス上の振る舞いには、以下が含まれます。

- 実際のユーザー価値があるように見えない、低労力、重複的、プレースホルダー、または機械生成のリスティングを大量に一括公開すること
- ほぼ同一の Skills や Plugin で検索またはカテゴリのサーフェスをあふれさせること
- 利用、保守、ソースの明確さ、または意味のある差別化がほとんどないリスティングを何百件も公開すること
- 自動化、自己インストールループ、偽アカウント、組織的活動、有償エンゲージメント、またはその他の非オーガニックな振る舞いを通じて、インストール、ダウンロード、スター、またはその他のエンゲージメント指標を人為的に膨らませること
- モデレーション、禁止、公開者制限、またはマーケットプレイスレビューを回避するためにアカウントを作成またはローテーションすること
- 所有権、ソース、機能、セキュリティ体制、インストール要件、または別のプロジェクトや公開者との関係についてユーザーに誤解を与えること
- すでに非表示、削除、またはブロックされたコンテンツを、根本的な問題を修正せずに繰り返しアップロードすること

大量公開が自動的に不正利用になるわけではありません。リスティングが意味のある違いを持ち、正確に説明され、保守され、実際のユーザーに使用されている場合、大規模なカタログは許容されます。量が薄い、重複的、誤解を招く、保守されていない、または人為的に宣伝されたリスティングと組み合わさると、大規模なカタログは信頼と安全性の問題になります。

## コンテンツの権利

ClawHub 上のコンテンツが自分の著作権またはその他の権利を侵害していると考える場合は、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を使用してください。リスティングが安全でない、悪意がある、または誤解を招く場合を除き、著作権または権利の申し立てに通常のマーケットプレイス報告を使用しないでください。

## レビューと執行

ClawHub は、安全でないコンテンツや不正な公開行動を特定するために、自動チェック、統計的な不正利用シグナル、ユーザー報告、スタッフレビューを使用する場合があります。シグナル単体で不正利用が証明されるわけではありません。それは ClawHub がレビューの必要な対象を判断する助けになります。

当社は以下を行う場合があります。

- 違反しているリスティングを非表示、保留、削除、ソフト削除、またはリソースタイプでサポートされている場合はハード削除する
- 安全でないリリースのダウンロードまたはインストールをブロックする
- API トークンを取り消す
- 関連コンテンツをソフト削除する
- 公開アクセスを制限する
- 繰り返し違反する者または重大な違反者を禁止する

明らかな不正利用に対して、警告を先に行う執行を保証するものではありません。報告、モデレーション保留、非表示のリスティング、禁止、アカウントの状態については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。
