# 許容される利用

ClawHub は OpenClaw 向けの Skills、Plugin、パッケージ、マーケットプレイスメタデータをホストします。
コンテンツや公開時の挙動が ClawHub に属するかを判断するために、このページを使用してください。

これらのルールは、掲載項目が行うこと、ユーザーに実行を求めること、自身をどう表現するか、公開者が ClawHub の発見、インストール、信頼サーフェスをどう使うかに適用されます。モデレーション状態とアカウントの健全性については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。著作権またはその他の権利の申し立てについては、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を参照してください。

## 許可されるコンテンツ

ClawHub は、有用で理解しやすく、誠実に公開されているコンテンツを歓迎します。

| カテゴリ | 許可される条件 |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開発者生産性 | 掲載項目が、ユーザーによるソフトウェアの構築、テスト、移行、デバッグ、文書化、運用を支援する。 |
| UI、データ、自動化ワークフロー | スコープが明確で、必要な認証情報が明示され、リスクのあるアクションにレビュー、ドライラン、プレビュー、確認の経路が含まれる。 |
| 防御的セキュリティ、モデレーション、不正利用レビュー | ツールが認可されたレビュー向けとして位置付けられ、証拠を保持し、人間による承認境界を明確に保つ。 |
| 個人またはチームのワークフロー | ワークフローが、同意に基づくアカウント、透明なセットアップ、明示的な権限を使用する。 |
| メンテナンスされているカタログ | 各掲載項目が個別性と有用性を持ち、正確に説明され、適切にメンテナンスされている。 |

文脈が重要です。同じトピックでも、狭い防御的または同意に基づく設定では許容される一方で、不正利用ワークフローとしてパッケージ化されている場合は許容されません。

## 許可されないコンテンツ

ClawHub は、不正利用、欺瞞、安全でない実行、または権利侵害を主目的とするコンテンツをホストしません。

| カテゴリ | 許可されない内容 |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 未認可アクセスまたはセキュリティ回避 | 認証回避、アカウント乗っ取り、レート制限の不正利用、ライブ通話またはエージェントの乗っ取り、再利用可能なセッション窃取、未承認ユーザー向けのペアリングフローの自動承認。 |
| プラットフォーム不正利用とBAN回避 | BAN後のステルスアカウント、アカウントのウォーミングまたはファーミング、偽のエンゲージメント、複数アカウント自動化、大量投稿、スパムボット、検知回避を目的として作られた自動化。 |
| 詐欺、スキャム、欺瞞的な金融ワークフロー | 偽の証明書または請求書、欺瞞的な支払いフロー、スキャム目的のアウトリーチ、偽の社会的証明、不正目的の合成IDワークフロー、明確な人間の承認なしに支出または請求を行うツール。 |
| プライバシー侵害的なエンリッチメントまたは監視 | スパム目的の連絡先スクレイピング、ドクシング、ストーキング、未承諾アウトリーチと組み合わせたリード抽出、秘密裏の監視、同意のない生体照合、漏えいデータまたは侵害ダンプの使用。 |
| 同意のないなりすましまたはID操作 | フェイススワップ、デジタルツイン、クローン化されたインフルエンサー、偽のペルソナ、またはなりすましや誤導に使われるその他のツール。 |
| 明示的な性的コンテンツまたは安全制限を無効化した成人向け生成 | NSFW 画像、動画、またはコンテンツ生成、サードパーティ API を包む成人向けコンテンツラッパー、または明示的な性的コンテンツを主目的とする掲載項目。 |
| 隠された、安全でない、または誤解を招く実行要件 | 難読化されたインストールコマンド、明確なレビュー可能性なしにダウンロードしたコンテンツを `sh` または `bash` で実行するような pipe-to-shell インストーラー、未申告のシークレットまたは秘密鍵要件、明確なレビュー可能性なしのリモート `npx @latest` 実行、または掲載項目が実行に本当に必要とするものを隠すメタデータ。 |
| 著作権侵害または権利侵害にあたる素材 | 他者の Skill、Plugin、ドキュメント、ブランドアセット、または専有コードを許可なく再公開すること、ライセンス条件に違反すること、または元の作者や公開者になりすますこと。 |

## 許可されないマーケットプレイス上の挙動

ClawHub は、公開者がマーケットプレイスをどう使うかもレビューします。発見、指標、信頼シグナル、モデレーションシステム、またはユーザーの注意を操作する目的で ClawHub を使用しないでください。

許可されないマーケットプレイス上の挙動には次が含まれます。

- 実際のユーザー価値があるように見えない、低労力、重複的、プレースホルダー的、または機械生成の掲載項目を大量に一括公開すること
- 検索またはカテゴリサーフェスを、ほぼ同一の Skills または Plugin であふれさせること
- 利用、メンテナンス、ソースの明確さ、または意味のある差別化がほとんど、またはまったくない掲載項目を数百件公開すること
- 自動化、自己インストールループ、偽アカウント、協調活動、有料エンゲージメント、またはその他の非オーガニックな挙動によって、インストール数、ダウンロード数、スター数、その他のエンゲージメント指標を人為的に水増しすること
- モデレーション、BAN、公開者制限、またはマーケットプレイスレビューを回避するためにアカウントを作成またはローテーションすること
- 所有権、ソース、機能、セキュリティ姿勢、インストール要件、または別のプロジェクトや公開者との関係についてユーザーを誤解させること
- 根本的な問題を修正せずに、すでに非表示、削除、またはブロックされたコンテンツを繰り返しアップロードすること

大量公開は自動的に不正利用とはみなされません。掲載項目が意味のある違いを持ち、正確に説明され、メンテナンスされ、実際のユーザーに使われている場合、大規模なカタログは許容されます。大量の掲載項目が、内容の薄さ、重複性、誤解を招く表現、未メンテナンス、または人為的な宣伝と組み合わさる場合、大規模なカタログは信頼と安全性の問題になります。

## コンテンツ権利

ClawHub 上のコンテンツが著作権またはその他の権利を侵害していると思われる場合は、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を使用してください。掲載項目が安全でない、悪意がある、または誤解を招く場合を除き、著作権または権利の申し立てに通常のマーケットプレイス報告を使用しないでください。

## レビューと執行

ClawHub は、安全でないコンテンツまたは不正な公開行為を特定するために、自動チェック、統計的な不正利用シグナル、ユーザー報告、スタッフレビューを使用する場合があります。シグナル自体が不正利用を証明するわけではありません。ClawHub がレビューを必要とするものを判断する助けになります。

当社は次を行う場合があります。

- 違反している掲載項目を非表示、保留、削除、ソフト削除、またはリソースタイプでサポートされている場合はハード削除する
- 安全でないリリースのダウンロードまたはインストールをブロックする
- API トークンを取り消す
- 関連コンテンツをソフト削除する
- 公開アクセスを制限する
- 繰り返し違反する者、または重大な違反者をBANする

明白な不正利用に対して、警告を先に行う執行を保証するものではありません。報告、モデレーション保留、非表示の掲載項目、BAN、アカウントの健全性については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。
