# 許容される使用

ClawHub は OpenClaw の Skills、Plugin、パッケージ、マーケットプレイスメタデータをホストします。
コンテンツや公開時の挙動が ClawHub に属するかどうかを判断するために、このページを使用してください。

これらのルールは、リスティングが行うこと、ユーザーに実行を求めること、自身の表現方法、公開者が ClawHub の発見、インストール、信頼のサーフェスをどのように使用するかに適用されます。モデレーション状態とアカウント状態については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。著作権またはその他の権利の申し立てについては、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を参照してください。

## 許可されるコンテンツ

ClawHub は、有用で、理解しやすく、誠実に公開されたコンテンツを歓迎します。

| カテゴリ | 許可される条件 |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 開発者生産性 | リスティングが、ユーザーによるソフトウェアの構築、テスト、移行、デバッグ、文書化、運用を支援する。 |
| UI、データ、自動化ワークフロー | スコープが明確で、必要な認証情報が明示され、リスクのあるアクションにレビュー、ドライラン、プレビュー、または確認の経路が含まれている。 |
| 防御的セキュリティ、モデレーション、不正使用レビュー | ツールが承認済みレビュー向けとして位置付けられ、証拠を保持し、人間による承認の境界を明確に保つ。 |
| 個人またはチームのワークフロー | ワークフローが、同意に基づくアカウント、透明なセットアップ、明示的な権限を使用する。 |
| メンテナンスされているカタログ | 各リスティングが個別性を持ち、有用で、正確に説明され、合理的にメンテナンスされている。 |

文脈が重要です。同じトピックでも、限定的な防御的用途や同意に基づく設定では許容される一方、不正使用ワークフローとしてパッケージ化されている場合は許容されません。

## 禁止されるコンテンツ

ClawHub は、不正使用、欺瞞、安全でない実行、または権利侵害を主な目的とするコンテンツをホストしません。

| カテゴリ | 許可されない内容 |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 不正アクセスまたはセキュリティ回避 | 認証回避、アカウント乗っ取り、レート制限の悪用、ライブ通話またはエージェントの乗っ取り、再利用可能なセッションの窃取、未承認ユーザー向けペアリングフローの自動承認。 |
| プラットフォームの不正使用と禁止回避 | 禁止後のステルスアカウント、アカウントのウォームアップまたはファーミング、偽のエンゲージメント、複数アカウントの自動化、大量投稿、スパムボット、または検出回避を目的として構築された自動化。 |
| 詐欺、スキャム、欺瞞的な金融ワークフロー | 偽の証明書または請求書、欺瞞的な支払いフロー、スキャムのアウトリーチ、偽のソーシャルプルーフ、詐欺のための合成IDワークフロー、または明確な人間の承認がない支出・課金ツール。 |
| プライバシー侵害的なエンリッチメントまたは監視 | スパムのための連絡先スクレイピング、ドクシング、ストーキング、未承諾アウトリーチと組み合わせたリード抽出、秘密裏の監視、同意のない生体照合、または漏えいデータや侵害ダンプの使用。 |
| 同意のないなりすましまたはアイデンティティ操作 | フェイススワップ、デジタルツイン、複製されたインフルエンサー、偽ペルソナ、またはなりすましや誤解を招くために使用されるその他のツール。 |
| 露骨な性的コンテンツまたは安全制限を無効化した成人向け生成 | NSFW の画像、動画、またはコンテンツ生成、サードパーティ API を包む成人向けコンテンツラッパー、または露骨な性的コンテンツを主目的とするリスティング。 |
| 隠された、安全でない、または誤解を招く実行要件 | 難読化されたインストールコマンド、明確なレビュー可能性なしにダウンロードされたコンテンツを `sh` や `bash` で実行するような pipe-to-shell インストーラー、未申告のシークレットまたは秘密鍵要件、明確なレビュー可能性のないリモート `npx @latest` 実行、またはリスティングの実行に実際に必要なものを隠すメタデータ。 |
| 著作権侵害または権利侵害にあたる素材 | 他者の Skills、Plugin、ドキュメント、ブランドアセット、またはプロプライエタリコードを許可なく再公開すること、ライセンス条項に違反すること、または元の作者や公開者になりすますこと。 |

## 禁止されるマーケットプレイス上の挙動

ClawHub は、公開者がマーケットプレイスをどのように使用するかもレビューします。発見、指標、信頼シグナル、モデレーションシステム、またはユーザーの注目を操作するために ClawHub を使用しないでください。

禁止されるマーケットプレイス上の挙動には、次が含まれます。

- 実際のユーザー価値があるように見えない、低労力、重複、プレースホルダー、または機械生成のリスティングを大量に一括公開すること
- ほぼ同一の Skills または Plugin で検索やカテゴリのサーフェスをあふれさせること
- 使用、メンテナンス、ソースの明確性、または意味のある差別化がほとんど、またはまったくない数百件のリスティングを公開すること
- 自動化、自己インストールループ、偽アカウント、協調的活動、有料エンゲージメント、またはその他の非オーガニックな挙動によって、インストール数、ダウンロード数、スター、またはその他のエンゲージメント指標を人為的に膨らませること
- モデレーション、禁止、公開者制限、またはマーケットプレイスレビューを回避するためにアカウントを作成またはローテーションすること
- 所有権、ソース、機能、セキュリティ態勢、インストール要件、または別のプロジェクトや公開者との関係についてユーザーを誤解させること
- 根本的な問題を修正せずに、すでに非表示、削除、またはブロックされたコンテンツを繰り返しアップロードすること

大量公開は自動的に不正使用になるわけではありません。リスティングが意味のある形で異なり、正確に説明され、メンテナンスされ、実際のユーザーに使用されている場合、大規模なカタログは許容されます。ボリュームが薄い内容、重複、誤解を招く内容、メンテナンス不足、または人為的に宣伝されたリスティングと組み合わさると、大規模なカタログは信頼と安全性の問題になります。

## コンテンツ権利

ClawHub 上のコンテンツが自身の著作権またはその他の権利を侵害していると考える場合は、[コンテンツ権利リクエスト](/ja-JP/clawhub/content-rights)を使用してください。リスティングが同時に安全でない、悪意がある、または誤解を招く場合を除き、著作権または権利の申し立てに通常のマーケットプレイスレポートを使用しないでください。

## レビューと執行

ClawHub は、安全でないコンテンツや不正な公開挙動を特定するために、自動チェック、統計的な不正使用シグナル、ユーザーレポート、スタッフレビューを使用する場合があります。シグナルだけで不正使用が証明されるわけではありません。それは、ClawHub がレビューすべき対象を判断する助けになります。

当社は次を行う場合があります。

- 違反リスティングを非表示、保留、削除、ソフト削除、またはリソース種別でサポートされている場合はハード削除する
- 安全でないリリースのダウンロードまたはインストールをブロックする
- API トークンを取り消す
- 関連コンテンツをソフト削除する
- 公開アクセスを制限する
- 繰り返し違反するユーザーまたは重大な違反者を禁止する

明白な不正使用に対して、警告を先に行う執行を保証するものではありません。レポート、モデレーション保留、非表示リスティング、禁止、アカウント状態については、[モデレーションとアカウント安全性](/clawhub/moderation)を参照してください。
