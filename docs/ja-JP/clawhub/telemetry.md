# テレメトリ

ClawHub は **最小限のテレメトリ** を使用して **インストール数**（実際に使用されているもの）を算出し、より良い並べ替え/フィルタリングに活用します。
これは CLI の `clawhub sync` コマンドに基づきます。

## テレメトリが収集されるタイミング

テレメトリは次の場合にのみ送信されます。

- CLI で **ログイン済み** である（同期/公開フローではすでに認証を必須にしています）。
- `clawhub sync` を実行する。
- テレメトリが **無効化されていない**（下の「無効化する方法」を参照）。

ログインしていない場合、何も報告されません。

## 収集する内容

各 `clawhub sync` で、CLI は検出した内容の **完全なスナップショット** を、スキャンルート（「フォルダー/ルート」）ごとにグループ化して報告します。

各ルートについて、次を保存します。

- `rootId`: 正規化されたルートパスの **SHA-256 ハッシュ**（サーバーが生のパスを見ることはありません）。
- `label`: 最後の 2 つのパスセグメントから派生した、人間が読めるラベル（ホームパスは `~` 付きで表示されます）。
- `firstSeenAt`, `lastSeenAt`, 任意の `expiredAt`。

ルート配下で見つかった各 Skill について、次を保存します。

- `skillId`（slug によって解決されます。レジストリに存在する Skills のみ追跡されます）。
- `firstSeenAt`, `lastSeenAt`。
- `lastVersion`（ベストエフォート。現在は既知の場合、レジストリで一致したバージョン）。
- 以前に報告されたインストールがルートから消えた場合の任意の `removedAt`。

### 収集_しない_もの

- 生の絶対フォルダーパスは収集しません（ハッシュ化された `rootId` + 短い表示ラベルのみ）。
- ファイル内容は収集しません。
- 実行ごとのログ、プロンプト、その他の CLI 出力は収集しません。
- レジストリにアップロードされていない Skills は追跡しません（不明な slug は無視されます）。

## インストール数

Skill ごとに 2 つのカウンターを維持します。

- `installsCurrent`: 現在、少なくとも 1 つのアクティブなルートにその Skill をインストールしているユニークユーザー数。
- `installsAllTime`: これまでにその Skill がインストールされていると報告したことがあるユニークユーザー数。

### 複数のルート

複数のフォルダーから同期する場合、各スキャンルートを独立して扱います。Skill が **いずれか** のアクティブなルートに存在する場合、「現在インストール済み」とみなされます。

### アンインストール検出

`sync` はルートごとの完全なセットを報告するため、次のようになります。

- 次回の同期で Skill がルートから消えた場合、そのルートでは削除済みとしてマークします。
- Skill がすべてのルートから削除されると、`installsCurrent` にはカウントされなくなります。
- テレメトリを削除しない限り、`installsAllTime` は減少しません（下記参照）。

### 古さ（120 日）

**120 日** にわたってテレメトリを報告しないルートは古いものとしてマークされ、そのインストールは `installsCurrent` にカウントされなくなります。
これはバックグラウンドジョブを避けるため、遅延評価されます（次回のテレメトリ報告時）。

## 透明性 + ユーザー制御

ClawHub は、あなた自身のプロフィールに非公開の「インストール済み」タブを提供します。

- 保存している正確なルート + インストール済み Skills を表示します。
- **JSON エクスポート** ビューを含みます。
- アカウントに保存されたすべてのテレメトリを削除する **テレメトリを削除** アクションを含みます。

他の全員に見えるのは **集計されたインストールカウンター** のみです。他の誰も、あなたのルート/フォルダーを見ることはできません。

アカウントを削除すると、テレメトリデータも削除されます。

## テレメトリを無効化する方法

環境変数を設定します。

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

これを設定すると、CLI は `clawhub sync` 中にテレメトリを送信しません。
