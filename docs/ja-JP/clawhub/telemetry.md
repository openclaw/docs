# テレメトリ

ClawHub は **最小限のテレメトリ** を使用して **インストール数**（実際に使われているもの）を計算し、より良い並べ替え/フィルタリングを実現します。
これは CLI の `clawhub sync` コマンドに基づいています。

## テレメトリが収集されるタイミング

テレメトリは次の場合にのみ送信されます。

- CLI で **ログイン済み** である（sync/publish フローではすでに認証を必須にしています）。
- `clawhub sync` を実行する。
- テレメトリが **無効化されていない**（下の「無効化する方法」を参照）。

ログインしていない場合、何も報告されません。

## 収集する内容

各 `clawhub sync` で、CLI は検出した内容の **完全なスナップショット** を、スキャンルート（「フォルダー/ルート」）ごとにグループ化して報告します。

各ルートについて保存する内容:

- `rootId`: 正規化されたルートパスの **SHA-256 ハッシュ**（サーバーが生のパスを見ることはありません）。
- `label`: 最後の 2 つのパスセグメントから派生した、人間が読めるラベル（ホームパスは `~` で表示されます）。
- `firstSeenAt`、`lastSeenAt`、任意の `expiredAt`。

ルート配下で検出された各 skill について保存する内容:

- `skillId`（slug で解決。レジストリに存在する skill のみ追跡されます）。
- `firstSeenAt`、`lastSeenAt`。
- `lastVersion`（ベストエフォート。現在は既知の場合、レジストリで一致したバージョン）。
- 以前に報告されたインストールがルートから消えた場合の任意の `removedAt`。

### 収集し_ない_内容

- 生の絶対フォルダーパスはありません（ハッシュ化された `rootId` + 短い表示ラベルのみ）。
- ファイル内容はありません。
- 実行ごとのログ、プロンプト、その他の CLI 出力はありません。
- レジストリにアップロードされていない skill は追跡されません（不明な slug は無視されます）。

## インストール数

skill ごとに 2 つのカウンターを保持します。

- `installsCurrent`: 少なくとも 1 つのアクティブなルートに現在その skill をインストールしているユニークユーザー。
- `installsAllTime`: これまでにその skill のインストールを報告したことがあるユニークユーザー。

### 複数のルート

複数のフォルダーから sync する場合、各スキャンルートを独立して扱います。skill は **いずれか** のアクティブなルートに存在する場合、「現在インストール済み」と見なされます。

### アンインストール検出

`sync` はルートごとの完全なセットを報告するため:

- 次回の sync で skill がルートから消えた場合、そのルートでは削除済みとしてマークします。
- その skill がすべてのルートから削除されると、`installsCurrent` にはカウントされなくなります。
- `installsAllTime` は、テレメトリを削除しない限り減りません（下記参照）。

### 古さ（120 日）

**120 日間** テレメトリを報告しないルートは stale としてマークされ、そのインストールは `installsCurrent` にカウントされなくなります。
これはバックグラウンドジョブを避けるため、遅延評価されます（次回のテレメトリ報告時）。

## 透明性 + ユーザーコントロール

ClawHub は、自分のプロフィールに非公開の「インストール済み」タブを提供します。

- 保存している正確なルート + インストール済み skill を表示します。
- **JSON エクスポート** ビューを含みます。
- アカウントに保存されているすべてのテレメトリを削除するための **テレメトリを削除** アクションを含みます。

他の全員に表示されるのは **集計されたインストールカウンター** のみです。他の誰もあなたのルート/フォルダーを見ることはできません。

アカウントを削除すると、テレメトリデータも削除されます。

## テレメトリを無効化する方法

環境変数を設定します。

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

これを設定すると、CLI は `clawhub sync` 中にテレメトリを送信しません。
