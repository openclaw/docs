# Org と名前空間の申し立て

ClawHub は、オーナーハンドル、org ハンドル、skill slug、Plugin パッケージ名、および
パッケージスコープを公開名前空間として使用します。名前空間が
実在するプロジェクト、ブランド、パッケージエコシステム、または組織に属しているように見えるものの、
ClawHub 上ですでに申し立て済み、予約済み、誤解を招く、または係争中である場合は、
[Org / 名前空間の申し立て issue フォーム](https://github.com/openclaw/clawhub/issues/new?template=org-namespace-claim.yml)で
スタッフにレビューを依頼してください。

公開の、機密ではない所有権レビューにはこの手順を使用してください。名前空間の申し立てには、
製品内レポートやアカウント異議申し立てフォームを使用しないでください。

## 申し立てを開くタイミング

実世界の所有権を理由に、名前空間を予約、移転、名前変更、非表示、隔離、エイリアス化、
またはその他の方法で変更すべきかどうかを ClawHub スタッフがレビューする必要があると考える場合は、
名前空間の申し立てを開いてください。

例:

- GitHub org、プロジェクト、会社、またはコミュニティと一致する org ハンドル
- 一致する ClawHub オーナーの下でのみ公開されるべき `@example-org/*` のようなパッケージスコープ
- プロジェクトになりすましているように見える skill slug または Plugin パッケージ名
- ブランド、商標、プロジェクト名変更、またはパッケージ履歴に関する係争
- 正当な名前空間オーナーを妨げている、削除済み、非アクティブ、または連絡不能なオーナー

掲載が所有権の係争を超えて安全でない、悪意がある、または誤解を招く場合は、
関連するモデレーションまたはセキュリティのガイダンスにも従ってください。名前空間申し立てフォームは
所有権レビューのためのものであり、緊急の脆弱性開示のためのものではありません。

## 提出前に

まず、名前空間と一致するオーナーで公開していることを確認してください。
Plugin パッケージの場合、`@example-org/example-plugin` のようなスコープ付き名前は、
一致する `example-org` オーナーとして公開する必要があります。

現在のオーナーを管理できる場合は、影響を受けるリソースを公開、名前変更、移転、非表示、
または削除して、名前空間を直接修正してください。現在のオーナーを管理できない場合、
またはスタッフが係争を解決する必要がある場合に、申し立てを使用してください。

## 含める証拠

公開可能で機密ではない証拠を使用してください。役立つ証拠には以下があります。

- GitHub org、リポジトリ、リリース、またはメンテナー履歴
- 名前空間を明記している公式プロジェクトドキュメント
- ドメインまたは公式メールドメインの証明
- npm、PyPI、crates.io、またはその他のパッケージレジストリのスコープ管理
- 公開で議論しても安全な、商標、ブランド、またはプロジェクト所有権の証拠
- ソースリポジトリ履歴、パッケージ履歴、または公開された名前変更通知
- 係争中の ClawHub オーナー、skill、Plugin、パッケージ、または issue へのリンク

各リンクが何を証明しているかを説明してください。スタッフが
秘密の認証情報やシークレットを必要とせずに関係を理解できるようにしてください。

## 含めてはいけないもの

公開 GitHub issue にシークレットや非公開の証拠を載せないでください。以下を含めないでください。

- API トークン、署名キー、または認証情報
- DNS チャレンジトークン
- 非公開の法的ファイルまたは契約
- 個人の身分証明書
- 非公開メール、非公開セキュリティレポート、または機密の顧客データ

申し立てフォームでは、機密証拠に非公開のスタッフチャネルが必要かどうかを尋ねます。
機密資料を公開投稿する代わりに、そのオプションを使用してください。

## 想定される結果

証拠とリスクに応じて、ClawHub スタッフは名前空間の予約、所有権の移転、
リソースの名前変更、既存掲載の非表示または隔離、エイリアスまたはリダイレクトの追加、
追加証拠の依頼、またはリクエストの却下を行う場合があります。

名前空間レビューは、一致するすべての名前が移転されることを保証するものではありません。
スタッフは公開証拠、既存の使用状況、セキュリティリスク、およびユーザーへの影響を考慮します。
