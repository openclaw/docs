# 組織と名前空間の申し立て

ClawHub は、owner ハンドル、org ハンドル、skill slug、Plugin パッケージ名、
パッケージスコープを公開名前空間として使用します。名前空間が実在する
プロジェクト、ブランド、パッケージエコシステム、または組織に属しているように見える一方で、
ClawHub 上ですでに申し立て済み、予約済み、誤解を招く状態、または係争中の場合は、
[組織 / 名前空間の申し立て issue フォーム](https://github.com/openclaw/clawhub/issues/new?template=org-namespace-claim.yml)
でスタッフにレビューを依頼してください。

公開の非機密な所有権レビューには、この経路を使用してください。名前空間の申し立てに、
製品内レポートやアカウント異議申し立てフォームを使用しないでください。

## 申し立てを開くタイミング

実世界の所有権を理由として、名前空間を予約、移転、改名、非表示、隔離、エイリアス化、
またはその他の形で変更すべきかどうかを ClawHub スタッフがレビューすべきだと考える場合は、
名前空間の申し立てを開いてください。

例:

- あなたの GitHub org、プロジェクト、会社、またはコミュニティと一致する org ハンドル
- 一致する ClawHub owner の下でのみ公開されるべき `@example-org/*` のようなパッケージスコープ
- プロジェクトになりすましているように見える skill slug または Plugin パッケージ名
- ブランド、商標、プロジェクト名変更、またはパッケージ履歴に関する係争
- 正当な名前空間 owner をブロックしている、削除済み、非アクティブ、または連絡不能な owner

リストが所有権の係争を超えて危険、悪意がある、または誤解を招く場合は、
関連するモデレーションまたはセキュリティのガイダンスにも従ってください。名前空間申し立てフォームは
所有権レビュー用であり、緊急の脆弱性開示用ではありません。

## 提出前に

まず、その名前空間と一致する owner で公開していることを確認してください。
Plugin パッケージの場合、`@example-org/example-plugin` のようなスコープ付き名前は、
一致する `example-org` owner として公開する必要があります。

現在の owner を管理できる場合は、影響を受けるリソースを公開、改名、移転、
非表示、または削除して、名前空間を直接修正してください。現在の owner を管理できない場合、
またはスタッフが係争を解決する必要がある場合に申し立てを使用してください。

## 含める証拠

公開の非機密な証拠を使用してください。有用な証拠には次のものがあります。

- GitHub org、リポジトリ、リリース、またはメンテナー履歴
- 名前空間を明示している公式プロジェクトドキュメント
- ドメインまたは公式メールドメインの証明
- npm、PyPI、crates.io、またはその他のパッケージレジストリのスコープ管理
- 公開で議論しても安全な、商標、ブランド、またはプロジェクト所有権の証拠
- ソースリポジトリ履歴、パッケージ履歴、または公開された改名通知
- 係争中の ClawHub owner、skill、Plugin、パッケージ、または issue へのリンク

各リンクが何を証明するかを説明してください。スタッフが非公開の認証情報やシークレットを必要とせずに、
関係を理解できるようにしてください。

## 含めてはいけないもの

シークレットや非公開の証拠を公開 GitHub issue に載せないでください。次のものを含めないでください。

- API トークン、署名キー、または認証情報
- DNS チャレンジトークン
- 非公開の法的ファイルまたは契約書
- 個人の本人確認書類
- 非公開メール、非公開セキュリティレポート、または機密の顧客データ

申し立てフォームでは、機密証拠に非公開のスタッフチャネルが必要かどうかを尋ねます。
機密資料を公開投稿する代わりに、その選択肢を使用してください。

## 想定される結果

証拠とリスクに応じて、ClawHub スタッフは名前空間の予約、所有権の移転、
リソースの改名、既存リストの非表示または隔離、エイリアスまたはリダイレクトの追加、
追加証拠の依頼、またはリクエストの却下を行う場合があります。

名前空間レビューは、一致するすべての名前が移転されることを保証するものではありません。
スタッフは公開証拠、既存の使用状況、セキュリティリスク、ユーザーへの影響を考慮します。
