# 組織と名前空間の申し立て

ClawHub は、所有者ハンドル、組織ハンドル、skill スラッグ、Plugin パッケージ名、および
パッケージスコープを公開名前空間として使用します。名前空間が
実在するプロジェクト、ブランド、パッケージエコシステム、または組織に属しているように見えるものの、
ClawHub 上ですでに取得済み、予約済み、誤解を招く、または係争中である場合は、
[組織 / 名前空間申し立て Issue フォーム](https://github.com/openclaw/clawhub/issues/new?template=org-namespace-claim.yml)
でスタッフにレビューを依頼してください。

公開されてもよい、機密性のない所有権レビューにはこの手順を使用してください。名前空間の申し立てに、
製品内レポートやアカウント異議申し立てフォームを使用しないでください。

## 申し立てを開くタイミング

実世界の所有権を理由として、名前空間を予約、移管、名前変更、非表示、隔離、エイリアス化、
またはその他の方法で変更すべきかどうかを ClawHub スタッフがレビューする必要があると考える場合は、
名前空間の申し立てを開いてください。

例:

- GitHub の組織、プロジェクト、会社、またはコミュニティと一致する組織ハンドル
- `@example-org/*` のように、一致する ClawHub 所有者の下でのみ公開されるべき
  パッケージスコープ
- プロジェクトになりすましているように見える skill スラッグまたは Plugin パッケージ名
- ブランド、商標、プロジェクト名の変更、またはパッケージ履歴に関する係争
- 正当な名前空間の所有者を妨げている、削除済み、非アクティブ、または連絡不能な所有者

掲載内容が、所有権の係争を超えて危険、悪意がある、または誤解を招く場合は、
関連するモデレーションまたはセキュリティのガイダンスにも従ってください。名前空間申し立てフォームは
所有権レビュー用であり、緊急の脆弱性開示用ではありません。

## 提出前に

まず、その名前空間と一致する所有者で公開していることを確認してください。
Plugin パッケージでは、`@example-org/example-plugin` のようなスコープ付き名前は、
一致する `example-org` 所有者として公開する必要があります。

現在の所有者を管理できる場合は、影響を受けるリソースを公開、名前変更、移管、非表示、または削除して、
名前空間を直接修正してください。現在の所有者を管理できない場合、またはスタッフが
係争を解決する必要がある場合は、申し立てを使用してください。

## 含める証拠

公開されてもよい、機密性のない証拠を使用してください。有用な証拠には次のものがあります。

- GitHub の組織、リポジトリ、リリース、またはメンテナー履歴
- 名前空間を明記している公式プロジェクトドキュメント
- ドメインまたは公式メールドメインの証明
- npm、PyPI、crates.io、またはその他のパッケージレジストリのスコープ管理
- 公開の場で議論しても安全な、商標、ブランド、またはプロジェクト所有権の証拠
- ソースリポジトリ履歴、パッケージ履歴、または公開された名前変更通知
- 係争中の ClawHub 所有者、skill、Plugin、パッケージ、または Issue へのリンク

各リンクが何を証明するかを説明してください。スタッフが、非公開の認証情報やシークレットを必要とせずに
関係を理解できる必要があります。

## 含めてはいけないもの

公開 GitHub Issue にシークレットや非公開の証拠を載せないでください。次のものを含めないでください。

- API トークン、署名鍵、または認証情報
- DNS チャレンジトークン
- 非公開の法的文書または契約
- 個人の本人確認書類
- 非公開メール、非公開セキュリティレポート、または機密の顧客データ

申し立てフォームでは、機密性の高い証拠に非公開のスタッフチャネルが必要かどうかを尋ねます。
機密性の高い資料を公開投稿する代わりに、その選択肢を使用してください。

## 想定される結果

証拠とリスクに応じて、ClawHub スタッフは、名前空間の予約、所有権の移管、
リソースの名前変更、既存掲載の非表示または隔離、エイリアスまたはリダイレクトの追加、
追加証拠の依頼、またはリクエストの却下を行う場合があります。

名前空間レビューは、一致するすべての名前が移管されることを保証するものではありません。
スタッフは、公開証拠、既存の使用状況、セキュリティリスク、およびユーザーへの影響を考慮します。
