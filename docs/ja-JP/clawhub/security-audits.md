# セキュリティ監査

ClawHub のセキュリティ監査は、Skill または Plugin がインストールしても十分に安全かどうかを判断するのに役立ちます。監査では、リリースが何を行うか、どのような権限を要求するか、ファイル、アカウント、認証情報、コード、外部サービスへのアクセスを許可する前に特に注意すべき点があるかを示します。

監査は安全性を判断するうえで有力な指標ですが、リリースにリスクがないことを保証するものではありません。機密性の高いアクセスを許可する前に、必ず自身で判断してください。

[セキュリティ](/clawhub/security)、[許容される使用方法](/ja-JP/clawhub/acceptable-usage)、[モデレーションとアカウントの安全性](/clawhub/moderation)も参照してください。

## インストール前に確認すること

インストールする前に、以下を確認してください。

- 監査ステータス全体
- リスクレベル
- 記載されている所見
- 必要な認証情報、権限、環境変数
- 所有者、ソース、バージョン、変更履歴、ダウンロード数、スター数、その他の信頼性の指標

内容を理解し、信頼できるものだけをインストールしてください。

## 監査ステータス

監査ステータスは、監査結果にどのように対応すべきかを示します。

| ステータス  | 意味                                                                      |
| ----------- | ------------------------------------------------------------------------- |
| `Pass`      | 低リスクを超える明らかな問題は見つかりませんでした。                      |
| `Review`    | インストール前に所見を確認してください。正当なリリースである可能性はあります。 |
| `Warn`      | 特に注意してください。ClawHub が影響の大きい懸念事項または警告シグナルを検出しました。 |
| `Malicious` | インストールしないでください。                                            |
| `Pending`   | 監査はまだ完了していません。                                              |
| `Error`     | 監査を完了できませんでした。                                              |

`Pass` は安心材料ですが、自身の判断に代わるものではありません。コンテンツの公開、データの編集、コマンドの実行、ファイルの読み取り、本番システムへのアクセスが可能なツールでは、特に重要です。

## リスクレベル

リスクレベルは影響範囲、つまりリリースを意図どおりに使用した場合に、どの程度の権限を持つと考えられるかを示します。

| リスクレベル | 意味                                                                            |
| ------------ | ------------------------------------------------------------------------------- |
| `Low`        | 機密性の高い権限やユーザーへの影響はほとんど見つかりませんでした。              |
| `Medium`     | アカウントへのアクセスやデータの変更など、重要な権限を持つリリースです。        |
| `High`       | 影響の大きい権限、重大な所見、または悪意を示すシグナルがあるリリースです。      |

リスクレベルと監査ステータスは、それぞれ異なる問いに答えます。

- リスクレベルの問い：「ここにはどの程度の権限があるか？」
- 監査ステータスの問い：「この結果を受けて何をすべきか？」

たとえば、公開用の Skill が `Medium` リスクで `Review` と表示される場合があります。これは悪意があるという意味ではありません。その Skill は目的に沿っていると考えられるものの、重要なアカウント権限を使用して操作できるという意味です。

## 所見

所見は、監査結果が表示された理由を説明します。通常、各所見には以下が含まれます。

- それが何を意味するか
- フラグが付けられた理由
- 関連する Skill または Plugin の内容
- 推奨事項

所見には `Info`、`Low`、`Medium`、`High`、`Critical` のラベルが付くことがあります。重大度の高い所見ほど、リスクレベルと監査ステータスに強く影響します。

信頼度の低い所見は、ページを有用な証拠に集中させるため、公開される監査の集計には表示されません。

## ClawHub が確認する内容

ClawHub は、提出されたリリース成果物の以下の項目を監査します。

- Skill の指示または Plugin のメタデータ
- 宣言された環境変数と権限
- インストール手順とパッケージのメタデータ
- 含まれるファイルとファイルマニフェスト
- 互換性と機能のメタデータ

主な観点は一貫性です。名前、概要、メタデータ、要求される権限、実際の内容が、ユーザーが合理的に期待するものと一致しているかを確認します。

強力な動作が自動的に悪いと判断されるわけではありません。多くの有用なツールには、認証情報、ローカルコマンド、プロバイダー API、パッケージのインストールが必要です。監査では、その権限が想定どおりで、開示され、目的に対して相応であるかを確認します。

成果物のページには、以下の完全な監査へのリンクがあります。

```text
/<owner>/skills/<slug>/security-audit
```

監査ページには以下がまとめられています。

1. SkillSpector
2. VirusTotal
3. リスク分析

## VirusTotal

ClawHub は、監査スタックでマルウェアのテレメトリとして VirusTotal を使用しています。VirusTotal はファイルの評価とマルウェアスキャンに関する信頼された業界標準であり、この提携により、ClawHub は Skill と Plugin のレビューに、より幅広いセキュリティインテリジェンスを追加できます。

VirusTotal は、既知の悪意ある成果物、スキャンエンジンによる検出、ClawHub のエージェントを考慮したレビューを補完する評価シグナルに特に有用です。ベンダーエンジンの件数が利用可能な場合、監査では以下のように平易な表現で要約します。

```text
62/62 のベンダーがこの Skill を安全と判定しました。
```

または：

```text
2/64 のベンダーがこの Skill を悪意あり、1/64 が疑わしい、61/64 が安全と判定しました。
```

ClawHub に要約可能なベンダー件数のテレメトリがない場合、監査には以下のように表示されます。

```text
VirusTotal の所見はありません
```

VirusTotal はあくまでテレメトリです。ClawHub 独自の成果物を考慮したリスク分析に代わるものではありません。

## リスク分析

リスク分析は、ClawHub 独自のセキュリティ監査システムである ClawScan によって内部的に実行されます。各リリースをエージェント向けの成果物として、指示、メタデータ、宣言された権限、ファイル、機能シグナル、静的スキャンシグナル、SkillSpector の所見、VirusTotal のテレメトリ、公開者が提供したコンテキストを確認します。静的スキャンシグナルはこのレビューの内部コンテキストであり、単独で公開される監査セクションでも、インストールをブロックする判定でもありません。

リスク分析では、
[OWASP Agentic Skills Top 10](https://owasp.org/www-project-agentic-skills-top-10/)
を、プロンプトインジェクション、ツールの誤用、認証情報の漏えい、安全でない実行、メモリまたはコンテキストの汚染、過剰な自律性などのリスクを検討する観点として使用します。

ClawScan は、一見危険に見える機能を自動的に悪意ありとは判断しません。その機能が開示され、目的に沿っており、リリースに記載されたユースケースによって裏付けられているかを確認します。
