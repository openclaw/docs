# コンテンツ権利リクエスト

ClawHub に公開されたコンテンツが、自分の著作権またはその他の権利を侵害していると考える場合は、[ClawHub コンテンツ権利リクエスト](https://forms.openclaw.ai/clawhub-content-rights)を送信してください。

以下を含めてください。

- 1つ以上の正確な `https://clawhub.ai/<owner>/skills/<skill>` URL
- 氏名、組織、連絡先メールアドレス
- 権利に関する懸念の簡潔な説明
- 裏付けとなる証拠（ある場合）

ClawHub スタッフがリクエストを手動で確認し、追加情報を得るためにリクエスト送信者または公開者に連絡することがあります。状況に応じて、対象コンテンツは非表示、復元、または変更なしとなる場合があります。

コンテンツ権利に関する懸念ではない安全でないマーケットプレイスコンテンツについては、[モデレーションとアカウント安全性](/clawhub/moderation)で説明されている通常の報告フローを使用してください。
ClawHub 自体の脆弱性については、[セキュリティ](/ja-JP/clawhub/security)を参照してください。
