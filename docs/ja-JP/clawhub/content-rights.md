# コンテンツ権利リクエスト

ClawHub に公開されているコンテンツが、自分の著作権またはその他の
権利を侵害していると思われる場合は、[ClawHub コンテンツ権利リクエスト](https://forms.openclaw.ai/clawhub-content-rights)を送信してください。

含める内容:

- 1つ以上の正確な `https://clawhub.ai/<owner>/skills/<skill>` URL
- 氏名、組織、連絡先メールアドレス
- 権利上の懸念についての簡潔な説明
- 入手可能な場合は裏付けとなる証拠

ClawHub スタッフはリクエストを手動で確認し、追加情報のためにリクエスト送信者または
公開者に連絡する場合があります。状況に応じて、影響を受ける
コンテンツは非表示、復元、または変更なしのままになる場合があります。

コンテンツ権利上の懸念ではない安全でないマーケットプレイスコンテンツについては、
[モデレーションとアカウントの安全性](/clawhub/moderation)で説明されている通常の報告フローを使用してください。
ClawHub 自体の脆弱性については、[セキュリティ](/ja-JP/clawhub/security)を参照してください。
