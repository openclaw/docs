# コンテンツ権利リクエスト

ClawHub に公開されているコンテンツが著作権またはその他の権利を侵害していると思われる場合は、[ClawHub コンテンツ権利リクエスト](https://forms.openclaw.ai/clawhub-content-rights)を送信してください。

含める内容:

- 1つ以上の正確な `https://clawhub.ai/<owner>/skills/<skill>` URL
- 氏名、組織、連絡先メールアドレス
- 権利に関する懸念の簡潔な説明
- 利用可能な場合は、裏付けとなる証拠

ClawHub スタッフはリクエストを手動で確認し、詳細情報を得るためにリクエスト送信者または公開者に連絡する場合があります。状況に応じて、影響を受けるコンテンツは非表示、復元、または変更なしのままとなる場合があります。

コンテンツ権利に関する懸念ではない安全でないマーケットプレイスコンテンツについては、[モデレーションとアカウントの安全性](/clawhub/moderation)で説明されている通常の報告フローを使用してください。
ClawHub 自体の脆弱性については、[セキュリティ](/ja-JP/clawhub/security)を参照してください。
