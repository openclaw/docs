# コンテンツ権利リクエスト

ClawHub で公開されているコンテンツがあなたの著作権またはその他の
権利を侵害していると思われる場合は、[ClawHubコンテンツ権利リクエスト](https://forms.openclaw.ai/clawhub-content-rights)を送信してください。

含める内容:

- 1つ以上の正確な `https://clawhub.ai/<owner>/skills/<skill>` URL
- あなたの名前、組織、連絡先メールアドレス
- 権利に関する懸念の簡単な説明
- 入手可能な場合は、裏付けとなる証拠

ClawHub スタッフはリクエストを手動で確認し、詳細情報を得るためにリクエスト送信者または
公開者に連絡する場合があります。状況に応じて、影響を受ける
コンテンツは非表示、復元、または変更なしのままになることがあります。

コンテンツ権利に関する懸念ではない安全でないマーケットプレイスコンテンツについては、[モデレーションとアカウントの安全性](/clawhub/moderation)で説明されている
通常の報告フローを使用してください。
ClawHub 自体の脆弱性については、[セキュリティ](/ja-JP/clawhub/security)を参照してください。
