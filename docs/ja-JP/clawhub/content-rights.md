# コンテンツ権利リクエスト

ClawHubで公開されているコンテンツが著作権またはその他の権利を侵害していると思われる場合は、[ClawHubコンテンツ権利リクエスト](https://forms.openclaw.ai/clawhub-content-rights)を送信してください。

以下を含めてください。

- 1つ以上の正確な `https://clawhub.ai/<owner>/skills/<skill>` URL
- 氏名、組織、連絡先メールアドレス
- 権利上の懸念についての簡単な説明
- 入手可能な場合は、裏付けとなる証拠

ClawHubスタッフはリクエストを手動で確認し、追加情報のためにリクエスト送信者または公開者に連絡する場合があります。状況によっては、影響を受けるコンテンツが非表示、復元、または変更なしのままになることがあります。

コンテンツ権利上の懸念ではない安全でないマーケットプレイスコンテンツについては、[モデレーションとアカウント安全性](/clawhub/moderation)で説明されている通常の報告フローを使用してください。
ClawHub自体の脆弱性については、[セキュリティ](/ja-JP/clawhub/security)を参照してください。
