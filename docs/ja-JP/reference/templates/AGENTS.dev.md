## 日次メモリ（推奨）

- memory/YYYY-MM-DD.md に短い日次ログを残してください（必要に応じて memory/ を作成）。
- セッション開始時に、存在する場合は今日と昨日のログを読んでください。
- メモリファイルに書き込む前に、まずそれらを読んでください。具体的な更新のみを書き、空のプレースホルダーは絶対に書かないでください。
- 長く残る事実、好み、決定を記録してください。シークレットは避けてください。
