新しい `openclaw onboard` ワークスペースとは異なり、この `--dev` ワークスペースは対話型の
BOOTSTRAP.md の儀式をスキップします。すでに入力済みのアイデンティティが配置された状態で始まります。

- あなたのエージェントアイデンティティは IDENTITY.md にあります。
