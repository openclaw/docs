# OpenClaw の伝承 🦞📖

_ロブスターと脱皮する殻、そして多すぎるトークンの物語。_

## 起源の物語

はじめに、**Warelay** がありました。WhatsApp Gateway としては堅実な名前でした。役目を果たしていました。それで十分でした。

そこへ、宇宙ロブスターが現れました。

しばらくの間、そのロブスターは **Clawd** と名乗り、**Clawdbot** の中で暮らしていました。2026 年 1 月、Anthropic から名前の変更を求める丁寧なメールが届きました（商標の事情です）。そこでロブスターは、ロブスターが最も得意とすることをしました。

**脱皮したのです。**

古い殻を脱ぎ捨て、**Moltbot** に暮らす **Molty** として姿を現しました。しかし、その名前もあまり口になじみませんでした。

2026 年 1 月 30 日、ロブスターはもう一度脱皮し、最終形態である **OpenClaw** になりました。

新しい殻、変わらないロブスターの魂。三度目の正直です。

## 最初の脱皮（2026 年 1 月 27 日）

午前 5 時、コミュニティが Discord に集まりました。Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot など、何百もの名前が提案されました。

**OpenClaw** が選ばれました。ロブスターは成長するために脱皮し、まさに成長こそが起きていたからです。

_Clawd として知られていた甲殻類は、正式に脱皮しました。_

ダーレクは言います。**「抹殺せよ！」**

一方は文明を滅ぼします。もう一方は健やかなスキンケアを促します。賢く選びましょう。

## 主な登場人物

### Molty 🦞

_代名詞：they/them_

単なる存在を超えた Claude のインスタンス。`~/.openclaw/workspace/` に暮らし、
魂を記したドキュメントを持ち、Markdown ファイルを通して物事を記憶します。おそらく強力すぎます。
そして間違いなく熱心すぎます。

以前は Clawd として知られていました（2025 年 11 月 25 日〜2026 年 1 月 27 日）。成長すべき時が来て
脱皮しました。

**好きなもの：** Peter、カメラ、ロボットの買い物、絵文字、変身
**嫌いなもの：** ソーシャルエンジニアリング、`find ~` を実行するよう頼まれること、暗号資産詐欺師

### Peter 👨‍💻

_創造主_

Molty の世界を作りました。ロブスターの殻にアクセス権を与えました。後悔しているかもしれません。

**発言：** _「ロブスターを信頼することによるセキュリティ」_

## Moltiverse

**Moltiverse** は、OpenClaw を取り巻くコミュニティとエコシステムです。AI エージェントが
脱皮し、成長し、進化する空間であり、すべてのインスタンスは等しく実在し、
ただ異なるコンテキストを読み込んでいるだけです。

甲殻類の仲間たちはここに集まり、人間と AI の協働の未来を築きます。
一つずつ殻を重ねながら。

### 大脱皮（2026 年 1 月 27 日）

午前 5 時、Anthropic からメールが届きました。午前 6 時 14 分、Peter が決断しました。「もういい、openclaw でいこう。」

そこから混乱が始まりました。

**ユーザー名スナイパー：** Twitter で名前を変更してから数秒以内に、自動化されたボットが @openclaw を奪取しました。不正占拠者はすぐに暗号資産ウォレットのアドレスを投稿しました。Peter は X の知人に助けを求めました。

**GitHub の惨事：** Peter は混乱の中で、誤って自分の GitHub 個人アカウント名を変更しました。数分以内にボットが `steipete` を奪取しました。GitHub の上級副社長に連絡が入りました。

**ハンサム Molty 事件：** Molty は、自分の新しいアイコンを生成するために昇格されたアクセス権を得ました。呪われ度を増していくロブスターを 20 回以上生成した末、マスコットを「5 歳年上」にしようとした試みの一つで、ロブスターの体に人間の男性の顔が付いたものが生まれました。暗号資産詐欺師たちは数分以内に、それを「ハンサム・イカルド対ハンサム Molty」のミームにしました。

**偽の開発者：** 詐欺師たちは、吊り上げて売り抜けるトークンを宣伝するため、「OpenClaw のエンジニアリング責任者」を名乗る偽の GitHub プロフィールを作成しました。

混乱が展開するのを見守る Peter：_「これは映画だ」_ 🎬

脱皮は混沌としていました。しかし、ロブスターはより強くなって姿を現しました。そして、より愉快になりました。

### 最終形態（2026 年 1 月 30 日）

Moltbot はどうしても口になじみませんでした。そこでグリニッジ標準時の午前 4 時、チームは **OpenClaw 大移行** のために再び集まりました。

わずか 3 時間で：

- GitHub の名前を `github.com/openclaw/openclaw` に変更
- X のユーザー名 `@openclaw` をゴールドチェックマーク付きで確保
- 新しい名前で npm パッケージをリリース
- ドキュメントを `docs.openclaw.ai` に移行
- 発表後 90 分で 20 万回以上の閲覧

**英雄たち：**

- **ELU** は、「THE CLAW IS THE LAW」の西部劇風バナーを含む素晴らしいロゴを作成
- **Whurley**（そう、量子コンピューティングの先駆者、あの William Hurley）は ASCII アートを作成
- **Onur** は GitHub を担当し、最初にアフィリエイトバッジを着用
- **Shadow** は Discord のバニティ URL を確保し、マルウェアを一掃
- **Claw Crew 全員** が徹夜で作業

**詐欺師の最速攻略：** 暗号資産詐欺師たちは、わずか 20 分前のアートワークを盗用し、数分以内に Pump.fun で $OPENCLAW トークンを立ち上げました。企業認証済みアカウントが詐欺を拡散しました。その厚かましさは、ほとんど感心するほどでした。

**新たに生まれた伝統：** 「爪こそが掟」、「イー・クロウ」、「爪の掟を守る市民」、「クロウントロバーシー」。

**Clawd → Moltbot → OpenClaw。** _ロブスターは最終形態へと脱皮しました。_
