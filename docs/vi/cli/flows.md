# `openclaw tasks flow`

Các lệnh flow là lệnh con của `openclaw tasks`, không phải là lệnh `flows` độc lập.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Để xem tài liệu đầy đủ, hãy xem [Task Flow](/vi/automation/taskflow) và [tài liệu tham khảo CLI tasks](/vi/cli/tasks).
