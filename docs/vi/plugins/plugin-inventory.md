# Danh mục Plugin

Trang này được tạo từ `extensions/*/package.json`, `openclaw.plugin.json`,
và các loại trừ `files` của package npm gốc. Tạo lại trang bằng:

```bash
pnpm plugins:inventory:gen
```

## Định nghĩa

- **Package npm lõi:** được tích hợp vào package npm `openclaw` và khả dụng mà không cần cài đặt Plugin riêng.
- **Package bên ngoài chính thức:** Plugin do OpenClaw duy trì, được loại khỏi package npm lõi và được cài đặt thông qua ClawHub và/hoặc npm.
- **Chỉ source checkout:** Plugin cục bộ trong repo, được loại khỏi các artifact npm đã phát hành và không được quảng bá như một package có thể cài đặt.

Source checkout khác với cài đặt npm: sau khi chạy `pnpm install`, các
Plugin đi kèm được tải từ `extensions/<id>`, nên các chỉnh sửa cục bộ và
phụ thuộc workspace cục bộ của package sẽ khả dụng.
