# Tham chiếu Plugin

Trang này được tạo từ `extensions/*/package.json` và
`openclaw.plugin.json`. Tạo lại trang bằng:

```bash
pnpm plugins:inventory:gen
```

Sử dụng [Danh mục Plugin](/vi/plugins/plugin-inventory) để duyệt toàn bộ 143
trang tham chiếu Plugin đã tạo theo bản phân phối, gói và mô tả.
