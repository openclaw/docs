# Plugin gói ngôn ngữ Diffs

Thêm tô sáng cú pháp cho các ngôn ngữ nằm ngoài bộ trình xem diffs mặc định.

- Gói: `@openclaw/diffs-language-pack`
- Tuyến cài đặt: npm; ClawHub: `clawhub:@openclaw/diffs-language-pack`

## Ngôn ngữ đã thêm

Plugin `diffs` cơ sở đã tô sáng các ngôn ngữ phổ biến được ghi trong [Diffs](/vi/tools/diffs). Cài đặt gói ngôn ngữ này khi bạn muốn tô sáng cú pháp cho một tập hợp ngôn ngữ rộng hơn được Shiki hỗ trợ. Nếu gói này chưa được cài đặt, các tệp đó vẫn hiển thị dưới dạng văn bản thuần dễ đọc.

Ví dụ bao gồm Astro, Vue, Svelte, MDX, GraphQL, Terraform/HCL, Nix, Clojure, Elixir, Haskell, OCaml, Scala, Zig, Solidity, Verilog/VHDL, Fortran, MATLAB, LaTeX, Mermaid, Sass/Less/SCSS, Nginx, Apache, CSV, dotenv, INI và các tệp diff.

Xem [ngôn ngữ Shiki](https://shiki.style/languages) để biết danh mục ngôn ngữ và bí danh thượng nguồn của Shiki.
