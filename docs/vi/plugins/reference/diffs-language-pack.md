# Plugin gói ngôn ngữ Diffs

Bổ sung tô sáng cú pháp cho các ngôn ngữ nằm ngoài tập hợp mặc định của trình xem diff.

## Các ngôn ngữ được bổ sung

Plugin `diffs` cơ sở đã tô sáng các ngôn ngữ phổ biến được ghi trong tài liệu [Diffs](/vi/tools/diffs). Hãy cài đặt gói ngôn ngữ này khi bạn muốn tô sáng cú pháp cho tập hợp ngôn ngữ rộng hơn mà Shiki hỗ trợ. Nếu chưa cài đặt gói này, các tệp đó vẫn được hiển thị dưới dạng văn bản thuần dễ đọc.

Ví dụ bao gồm Astro, Vue, Svelte, MDX, GraphQL, Terraform/HCL, Nix, Clojure, Elixir, Haskell, OCaml, Scala, Zig, Solidity, Verilog/VHDL, Fortran, MATLAB, LaTeX, Mermaid, Sass/Less/SCSS, Nginx, Apache, CSV, dotenv, INI và các tệp diff.

Xem [các ngôn ngữ Shiki](https://shiki.style/languages) để biết danh mục ngôn ngữ và bí danh nguồn của Shiki.
