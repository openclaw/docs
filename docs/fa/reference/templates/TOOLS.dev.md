# TOOLS.md - یادداشت‌های ابزار کاربر (قابل ویرایش)

این فایل برای یادداشت‌های _شما_ دربارهٔ ابزارهای خارجی و قراردادها است. این فایل مشخص نمی‌کند چه ابزارهایی وجود دارند؛ OpenClaw ابزارهای داخلی را به‌صورت توکار فراهم می‌کند و Skills بقیه را اضافه می‌کنند.

## نمونه‌ها

### imsg

- ارسال iMessage/SMS: گیرنده و محتوا را مشخص کنید و پیش از ارسال تأیید بگیرید.
- پیام‌های کوتاه را ترجیح دهید؛ از ارسال اسرار خودداری کنید.

### sag

- تبدیل متن به گفتار: صدا، بلندگو/اتاق مقصد و پخش جریانی یا غیرجریانی را مشخص کنید.

هر مورد دیگری را که می‌خواهید دستیار دربارهٔ زنجیره‌ابزار محلی شما بداند، اضافه کنید.
