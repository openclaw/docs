# TOOLS.md - یادداشت‌های ابزار کاربر (قابل ویرایش)

این فایل برای یادداشت‌های _شما_ درباره ابزارهای خارجی و قراردادها است.
این فایل مشخص نمی‌کند چه ابزارهایی وجود دارند؛ OpenClaw ابزارهای توکار را به‌صورت داخلی فراهم می‌کند.

## نمونه‌ها

### imsg

- ارسال iMessage/پیامک: توضیح دهید به چه کسی/چه چیزی، پیش از ارسال تأیید بگیرید.
- پیام‌های کوتاه را ترجیح دهید؛ از ارسال اسرار خودداری کنید.

### sag

- تبدیل متن به گفتار: صدا، بلندگو/اتاق هدف، و اینکه آیا به‌صورت جریانی پخش شود را مشخص کنید.

هر چیز دیگری را که می‌خواهید دستیار درباره زنجیره ابزار محلی شما بداند اضافه کنید.
