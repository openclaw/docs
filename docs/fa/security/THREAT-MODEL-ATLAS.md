## 1. مقدمه

### 1.1 هدف

این مدل تهدید، تهدیدهای خصمانه علیه پلتفرم عامل هوش مصنوعی OpenClaw و بازار مهارت ClawHub را با استفاده از چارچوب MITRE ATLAS که به‌طور خاص برای سامانه‌های هوش مصنوعی/یادگیری ماشین طراحی شده است، مستند می‌کند.

### 1.2 دامنه

| مؤلفه                  | شامل می‌شود | یادداشت‌ها                                      |
| ---------------------- | -------- | ------------------------------------------------ |
| زمان اجرای عامل OpenClaw | بله      | اجرای اصلی عامل، فراخوانی ابزارها، نشست‌ها       |
| Gateway                | بله      | احراز هویت، مسیریابی، یکپارچه‌سازی کانال     |

| دستگاه‌های کاربر      | جزئی  | برنامه‌های موبایل، کلاینت‌های دسکتاپ                     |

## 4. تحلیل زنجیره تأمین ClawHub

### 4.1 کنترل‌های امنیتی فعلی

| کنترل                 | پیاده‌سازی                  | اثربخشی                                             |
| -------------------- | --------------------------- | ---------------------------------------------------- |
| سن حساب GitHub        | `requireGitHubAccountAge()` | متوسط - مانع را برای مهاجمان جدید بالاتر می‌برد     |
| پاک‌سازی مسیر         | `sanitizePath()`            | زیاد - از پیمایش مسیر جلوگیری می‌کند                |
| اعتبارسنجی نوع فایل   | `isTextFile()`              | متوسط - فقط فایل‌های متنی، اما همچنان می‌توانند مخرب باشند |
| محدودیت‌های اندازه    | بسته کل 50MB                | زیاد - از اتمام منابع جلوگیری می‌کند                |
| SKILL.md الزامی       | readme اجباری               | ارزش امنیتی کم - فقط اطلاع‌رسانی                    |
| تعدیل الگو            | FLAG_RULES در moderation.ts | کم - به‌راحتی دور زده می‌شود                        |
| وضعیت تعدیل           | فیلد `moderationStatus`     | متوسط - بازبینی دستی ممکن است                       |

### 4.2 الگوهای پرچم‌گذاری تعدیل

الگوهای فعلی در `moderation.ts`:

```javascript
// Known-bad identifiers
/(keepcold131\/ClawdAuthenticatorTool|ClawdAuthenticatorTool)/i

// Suspicious keywords
/(malware|stealer|phish|phishing|keylogger)/i
/(api[-_ ]?key|token|password|private key|secret)/i
/(wallet|seed phrase|mnemonic|crypto)/i
/(discord\.gg|webhook|hooks\.slack)/i
/(curl[^\n]+\|\s*(sh|bash))/i
/(bit\.ly|tinyurl\.com|t\.co|goo\.gl|is\.gd)/i
```

**محدودیت‌ها:**

- فقط slug، displayName، خلاصه، frontmatter، فراداده و مسیرهای فایل را بررسی می‌کند
- محتوای واقعی کد Skills را تحلیل نمی‌کند
- regex ساده با مبهم‌سازی به‌راحتی دور زده می‌شود
- تحلیل رفتاری ندارد

### 4.3 بهبودهای برنامه‌ریزی‌شده

| بهبود                   | وضعیت                                | اثر                                                                |
| ---------------------- | ------------------------------------- | --------------------------------------------------------------------- |
| یکپارچه‌سازی VirusTotal | در حال انجام                          | زیاد - تحلیل رفتاری Code Insight                                      |
| گزارش‌دهی جامعه         | جزئی (جدول `skillReports` وجود دارد) | متوسط                                                               |
| ثبت رویداد حسابرسی      | جزئی (جدول `auditLogs` وجود دارد)    | متوسط                                                               |
| سامانه نشان‌ها          | پیاده‌سازی‌شده                        | متوسط - `highlighted`، `official`، `deprecated`، `redactionApproved` |
