نشانگرهای در حال تایپ هنگام فعال بودن یک اجرا به کانال گفت‌وگو ارسال می‌شوند. از
`agents.defaults.typingMode` برای کنترل این‌که تایپ **چه زمانی** شروع شود و از `typingIntervalSeconds`
برای کنترل این‌که **هر چند وقت یک‌بار** تازه‌سازی شود استفاده کنید.

## پیش‌فرض‌ها

وقتی `agents.defaults.typingMode` **تنظیم نشده باشد**، OpenClaw رفتار قدیمی را حفظ می‌کند:

- **گپ‌های مستقیم**: تایپ بلافاصله پس از شروع حلقه مدل آغاز می‌شود.
- **گپ‌های گروهی با اشاره**: تایپ بلافاصله شروع می‌شود.
- **گپ‌های گروهی بدون اشاره**: تایپ فقط زمانی شروع می‌شود که جریان‌دهی متن پیام آغاز شود.
- **اجراهای Heartbeat**: اگر هدف Heartbeat حل‌شده یک گپ دارای قابلیت تایپ باشد و تایپ غیرفعال نشده باشد، تایپ هنگام شروع اجرای Heartbeat آغاز می‌شود.

## حالت‌ها

`agents.defaults.typingMode` را روی یکی از موارد زیر تنظیم کنید:

- `never` — هرگز هیچ نشانگر تایپی نمایش داده نمی‌شود.
- `instant` — تایپ را **به‌محض شروع حلقه مدل** آغاز می‌کند، حتی اگر اجرا
  بعداً فقط توکن پاسخ خاموش را برگرداند.
- `thinking` — تایپ را با **اولین دلتای استدلال** آغاز می‌کند (برای اجرا به
  `reasoningLevel: "stream"` نیاز دارد).
- `message` — تایپ را با **اولین دلتای متن غیرخاموش** آغاز می‌کند (توکن خاموش
  `NO_REPLY` را نادیده می‌گیرد).

ترتیب از نظر «زودتر فعال شدن»:
`never` → `message` → `thinking` → `instant`

## پیکربندی

```json5
{
  agent: {
    typingMode: "thinking",
    typingIntervalSeconds: 6,
  },
}
```

می‌توانید حالت یا ریتم را برای هر نشست بازنویسی کنید:

```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4,
  },
}
```

## نکته‌ها

- حالت `message` برای پاسخ‌هایی که فقط خاموش هستند، وقتی کل
  payload دقیقاً همان توکن خاموش باشد (برای مثال `NO_REPLY` / `no_reply`،
  با تطبیق بدون حساسیت به بزرگی و کوچکی حروف)، تایپ را نشان نمی‌دهد.
- `thinking` فقط زمانی فعال می‌شود که اجرا استدلال را به‌صورت جریانی ارسال کند (`reasoningLevel: "stream"`).
  اگر مدل دلتاهای استدلال منتشر نکند، تایپ شروع نمی‌شود.
- تایپ Heartbeat یک سیگنال زنده‌بودن برای هدف تحویل حل‌شده است. این تایپ
  هنگام شروع اجرای Heartbeat آغاز می‌شود، نه بر اساس زمان‌بندی جریان `message` یا `thinking`.
  برای غیرفعال کردن آن، `typingMode: "never"` را تنظیم کنید.
- Heartbeatها وقتی `target: "none"` باشد، وقتی هدف قابل حل نباشد، وقتی تحویل گپ برای Heartbeat غیرفعال باشد، یا وقتی کانال از تایپ پشتیبانی نکند، تایپ را نشان نمی‌دهند.
- `typingIntervalSeconds` **ریتم تازه‌سازی** را کنترل می‌کند، نه زمان شروع را.
  مقدار پیش‌فرض ۶ ثانیه است.
