# استفاده قابل‌قبول

ClawHub میزبان Skills، Pluginها، بسته‌ها و فراداده‌های بازارچه برای OpenClaw است.
از این صفحه برای تشخیص اینکه آیا محتوا یا رفتار انتشار به ClawHub تعلق دارد استفاده کنید.

این قوانین بر کارکرد یک فهرست، دستورهایی که از کاربران می‌خواهد اجرا کنند، نحوه معرفی خود و شیوه استفاده ناشران از قابلیت‌های کشف، نصب و اعتماد ClawHub اعمال می‌شوند. برای وضعیت‌های نظارت و اعتبار حساب، به [نظارت و ایمنی حساب](/clawhub/moderation) مراجعه کنید. برای ادعاهای حق نشر یا سایر حقوق، به [درخواست‌های حقوق محتوا](/clawhub/content-rights) مراجعه کنید.

## محتوای مجاز

ClawHub از محتوایی استقبال می‌کند که مفید، قابل‌فهم و با حسن نیت منتشر شده باشد.

| دسته‌بندی | در صورتی مجاز است که |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| بهره‌وری توسعه‌دهندگان | فهرست به کاربران در ساخت، آزمایش، مهاجرت، اشکال‌زدایی، مستندسازی یا بهره‌برداری از نرم‌افزار کمک کند. |
| گردش‌کارهای رابط کاربری، داده و خودکارسازی | دامنه کار روشن باشد، اطلاعات احراز هویت موردنیاز به‌صراحت ذکر شوند و اقدامات پرخطر شامل مسیرهای بازبینی، اجرای آزمایشی، پیش‌نمایش یا تأیید باشند. |
| امنیت دفاعی، نظارت و بررسی سوءاستفاده | ابزار برای بررسی مجاز ارائه شود، شواهد را حفظ کند و مرزهای تأیید انسانی را روشن نگه دارد. |
| گردش‌کارهای شخصی یا تیمی | گردش‌کار از حساب‌های مبتنی بر رضایت، راه‌اندازی شفاف و مجوزهای صریح استفاده کند. |
| فهرست‌های نگه‌داری‌شده | هر فهرست متمایز، مفید، با توصیف دقیق و به‌طور معقول نگه‌داری‌شده باشد. |

بافت اهمیت دارد. یک موضوع یکسان ممکن است در محیطی محدود، دفاعی یا مبتنی بر رضایت قابل‌قبول باشد، اما هنگامی که به‌صورت گردش‌کار سوءاستفاده بسته‌بندی شود، غیرقابل‌قبول است.

## محتوای غیرمجاز

ClawHub محتوایی را که هدف اصلی آن سوءاستفاده، فریب، اجرای ناامن یا نقض حقوق باشد، میزبانی نمی‌کند.

| دسته‌بندی | غیرمجاز |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| دسترسی غیرمجاز یا دور زدن امنیت | دور زدن احراز هویت، تصاحب حساب، سوءاستفاده از محدودیت نرخ، تصاحب تماس زنده یا عامل، سرقت نشست قابل‌استفاده مجدد یا تأیید خودکار فرایندهای جفت‌سازی برای کاربران تأییدنشده. |
| سوءاستفاده از پلتفرم و دور زدن محرومیت | حساب‌های پنهانی پس از محرومیت، گرم‌کردن یا پرورش حساب، تعامل جعلی، خودکارسازی چندحسابی، ارسال انبوه، ربات‌های هرزنامه یا خودکارسازی ساخته‌شده برای جلوگیری از شناسایی. |
| تقلب، کلاهبرداری و گردش‌کارهای مالی فریبکارانه | گواهی‌ها یا صورت‌حساب‌های جعلی، فرایندهای پرداخت فریبکارانه، ارتباط‌گیری کلاهبردارانه، اعتبار اجتماعی جعلی، گردش‌کارهای هویت مصنوعی برای تقلب یا ابزارهای خرج‌کردن و دریافت وجه بدون تأیید روشن انسانی. |
| غنی‌سازی ناقض حریم خصوصی یا نظارت | استخراج اطلاعات تماس برای هرزنامه، افشای اطلاعات شخصی، تعقیب، استخراج سرنخ همراه با ارتباط ناخواسته، پایش مخفیانه، تطبیق زیست‌سنجی بدون رضایت یا استفاده از داده‌های افشاشده یا مجموعه‌داده‌های رخنه امنیتی. |
| جعل هویت یا دست‌کاری هویت بدون رضایت | تعویض چهره، دوقلوهای دیجیتال، چهره‌های تأثیرگذار شبیه‌سازی‌شده، شخصیت‌های جعلی یا سایر ابزارهایی که برای جعل هویت یا گمراه‌کردن استفاده می‌شوند. |
| محتوای جنسی صریح یا تولید محتوای بزرگسال با ایمنی غیرفعال | تولید تصویر، ویدئو یا محتوای نامناسب برای محیط کار؛ پوشش‌های محتوای بزرگسال پیرامون APIهای شخص ثالث؛ یا فهرست‌هایی که هدف اصلی آن‌ها محتوای جنسی صریح است. |
| الزامات اجرایی پنهان، ناامن یا گمراه‌کننده | فرمان‌های نصب مبهم‌سازی‌شده، نصب‌کننده‌های انتقال مستقیم به پوسته مانند اجرای محتوای بارگیری‌شده با `sh` یا `bash` بدون امکان بازبینی روشن، الزامات اعلام‌نشده برای رمز یا کلید خصوصی، اجرای راه دور `npx @latest` بدون امکان بازبینی روشن، یا فراداده‌ای که نیازهای واقعی فهرست برای اجرا را پنهان کند. |
| مطالب ناقض حق نشر یا سایر حقوق | بازنشر Skill، Plugin، مستندات، دارایی‌های برند یا کد اختصاصی شخص دیگری بدون اجازه؛ نقض شرایط مجوز؛ یا جعل هویت نویسنده یا ناشر اصلی. |

## رفتار غیرمجاز در بازارچه

ClawHub همچنین نحوه استفاده ناشران از بازارچه را بررسی می‌کند. از ClawHub برای دست‌کاری قابلیت کشف، معیارها، نشانه‌های اعتماد، سامانه‌های نظارت یا توجه کاربران استفاده نکنید.

رفتار غیرمجاز در بازارچه شامل موارد زیر است:

- انتشار انبوه تعداد زیادی فهرست کم‌زحمت، تکراری، جای‌نگهدار یا
  تولیدشده با ماشین که به نظر نمی‌رسد ارزش واقعی برای کاربران داشته باشند
- اشباع سطوح جست‌وجو یا دسته‌بندی با Skillها یا Pluginهای تقریباً یکسان
- انتشار صدها فهرست با استفاده، نگه‌داری، شفافیت منبع یا
  تمایز معنادار اندک یا بدون آن
- افزایش مصنوعی تعداد نصب‌ها، بارگیری‌ها، ستاره‌ها یا سایر معیارهای
  تعامل از طریق خودکارسازی، چرخه‌های نصب توسط خود ناشر، حساب‌های جعلی، فعالیت
  هماهنگ‌شده، تعامل پولی یا سایر رفتارهای غیرطبیعی
- ایجاد یا چرخش حساب‌ها برای دور زدن نظارت، محرومیت‌ها، محدودیت‌های ناشر یا
  بررسی بازارچه
- گمراه‌کردن کاربران درباره مالکیت، منبع، قابلیت‌ها، وضعیت امنیتی،
  الزامات نصب یا وابستگی به پروژه یا ناشر دیگری
- بارگذاری مکرر محتوایی که پیش‌تر پنهان، حذف یا مسدود شده است،
  بدون رفع مشکل زیربنایی

انتشار با حجم بالا به‌طور خودکار سوءاستفاده محسوب نمی‌شود. فهرست‌های بزرگ زمانی قابل‌قبول‌اند که موارد فهرست‌شده تفاوت معنادار داشته باشند، به‌دقت توصیف و نگه‌داری شوند و کاربران واقعی از آن‌ها استفاده کنند. فهرست‌های بزرگ هنگامی به مشکل اعتماد و ایمنی تبدیل می‌شوند که حجم بالا با موارد کم‌محتوا، تکراری، گمراه‌کننده، نگه‌داری‌نشده یا دارای ترویج مصنوعی همراه باشد.

## حقوق محتوا

اگر باور دارید محتوایی در ClawHub حق نشر یا سایر حقوق شما را نقض می‌کند، از [درخواست‌های حقوق محتوا](/clawhub/content-rights) استفاده کنید. برای ادعاهای حق نشر یا سایر حقوق از گزارش‌های عادی بازارچه استفاده نکنید، مگر اینکه فهرست موردنظر ناامن، مخرب یا گمراه‌کننده نیز باشد.

## بررسی و اعمال مقررات

ClawHub ممکن است برای شناسایی محتوای ناامن یا رفتار انتشاراتی سوءاستفاده‌گرانه از بررسی‌های خودکار، نشانه‌های آماری سوءاستفاده، گزارش‌های کاربران و بررسی کارکنان استفاده کند. یک نشانه به‌تنهایی سوءاستفاده را اثبات نمی‌کند؛ بلکه به ClawHub کمک می‌کند تصمیم بگیرد چه چیزی نیازمند بررسی است.

ممکن است:

- فهرست‌های ناقض را پنهان، متوقف، حذف، حذف نرم یا، در صورت پشتیبانی
  نوع منبع، حذف قطعی کنیم
- بارگیری یا نصب نسخه‌های ناامن را مسدود کنیم
- توکن‌های API را باطل کنیم
- محتوای مرتبط را به‌صورت نرم حذف کنیم
- دسترسی انتشار را محدود کنیم
- متخلفان تکراری یا شدید را محروم کنیم

برای سوءاستفاده آشکار، اعمال مقررات همراه با هشدار قبلی را تضمین نمی‌کنیم. برای گزارش‌ها، توقف‌های نظارتی، فهرست‌های پنهان، محرومیت‌ها و اعتبار حساب به [نظارت و ایمنی حساب](/clawhub/moderation) مراجعه کنید.
