# استفاده قابل قبول

ClawHub میزبان Skills، Pluginها، بسته‌ها و فرادادهٔ بازارچه برای OpenClaw است.
از این صفحه برای تصمیم‌گیری دربارهٔ این‌که محتوا یا رفتار انتشار به ClawHub
تعلق دارد یا نه استفاده کنید.

این قواعد دربارهٔ کاری که یک فهرست انجام می‌دهد، چیزی که از کاربران می‌خواهد اجرا کنند، نحوهٔ
معرفی خودش، و شیوهٔ استفادهٔ ناشران از سطوح کشف، نصب و اعتماد ClawHub اعمال می‌شود. برای وضعیت‌های تعدیل و اعتبار حساب، به
[تعدیل و ایمنی حساب](/clawhub/moderation) مراجعه کنید. برای ادعاهای حق نشر یا حقوق دیگر،
[درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) را ببینید.

## محتوای مجاز

ClawHub از محتوایی که مفید، قابل فهم و با حسن نیت منتشر شده باشد استقبال می‌کند.

| دسته | مجاز است وقتی |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| بهره‌وری توسعه‌دهنده | فهرست به کاربران کمک کند نرم‌افزار بسازند، آزمایش کنند، مهاجرت دهند، اشکال‌زدایی کنند، مستندسازی کنند یا به‌کار بگیرند. |
| جریان‌های کاری رابط کاربری، داده و خودکارسازی | دامنه روشن باشد، اعتبارنامه‌های لازم صریح باشند، و اقدام‌های پرخطر مسیرهای بازبینی، اجرای آزمایشی، پیش‌نمایش یا تأیید داشته باشند. |
| امنیت دفاعی، تعدیل و بازبینی سوءاستفاده | ابزار برای بازبینی مجاز چارچوب‌بندی شده باشد، شواهد را حفظ کند و مرزهای تأیید انسانی را روشن نگه دارد. |
| جریان‌های کاری شخصی یا تیمی | جریان کاری از حساب‌های مبتنی بر رضایت، راه‌اندازی شفاف و مجوزهای صریح استفاده کند. |
| کاتالوگ‌های نگهداری‌شده | هر فهرست متمایز، مفید، دقیقاً توصیف‌شده و به‌طور معقولی نگهداری‌شده باشد. |

زمینه مهم است. یک موضوع واحد می‌تواند در یک چارچوب دفاعی محدود یا
مبتنی بر رضایت قابل قبول باشد و وقتی به‌صورت جریان کاری سوءاستفاده بسته‌بندی شود غیرقابل قبول باشد.

## محتوای غیرمجاز

ClawHub محتوایی را که هدف اصلی آن سوءاستفاده، فریب، اجرای ناامن
یا نقض حقوق باشد میزبانی نمی‌کند.

| دسته | مجاز نیست |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| دسترسی غیرمجاز یا دور زدن امنیت | دور زدن احراز هویت، تصاحب حساب، سوءاستفاده از محدودیت نرخ، تصاحب تماس زنده یا عامل، سرقت نشست قابل استفادهٔ مجدد، یا تأیید خودکار جریان‌های جفت‌سازی برای کاربران تأییدنشده. |
| سوءاستفاده از پلتفرم و دور زدن مسدودسازی | حساب‌های پنهان پس از مسدودسازی، آماده‌سازی یا پرورش حساب، تعامل جعلی، خودکارسازی چندحسابی، ارسال انبوه، ربات‌های هرزنامه، یا خودکارسازی ساخته‌شده برای اجتناب از شناسایی. |
| کلاهبرداری، تقلب و جریان‌های کاری مالی فریبکارانه | گواهی‌ها یا فاکتورهای جعلی، جریان‌های پرداخت فریبکارانه، ارتباط‌گیری کلاهبردارانه، اثبات اجتماعی جعلی، جریان‌های کاری هویت مصنوعی برای تقلب، یا ابزارهای خرج‌کردن/شارژ بدون تأیید انسانی روشن. |
| غنی‌سازی یا نظارت ناقض حریم خصوصی | جمع‌آوری مخاطبان برای هرزنامه، افشای اطلاعات شخصی، تعقیب، استخراج سرنخ همراه با ارتباط‌گیری ناخواسته، پایش مخفیانه، تطبیق زیست‌سنجی بدون رضایت، یا استفاده از داده‌های افشاشده یا بسته‌های رخنه. |
| جعل هویت یا دست‌کاری هویت بدون رضایت | تعویض چهره، دوقلوهای دیجیتال، اینفلوئنسرهای شبیه‌سازی‌شده، شخصیت‌های جعلی، یا ابزارهای دیگری که برای جعل هویت یا گمراه‌کردن استفاده می‌شوند. |
| محتوای جنسی صریح یا تولید بزرگسالانه با ایمنی غیرفعال | تولید تصویر، ویدئو یا محتواهای NSFW؛ پوشش‌های محتوای بزرگسالان پیرامون APIهای شخص ثالث؛ یا فهرست‌هایی که هدف اصلی آن‌ها محتوای جنسی صریح است. |
| الزامات اجرای پنهان، ناامن یا گمراه‌کننده | فرمان‌های نصب مبهم‌سازی‌شده، نصب‌کننده‌هایی که محتوای دانلودشده را بدون قابلیت بازبینی روشن با `sh` یا `bash` اجرا می‌کنند، الزامات اعلام‌نشدهٔ راز یا کلید خصوصی، اجرای راه دور `npx @latest` بدون قابلیت بازبینی روشن، یا فراداده‌ای که نیاز واقعی فهرست برای اجرا را پنهان می‌کند. |
| مواد ناقض حق نشر یا ناقض حقوق | بازنشر Skill، Plugin، مستندات، دارایی‌های برند یا کد اختصاصی فردی دیگر بدون مجوز؛ نقض شرایط مجوز؛ یا جعل هویت نویسنده یا ناشر اصلی. |

## رفتار غیرمجاز در بازارچه

ClawHub همچنین بررسی می‌کند ناشران چگونه از بازارچه استفاده می‌کنند. از ClawHub برای
دست‌کاری کشف، معیارها، نشانه‌های اعتماد، سامانه‌های تعدیل یا توجه کاربران
استفاده نکنید.

رفتار غیرمجاز در بازارچه شامل این موارد است:

- انتشار انبوه تعداد زیادی فهرست کم‌زحمت، تکراری، جای‌نگهدار یا
  تولیدشده با ماشین که به نظر نمی‌رسد ارزش واقعی برای کاربر داشته باشند
- پر کردن سطوح جست‌وجو یا دسته‌بندی با Skills یا Pluginهای تقریباً یکسان
- انتشار صدها فهرست با استفاده، نگهداری، شفافیت منبع
  یا تمایز معنادار اندک یا بدون آن
- افزایش مصنوعی نصب‌ها، دانلودها، ستاره‌ها یا سایر معیارهای تعامل
  از طریق خودکارسازی، حلقه‌های نصب توسط خود، حساب‌های جعلی، فعالیت
  هماهنگ، تعامل پولی یا رفتار غیرارگانیک دیگر
- ایجاد یا چرخاندن حساب‌ها برای دور زدن تعدیل، مسدودسازی‌ها، محدودیت‌های ناشر یا
  بازبینی بازارچه
- گمراه کردن کاربران دربارهٔ مالکیت، منبع، قابلیت‌ها، وضعیت امنیتی،
  الزامات نصب یا وابستگی به پروژه یا ناشر دیگر
- بارگذاری مکرر محتوایی که قبلاً پنهان، حذف یا مسدود شده است
  بدون رفع مشکل زیربنایی

انتشار با حجم بالا به‌طور خودکار سوءاستفاده نیست. کاتالوگ‌های بزرگ زمانی قابل قبول‌اند
که فهرست‌ها به‌طور معناداری متفاوت، دقیقاً توصیف‌شده، نگهداری‌شده
و توسط کاربران واقعی استفاده شوند. کاتالوگ‌های بزرگ وقتی به مشکل اعتماد و ایمنی تبدیل می‌شوند
که حجم با فهرست‌های سطحی، تکراری، گمراه‌کننده، نگهداری‌نشده یا
به‌طور مصنوعی ترویج‌شده همراه باشد.

## حقوق محتوا

اگر باور دارید محتوایی در ClawHub حق نشر یا حقوق دیگر شما را نقض می‌کند، از
[درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) استفاده کنید. برای ادعاهای حق نشر یا حقوق،
از گزارش‌های عادی بازارچه استفاده نکنید مگر این‌که فهرست همچنین ناامن،
مخرب یا گمراه‌کننده باشد.

## بازبینی و اجرا

ClawHub ممکن است از بررسی‌های خودکار، نشانه‌های آماری سوءاستفاده، گزارش‌های کاربران و
بازبینی کارکنان برای شناسایی محتوای ناامن یا رفتار انتشار سوءاستفاده‌آمیز استفاده کند. یک نشانه
به‌تنهایی سوءاستفاده را ثابت نمی‌کند؛ به ClawHub کمک می‌کند تصمیم بگیرد چه چیزی نیاز به بازبینی دارد.

ما ممکن است:

- فهرست‌های متخلف را پنهان، نگه‌داری، حذف، نرم‌حذف، یا، در مواردی که برای نوع منبع پشتیبانی می‌شود،
  سخت‌حذف کنیم
- دانلودها یا نصب‌ها را برای انتشارهای ناامن مسدود کنیم
- توکن‌های API را لغو کنیم
- محتوای مرتبط را نرم‌حذف کنیم
- دسترسی انتشار را محدود کنیم
- متخلفان تکراری یا شدید را مسدود کنیم

ما اجرای همراه با هشدار اولیه را برای سوءاستفادهٔ آشکار تضمین نمی‌کنیم. برای گزارش‌ها، توقف‌های تعدیل،
فهرست‌های پنهان، مسدودسازی‌ها و اعتبار حساب، به
[تعدیل و ایمنی حساب](/clawhub/moderation) مراجعه کنید.
