# استفاده قابل قبول

ClawHub میزبان Skills، plugins، بسته‌ها و فراداده بازار برای OpenClaw است.
از این صفحه برای تصمیم‌گیری درباره اینکه آیا محتوا یا رفتار انتشار به
ClawHub تعلق دارد یا نه استفاده کنید.

این قواعد درباره کاری که یک فهرست انجام می‌دهد، چیزی که از کاربران می‌خواهد اجرا کنند، اینکه چگونه
خود را معرفی می‌کند، و اینکه ناشران چگونه از سطوح کشف، نصب و
اعتماد ClawHub استفاده می‌کنند اعمال می‌شود. برای وضعیت‌های نظارت و وضعیت حساب، به
[نظارت و ایمنی حساب](/clawhub/moderation) مراجعه کنید. برای حق نشر یا سایر ادعاهای حقوقی،
[درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) را ببینید.

## محتوای مجاز

ClawHub از محتوایی استقبال می‌کند که مفید، قابل فهم و با حسن نیت
منتشر شده باشد.

| دسته‌بندی                                         | مجاز است وقتی                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| بهره‌وری توسعه‌دهنده                           | فهرست به کاربران کمک می‌کند نرم‌افزار را بسازند، آزمایش کنند، مهاجرت دهند، اشکال‌زدایی کنند، مستند کنند یا بهره‌برداری کنند.                                               |
| گردش‌کارهای UI، داده و اتوماسیون               | دامنه روشن است، اعتبارنامه‌های لازم صریح هستند، و اقدام‌های پرخطر شامل مسیرهای بازبینی، اجرای آزمایشی، پیش‌نمایش یا تأیید هستند. |
| امنیت دفاعی، نظارت و بازبینی سوءاستفاده | ابزار برای بازبینی مجاز چارچوب‌بندی شده، شواهد را حفظ می‌کند، و مرزهای تأیید انسانی را روشن نگه می‌دارد.                          |
| گردش‌کارهای شخصی یا تیمی                       | گردش‌کار از حساب‌های مبتنی بر رضایت، راه‌اندازی شفاف و مجوزهای صریح استفاده می‌کند.                                            |
| کاتالوگ‌های نگه‌داری‌شده                              | هر فهرست متمایز، مفید، دقیقاً توصیف‌شده و به‌طور معقول نگه‌داری‌شده است.                                                |

زمینه اهمیت دارد. یک موضوع یکسان می‌تواند در یک چارچوب دفاعی محدود یا
مبتنی بر رضایت قابل قبول باشد و وقتی به‌صورت یک گردش‌کار سوءاستفاده بسته‌بندی شود غیرقابل قبول باشد.

## رفتار غیرمجاز بازار

ClawHub همچنین نحوه استفاده ناشران از بازار را بازبینی می‌کند. از ClawHub برای
دستکاری کشف، معیارها، نشانه‌های اعتماد، سامانه‌های نظارت یا
توجه کاربر استفاده نکنید.

رفتار غیرمجاز بازار شامل این موارد است:

- انتشار انبوه تعداد زیادی فهرست کم‌زحمت، تکراری، جای‌نگهدار یا
  تولیدشده با ماشین که به نظر نمی‌رسد ارزش واقعی برای کاربر داشته باشند
- پرکردن سطوح جست‌وجو یا دسته‌بندی با Skills یا plugins تقریباً یکسان
- انتشار صدها فهرست با استفاده، نگه‌داری، شفافیت منبع
  یا تمایز معنادار کم یا بدون آن
- افزایش مصنوعی نصب‌ها، دانلودها، ستاره‌ها یا سایر معیارهای
  تعامل از طریق اتوماسیون، حلقه‌های خودنصب، حساب‌های جعلی، فعالیت
  هماهنگ، تعامل پولی یا سایر رفتارهای غیرطبیعی
- ایجاد یا چرخاندن حساب‌ها برای دور زدن نظارت، محرومیت‌ها، محدودیت‌های ناشر یا
  بازبینی بازار
- گمراه‌کردن کاربران درباره مالکیت، منبع، قابلیت‌ها، وضعیت امنیتی،
  الزامات نصب یا وابستگی به پروژه یا ناشر دیگر
- بارگذاری مکرر محتوایی که پیش‌تر پنهان، حذف یا مسدود شده است
  بدون رفع مشکل زیربنایی

انتشار با حجم بالا به‌طور خودکار سوءاستفاده نیست. کاتالوگ‌های بزرگ زمانی قابل قبول‌اند
که فهرست‌ها به‌طور معنادار متفاوت، دقیقاً توصیف‌شده، نگه‌داری‌شده
و توسط کاربران واقعی استفاده شوند. کاتالوگ‌های بزرگ وقتی به مشکل اعتماد و ایمنی تبدیل می‌شوند که
حجم با فهرست‌های کم‌محتوا، تکراری، گمراه‌کننده، نگه‌داری‌نشده یا
به‌طور مصنوعی تبلیغ‌شده همراه باشد.

## حقوق محتوا

اگر باور دارید محتوایی در ClawHub حق نشر یا سایر حقوق شما را نقض می‌کند، از
[درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) استفاده کنید. از گزارش‌های معمول بازار
برای ادعاهای حق نشر یا حقوق استفاده نکنید مگر اینکه فهرست همچنین ناامن،
مخرب یا گمراه‌کننده باشد.

## بازبینی و اعمال

ClawHub ممکن است از بررسی‌های خودکار، نشانه‌های آماری سوءاستفاده، گزارش‌های کاربران و
بازبینی کارکنان برای شناسایی محتوای ناامن یا رفتار انتشار سوءاستفاده‌آمیز استفاده کند. یک نشانه
به‌تنهایی سوءاستفاده را ثابت نمی‌کند؛ به ClawHub کمک می‌کند تصمیم بگیرد چه چیزی نیاز به بازبینی دارد.

ممکن است:

- فهرست‌های ناقض را پنهان کنیم، نگه داریم، حذف کنیم، حذف نرم کنیم، یا، در مواردی که برای نوع منبع پشتیبانی می‌شود،
  حذف سخت کنیم
- دانلودها یا نصب‌ها را برای انتشارهای ناامن مسدود کنیم
- توکن‌های API را لغو کنیم
- محتوای مرتبط را حذف نرم کنیم
- دسترسی انتشار را محدود کنیم
- متخلفان تکراری یا شدید را محروم کنیم

ما اعمال با هشدار اولیه را برای سوءاستفاده آشکار تضمین نمی‌کنیم. برای گزارش‌ها، نگه‌داشت‌های نظارت،
فهرست‌های پنهان، محرومیت‌ها و وضعیت حساب، به
[نظارت و ایمنی حساب](/clawhub/moderation) مراجعه کنید.
