# استفادهٔ قابل قبول

ClawHub میزبان Skills، Pluginها، بسته‌ها و فرادادهٔ بازارچه برای OpenClaw است.
از این صفحه استفاده کنید تا تصمیم بگیرید آیا محتوا یا رفتار انتشار به
ClawHub تعلق دارد یا نه.

این قواعد دربارهٔ کاری که یک فهرست انجام می‌دهد، چیزی که از کاربران می‌خواهد اجرا کنند، نحوهٔ
معرفی خودش، و نحوهٔ استفادهٔ ناشران از سطوح کشف، نصب و اعتماد
ClawHub اعمال می‌شوند. برای وضعیت‌های نظارت و وضعیت حساب، به
[نظارت و ایمنی حساب](/fa/clawhub/moderation) مراجعه کنید. برای ادعاهای حق نشر یا سایر حقوق،
به [درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) مراجعه کنید.

## محتوای مجاز

ClawHub از محتوایی استقبال می‌کند که مفید، قابل فهم، و با حسن نیت منتشر شده باشد.

| دسته‌بندی                                         | زمانی مجاز است که                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| بهره‌وری توسعه‌دهنده                           | فهرست به کاربران کمک کند نرم‌افزار را بسازند، آزمایش کنند، مهاجرت دهند، اشکال‌زدایی کنند، مستندسازی کنند یا عملیاتی کنند.                                               |
| گردش‌کارهای رابط کاربری، داده و خودکارسازی               | دامنه روشن باشد، اعتبارنامه‌های لازم صریح باشند، و اقدامات پرخطر شامل مسیرهای بازبینی، اجرای آزمایشی، پیش‌نمایش یا تأیید باشند. |
| امنیت دفاعی، نظارت و بررسی سوءاستفاده | ابزار برای بررسی مجاز چارچوب‌بندی شده باشد، شواهد را حفظ کند، و مرزهای تأیید انسانی را روشن نگه دارد.                          |

| کاتالوگ‌های نگهداری‌شده                              | هر فهرست متمایز، مفید، دقیق توصیف‌شده، و به‌طور معقول نگهداری‌شده باشد.                                                |

زمینه اهمیت دارد. همان موضوع می‌تواند در یک محیط دفاعی محدود یا
مبتنی بر رضایت قابل قبول باشد و وقتی به‌صورت گردش‌کار سوءاستفاده بسته‌بندی شود غیرقابل قبول باشد.

## رفتار غیرمجاز بازارچه

ClawHub همچنین نحوهٔ استفادهٔ ناشران از بازارچه را بررسی می‌کند. از ClawHub برای
دستکاری کشف، سنجه‌ها، سیگنال‌های اعتماد، سامانه‌های نظارت یا توجه
کاربر استفاده نکنید.

رفتار غیرمجاز بازارچه شامل موارد زیر است:

- انتشار انبوه تعداد زیادی فهرست کم‌تلاش، تکراری، جای‌نگهدار یا
  تولیدشده با ماشین که به نظر نمی‌رسد ارزش واقعی برای کاربر داشته باشند
- پرکردن سطوح جست‌وجو یا دسته‌بندی با Skills یا Pluginهای تقریباً یکسان
- انتشار صدها فهرست با استفاده، نگهداری، شفافیت منبع
  یا تمایز معنادار کم یا بدون آن
- افزایش مصنوعی نصب‌ها، دانلودها، ستاره‌ها یا سایر سنجه‌های
  تعامل از طریق خودکارسازی، حلقه‌های نصب خود، حساب‌های جعلی، فعالیت
  هماهنگ، تعامل پولی یا رفتار غیرارگانیک دیگر
- ایجاد یا چرخاندن حساب‌ها برای دور زدن نظارت، ممنوعیت‌ها، محدودیت‌های ناشر یا
  بازبینی بازارچه
- گمراه‌کردن کاربران دربارهٔ مالکیت، منبع، قابلیت‌ها، وضعیت امنیتی،
  الزامات نصب، یا وابستگی به پروژه یا ناشر دیگر
- بارگذاری مکرر محتوایی که پیش‌تر پنهان، حذف یا مسدود شده است
  بدون رفع مشکل زیربنایی

انتشار با حجم بالا به‌طور خودکار سوءاستفاده نیست. کاتالوگ‌های بزرگ زمانی قابل قبول‌اند
که فهرست‌ها به‌طور معنادار متفاوت، دقیق توصیف‌شده، نگهداری‌شده
و توسط کاربران واقعی استفاده شوند. کاتالوگ‌های بزرگ زمانی به مشکل اعتماد و ایمنی تبدیل می‌شوند که
حجم با فهرست‌های کم‌مایه، تکراری، گمراه‌کننده، نگهداری‌نشده یا
به‌طور مصنوعی ترویج‌شده همراه باشد.

## حقوق محتوا

اگر باور دارید محتوایی در ClawHub حق نشر یا سایر حقوق شما را نقض می‌کند، از
[درخواست‌های حقوق محتوا](/fa/clawhub/content-rights) استفاده کنید. از گزارش‌های عادی بازارچه
برای ادعاهای حق نشر یا حقوق استفاده نکنید، مگر اینکه فهرست همچنین ناامن،
مخرب یا گمراه‌کننده باشد.

## بازبینی و اجرا

ClawHub ممکن است از بررسی‌های خودکار، سیگنال‌های آماری سوءاستفاده، گزارش‌های کاربران و
بازبینی کارکنان برای شناسایی محتوای ناامن یا رفتار انتشار سوءاستفاده‌گرانه استفاده کند. یک سیگنال
به‌تنهایی سوءاستفاده را ثابت نمی‌کند؛ به ClawHub کمک می‌کند تصمیم بگیرد چه چیزی نیاز به بازبینی دارد.

ممکن است:

- فهرست‌های ناقض را پنهان، نگه‌داری، حذف، نرم‌حذف، یا، در جایی که برای نوع منبع پشتیبانی می‌شود،
  سخت‌حذف کنیم
- دانلودها یا نصب‌ها را برای انتشارهای ناامن مسدود کنیم
- توکن‌های API را لغو کنیم
- محتوای مرتبط را نرم‌حذف کنیم
- دسترسی انتشار را محدود کنیم
- متخلفان تکراری یا شدید را ممنوع کنیم

ما اجرای هشدار-نخست را برای سوءاستفادهٔ آشکار تضمین نمی‌کنیم. برای گزارش‌ها، نگه‌داشت‌های نظارت،
فهرست‌های پنهان، ممنوعیت‌ها و وضعیت حساب به
[نظارت و ایمنی حساب](/fa/clawhub/moderation) مراجعه کنید.
