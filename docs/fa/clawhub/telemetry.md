# تله‌متری

ClawHub از حداقل تله‌متری CLI برای محاسبه تعداد تجمیعی نصب‌ها استفاده می‌کند.

## چه زمانی تله‌متری جمع‌آوری می‌شود

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد حساب خود شده باشید.
- دستور `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوه غیرفعال‌کردن» را در ادامه ببینید).

اگر وارد حساب خود نشده باشید، هیچ اطلاعاتی گزارش نمی‌شود.

## چه اطلاعاتی جمع‌آوری می‌کنیم

برای هر اجرای گزارش‌شده `clawhub install`، CLI یک رویداد نصب را به‌صورت تلاش در حد امکان ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: نامک Skills نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص‌بودن.

### چه اطلاعاتی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه‌ای.
- هیچ محتوای فایلی.
- هیچ گزارش اجرای جداگانه، پرامپت یا خروجی دیگری از CLI.

## تعداد نصب‌ها

ClawHub برای هر Skills شمارنده‌های تجمیعی نگه می‌دارد:

- `installsAllTime`: کاربران منحصربه‌فردی که حداقل یک نصب CLI برای آن Skills گزارش کرده‌اند.
- `installsCurrent`: کاربران منحصربه‌فردی که یک نصب را گزارش کرده‌اند و داده‌های تله‌متری خود را حذف نکرده‌اند.

## شفافیت و کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را مشاهده می‌کنند.

با حذف حساب، داده‌های تله‌متری شما نیز حذف می‌شوند.

## نحوه غیرفعال‌کردن تله‌متری

متغیر محیطی زیر را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این متغیر، CLI تله‌متری نصب را ارسال نخواهد کرد.
