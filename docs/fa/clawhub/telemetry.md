# دورسنجی

ClawHub از دورسنجی حداقلی CLI برای محاسبهٔ شمارش‌های تجمیعی نصب استفاده می‌کند.

## زمان جمع‌آوری دورسنجی

دورسنجی فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- دورسنجی **غیرفعال نشده باشد** (بخش «نحوهٔ غیرفعال‌سازی» در پایین را ببینید).

اگر وارد نشده باشید، چیزی گزارش نمی‌شود.

## آنچه جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب با بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ مهارت نصب‌شده.
- `version`: نسخهٔ نصب‌شده، اگر مشخص باشد.

### آنچه _جمع‌آوری نمی‌کنیم_

- هیچ مسیر پوشه یا شناسهٔ برگرفته از پوشه.
- هیچ محتوای فایل.
- هیچ گزارش اجرای جداگانه، پرامپت، یا خروجی دیگر CLI.

## شمارش نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر مهارت نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که حداقل یک نصب CLI برای مهارت را گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و دورسنجی خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را می‌بینند.

حذف حساب شما، داده‌های دورسنجی شما را نیز حذف می‌کند.

## نحوهٔ غیرفعال‌سازی دورسنجی

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مورد، CLI دورسنجی نصب را ارسال نخواهد کرد.
