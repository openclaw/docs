# تله‌متری

ClawHub از تله‌متری حداقلی CLI برای محاسبه تعداد نصب‌های تجمیعی استفاده می‌کند.

## زمان جمع‌آوری تله‌متری

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد حساب شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوه غیرفعال‌سازی» را در پایین ببینید).

اگر وارد حساب نشده باشید، هیچ چیزی گزارش نمی‌شود.

## آنچه جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب به‌صورت بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ مهارت نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص بودن.

### آنچه _جمع‌آوری نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه.
- هیچ محتوای فایل.
- هیچ گزارش هر اجرا، اعلان، یا خروجی دیگر CLI.

## تعداد نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر مهارت نگه می‌دارد:

- `installsAllTime`: کاربران منحصربه‌فردی که دست‌کم یک نصب CLI را برای آن مهارت گزارش کرده‌اند.
- `installsCurrent`: کاربران منحصربه‌فردی که نصبی را گزارش کرده‌اند و داده‌های
  تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های نصب تجمیعی** را می‌بینند.

حذف حساب شما داده‌های تله‌متری شما را نیز حذف می‌کند.

## نحوه غیرفعال‌سازی تله‌متری

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مورد، CLI تله‌متری نصب را ارسال نخواهد کرد.
