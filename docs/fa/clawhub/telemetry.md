# تله‌متری

ClawHub از حداقل تله‌متری CLI برای محاسبه تعداد تجمیعی نصب‌ها استفاده می‌کند.

## تله‌متری چه زمانی جمع‌آوری می‌شود

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد حساب شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوه غیرفعال‌کردن» را در ادامه ببینید).

اگر وارد حساب نشده باشید، هیچ‌چیز گزارش نمی‌شود.

## چه اطلاعاتی جمع‌آوری می‌کنیم

برای هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب را به‌صورت بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: نامک Skill نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص‌بودن.

### چه اطلاعاتی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه‌ای که از پوشه مشتق شده باشد.
- هیچ محتوای فایلی.
- هیچ گزارش هر اجرا، پرامپت یا خروجی دیگری از CLI.

## تعداد نصب‌ها

ClawHub شمارنده‌های تجمیعی زیر را برای هر Skill نگه‌داری می‌کند:

- `installsAllTime`: کاربران یکتایی که حداقل یک نصب CLI از Skill را گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و داده‌های
  تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را مشاهده می‌کنند.

با حذف حساب، داده‌های تله‌متری شما نیز حذف می‌شوند.

## نحوه غیرفعال‌کردن تله‌متری

متغیر محیطی زیر را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این متغیر، CLI تله‌متری نصب را ارسال نخواهد کرد.
