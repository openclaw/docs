# تله‌متری

ClawHub از تله‌متری حداقلی CLI برای محاسبه شمارش‌های تجمیعی نصب استفاده می‌کند.

## تله‌متری چه زمانی جمع‌آوری می‌شود

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- دستور `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوه غیرفعال‌سازی» را در پایین ببینید).

اگر وارد نشده باشید، هیچ چیزی گزارش نمی‌شود.

## چه چیزهایی جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب با بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ مهارت نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص بودن.

### چه چیزهایی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه.
- هیچ محتوای فایلی.
- هیچ لاگ، پرامپت، یا خروجی دیگر CLI به‌ازای هر اجرا.

## شمارش نصب‌ها

ClawHub شمارنده‌های تجمیعی را به‌ازای هر مهارت نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که حداقل یک نصب CLI را برای این مهارت گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و داده‌های تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را می‌بینند.

حذف حساب کاربری شما همچنین داده‌های تله‌متری شما را حذف می‌کند.

## نحوه غیرفعال‌سازی تله‌متری

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مقدار، CLI تله‌متری نصب را ارسال نخواهد کرد.
