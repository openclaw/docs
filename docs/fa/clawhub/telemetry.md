# دورسنجی

ClawHub از دورسنجی حداقلی CLI برای محاسبهٔ شمارش تجمیعی نصب‌ها استفاده می‌کند.

## چه زمانی دورسنجی جمع‌آوری می‌شود

دورسنجی فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- دورسنجی **غیرفعال نشده باشد** (بخش «نحوهٔ غیرفعال‌سازی» را در ادامه ببینید).

اگر وارد نشده باشید، هیچ چیزی گزارش نمی‌شود.

## چه چیزهایی را جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب را به‌صورت best-effort ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: slug مهارت نصب‌شده.
- `version`: نسخهٔ نصب‌شده، در صورت مشخص بودن.

### چه چیزهایی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسهٔ مشتق‌شده از پوشه‌ای.
- هیچ محتوای فایلی.
- هیچ گزارش هر اجرا، prompt، یا خروجی دیگر CLI.

## شمارش نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر مهارت نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که دست‌کم یک نصب CLI را برای مهارت گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و دورسنجی خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را می‌بینند.

حذف حساب شما داده‌های دورسنجی شما را نیز حذف می‌کند.

## نحوهٔ غیرفعال‌سازی دورسنجی

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مقدار، CLI دورسنجی نصب را ارسال نخواهد کرد.
