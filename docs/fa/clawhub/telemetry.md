# دورسنجی

ClawHub از دورسنجی حداقلی CLI برای محاسبه شمار نصب‌های تجمیعی استفاده می‌کند.

## چه زمانی دورسنجی جمع‌آوری می‌شود

دورسنجی فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- دورسنجی **غیرفعال نشده باشد** (در ادامه، «نحوه غیرفعال‌سازی» را ببینید).

اگر وارد نشده باشید، چیزی گزارش نمی‌شود.

## چه چیزهایی جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب با بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ skill نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص بودن.

### چه چیزهایی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه.
- هیچ محتوای فایلی.
- هیچ گزارش هر اجرا، پرامپت، یا خروجی دیگر CLI.

## شمار نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر skill نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که دست‌کم یک نصب CLI را برای آن skill گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و دورسنجی خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های نصب تجمیعی** را می‌بینند.

حذف حساب شما همچنین داده‌های دورسنجی شما را حذف می‌کند.

## نحوه غیرفعال‌سازی دورسنجی

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مقدار، CLI دورسنجی نصب را ارسال نخواهد کرد.
