# تله‌متری

ClawHub از حداقل تله‌متری CLI برای محاسبه تعداد نصب‌های تجمیعی استفاده می‌کند.

## چه زمانی تله‌متری جمع‌آوری می‌شود

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوه غیرفعال کردن» را در ادامه ببینید).

اگر وارد نشده باشید، هیچ چیزی گزارش نمی‌شود.

## چه چیزهایی جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب بر پایه بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ مهارت نصب‌شده.
- `version`: نسخه نصب‌شده، در صورت مشخص بودن.

### چه چیزهایی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه.
- هیچ محتوای فایلی.
- هیچ گزارش هر اجرا، پرامپت، یا خروجی دیگر CLI.

## تعداد نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر مهارت نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که دست‌کم یک نصب CLI را برای مهارت گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های نصب تجمیعی** را می‌بینند.

حذف حساب شما، داده‌های تله‌متری شما را نیز حذف می‌کند.

## نحوه غیرفعال کردن تله‌متری

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مورد، CLI تله‌متری نصب را ارسال نخواهد کرد.
