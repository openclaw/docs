# تله‌متری

ClawHub از تله‌متری حداقلی CLI برای محاسبهٔ شمارش‌های تجمیعی نصب استفاده می‌کند.

## زمان جمع‌آوری تله‌متری

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «نحوهٔ غیرفعال‌سازی» را در پایین ببینید).

اگر وارد نشده باشید، هیچ چیزی گزارش نمی‌شود.

## آنچه جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب با بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ Skills نصب‌شده.
- `version`: نسخهٔ نصب‌شده، در صورت مشخص بودن.

### آنچه _جمع‌آوری نمی‌کنیم_

- هیچ مسیر پوشه یا شناسهٔ مشتق‌شده از پوشه.
- هیچ محتوای فایلی.
- هیچ لاگ هر اجرا، پرامپت، یا خروجی دیگر CLI.

## شمارش نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر Skills نگه می‌دارد:

- `installsAllTime`: کاربران یکتایی که دست‌کم یک نصب CLI را برای آن Skills گزارش کرده‌اند.
- `installsCurrent`: کاربران یکتایی که یک نصب را گزارش کرده‌اند و داده‌های تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های تجمیعی نصب** را می‌بینند.

حذف حساب شما داده‌های تله‌متری شما را نیز حذف می‌کند.

## نحوهٔ غیرفعال‌سازی تله‌متری

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مقدار، CLI تله‌متری نصب را ارسال نخواهد کرد.
