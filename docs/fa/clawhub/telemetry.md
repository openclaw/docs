# تله‌متری

ClawHub از تله‌متری حداقلی CLI برای محاسبه شمار نصب‌های تجمیعی استفاده می‌کند.

## چه زمانی تله‌متری جمع‌آوری می‌شود

تله‌متری فقط زمانی ارسال می‌شود که:

- در CLI وارد شده باشید.
- `clawhub install <slug>` را اجرا کنید.
- تله‌متری **غیرفعال نشده باشد** (بخش «چگونه غیرفعال کنیم» را در پایین ببینید).

اگر وارد نشده باشید، چیزی گزارش نمی‌شود.

## چه چیزهایی جمع‌آوری می‌کنیم

در هر `clawhub install` گزارش‌شده، CLI یک رویداد نصب با بهترین تلاش ارسال می‌کند.

این رویداد شامل موارد زیر است:

- `slug`: اسلاگ Skills نصب‌شده.
- `version`: نسخه نصب‌شده، وقتی مشخص باشد.

### چه چیزهایی را جمع‌آوری _نمی‌کنیم_

- هیچ مسیر پوشه یا شناسه مشتق‌شده از پوشه.
- هیچ محتوای فایلی.
- هیچ گزارش اجرای جداگانه، پرامپت، یا خروجی دیگر CLI.

## شمار نصب‌ها

ClawHub شمارنده‌های تجمیعی را برای هر Skills نگه می‌دارد:

- `installsAllTime`: کاربران منحصربه‌فردی که حداقل یک نصب CLI برای Skills گزارش کرده‌اند.
- `installsCurrent`: کاربران منحصربه‌فردی که یک نصب گزارش کرده‌اند و تله‌متری خود را حذف نکرده‌اند.

## شفافیت + کنترل‌های کاربر

همه فقط **شمارنده‌های نصب تجمیعی** را می‌بینند.

حذف حساب شما داده‌های تله‌متری شما را نیز حذف می‌کند.

## چگونه تله‌متری را غیرفعال کنیم

متغیر محیطی را تنظیم کنید:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

با تنظیم این مورد، CLI تله‌متری نصب را ارسال نخواهد کرد.
