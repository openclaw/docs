# احراز هویت

ClawHub برای ورود به وب از GitHub استفاده می‌کند. CLI از توکن‌های API ‏ClawHub استفاده می‌کند که از طریق آن حسابِ واردشده ایجاد شده‌اند.

## ورود به وب

برای ورود به [clawhub.ai](https://clawhub.ai) از GitHub استفاده کنید.

حساب‌های حذف‌شده، مسدودشده یا غیرفعال نمی‌توانند فرایند عادی ورود به ClawHub را تکمیل کنند. اگر پس از ورود دوباره به حالت خارج‌شده از حساب بازمی‌گردید، ممکن است حساب شما وضعیت مناسبی نداشته باشد. اگر حسابتان مسدود یا غیرفعال شده است و فکر می‌کنید این اتفاق اشتباهی رخ داده، از [فرم درخواست تجدیدنظر ClawHub](https://appeals.openclaw.ai/) استفاده کنید.

## ورود با CLI

فرایند پیش‌فرض ورود با CLI مرورگر شما را باز می‌کند:

```bash
clawhub login
clawhub whoami
```

آنچه رخ می‌دهد:

1. CLI یک سرور موقت فراخوانی برگشتی را روی `127.0.0.1` راه‌اندازی می‌کند.
2. مرورگر شما صفحه ورود ClawHub را باز می‌کند.
3. پس از ورود با GitHub، ‏ClawHub یک توکن API ایجاد می‌کند.
4. مرورگر به فراخوانی برگشتی محلی هدایت می‌شود.
5. CLI توکن را در فایل پیکربندی ClawHub شما ذخیره می‌کند.

اگر مرورگر شما به‌دلیل قواعد دیوار آتش، VPN یا پراکسی نمی‌تواند به فراخوانی برگشتی محلی دسترسی پیدا کند، از فرایند توکن بدون رابط گرافیکی استفاده کنید.

## ورود بدون رابط گرافیکی

یک توکن در رابط وب ClawHub ایجاد کنید، سپس آن را به CLI بدهید:

```bash
clawhub login --token clh_...
```

از این فرایند برای سرورها، کارهای CI یا محیط‌های صرفاً مبتنی بر پایانه استفاده کنید.

برای پوسته‌های راه‌دور که می‌توانید مرورگر را در جای دیگری باز کنید، دستور زیر را اجرا کنید:

```bash
clawhub login --device
```

CLI یک کد یک‌بارمصرف نمایش می‌دهد و منتظر می‌ماند تا آن را در `https://clawhub.ai/cli/device` مجاز کنید.

## ذخیره‌سازی توکن

مسیرهای پیش‌فرض پیکربندی:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` یا `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

مسیر را با دستور زیر تغییر دهید:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

برای راه‌اندازی CI، توکن ذخیره‌شده را با دستور زیر نمایش دهید:

```bash
clawhub token
```

## لغو اعتبار

می‌توانید توکن‌های API را در رابط وب ClawHub لغو کنید.

توکن‌های لغوشده، نامعتبر یا مفقود پاسخ `401 Unauthorized` را برمی‌گردانند. با `clawhub login` دوباره وارد شوید یا با `clawhub login --token` یک توکن تازه ارائه دهید.

حساب‌های حذف‌شده، مسدودشده یا غیرفعال نمی‌توانند به استفاده از توکن‌های API موجود ادامه دهند. اگر حسابتان مسدود یا غیرفعال شده است و فکر می‌کنید این اتفاق اشتباهی رخ داده، از [فرم درخواست تجدیدنظر ClawHub](https://appeals.openclaw.ai/) استفاده کنید.
