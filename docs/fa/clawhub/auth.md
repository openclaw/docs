# احراز هویت

ClawHub برای ورود وب از GitHub استفاده می‌کند. CLI از توکن‌های رابط برنامه‌نویسی کاربردی ClawHub استفاده می‌کند که
از طریق همان حساب واردشده ساخته می‌شوند.

## ورود وب

برای ورود، از GitHub در [clawhub.ai](https://clawhub.ai) استفاده کنید.

حساب‌های حذف‌شده، مسدودشده، یا غیرفعال نمی‌توانند ورود عادی ClawHub را کامل کنند.
اگر ورود شما را به وضعیت خارج‌شده برمی‌گرداند، ممکن است حساب شما در وضعیت مناسبی
نباشد.

## ورود CLI

جریان پیش‌فرض ورود CLI مرورگر شما را باز می‌کند:

```bash
clawhub login
clawhub whoami
```

چه اتفاقی می‌افتد:

1. CLI یک سرور فراخوان موقت روی `127.0.0.1` راه‌اندازی می‌کند.
2. مرورگر شما صفحه ورود ClawHub را باز می‌کند.
3. پس از ورود با GitHub، ClawHub یک توکن رابط برنامه‌نویسی کاربردی می‌سازد.
4. مرورگر به فراخوان محلی بازهدایت می‌شود.
5. CLI توکن را در فایل پیکربندی ClawHub شما ذخیره می‌کند.

اگر مرورگر شما به‌دلیل دیوار آتش، شبکه خصوصی مجازی، یا
قواعد پروکسی نتواند به فراخوان محلی دسترسی پیدا کند، از جریان توکن بدون واسط گرافیکی استفاده کنید.

## ورود بدون واسط گرافیکی

یک توکن در رابط کاربری وب ClawHub بسازید، سپس آن را به CLI بدهید:

```bash
clawhub login --token clh_...
```

از این جریان برای سرورها، کارهای یکپارچه‌سازی مداوم، یا محیط‌های فقط ترمینال استفاده کنید.

برای پوسته‌های راه دور که می‌توانید مرورگری را در جای دیگری باز کنید، اجرا کنید:

```bash
clawhub login --device
```

CLI یک کد یک‌بارمصرف چاپ می‌کند و منتظر می‌ماند تا شما آن را در
`https://clawhub.ai/cli/device` مجاز کنید.

## ذخیره‌سازی توکن

مسیرهای پیش‌فرض پیکربندی:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` یا `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

مسیر را با این مورد بازنویسی کنید:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

## ابطال

می‌توانید توکن‌های رابط برنامه‌نویسی کاربردی را در رابط کاربری وب ClawHub باطل کنید.

توکن‌های باطل‌شده، نامعتبر، یا موجودنبودن توکن، `401 Unauthorized` برمی‌گردانند. دوباره
با `clawhub login` وارد شوید یا با `clawhub login --token` یک توکن تازه ارائه کنید.

حساب‌های حذف‌شده، مسدودشده، یا غیرفعال نمی‌توانند به استفاده از توکن‌های رابط برنامه‌نویسی کاربردی موجود ادامه دهند.
