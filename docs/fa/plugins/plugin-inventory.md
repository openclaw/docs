# موجودی Plugin

این صفحه از `extensions/*/package.json`، `openclaw.plugin.json`،
و موارد مستثناشده‌ی `files` در بسته‌ی npm ریشه تولید می‌شود. آن را با دستور زیر دوباره تولید کنید:

```bash
pnpm plugins:inventory:gen
```

## تعریف‌ها

- **بسته‌ی npm هسته:** در بسته‌ی npm `openclaw` ساخته شده و بدون نصب جداگانه‌ی Plugin در دسترس است.
- **بسته‌ی رسمی خارجی:** Plugin تحت نگهداری OpenClaw که از بسته‌ی npm هسته حذف شده و از طریق ClawHub و/یا npm نصب می‌شود.
- **فقط checkout منبع:** Plugin محلیِ مخزن که از آرتیفکت‌های npm منتشرشده حذف شده و به‌عنوان بسته‌ی قابل نصب معرفی نمی‌شود.

checkoutهای منبع با نصب‌های npm متفاوت‌اند: پس از `pnpm install`، Pluginهای
همراه از `extensions/<id>` بارگذاری می‌شوند تا ویرایش‌های محلی و وابستگی‌های workspace
محلیِ بسته در دسترس باشند.
