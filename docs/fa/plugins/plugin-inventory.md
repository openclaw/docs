# فهرست Plugin

این صفحه از `extensions/*/package.json`، `openclaw.plugin.json`،
و استثناهای `files` در بستهٔ npm ریشه تولید می‌شود. آن را با دستور زیر دوباره تولید کنید:

```bash
pnpm plugins:inventory:gen
```

## تعریف‌ها

- **بستهٔ اصلی npm:** در بستهٔ npm با نام `openclaw` ساخته شده و بدون نصب جداگانهٔ Plugin در دسترس است.
- **بستهٔ خارجی رسمی:** Plugin نگهداری‌شده توسط OpenClaw که از بستهٔ اصلی npm حذف شده و از طریق ClawHub و/یا npm نصب می‌شود.
- **فقط برداشت کد منبع:** Plugin محلیِ مخزن که از مصنوعات منتشرشدهٔ npm حذف شده و به‌عنوان بستهٔ قابل نصب معرفی نمی‌شود.

برداشت‌های کد منبع با نصب‌های npm متفاوت هستند: پس از `pnpm install`، Pluginهای همراه
از `extensions/<id>` بارگذاری می‌شوند تا ویرایش‌های محلی و وابستگی‌های فضای کاریِ محلیِ بسته
در دسترس باشند.
