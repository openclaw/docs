OpenClaw Gateway پایدار را روی [Hostinger](https://www.hostinger.com/openclaw) از طریق استقرار مدیریت‌شده **۱-کلیکی** یا نصب روی **VPS** اجرا کنید.

## پیش‌نیازها

- حساب Hostinger ([ثبت‌نام](https://www.hostinger.com/openclaw))
- حدود ۵ تا ۱۰ دقیقه

## گزینه A: OpenClaw ۱-کلیکی

سریع‌ترین راه برای شروع. Hostinger زیرساخت، Docker و به‌روزرسانی‌های خودکار را مدیریت می‌کند.

<Steps>
  <Step title="خرید و راه‌اندازی">
    1. از [صفحه OpenClaw در Hostinger](https://www.hostinger.com/openclaw)، یک طرح OpenClaw مدیریت‌شده انتخاب کنید و فرایند پرداخت را کامل کنید.

    <Note>
    هنگام پرداخت می‌توانید اعتبارهای **هوش مصنوعی آماده استفاده** را انتخاب کنید که از پیش خریداری شده‌اند و بلافاصله در OpenClaw یکپارچه می‌شوند -- بدون نیاز به حساب‌های خارجی یا کلیدهای API از ارائه‌دهندگان دیگر. می‌توانید بلافاصله گفتگو را شروع کنید. همچنین می‌توانید هنگام راه‌اندازی، کلید خودتان را از Anthropic، OpenAI، Google Gemini یا xAI وارد کنید.
    </Note>

  </Step>

  <Step title="انتخاب کانال پیام‌رسانی">
    یک یا چند کانال را برای اتصال انتخاب کنید:

    - **WhatsApp** -- کد QR نمایش‌داده‌شده در راهنمای راه‌اندازی را اسکن کنید.
    - **Telegram** -- توکن ربات را از [BotFather](https://t.me/BotFather) جای‌گذاری کنید.

  </Step>

  <Step title="تکمیل نصب">
    برای استقرار نمونه، روی **پایان** کلیک کنید. پس از آماده شدن، از **نمای کلی OpenClaw** در hPanel به داشبورد OpenClaw دسترسی پیدا کنید.
  </Step>

</Steps>

## گزینه B: OpenClaw روی VPS

کنترل بیشتر روی سرور شما. Hostinger، OpenClaw را از طریق Docker روی VPS شما مستقر می‌کند و شما آن را از طریق **مدیر Docker** در hPanel مدیریت می‌کنید.

<Steps>
  <Step title="خرید VPS">
    1. از [صفحه OpenClaw در Hostinger](https://www.hostinger.com/openclaw)، یک طرح OpenClaw روی VPS انتخاب کنید و فرایند پرداخت را کامل کنید.

    <Note>
    می‌توانید هنگام پرداخت اعتبارهای **هوش مصنوعی آماده استفاده** را انتخاب کنید -- این اعتبارها از پیش خریداری شده‌اند و بلافاصله در OpenClaw یکپارچه می‌شوند، بنابراین می‌توانید بدون هیچ حساب خارجی یا کلید API از ارائه‌دهندگان دیگر، گفتگو را شروع کنید.
    </Note>

  </Step>

  <Step title="پیکربندی OpenClaw">
    پس از آماده‌سازی VPS، فیلدهای پیکربندی را پر کنید:

    - **توکن Gateway** -- به‌صورت خودکار تولید می‌شود؛ آن را برای استفاده بعدی ذخیره کنید.
    - **شماره WhatsApp** -- شماره شما همراه با کد کشور (اختیاری).
    - **توکن ربات Telegram** -- از [BotFather](https://t.me/BotFather) (اختیاری).
    - **کلیدهای API** -- فقط در صورتی لازم است که هنگام پرداخت، اعتبارهای هوش مصنوعی آماده استفاده را انتخاب نکرده باشید.

  </Step>

  <Step title="شروع OpenClaw">
    روی **استقرار** کلیک کنید. پس از اجرا، داشبورد OpenClaw را از hPanel با کلیک روی **باز کردن** باز کنید.
  </Step>

</Steps>

لاگ‌ها، راه‌اندازی‌های مجدد و به‌روزرسانی‌ها مستقیماً از رابط مدیر Docker در hPanel مدیریت می‌شوند. برای به‌روزرسانی، در مدیر Docker روی **به‌روزرسانی** بزنید تا آخرین تصویر دریافت شود.

## تأیید راه‌اندازی

در کانالی که متصل کرده‌اید، برای دستیار خود «Hi» بفرستید. OpenClaw پاسخ می‌دهد و شما را در تنظیم ترجیحات اولیه راهنمایی می‌کند.

## عیب‌یابی

**داشبورد بارگذاری نمی‌شود** -- چند دقیقه صبر کنید تا آماده‌سازی کانتینر تمام شود. لاگ‌های مدیر Docker را در hPanel بررسی کنید.

**کانتینر Docker مدام راه‌اندازی مجدد می‌شود** -- لاگ‌های مدیر Docker را باز کنید و به دنبال خطاهای پیکربندی بگردید (توکن‌های گمشده، کلیدهای API نامعتبر).

**ربات Telegram پاسخ نمی‌دهد** -- پیام کد جفت‌سازی خود را از Telegram مستقیماً به‌صورت یک پیام داخل گفتگوی OpenClaw خود ارسال کنید تا اتصال کامل شود.
