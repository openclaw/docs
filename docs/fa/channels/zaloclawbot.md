OpenClaw از طریق Plugin خارجی فهرست‌شده در کاتالوگ
`@zalo-platforms/openclaw-zaloclawbot` به Zalo ClawBot متصل می‌شود. ورود با کد QR
یک Zalo Mini App انجام می‌شود.

## سازگاری

| نسخه Plugin | نسخه OpenClaw | npm dist-tag | وضعیت        |
| -------------- | ---------------- | ------------ | ------------- |
| 0.1.x          | >=2026.4.10      | `latest`     | فعال / بتا |

## پیش‌نیازها

- Node.js **>= 22**
- [OpenClaw](https://docs.openclaw.ai/install) باید نصب شده باشد (`openclaw` CLI در دسترس باشد).
- یک حساب Zalo روی دستگاه موبایل برای اسکن کد QR ورود.

## نصب با onboard (توصیه‌شده)

جادوگر راه‌اندازی OpenClaw را اجرا کنید و از منوی کانال، **Zalo ClawBot** را انتخاب کنید:

```bash
openclaw onboard
```

جادوگر، Plugin را از کاتالوگ رسمی نصب می‌کند (با صحت‌سنجی یکپارچگی)، کد QR ورود را مستقیماً در ترمینال نمایش می‌دهد، و پس از اسکن آن با برنامه Zalo، راه‌اندازی کانال را تکمیل می‌کند. هیچ دستور اضافه‌ای لازم نیست.

## نصب دستی

برای افزودن کانال به Gatewayی که قبلاً راه‌اندازی شده است، این مراحل را دنبال کنید:

### 1. Plugin را نصب کنید

```bash
openclaw plugins install "@zalo-platforms/openclaw-zaloclawbot@0.1.4"
```

از نسخه دقیق پین‌شده‌ای که در بالا نشان داده شده استفاده کنید (با ورودی کاتالوگ رسمی مطابقت دارد)، تا OpenClaw هنگام نصب، بسته را با هش یکپارچگی کاتالوگ بررسی کند.

### 2. Plugin را در پیکربندی فعال کنید

```bash
openclaw config set plugins.entries.openclaw-zaloclawbot.enabled true
```

### 3. کد QR را ایجاد کنید و وارد شوید

```bash
openclaw channels login --channel openclaw-zaloclawbot
```

کد QR نمایش‌داده‌شده در ترمینال را با برنامه موبایل Zalo اسکن کنید، شرایط استفاده را داخل Zalo Mini App بپذیرید، و نشست را مجاز کنید.

### 4. Gateway را راه‌اندازی مجدد کنید

```bash
openclaw gateway restart
```

## نحوه کارکرد

برخلاف کانال استاندارد توسعه‌دهنده Zalo که از شما می‌خواهد حساب رسمی Zalo (OA) خودتان را ثبت کنید و اعتبارنامه‌های ثابت توسعه‌دهنده را وارد کنید، Zalo ClawBot با استفاده از زیرساخت رسمی و مشترک، به‌عنوان یک **دستیار شخصی وابسته به مالک** کار می‌کند:

1. **راه‌اندازی امن:** کد QR به یک Zalo Mini App امن منتهی می‌شود که یک ربات خصوصی تازه تأمین‌شده را، زیر یک OA رسمی مشترک، مستقیماً به شناسه کاربری Zalo شما متصل می‌کند.
2. **حریم خصوصی وابسته به مالک:** طبق طراحی، ربات فقط مجاز است با مالک خود ارتباط برقرار کند. پیام‌های کاربران دیگر در سطح پلتفرم کنار گذاشته می‌شوند، که اتصال را خصوصی و امن می‌کند.
3. **مسیر API رسمی:** این Plugin به‌جای خودکارسازی مرورگر یا نشست وب، از APIهای Zalo Bot Platform استفاده می‌کند.

## پشت صحنه

Plugin مربوط به Zalo ClawBot از طریق یک حلقه پیام پایدار مبتنی بر نظرسنجی طولانی با APIهای Zalo ارتباط برقرار می‌کند. برای حفظ زمان اجرای سبک و تمیز:

- اتصال‌های نظرسنجی طولانی از endpoint `getUpdates` استفاده می‌کنند.
- Webhookها به‌طور پیش‌فرض برای اجرای Gateway محلی دسکتاپ/ترمینال غیرفعال هستند.
- پیام‌ها در سمت کلاینت پردازش می‌شوند و مستقیماً به زمان اجرای عامل محلی شما نگاشت می‌شوند.

Plugin خارجی، اعتبارنامه‌های ربات را زیر دایرکتوری وضعیت OpenClaw مدیریت می‌کند.
با آن دایرکتوری به‌عنوان داده حساس رفتار کنید و آن را در همان سیاست کنترل دسترسی و
پشتیبان‌گیریِ سایر وضعیت OpenClaw قرار دهید.

## عیب‌یابی

- **پایان مهلت ورود QR:** توکن ورود (`zbsk`) به دلایل امنیتی پس از 5 دقیقه منقضی می‌شود. اگر کد QR پیش از اسکن منقضی شد، کافی است دستور ورود را دوباره اجرا کنید تا کد جدیدی ایجاد شود.
- **Gateway بارگذاری نمی‌شود:** مطمئن شوید نسخه میزبان OpenClaw شما `2026.4.10` یا بالاتر است. نسخه‌های قدیمی‌تر از دفتر ثبت نصب Plugin خارجی npm پشتیبانی نمی‌کنند.
