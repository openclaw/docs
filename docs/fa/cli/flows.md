# `openclaw tasks flow`

دستورهای Flow زیر‌دستورهای `openclaw tasks` هستند، نه یک دستور مستقل `flows`.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

برای مستندات کامل، [TaskFlow](/fa/automation/taskflow) و [مرجع CLI برای tasks](/fa/cli/tasks) را ببینید.
