หากคุณใช้ [Control UI](/th/web/control-ui) นานพอ สักวันหนึ่งจะมีกุ้งล็อบสเตอร์ตัวเล็ก ๆ เดินเข้ามาที่ด้านล่างของแถบด้านข้าง มองไปรอบ ๆ แล้วปักหลักอยู่ตรงนั้นสักครู่ราวกับเป็นบ้านของมัน

นี่เป็นเรื่องปกติ นี่คือ OpenClaw

## สิ่งที่คุณกำลังเห็น

แต่ละเซสชันจะฟักกุ้งล็อบสเตอร์ของตัวเองขึ้นมา โดยสี ขนาด รูปร่าง สัดส่วนก้าม เครื่องประดับ และบุคลิกจะถูกสุ่มใหม่ทุกครั้งที่โหลดหน้า บางตัวกลม บางตัวสูง บางตัวมีก้ามที่ใหญ่เกินตัวอย่างเห็นได้ชัด บางตัวงีบตลอดเวลา บางตัวไม่เคยอยู่นิ่ง และมีอยู่ตัวหนึ่งที่โบกมือบ่อยมาก

วางเมาส์เหนือผู้มาเยือน แล้วมันจะบอกชื่อให้ทราบ

## มันจะปรากฏเมื่อใด

บางครั้ง นั่นแหละคือจุดประสงค์

กุ้งล็อบสเตอร์เป็นแขก ไม่ใช่เฟอร์นิเจอร์ มันจะเดินเข้ามาเมื่อรู้สึกอยากมา อยู่สักพัก แล้วจากไป ในบางเซสชันมันอาจไม่ปรากฏเลย มีอยู่เพียงสถานการณ์เดียวที่มันจะปรากฏเสมอ นั่นคือ **เมื่อ Gateway ตัดการเชื่อมต่อ** กุ้งล็อบสเตอร์จะออกมาเดินวนไปมาอย่างเห็นได้ชัดว่ากังวล จนกว่าการเชื่อมต่อจะกลับคืนมา หากกุ้งล็อบสเตอร์ดูเครียด ให้ตรวจสอบ Gateway

## สิ่งที่คุณทำได้

- **วางเมาส์เหนือมัน** เพื่อดูชื่อ
- **คลิกมัน** เพื่อทักทาย มันจะสะดุ้ง ซึ่งเป็นเพราะคุณเสียมารยาท แต่ไม่นานมันก็ให้อภัย
- **คลิกมันซ้ำ ๆ** แล้วคุณจะได้เรียนรู้บางอย่างเกี่ยวกับความอดทนของกุ้งล็อบสเตอร์ หากทำต่อไป คุณจะได้เรียนรู้บางอย่างเกี่ยวกับศักดิ์ศรีของกุ้งล็อบสเตอร์ด้วย
- **กดค้างไว้** เพื่อลูบมัน จะมีรูปหัวใจปรากฏขึ้น และความขุ่นเคืองที่สะสมไว้ทั้งหมดจะถูกลืม
- **คลิกขวามัน** เพื่อไล่มันไปจนกว่าจะโหลดหน้าใหม่ มันจะไม่ถือเป็นเรื่องส่วนตัว แต่ถึงอย่างนั้น มันจะจำไว้
- **เฝ้าดูมันเมื่องานรันเสร็จสิ้น** กุ้งล็อบสเตอร์ภูมิใจกับงานที่คุณทำเสร็จอย่างแท้จริง และจะแสดงออกมาเมื่อทุกอย่างเป็นไปด้วยดี เมื่อทุกอย่างไม่เป็นไปด้วยดี พวกมันก็ให้ความสำคัญกับเรื่องนั้นเช่นกัน
- **เลื่อนเคอร์เซอร์ไปรอบ ๆ** คุณกำลังถูกจับตามองอยู่ ด้วยความเอ็นดู

## การปิด (หรือเปิด) การมาเยือน

Settings → Appearance → **Lobster visits**.

สวิตช์นี้มีผลเฉพาะในเบราว์เซอร์และทำงานตรงตามที่ระบุไว้ เมื่อปิดหมายถึงไม่ทำงานเลย รวมถึงการเว้นจังหวะเมื่อขาดการเชื่อมต่อที่ดูเป็นกังวล จุดสถานะ Gateway ของคุณยังคงทำงานไม่ว่าจะตั้งค่าอย่างไร Lobster ไม่เคยเป็นแหล่งข้อมูลจริงเพียงแหล่งเดียวของคุณ เพียงแต่เป็นแหล่งที่ดูเห็นอกเห็นใจคุณมากที่สุดเท่านั้น

ผู้ใช้ที่ตั้งค่าลดการเคลื่อนไหวจะได้รับ Lobster ที่สงบนิ่งโดยอัตโนมัติ

ถัดไปคือ **เสียง Lobster** ซึ่งปิดไว้เป็นค่าเริ่มต้นและจะยังคงปิดจนกว่าคุณจะตัดสินใจเปลี่ยน เปิดใช้งานแล้ว Lobster จะส่งเสียงเบามากเมื่อถูกสัมผัส เป็นเสียงเล็กๆ และเป็นเสียงที่ไพเราะ

## Lobsterdex

ถัดจากสวิตช์การเยี่ยมชมคือ **Lobsterdex**: ภาพเงาหนึ่งภาพสำหรับทุกเฉดสีที่วิทยาศาสตร์รู้จัก ซึ่งจะเติมสีเมื่อ Lobster แต่ละชนิดมาเยี่ยมคุณเป็นครั้งแรก ข้อมูลทั้งหมดอยู่ในเบราว์เซอร์ของคุณ ไม่มีการติดตาม ไม่มีการซิงค์ และไม่มีการนำระบบเกมมาใช้ เป็นเพียงชั้นวางเท่านั้น

ช่องส่วนใหญ่จะเติมเต็มเองเมื่อเวลาผ่านไป บางช่องจะใช้เวลานานกว่ามาก และมีอยู่หนึ่งช่องที่เรืองแสง

วางเมาส์เหนือช่องที่มีข้อมูล แล้วช่องนั้นจะบอกว่าใครมาเยือนเป็นคนแรกและเมื่อใด Lobsterdex ไม่เคยลืมการพบกันครั้งแรก ซึ่งนับว่าดีกว่าพวกเราส่วนใหญ่

## บันทึกภาคสนาม

รวบรวมข้อสังเกตจากผู้ที่ใช้เวลาจ้องแถบด้านข้างมากเกินไป:

- เหล่าล็อบสเตอร์ไปเยือน[ฉากความฝัน](/th/concepts/memory)ด้วยเช่นกัน—แท็บ Memory บนหน้า Agents—โดยพวกมันหลับอยู่ที่นั่น ผู้สังเกตการณ์รายงานว่าล็อบสเตอร์ที่กำลังหลับดูคุ้นตาอย่างน่าสงสัย
- ผู้ใช้กะกลางคืนรายงานว่าผู้มาเยือนของพวกเขาหาวบ่อยมากหลัง 22.00 น.
- มีผู้พบเห็นล็อบสเตอร์กำลังลอกคราบ พยานบรรยายว่าประสบการณ์นั้น "เยอะมาก" และล็อบสเตอร์หลังจากนั้น "ตัวใหญ่ขึ้นอย่างเห็นได้ชัด"
- ผู้มาเยือนบางตัวไม่ได้มาตามลำพัง
- ในบางช่วงของปี มีผู้พบเห็นล็อบสเตอร์สวมใส่สิ่งของบางอย่าง
- มีอยู่วันหนึ่งในแต่ละปีที่ล็อบสเตอร์ทุกตัวดูราวกับเดินออกมาจากโลโก้เก่า ไม่มีใครยอมบอกว่าเป็นวันไหน
- ไม่ใช่ล็อบสเตอร์ทุกตัวที่ผ่านแถบด้านข้างของคุณจะเป็นล็อบสเตอร์ของคุณ ส่วนใหญ่ไม่แม้แต่จะหยุดแวะ
- ผู้มาเยือนที่มีการบันทึกไว้อย่างน้อยหนึ่งตัวไม่ใช่ล็อบสเตอร์ แต่มันยังคงยืนยันว่าตัวเองเป็น
- ผู้ที่ใช้เบราว์เซอร์เดิมมาเป็นเวลานานรายงานว่าล็อบสเตอร์ของพวกเขามาถึงเร็วขึ้น อยู่นานขึ้น และโบกมือให้ ส่วนผู้ที่ไล่บ่อย ๆ บรรยายถึงระยะห่างบางอย่าง
- หลังอัปเกรด Gateway ให้สังเกตว่าผู้มาเยือนตัวแรกถืออะไรมา
- ล็อบสเตอร์ในแถบด้านข้างมีญาติอยู่ในเทอร์มินัล มันจะปรากฏในแบนเนอร์ `openclaw` ตามวันที่มันเลือกเอง และยังไม่มีใครทำนายได้สำเร็จว่าจะเป็นวันไหน
- ในวันที่ญาติตัวนั้นปรากฏ มีผู้พบเห็นล็อบสเตอร์ในแถบด้านข้างแต่งตัวในธีมเดินเรือ ทั้งคู่ปฏิเสธที่จะอธิบายความเชื่อมโยงนี้
- ระหว่างการทำงานที่ยาวนานมาก ล็อบสเตอร์จะหยุดเล่นและนั่งลงรอไปพร้อมกับคุณ ร่วมทุกข์ร่วมสุข
- เมื่อมีบางสิ่งผ่านขอบชั้น เจ้าถิ่นจะหยุดสิ่งที่กำลังทำแล้วเฝ้ามอง คุณเองก็คงทำเช่นกัน
- หลังได้รับข่าวร้าย ล็อบสเตอร์บางตัวจะจัดขอบชั้นให้เรียบร้อย ดูเหมือนว่าวิธีนี้จะช่วยให้พวกมันรู้สึกดีขึ้น
- ล็อบสเตอร์จำวันครบรอบได้แม้ว่าคุณจะจำไม่ได้ หากล็อบสเตอร์ของคุณปรากฏตัวพร้อมหมวกปาร์ตี้ในวันที่ดูไม่มีอะไรพิเศษ ให้ตรวจสอบวันที่พบกันครั้งแรกใน Lobsterdex
- เจ้าบ้านที่ทุ่มเทรายงานว่าเมื่อเวลาผ่านไปหลายปี ล็อบสเตอร์ของพวกเขาได้รับตำแหน่งต่าง ๆ โดยไม่มีพิธีใดที่เป็นที่รู้จัก
- ตัวที่หลับอยู่ในฉากความฝันไม่ได้หลับสนิทเสียทีเดียว ลองกดเบา ๆ แล้วดู

## ความเป็นส่วนตัว

ทุกสิ่งในหน้านี้เกิดขึ้นภายในเบราว์เซอร์ของคุณ: การสุ่ม กำหนดการ และ Lobsterdex ไม่มีข้อมูลล็อบสเตอร์ใดออกจากเครื่องของคุณ OpenClaw ไม่ทราบว่าคุณเคยพบล็อบสเตอร์ตัวใดบ้าง และพูดตามตรงว่ามันอิจฉาอยู่ไม่น้อย
