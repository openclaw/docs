# บัญชีรายการ Plugin

หน้านี้สร้างจาก `extensions/*/package.json`, `openclaw.plugin.json`,
และการยกเว้น `files` ของแพ็กเกจ npm ราก สร้างใหม่ได้ด้วย:

```bash
pnpm plugins:inventory:gen
```

## คำจำกัดความ

- **แพ็กเกจ npm หลัก:** รวมอยู่ในแพ็กเกจ npm `openclaw` และพร้อมใช้งานโดยไม่ต้องติดตั้ง plugin แยกต่างหาก
- **แพ็กเกจภายนอกอย่างเป็นทางการ:** plugin ที่ดูแลโดย OpenClaw ซึ่งไม่รวมอยู่ในแพ็กเกจ npm หลัก และติดตั้งผ่าน ClawHub และ/หรือ npm
- **เฉพาะซอร์สเช็กเอาต์:** plugin ภายใน repo ที่ไม่รวมอยู่ในอาร์ติแฟกต์ npm ที่เผยแพร่ และไม่ได้โฆษณาว่าเป็นแพ็กเกจที่ติดตั้งได้

ซอร์สเช็กเอาต์แตกต่างจากการติดตั้ง npm: หลังจาก `pnpm install` แล้ว plugin
ที่รวมมาจะโหลดจาก `extensions/<id>` เพื่อให้การแก้ไขภายในเครื่องและ dependency
ของ workspace ภายในแพ็กเกจพร้อมใช้งาน
