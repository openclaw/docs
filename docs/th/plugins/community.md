Plugin ชุมชนคือแพ็กเกจจากบุคคลที่สามที่ขยายความสามารถของ OpenClaw ด้วย
ช่องทาง เครื่องมือ ผู้ให้บริการ ฮุก หรือความสามารถอื่น ๆ ใช้
[ClawHub](/clawhub) เป็นช่องทางหลักในการค้นหา Plugin ชุมชนสาธารณะ

## ค้นหา Plugin

ค้นหา ClawHub จาก CLI:

```bash
openclaw plugins search "calendar"
```

ติดตั้ง Plugin จาก ClawHub โดยระบุคำนำหน้าแหล่งที่มาอย่างชัดเจน:

```bash
openclaw plugins install clawhub:<package-name>
```

npm ยังคงเป็นช่องทางติดตั้งโดยตรงที่รองรับในช่วงเปลี่ยนผ่านการเปิดตัว:

```bash
openclaw plugins install npm:<package-name>
```

ใช้ [จัดการ Plugin](/th/plugins/manage-plugins) สำหรับตัวอย่างทั่วไปในการติดตั้ง อัปเดต
ตรวจสอบ และถอนการติดตั้ง ใช้ [`openclaw plugins`](/th/cli/plugins) สำหรับ
เอกสารอ้างอิงคำสั่งทั้งหมดและกฎการเลือกแหล่งที่มา

## เผยแพร่ Plugin

เผยแพร่ Plugin ชุมชนสาธารณะบน ClawHub เพื่อให้ผู้ใช้ OpenClaw สามารถค้นพบ
และติดตั้งได้ ClawHub ดูแลรายการแพ็กเกจที่ใช้งานจริง ประวัติรุ่นเผยแพร่
สถานะการสแกน และคำแนะนำในการติดตั้ง ส่วนเอกสารจะไม่ดูแลแค็ตตาล็อก
Plugin จากบุคคลที่สามแบบคงที่

```bash
clawhub package publish your-org/your-plugin --dry-run
clawhub package publish your-org/your-plugin
```

ก่อนเผยแพร่ ตรวจสอบให้แน่ใจว่า Plugin มีข้อมูลเมตาของแพ็กเกจ แมนิเฟสต์ของ Plugin
เอกสารการตั้งค่า และผู้รับผิดชอบการบำรุงรักษาที่ชัดเจน ClawHub จะตรวจสอบขอบเขตของเจ้าของ
ชื่อแพ็กเกจ เวอร์ชัน ขีดจำกัดไฟล์ และข้อมูลเมตาของแหล่งที่มาก่อน
สร้างรุ่นเผยแพร่ จากนั้นจะซ่อนรุ่นเผยแพร่ใหม่จากช่องทางติดตั้งและ
ดาวน์โหลดตามปกติจนกว่าการตรวจทานและการยืนยันจะเสร็จสิ้น

รายการตรวจสอบก่อนเผยแพร่:

| ข้อกำหนด             | เหตุผล                                                        |
| -------------------- | ------------------------------------------------------------- |
| เผยแพร่บน ClawHub    | ผู้ใช้ต้องใช้คำแนะนำ `openclaw plugins install` ได้           |
| ที่เก็บ GitHub สาธารณะ | การตรวจสอบซอร์ส การติดตามปัญหา และความโปร่งใส                 |
| เอกสารการตั้งค่าและการใช้งาน | ผู้ใช้ต้องทราบวิธีกำหนดค่า                              |
| การบำรุงรักษาอย่างต่อเนื่อง | มีการอัปเดตล่าสุดหรือการจัดการปัญหาอย่างรวดเร็ว           |

ข้อกำหนดการเผยแพร่ฉบับเต็ม:

- [การเผยแพร่บน ClawHub](/th/clawhub/publishing) - เจ้าของ ขอบเขต รุ่นเผยแพร่
  การตรวจทาน การตรวจสอบความถูกต้องของแพ็กเกจ และการโอนแพ็กเกจ
- [การสร้าง Plugin](/th/plugins/building-plugins) - โครงสร้างแพ็กเกจ Plugin
  และขั้นตอนการเผยแพร่ครั้งแรก
- [แมนิเฟสต์ของ Plugin](/th/plugins/manifest) - ฟิลด์แมนิเฟสต์แบบเนทีฟของ Plugin
