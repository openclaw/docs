# ข้อมูลอ้างอิง Plugin

หน้านี้สร้างจาก `extensions/*/package.json` และ
`openclaw.plugin.json` สร้างใหม่ด้วย:

```bash
pnpm plugins:inventory:gen
```

ใช้ [รายการ Plugin](/th/plugins/plugin-inventory) เพื่อเรียกดูหน้าอ้างอิง Plugin ที่สร้างขึ้นทั้งหมด 130 หน้า ตามการแจกจ่าย แพ็กเกจ และคำอธิบาย
