# ข้อมูลอ้างอิง Plugin

หน้านี้สร้างขึ้นจาก `extensions/*/package.json` และ
`openclaw.plugin.json` สร้างใหม่ด้วย:

```bash
pnpm plugins:inventory:gen
```

ใช้ [รายการ Plugin](/th/plugins/plugin-inventory) เพื่อเรียกดูหน้าอ้างอิง Plugin
