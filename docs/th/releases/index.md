# บันทึกประจำรุ่น

หน้าเหล่านี้เป็นบันทึกประจำรุ่นของ OpenClaw ฉบับสำหรับผู้อ่าน โดยรวบรวม
เรื่องราวหลัก ผลกระทบในทางปฏิบัติ ลิงก์แหล่งที่มา และการให้เครดิตผู้มีส่วนร่วมไว้ด้วยกัน
โดยไม่ต้องไล่อ่านบันทึกการเปลี่ยนแปลงดิบก่อน

## รุ่นที่เผยแพร่

- [v2026.7.1](/releases/2026.7.1) - การปรับปรุง Control UI และขั้นตอนการเริ่มต้นใช้งานครั้งใหญ่
  การอัปเดต iOS และ Android ครั้งสำคัญ การรองรับโมเดลที่กว้างขึ้น และการกู้คืน Gateway
- [v2026.6.11](/th/releases/2026.6.11) - การแก้ไขความน่าเชื่อถือสำหรับการตอบกลับ การส่ง
  การเชื่อมต่อใหม่ การตั้งค่าโมเดล เซสชัน และค่าเริ่มต้นสำหรับผู้ดูแลระบบที่ปลอดภัยยิ่งขึ้น
