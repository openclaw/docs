# TOOLS.md - บันทึกเฉพาะเครื่อง

Skills กำหนดว่าเครื่องมือทำงาน _อย่างไร_ ไฟล์นี้มีไว้สำหรับรายละเอียด _เฉพาะของคุณ_ ซึ่งเป็นสิ่งที่มีลักษณะเฉพาะสำหรับการตั้งค่าของคุณ เช่น ชื่อและตำแหน่งกล้อง โฮสต์และนามแฝง SSH เสียง TTS ที่ต้องการ ชื่อลำโพง/ห้อง ชื่อเล่นของอุปกรณ์ และรายละเอียดอื่น ๆ ที่เฉพาะเจาะจงกับสภาพแวดล้อม

## ตัวอย่าง

```markdown
### กล้อง

- living-room → พื้นที่หลัก มุมกว้าง 180°
- front-door → ทางเข้า ทำงานเมื่อตรวจพบการเคลื่อนไหว

### SSH

- home-server → 192.168.1.100, ผู้ใช้: admin

### TTS

- เสียงที่ต้องการ: "Nova" (อบอุ่น สำเนียงอังกฤษเล็กน้อย)
- ลำโพงเริ่มต้น: Kitchen HomePod
```

## เหตุใดจึงแยกไว้ต่างหาก

Skills ใช้ร่วมกันได้ ส่วนการตั้งค่าของคุณเป็นของคุณเอง การแยกทั้งสองส่วนออกจากกันช่วยให้คุณอัปเดต Skills ได้โดยไม่สูญเสียบันทึก และแบ่งปัน Skills ได้โดยไม่เปิดเผยโครงสร้างพื้นฐานของคุณ
