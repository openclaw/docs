# TOOLS.md - บันทึกเครื่องมือของผู้ใช้ (แก้ไขได้)

ไฟล์นี้มีไว้สำหรับบันทึกของ_คุณ_เกี่ยวกับเครื่องมือภายนอกและแนวปฏิบัติ ไฟล์นี้ไม่ได้กำหนดว่ามีเครื่องมือใดบ้าง OpenClaw มีเครื่องมือในตัวให้ใช้ภายใน และ Skills จะเพิ่มเครื่องมือส่วนที่เหลือ

## ตัวอย่าง

### imsg

- ส่ง iMessage/SMS: ระบุผู้รับและเนื้อหา แล้วขอการยืนยันก่อนส่ง
- ควรใช้ข้อความสั้น และหลีกเลี่ยงการส่งข้อมูลลับ

### sag

- การแปลงข้อความเป็นเสียงพูด: ระบุเสียง ผู้ฟังหรือห้องเป้าหมาย และต้องการสตรีมหรือไม่

เพิ่มข้อมูลอื่นใดที่คุณต้องการให้ผู้ช่วยทราบเกี่ยวกับชุดเครื่องมือภายในระบบของคุณ
