# TOOLS.md - บันทึกเครื่องมือของผู้ใช้ (แก้ไขได้)

ไฟล์นี้มีไว้สำหรับบันทึก _ของคุณเอง_ เกี่ยวกับเครื่องมือภายนอกและแนวทางปฏิบัติ
ไฟล์นี้ไม่ได้กำหนดว่ามีเครื่องมือใดบ้าง; OpenClaw มีเครื่องมือในตัวให้ภายในอยู่แล้ว

## ตัวอย่าง

### imsg

- ส่ง iMessage/SMS: อธิบายว่าจะส่งถึงใคร/ส่งอะไร และยืนยันก่อนส่ง
- ควรใช้ข้อความสั้น ๆ; หลีกเลี่ยงการส่งความลับ

### sag

- แปลงข้อความเป็นเสียงพูด: ระบุเสียงเป้าหมาย, ลำโพง/ห้องปลายทาง และว่าจะสตรีมหรือไม่

เพิ่มสิ่งอื่นใดก็ได้ที่คุณต้องการให้ผู้ช่วยรู้เกี่ยวกับ toolchain ในเครื่องของคุณ
