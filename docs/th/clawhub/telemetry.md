# เทเลเมทรี

ClawHub ใช้เทเลเมทรี CLI แบบน้อยที่สุดเพื่อคำนวณจำนวนการติดตั้งรวม

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณรัน `clawhub install <slug>`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานสิ่งใด

## สิ่งที่เราเก็บ

ในแต่ละ `clawhub install` ที่มีการรายงาน CLI จะส่งเหตุการณ์การติดตั้งแบบพยายามให้ดีที่สุดหนึ่งรายการ

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ skill ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์หรือตัวระบุที่ได้จากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุต CLI อื่นๆ

## จำนวนการติดตั้ง

ClawHub รักษาตัวนับรวมต่อ skill:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้ง CLI อย่างน้อยหนึ่งครั้งสำหรับ skill นี้
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้งและยังไม่ได้ลบ
  เทเลเมทรีของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม**

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีการติดตั้ง
