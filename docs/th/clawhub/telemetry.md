# การวัดและส่งข้อมูลการใช้งาน

ClawHub ใช้การวัดและส่งข้อมูลการใช้งานของ CLI เพียงเล็กน้อยเพื่อคำนวณจำนวนการติดตั้งโดยรวม

## เวลาที่มีการเก็บข้อมูลการใช้งาน

ข้อมูลการใช้งานจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI
- คุณเรียกใช้ `clawhub install <slug>`
- การวัดและส่งข้อมูลการใช้งาน **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใดๆ

## ข้อมูลที่เราเก็บรวบรวม

สำหรับแต่ละ `clawhub install` ที่มีการรายงาน CLI จะพยายามส่งเหตุการณ์การติดตั้งหนึ่งรายการโดยไม่รับประกันว่าจะสำเร็จ

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ skill ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง หากทราบ

### ข้อมูลที่เรา _ไม่_ เก็บรวบรวม

- ไม่มีพาธโฟลเดอร์หรือตัวระบุที่สร้างจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึก พรอมต์ หรือเอาต์พุตอื่นๆ ของ CLI จากการทำงานแต่ละครั้ง

## จำนวนการติดตั้ง

ClawHub เก็บรักษาตัวนับโดยรวมสำหรับแต่ละ skill:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้ง skill ผ่าน CLI อย่างน้อยหนึ่งครั้ง
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งและยังไม่ได้ลบ
  ข้อมูลการใช้งานของตน

## ความโปร่งใสและการควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งโดยรวม** เท่านั้น

การลบบัญชีจะลบข้อมูลการใช้งานของคุณด้วย

## วิธีปิดใช้งานการวัดและส่งข้อมูลการใช้งาน

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งข้อมูลการใช้งานเกี่ยวกับการติดตั้ง
