# การวัดและส่งข้อมูลทางไกล

ClawHub ใช้**การวัดและส่งข้อมูลทางไกลขั้นต่ำ**เพื่อคำนวณ**จำนวนการติดตั้ง** (สิ่งที่ใช้งานอยู่จริง) และช่วยให้การจัดเรียง/การกรองดีขึ้น
สิ่งนี้อิงตามคำสั่ง CLI `clawhub sync`

## เวลาที่มีการเก็บรวบรวมข้อมูลทางไกล

ข้อมูลทางไกลจะถูกส่งเฉพาะเมื่อ:

- คุณ**เข้าสู่ระบบแล้ว**ใน CLI (เรากำหนดให้ต้องยืนยันตัวตนสำหรับขั้นตอน sync/publish อยู่แล้ว)
- คุณรัน `clawhub sync`
- ข้อมูลทางไกล**ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานอะไร

## สิ่งที่เราเก็บรวบรวม

ในแต่ละ `clawhub sync` CLI จะรายงาน**สแนปช็อตฉบับเต็ม**ของสิ่งที่พบ โดยจัดกลุ่มตามรูทการสแกน (“โฟลเดอร์/รูท”)

สำหรับแต่ละรูท เราจะจัดเก็บ:

- `rootId`: **แฮช SHA-256** ของพาธรูทมาตรฐาน (เซิร์ฟเวอร์จะไม่เห็นพาธดิบ)
- `label`: ป้ายกำกับที่มนุษย์อ่านได้ ซึ่งได้มาจากสองเซกเมนต์สุดท้ายของพาธ (พาธ home จะแสดงด้วย `~`)
- `firstSeenAt`, `lastSeenAt`, `expiredAt` แบบไม่บังคับ

สำหรับแต่ละ skill ที่พบใต้รูท เราจะจัดเก็บ:

- `skillId` (แก้ไขจาก slug; ติดตามเฉพาะ skills ที่มีอยู่ในรีจิสทรีเท่านั้น)
- `firstSeenAt`, `lastSeenAt`
- `lastVersion` (พยายามให้ดีที่สุด; ปัจจุบันคือเวอร์ชันที่ตรงกับรีจิสทรี หากทราบ)
- `removedAt` แบบไม่บังคับ เมื่อการติดตั้งที่เคยรายงานไว้หายไปจากรูท

### สิ่งที่เรา_ไม่_เก็บรวบรวม

- ไม่มีพาธโฟลเดอร์แบบสัมบูรณ์ดิบ (มีเพียง `rootId` ที่ถูกแฮช + ป้ายกำกับสั้นสำหรับแสดงผล)
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน, พรอมป์ หรือเอาต์พุต CLI อื่นใด
- ไม่มีการติดตาม skills ที่ไม่ได้อัปโหลดไปยังรีจิสทรี (slug ที่ไม่รู้จักจะถูกละเว้น)

## จำนวนการติดตั้ง

เราดูแลตัวนับสองตัวต่อ skill:

- `installsCurrent`: ผู้ใช้ไม่ซ้ำที่ปัจจุบันมี skill ติดตั้งอยู่ในรูทที่ใช้งานอยู่อย่างน้อยหนึ่งรูท
- `installsAllTime`: ผู้ใช้ไม่ซ้ำที่เคยรายงานว่าติดตั้ง skill

### หลายรูท

หากคุณ sync จากหลายโฟลเดอร์ เราจะถือว่าแต่ละรูทการสแกนแยกจากกัน skill จะถือว่า “ติดตั้งอยู่ในปัจจุบัน” หากมีอยู่ในรูทที่ใช้งานอยู่**ใดๆ**

### การตรวจจับการถอนการติดตั้ง

เนื่องจาก `sync` รายงานชุดข้อมูลทั้งหมดต่อรูท:

- หาก skill หายไปจากรูทในการ sync ครั้งถัดไป เราจะทำเครื่องหมายว่าถูกลบสำหรับรูทนั้น
- หาก skill ถูกลบออกจากรูททั้งหมดของคุณ จะไม่นับรวมใน `installsCurrent` อีกต่อไป
- `installsAllTime` จะไม่ลดลง เว้นแต่คุณจะลบข้อมูลทางไกล (ดูด้านล่าง)

### ความเก่าเกินกำหนด (120 วัน)

รูทที่ไม่รายงานข้อมูลทางไกลเป็นเวลา**120 วัน**จะถูกทำเครื่องหมายว่าเก่าเกินกำหนด และการติดตั้งของรูทเหล่านั้นจะหยุดนับรวมใน `installsCurrent`
การประเมินนี้ทำแบบเลื่อนเวลา (ในการรายงานข้อมูลทางไกลครั้งถัดไป) เพื่อหลีกเลี่ยงงานเบื้องหลัง

## ความโปร่งใส + การควบคุมของผู้ใช้

ClawHub มีแท็บ “ติดตั้งแล้ว” แบบส่วนตัวบนโปรไฟล์ของคุณเอง:

- แสดงรูท + skills ที่ติดตั้งซึ่งเราจัดเก็บไว้อย่างตรงทั้งหมด
- มีมุมมอง**ส่งออก JSON**
- มีการดำเนินการ**ลบข้อมูลทางไกล**เพื่อลบข้อมูลทางไกลที่จัดเก็บไว้ทั้งหมดสำหรับบัญชีของคุณ

คนอื่นๆ จะเห็นเฉพาะ**ตัวนับการติดตั้งแบบรวม**เท่านั้น; ไม่มีใครอื่นสามารถเห็นรูท/โฟลเดอร์ของคุณได้

การลบบัญชีของคุณจะลบข้อมูลทางไกลของคุณด้วย

## วิธีปิดใช้งานข้อมูลทางไกล

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งข้อมูลทางไกลระหว่าง `clawhub sync`
