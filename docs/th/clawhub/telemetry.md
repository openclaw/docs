# การวัดและส่งข้อมูลทางไกล

ClawHub ใช้การวัดและส่งข้อมูลทางไกลของ CLI แบบน้อยที่สุด เพื่อคำนวณยอดติดตั้งรวม

## เวลาที่มีการเก็บข้อมูลการวัดและส่งข้อมูลทางไกล

การวัดและส่งข้อมูลทางไกลจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณรัน `clawhub install <slug>`
- การวัดและส่งข้อมูลทางไกล **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใดๆ

## สิ่งที่เราเก็บรวบรวม

ในแต่ละ `clawhub install` ที่มีการรายงาน CLI จะส่งเหตุการณ์การติดตั้งแบบพยายามให้ดีที่สุดหนึ่งรายการ

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ Skills ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บรวบรวม

- ไม่มีเส้นทางโฟลเดอร์หรือตัวระบุที่ได้มาจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุต CLI อื่นๆ

## ยอดติดตั้ง

ClawHub ดูแลตัวนับรวมต่อ Skills:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้ง CLI อย่างน้อยหนึ่งครั้งสำหรับ Skills นั้น
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้งและยังไม่ได้ลบ
  ข้อมูลการวัดและส่งข้อมูลทางไกลของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนเห็นเฉพาะ **ตัวนับยอดติดตั้งแบบรวม** เท่านั้น

การลบบัญชีของคุณจะลบข้อมูลการวัดและส่งข้อมูลทางไกลของคุณด้วย

## วิธีปิดใช้งานการวัดและส่งข้อมูลทางไกล

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งข้อมูลการวัดและส่งข้อมูลทางไกลของการติดตั้ง
