# เทเลเมทรี

ClawHub ใช้ **เทเลเมทรีขั้นต่ำ** เพื่อคำนวณ **จำนวนการติดตั้ง** (สิ่งที่ใช้งานจริง) และช่วยให้การจัดเรียง/กรองดีขึ้น
สิ่งนี้อิงจากคำสั่ง CLI `clawhub sync`

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณ **เข้าสู่ระบบ** ใน CLI (เรากำหนดให้ต้องยืนยันตัวตนอยู่แล้วสำหรับโฟลว์ sync/publish)
- คุณเรียกใช้ `clawhub sync`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใดๆ

## สิ่งที่เราเก็บ

ในแต่ละครั้งที่เรียก `clawhub sync` CLI จะรายงาน **สแนปช็อตแบบเต็ม** ของสิ่งที่พบ โดยจัดกลุ่มตามรูทการสแกน (“โฟลเดอร์/รูท”)

สำหรับแต่ละรูท เราจัดเก็บ:

- `rootId`: **แฮช SHA-256** ของพาธรูทแบบมาตรฐาน (เซิร์ฟเวอร์จะไม่เห็นพาธดิบ)
- `label`: ป้ายกำกับที่มนุษย์อ่านได้ ซึ่งได้มาจากสองส่วนท้ายสุดของพาธ (พาธ home จะแสดงด้วย `~`)
- `firstSeenAt`, `lastSeenAt`, `expiredAt` แบบไม่บังคับ

สำหรับแต่ละ Skills ที่พบภายใต้รูท เราจัดเก็บ:

- `skillId` (แก้จาก slug; ติดตามเฉพาะ Skills ที่มีอยู่ในรีจิสทรี)
- `firstSeenAt`, `lastSeenAt`
- `lastVersion` (ตามความพยายามที่ทำได้; ปัจจุบันคือเวอร์ชันที่ตรงกับรีจิสทรีหากทราบ)
- `removedAt` แบบไม่บังคับ เมื่อการติดตั้งที่เคยรายงานไว้หายไปจากรูท

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์แบบสัมบูรณ์ดิบ (มีเพียง `rootId` ที่ถูกแฮช + ป้ายกำกับแสดงผลแบบสั้น)
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุต CLI อื่นๆ
- ไม่มีการติดตาม Skills ที่ไม่ได้อัปโหลดไปยังรีจิสทรี (slug ที่ไม่รู้จักจะถูกละเว้น)

## จำนวนการติดตั้ง

เราดูแลตัวนับสองตัวต่อ Skills:

- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งปัจจุบันมี Skills ติดตั้งอยู่ในรูทที่ใช้งานอยู่ อย่างน้อยหนึ่งรูท
- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานว่ามี Skills ติดตั้งอยู่

### หลายรูท

หากคุณ sync จากหลายโฟลเดอร์ เราจะถือว่าแต่ละรูทการสแกนเป็นอิสระจากกัน Skills จะถือว่า “ติดตั้งอยู่ในปัจจุบัน” หากมีอยู่ในรูทที่ใช้งานอยู่ **รูทใดก็ได้**

### การตรวจจับการถอนการติดตั้ง

เนื่องจาก `sync` รายงานชุดข้อมูลทั้งหมดต่อรูท:

- หาก Skills หายไปจากรูทในการ sync ครั้งถัดไป เราจะทำเครื่องหมายว่าถูกนำออกสำหรับรูทนั้น
- หาก Skills ถูกนำออกจากรูททั้งหมดของคุณ จะไม่นับรวมใน `installsCurrent` อีกต่อไป
- `installsAllTime` จะไม่ลดลง เว้นแต่คุณจะลบเทเลเมทรี (ดูด้านล่าง)

### ความเก่าเกินกำหนด (120 วัน)

รูทที่ไม่รายงานเทเลเมทรีเป็นเวลา **120 วัน** จะถูกทำเครื่องหมายว่าเก่าเกินกำหนด และการติดตั้งของรูทเหล่านั้นจะหยุดนับรวมใน `installsCurrent`
สิ่งนี้จะถูกประเมินแบบ lazy (เมื่อมีรายงานเทเลเมทรีครั้งถัดไป) เพื่อหลีกเลี่ยงงานเบื้องหลัง

## ความโปร่งใส + การควบคุมของผู้ใช้

ClawHub มีแท็บ “ติดตั้งแล้ว” แบบส่วนตัวบนโปรไฟล์ของคุณเอง:

- แสดงรูท + Skills ที่ติดตั้งไว้อย่างตรงตามที่เราจัดเก็บ
- มีมุมมอง **ส่งออก JSON**
- มีการดำเนินการ **ลบเทเลเมทรี** เพื่อลบเทเลเมทรีที่จัดเก็บทั้งหมดสำหรับบัญชีของคุณ

คนอื่นจะเห็นเพียง **ตัวนับการติดตั้งแบบรวม** เท่านั้น ไม่มีใครเห็นรูท/โฟลเดอร์ของคุณได้

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีระหว่าง `clawhub sync`
