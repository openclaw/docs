# เทเลเมทรี

ClawHub ใช้เทเลเมทรีของ CLI ขั้นต่ำเพื่อคำนวณจำนวนการติดตั้งรวม

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณรัน `clawhub install <slug>`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใด ๆ

## สิ่งที่เราเก็บ

สำหรับ `clawhub install` แต่ละครั้งที่มีการรายงาน CLI จะส่งเหตุการณ์การติดตั้งแบบพยายามเต็มที่หนึ่งรายการ

เหตุการณ์นี้ประกอบด้วย:

- `slug`: slug ของ Skills ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีเส้นทางโฟลเดอร์หรือตัวระบุที่ได้มาจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกการรันแต่ละครั้ง พรอมป์ หรือเอาต์พุตอื่น ๆ ของ CLI

## จำนวนการติดตั้ง

ClawHub เก็บตัวนับรวมต่อ Skills:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำซึ่งรายงานการติดตั้งผ่าน CLI อย่างน้อยหนึ่งครั้งสำหรับ Skills นั้น
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำซึ่งรายงานการติดตั้งและยังไม่ได้ลบ
  เทเลเมทรีของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีการติดตั้ง
