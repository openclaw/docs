# เทเลเมทรี

ClawHub ใช้เทเลเมทรีของ CLI ในระดับต่ำสุดเพื่อคำนวณจำนวนการติดตั้งแบบรวม

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณรัน `clawhub install <slug>`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานสิ่งใด

## สิ่งที่เราเก็บ

ในแต่ละ `clawhub install` ที่มีการรายงาน CLI จะส่งเหตุการณ์การติดตั้งหนึ่งรายการแบบพยายามอย่างดีที่สุด

เหตุการณ์นี้ประกอบด้วย:

- `slug`: slug ของทักษะที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์หรือตัวระบุที่ได้มาจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุตอื่นใดของ CLI

## จำนวนการติดตั้ง

ClawHub รักษาตัวนับแบบรวมต่อทักษะ:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งผ่าน CLI อย่างน้อยหนึ่งครั้งสำหรับทักษะนั้น
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งและยังไม่ได้ลบ
  เทเลเมทรีของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อมีการตั้งค่านี้ CLI จะไม่ส่งเทเลเมทรีการติดตั้ง
