# เทเลเมทรี

ClawHub ใช้ **เทเลเมทรีขั้นต่ำ** เพื่อคำนวณ **จำนวนการติดตั้ง** (สิ่งที่ใช้งานจริง) และช่วยให้การจัดเรียง/กรองดีขึ้น
สิ่งนี้อิงจากคำสั่ง CLI `clawhub sync`

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณ **เข้าสู่ระบบแล้ว** ใน CLI (เรา require การยืนยันตัวตนสำหรับโฟลว์ sync/publish อยู่แล้ว)
- คุณเรียกใช้ `clawhub sync`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานสิ่งใด

## สิ่งที่เราเก็บ

ในแต่ละครั้งที่ `clawhub sync` CLI จะรายงาน **สแนปช็อตเต็ม** ของสิ่งที่พบ โดยจัดกลุ่มตามรากการสแกน (“โฟลเดอร์/รูท”)

สำหรับแต่ละรูท เราจะจัดเก็บ:

- `rootId`: **แฮช SHA-256** ของพาธรูทแบบ canonical (เซิร์ฟเวอร์จะไม่เห็นพาธดิบ)
- `label`: ป้ายกำกับที่มนุษย์อ่านได้ ซึ่งได้มาจากสองเซกเมนต์สุดท้ายของพาธ (พาธ home แสดงด้วย `~`)
- `firstSeenAt`, `lastSeenAt`, `expiredAt` แบบไม่บังคับ

สำหรับแต่ละ Skills ที่พบใต้รูท เราจะจัดเก็บ:

- `skillId` (ระบุจาก slug; จะติดตามเฉพาะ Skills ที่มีอยู่ในรีจิสทรี)
- `firstSeenAt`, `lastSeenAt`
- `lastVersion` (แบบ best-effort; ปัจจุบันคือเวอร์ชันที่จับคู่กับรีจิสทรี หากทราบ)
- `removedAt` แบบไม่บังคับ เมื่อการติดตั้งที่เคยรายงานไว้หายไปจากรูท

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์แบบ absolute ดิบ (มีเพียง `rootId` ที่ถูกแฮช + ป้ายกำกับแสดงผลสั้น ๆ)
- ไม่มีเนื้อหาไฟล์
- ไม่มีล็อกต่อการรัน, พรอมป์, หรือเอาต์พุต CLI อื่น ๆ
- ไม่มีการติดตาม Skills ที่ไม่ได้อัปโหลดไปยังรีจิสทรี (slug ที่ไม่รู้จักจะถูกละเว้น)

## จำนวนการติดตั้ง

เราดูแลตัวนับสองตัวต่อหนึ่ง Skills:

- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งปัจจุบันมี Skills ติดตั้งอยู่ในรูทที่ active อย่างน้อยหนึ่งรูท
- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานว่ามี Skills ติดตั้งอยู่

### หลายรูท

หากคุณ sync จากหลายโฟลเดอร์ เราจะถือว่าแต่ละรากการสแกนเป็นอิสระจากกัน Skills จะถือว่า “ติดตั้งอยู่ในปัจจุบัน” หากมีอยู่ในรูทที่ active **ใดก็ได้**

### การตรวจจับการถอนการติดตั้ง

เนื่องจาก `sync` รายงานชุดข้อมูลทั้งหมดต่อรูท:

- หาก Skills หายไปจากรูทในการ sync ครั้งถัดไป เราจะทำเครื่องหมายว่าถูกลบสำหรับรูทนั้น
- หาก Skills ถูกลบออกจากรูททั้งหมดของคุณ จะไม่ถูกนับรวมใน `installsCurrent` อีกต่อไป
- `installsAllTime` จะไม่ลดลง เว้นแต่คุณจะลบเทเลเมทรี (ดูด้านล่าง)

### ความเก่าเก็บ (120 วัน)

รูทที่ไม่รายงานเทเลเมทรีเป็นเวลา **120 วัน** จะถูกทำเครื่องหมายว่า stale และการติดตั้งของรูทเหล่านั้นจะหยุดถูกนับรวมใน `installsCurrent`
การประเมินนี้ทำแบบ lazy (เมื่อมีรายงานเทเลเมทรีครั้งถัดไป) เพื่อหลีกเลี่ยงงานเบื้องหลัง

## ความโปร่งใส + การควบคุมของผู้ใช้

ClawHub มีแท็บ “ติดตั้งแล้ว” แบบส่วนตัวบนโปรไฟล์ของคุณเอง:

- แสดงรูท + Skills ที่ติดตั้งซึ่งเราจัดเก็บไว้อย่างแม่นยำ
- มีมุมมอง **ส่งออก JSON**
- มีการดำเนินการ **ลบเทเลเมทรี** เพื่อลบเทเลเมทรีทั้งหมดที่จัดเก็บไว้สำหรับบัญชีของคุณ

คนอื่นจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น ไม่มีคนอื่นที่เห็นรูท/โฟลเดอร์ของคุณได้

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีระหว่าง `clawhub sync`
