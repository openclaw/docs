# Telemetry

ClawHub ใช้ Telemetry ของ CLI แบบขั้นต่ำเพื่อคำนวณจำนวนการติดตั้งโดยรวม

## เมื่อมีการเก็บ Telemetry

Telemetry จะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณเรียกใช้ `clawhub install <slug>`
- Telemetry **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใดๆ

## สิ่งที่เราเก็บ

ในแต่ละ `clawhub install` ที่ถูกรายงาน CLI จะส่งเหตุการณ์การติดตั้งหนึ่งรายการแบบ best-effort

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ skill ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์หรือตัวระบุที่ได้จากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน prompt หรือเอาต์พุต CLI อื่นๆ

## จำนวนการติดตั้ง

ClawHub ดูแลตัวนับรวมต่อ skill:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งผ่าน CLI อย่างน้อยหนึ่งครั้งสำหรับ skill
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งและยังไม่ได้ลบ
  Telemetry ของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น

การลบบัญชีของคุณจะลบข้อมูล Telemetry ของคุณด้วย

## วิธีปิดใช้งาน Telemetry

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่ง Telemetry การติดตั้ง
