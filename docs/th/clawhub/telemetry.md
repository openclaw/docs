# เทเลเมทรี

ClawHub ใช้เทเลเมทรีของ CLI ในระดับน้อยที่สุดเพื่อคำนวณจำนวนการติดตั้งแบบรวม

## เมื่อมีการเก็บรวบรวมเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณเข้าสู่ระบบใน CLI แล้ว
- คุณเรียกใช้ `clawhub install <slug>`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานข้อมูลใดๆ

## สิ่งที่เราเก็บรวบรวม

ในแต่ละ `clawhub install` ที่ถูกรายงาน CLI จะส่งเหตุการณ์การติดตั้งแบบพยายามให้ดีที่สุดหนึ่งรายการ

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ skill ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง เมื่อทราบ

### สิ่งที่เรา _ไม่_ เก็บรวบรวม

- ไม่มีพาธโฟลเดอร์หรือตัวระบุที่ได้มาจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุตอื่นๆ ของ CLI

## จำนวนการติดตั้ง

ClawHub รักษาตัวนับแบบรวมต่อ skill:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้ง CLI อย่างน้อยหนึ่งครั้งสำหรับ skill
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งได้รายงานการติดตั้งและยังไม่ได้ลบ
  เทเลเมทรีของตน

## ความโปร่งใส + การควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีการติดตั้ง
