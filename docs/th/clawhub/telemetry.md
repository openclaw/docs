# เทเลเมทรี

ClawHub ใช้ **เทเลเมทรีขั้นต่ำ** เพื่อคำนวณ **จำนวนการติดตั้ง** (สิ่งที่ใช้งานจริง) และช่วยให้การจัดเรียง/กรองทำได้ดีขึ้น
สิ่งนี้อ้างอิงจากคำสั่ง CLI `clawhub sync`

## เก็บเทเลเมทรีเมื่อใด

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณ **เข้าสู่ระบบแล้ว** ใน CLI (เรากำหนดให้ต้องยืนยันตัวตนสำหรับโฟลว์ sync/publish อยู่แล้ว)
- คุณเรียกใช้ `clawhub sync`
- เทเลเมทรี **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานสิ่งใด

## สิ่งที่เราเก็บ

ในแต่ละครั้งที่เรียกใช้ `clawhub sync` CLI จะรายงาน **สแนปช็อตแบบครบถ้วน** ของสิ่งที่พบ โดยจัดกลุ่มตามรูทการสแกน (“โฟลเดอร์/รูท”)

สำหรับแต่ละรูท เราเก็บ:

- `rootId`: **แฮช SHA-256** ของพาธรูทแบบ canonical (เซิร์ฟเวอร์จะไม่เห็นพาธดิบ)
- `label`: ป้ายกำกับที่มนุษย์อ่านได้ ซึ่งได้มาจากสองเซกเมนต์สุดท้ายของพาธ (พาธโฮมจะแสดงด้วย `~`)
- `firstSeenAt`, `lastSeenAt`, `expiredAt` ที่มีหรือไม่มีก็ได้

สำหรับแต่ละ skill ที่พบใต้รูท เราเก็บ:

- `skillId` (แก้ไขจาก slug; ติดตามเฉพาะ skills ที่มีอยู่ใน registry)
- `firstSeenAt`, `lastSeenAt`
- `lastVersion` (ทำแบบ best-effort; ปัจจุบันคือเวอร์ชันที่ตรงกับ registry หากทราบ)
- `removedAt` ที่มีหรือไม่มีก็ได้ เมื่อการติดตั้งที่เคยรายงานไว้หายไปจากรูท

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์แบบสัมบูรณ์ดิบ (มีเฉพาะ `rootId` ที่แฮชแล้ว + ป้ายกำกับแสดงผลแบบสั้น)
- ไม่มีเนื้อหาไฟล์
- ไม่มีล็อกต่อการรัน, พรอมป์ หรือเอาต์พุต CLI อื่นๆ
- ไม่มีการติดตาม skills ที่ไม่ได้อัปโหลดไปยัง registry (slug ที่ไม่รู้จักจะถูกละเว้น)

## จำนวนการติดตั้ง

เราดูแลตัวนับสองรายการต่อ skill:

- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งปัจจุบันมี skill ติดตั้งอยู่ในรูทที่ใช้งานอยู่ตั้งแต่หนึ่งรูทขึ้นไป
- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานว่าติดตั้ง skill

### หลายรูท

หากคุณ sync จากหลายโฟลเดอร์ เราจะปฏิบัติต่อแต่ละรูทการสแกนอย่างอิสระ skill จะถือว่า “ติดตั้งอยู่ในปัจจุบัน” หากมีอยู่ในรูทที่ใช้งานอยู่ **ใดๆ**

### การตรวจจับการถอนการติดตั้ง

เนื่องจาก `sync` รายงานชุดข้อมูลครบถ้วนต่อรูท:

- หาก skill หายไปจากรูทในการ sync ครั้งถัดไป เราจะทำเครื่องหมายว่าถูกลบออกสำหรับรูทนั้น
- หาก skill ถูกลบออกจากรูททั้งหมดของคุณ จะไม่ถูกนับรวมใน `installsCurrent` อีกต่อไป
- `installsAllTime` จะไม่ลดลง เว้นแต่คุณจะลบเทเลเมทรี (ดูด้านล่าง)

### ความเก่าเก็บ (120 วัน)

รูทที่ไม่รายงานเทเลเมทรีเป็นเวลา **120 วัน** จะถูกทำเครื่องหมายว่าเก่าเก็บ และการติดตั้งของรูทนั้นจะหยุดนับรวมใน `installsCurrent`
การประเมินนี้ทำแบบ lazy (เมื่อมีรายงานเทเลเมทรีครั้งถัดไป) เพื่อหลีกเลี่ยงงานเบื้องหลัง

## ความโปร่งใส + การควบคุมของผู้ใช้

ClawHub มีแท็บ “ติดตั้งแล้ว” แบบส่วนตัวบนโปรไฟล์ของคุณเอง:

- แสดงรูท + skills ที่ติดตั้งซึ่งเราเก็บไว้อย่างตรงตามจริง
- มีมุมมอง **การส่งออก JSON**
- มีการดำเนินการ **ลบเทเลเมทรี** เพื่อลบเทเลเมทรีทั้งหมดที่เก็บไว้สำหรับบัญชีของคุณ

คนอื่นๆ จะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น; ไม่มีใครอื่นที่เห็นรูท/โฟลเดอร์ของคุณได้

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีระหว่าง `clawhub sync`
