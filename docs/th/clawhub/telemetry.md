# การวัดและส่งข้อมูล

ClawHub ใช้การวัดและส่งข้อมูลจาก CLI ในระดับขั้นต่ำเพื่อคำนวณจำนวนการติดตั้งโดยรวม

## เมื่อใดที่มีการเก็บข้อมูลการวัดและส่งข้อมูล

ระบบจะส่งข้อมูลการวัดและส่งข้อมูลเฉพาะเมื่อ:

- เข้าสู่ระบบใน CLI แล้ว
- เรียกใช้ `clawhub install <slug>`
- การวัดและส่งข้อมูล **ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากไม่ได้เข้าสู่ระบบ ระบบจะไม่รายงานข้อมูลใดๆ

## ข้อมูลที่เราเก็บ

สำหรับ `clawhub install` แต่ละครั้งที่มีการรายงาน CLI จะส่งเหตุการณ์การติดตั้งหนึ่งเหตุการณ์แบบพยายามให้ดีที่สุด

เหตุการณ์ประกอบด้วย:

- `slug`: slug ของ skill ที่ติดตั้ง
- `version`: เวอร์ชันที่ติดตั้ง หากทราบ

### ข้อมูลที่เรา_ไม่_เก็บ

- ไม่มีเส้นทางโฟลเดอร์หรือตัวระบุที่ได้มาจากโฟลเดอร์
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึก พรอมต์ หรือเอาต์พุตอื่นๆ ของ CLI ในแต่ละครั้งที่เรียกใช้

## จำนวนการติดตั้ง

ClawHub เก็บรักษาตัวนับรวมสำหรับแต่ละ skill:

- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้ง skill ผ่าน CLI อย่างน้อยหนึ่งครั้ง
- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานการติดตั้งและยังไม่ได้ลบข้อมูล
  การวัดและส่งข้อมูลของตน

## ความโปร่งใสและการควบคุมของผู้ใช้

ทุกคนจะเห็นเฉพาะ **ตัวนับการติดตั้งแบบรวม** เท่านั้น

การลบบัญชีจะลบข้อมูลการวัดและส่งข้อมูลด้วย

## วิธีปิดใช้งานการวัดและส่งข้อมูล

กำหนดตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อกำหนดค่านี้แล้ว CLI จะไม่ส่งข้อมูลการวัดและส่งข้อมูลเกี่ยวกับการติดตั้ง
