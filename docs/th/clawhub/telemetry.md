# เทเลเมทรี

ClawHub ใช้**เทเลเมทรีขั้นต่ำ**เพื่อคำนวณ**จำนวนการติดตั้ง** (สิ่งที่ใช้งานจริง) และเพื่อให้การจัดเรียง/การกรองดีขึ้น
สิ่งนี้อิงตามคำสั่ง CLI `clawhub sync`

## เมื่อมีการเก็บเทเลเมทรี

เทเลเมทรีจะถูกส่งเฉพาะเมื่อ:

- คุณ**เข้าสู่ระบบ**ใน CLI แล้ว (เรากำหนดให้ต้องยืนยันตัวตนสำหรับโฟลว์ sync/publish อยู่แล้ว)
- คุณรัน `clawhub sync`
- เทเลเมทรี**ไม่ได้ถูกปิดใช้งาน** (ดู “วิธีปิดใช้งาน” ด้านล่าง)

หากคุณไม่ได้เข้าสู่ระบบ จะไม่มีการรายงานสิ่งใด

## สิ่งที่เราเก็บ

ในแต่ละครั้งที่รัน `clawhub sync` CLI จะรายงาน**สแนปช็อตเต็มรูปแบบ**ของสิ่งที่พบ โดยจัดกลุ่มตามรากการสแกน (“โฟลเดอร์/ราก”)

สำหรับแต่ละราก เราจัดเก็บ:

- `rootId`: **แฮช SHA-256** ของพาธรากมาตรฐาน (เซิร์ฟเวอร์จะไม่เห็นพาธดิบ)
- `label`: ป้ายกำกับที่มนุษย์อ่านได้ ซึ่งได้มาจากสองเซกเมนต์สุดท้ายของพาธ (พาธ home จะแสดงด้วย `~`)
- `firstSeenAt`, `lastSeenAt`, และ `expiredAt` ที่ไม่บังคับ

สำหรับ Skills แต่ละรายการที่พบภายใต้ราก เราจัดเก็บ:

- `skillId` (แก้ค่าโดยใช้ slug; ติดตามเฉพาะ Skills ที่มีอยู่ใน registry)
- `firstSeenAt`, `lastSeenAt`
- `lastVersion` (ตามความพยายามที่ทำได้; ปัจจุบันคือเวอร์ชันที่ตรงกับ registry หากทราบ)
- `removedAt` ที่ไม่บังคับ เมื่อการติดตั้งที่เคยรายงานไว้ก่อนหน้านี้หายไปจากราก

### สิ่งที่เรา _ไม่_ เก็บ

- ไม่มีพาธโฟลเดอร์สัมบูรณ์แบบดิบ (มีเพียง `rootId` ที่ถูกแฮช + ป้ายกำกับแสดงผลแบบสั้น)
- ไม่มีเนื้อหาไฟล์
- ไม่มีบันทึกต่อการรัน พรอมป์ หรือเอาต์พุต CLI อื่นๆ
- ไม่มีการติดตาม Skills ที่ไม่ได้อัปโหลดไปยัง registry (slug ที่ไม่รู้จักจะถูกละเว้น)

## จำนวนการติดตั้ง

เราดูแลตัวนับสองตัวต่อ Skills:

- `installsCurrent`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งปัจจุบันติดตั้ง Skills ไว้อย่างน้อยหนึ่งรากที่ใช้งานอยู่
- `installsAllTime`: ผู้ใช้ที่ไม่ซ้ำกันซึ่งเคยรายงานว่าติดตั้ง Skills แล้ว

### หลายราก

หากคุณ sync จากหลายโฟลเดอร์ เราจะจัดการรากการสแกนแต่ละรายการแยกจากกัน Skills จะถือว่า “ติดตั้งอยู่ในปัจจุบัน” หากมีอยู่ในรากที่ใช้งานอยู่**ใดๆ**

### การตรวจจับการถอนการติดตั้ง

เนื่องจาก `sync` รายงานชุดข้อมูลเต็มต่อราก:

- หาก Skills หายไปจากรากในการ sync ครั้งถัดไป เราจะทำเครื่องหมายว่าถูกลบสำหรับรากนั้น
- หาก Skills ถูกลบออกจากรากทั้งหมดของคุณ จะไม่นับรวมใน `installsCurrent` อีกต่อไป
- `installsAllTime` จะไม่ลดลง เว้นแต่คุณจะลบเทเลเมทรี (ดูด้านล่าง)

### ความเก่าเกินกำหนด (120 วัน)

รากที่ไม่รายงานเทเลเมทรีเป็นเวลา **120 วัน** จะถูกทำเครื่องหมายว่าเก่าเกินกำหนด และการติดตั้งของรากนั้นจะหยุดนับรวมใน `installsCurrent`
สิ่งนี้จะถูกประเมินแบบหน่วงเวลา (ในการรายงานเทเลเมทรีครั้งถัดไป) เพื่อหลีกเลี่ยงงานเบื้องหลัง

## ความโปร่งใส + การควบคุมของผู้ใช้

ClawHub มีแท็บส่วนตัว “ติดตั้งแล้ว” บนโปรไฟล์ของคุณเอง:

- แสดงราก + Skills ที่ติดตั้งตามจริงที่เราจัดเก็บ
- รวมมุมมอง**ส่งออก JSON**
- รวมการดำเนินการ**ลบเทเลเมทรี**เพื่อลบเทเลเมทรีที่จัดเก็บทั้งหมดสำหรับบัญชีของคุณ

คนอื่นๆ จะเห็นเฉพาะ**ตัวนับการติดตั้งแบบรวม**เท่านั้น ไม่มีใครอื่นเห็นราก/โฟลเดอร์ของคุณได้

การลบบัญชีของคุณจะลบข้อมูลเทเลเมทรีของคุณด้วย

## วิธีปิดใช้งานเทเลเมทรี

ตั้งค่าตัวแปรสภาพแวดล้อม:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

เมื่อตั้งค่านี้แล้ว CLI จะไม่ส่งเทเลเมทรีระหว่าง `clawhub sync`
