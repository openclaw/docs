# การยืนยันตัวตน

ClawHub ใช้ GitHub สำหรับการเข้าสู่ระบบผ่านเว็บ ส่วน CLI ใช้โทเค็น API ของ ClawHub ที่สร้างผ่านบัญชีที่เข้าสู่ระบบนั้น

## การเข้าสู่ระบบผ่านเว็บ

ใช้ GitHub เพื่อเข้าสู่ระบบที่ [clawhub.ai](https://clawhub.ai)

บัญชีที่ถูกลบ ถูกแบน หรือถูกปิดใช้งานไม่สามารถเข้าสู่ระบบ ClawHub ตามปกติได้ หากการเข้าสู่ระบบพาคุณกลับไปยังสถานะที่ยังไม่ได้เข้าสู่ระบบ บัญชีของคุณอาจไม่ได้อยู่ในสถานะที่ดี หากบัญชีของคุณถูกแบนหรือถูกปิดใช้งาน ให้ใช้
[แบบฟอร์มอุทธรณ์ของ ClawHub](https://appeals.openclaw.ai/) หากคุณเชื่อว่านี่เป็นข้อผิดพลาด

## การเข้าสู่ระบบ CLI

โฟลว์การเข้าสู่ระบบ CLI เริ่มต้นจะเปิดเบราว์เซอร์ของคุณ:

```bash
clawhub login
clawhub whoami
```

สิ่งที่เกิดขึ้น:

1. CLI เริ่มเซิร์ฟเวอร์ callback ชั่วคราวบน `127.0.0.1`
2. เบราว์เซอร์ของคุณเปิดหน้าเข้าสู่ระบบ ClawHub
3. หลังจากเข้าสู่ระบบ GitHub แล้ว ClawHub จะสร้างโทเค็น API
4. เบราว์เซอร์เปลี่ยนเส้นทางกลับไปยัง callback ในเครื่อง
5. CLI จัดเก็บโทเค็นไว้ในไฟล์ config ของ ClawHub ของคุณ

หากเบราว์เซอร์ของคุณเข้าถึง callback ในเครื่องไม่ได้เนื่องจากกฎของไฟร์วอลล์ VPN หรือพร็อกซี ให้ใช้โฟลว์โทเค็นแบบไม่มีส่วนติดต่อกราฟิก

## การเข้าสู่ระบบแบบไม่มีส่วนติดต่อกราฟิก

สร้างโทเค็นใน UI เว็บของ ClawHub จากนั้นส่งให้ CLI:

```bash
clawhub login --token clh_...
```

ใช้โฟลว์นี้สำหรับเซิร์ฟเวอร์ งาน CI หรือสภาพแวดล้อมที่มีเฉพาะเทอร์มินัล

สำหรับเชลล์ระยะไกลที่คุณสามารถเปิดเบราว์เซอร์จากที่อื่นได้ ให้รัน:

```bash
clawhub login --device
```

CLI จะแสดงโค้ดแบบใช้ครั้งเดียวและรอระหว่างที่คุณอนุมัติที่
`https://clawhub.ai/cli/device`

## การจัดเก็บโทเค็น

เส้นทาง config เริ่มต้น:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` หรือ `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

แทนที่เส้นทางด้วย:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

พิมพ์โทเค็นที่จัดเก็บไว้สำหรับการตั้งค่า CI ด้วย:

```bash
clawhub token
```

## การเพิกถอน

คุณสามารถเพิกถอนโทเค็น API ได้ใน UI เว็บของ ClawHub

โทเค็นที่ถูกเพิกถอน ไม่ถูกต้อง หรือขาดหายไปจะส่งคืน `401 Unauthorized` เข้าสู่ระบบอีกครั้งด้วย `clawhub login` หรือระบุโทเค็นใหม่ด้วย `clawhub login --token`

บัญชีที่ถูกลบ ถูกแบน หรือถูกปิดใช้งานไม่สามารถใช้โทเค็น API ที่มีอยู่ต่อไปได้ หากบัญชีของคุณถูกแบนหรือถูกปิดใช้งาน ให้ใช้
[แบบฟอร์มอุทธรณ์ของ ClawHub](https://appeals.openclaw.ai/) หากคุณเชื่อว่านี่เป็นข้อผิดพลาด
