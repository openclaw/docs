# การยืนยันตัวตน

ClawHub ใช้ GitHub สำหรับการลงชื่อเข้าใช้ผ่านเว็บ ส่วน CLI ใช้โทเค็น API ของ ClawHub ที่สร้าง
ผ่านบัญชีซึ่งลงชื่อเข้าใช้นั้น

## การลงชื่อเข้าใช้ผ่านเว็บ

ใช้ GitHub เพื่อลงชื่อเข้าใช้ที่ [clawhub.ai](https://clawhub.ai)

บัญชีที่ถูกลบ แบน หรือปิดใช้งานจะไม่สามารถลงชื่อเข้าใช้ ClawHub ตามปกติได้
หากหลังลงชื่อเข้าใช้แล้วกลับสู่สถานะที่ยังไม่ได้เข้าสู่ระบบ บัญชีอาจมีสถานะ
ไม่ปกติ หากบัญชีถูกแบนหรือปิดใช้งาน ให้ใช้
[แบบฟอร์มอุทธรณ์ของ ClawHub](https://appeals.openclaw.ai/) หากเชื่อว่าเกิดจาก
ความผิดพลาด

## การเข้าสู่ระบบ CLI

ขั้นตอนการเข้าสู่ระบบ CLI เริ่มต้นจะเปิดเบราว์เซอร์:

```bash
clawhub login
clawhub whoami
```

สิ่งที่เกิดขึ้น:

1. CLI เริ่มเซิร์ฟเวอร์ callback ชั่วคราวบน `127.0.0.1`
2. เบราว์เซอร์จะเปิดหน้าลงชื่อเข้าใช้ ClawHub
3. หลังจากลงชื่อเข้าใช้ GitHub แล้ว ClawHub จะสร้างโทเค็น API
4. เบราว์เซอร์จะเปลี่ยนเส้นทางกลับไปยัง callback ภายในเครื่อง
5. CLI จะจัดเก็บโทเค็นไว้ในไฟล์การกำหนดค่า ClawHub

หากเบราว์เซอร์ไม่สามารถเข้าถึง callback ภายในเครื่องได้เนื่องจากกฎของไฟร์วอลล์ VPN หรือ
พร็อกซี ให้ใช้ขั้นตอนโทเค็นแบบไม่ใช้ส่วนติดต่อกราฟิก

## การเข้าสู่ระบบแบบไม่ใช้ส่วนติดต่อกราฟิก

สร้างโทเค็นใน UI เว็บของ ClawHub แล้วส่งให้ CLI:

```bash
clawhub login --token clh_...
```

ใช้ขั้นตอนนี้สำหรับเซิร์ฟเวอร์ งาน CI หรือสภาพแวดล้อมที่ใช้ได้เฉพาะเทอร์มินัล

สำหรับเชลล์ระยะไกลที่สามารถเปิดเบราว์เซอร์จากที่อื่นได้ ให้เรียกใช้:

```bash
clawhub login --device
```

CLI จะแสดงรหัสแบบใช้ครั้งเดียวและรอระหว่างที่อนุญาตสิทธิ์ที่
`https://clawhub.ai/cli/device`

## การจัดเก็บโทเค็น

พาธการกำหนดค่าเริ่มต้น:

- macOS: `~/Library/Application Support/clawhub/config.json`
- Linux/XDG: `$XDG_CONFIG_HOME/clawhub/config.json` หรือ `~/.config/clawhub/config.json`
- Windows: `%APPDATA%\\clawhub\\config.json`

กำหนดพาธทับด้วย:

```bash
export CLAWHUB_CONFIG_PATH=/path/to/config.json
```

แสดงโทเค็นที่จัดเก็บไว้สำหรับการตั้งค่า CI ด้วย:

```bash
clawhub token
```

## การเพิกถอน

สามารถเพิกถอนโทเค็น API ได้ใน UI เว็บของ ClawHub

โทเค็นที่ถูกเพิกถอน ไม่ถูกต้อง หรือไม่มีอยู่จะส่งคืน `401 Unauthorized` ให้ลงชื่อเข้าใช้อีกครั้ง
ด้วย `clawhub login` หรือระบุโทเค็นใหม่ด้วย `clawhub login --token`

บัญชีที่ถูกลบ แบน หรือปิดใช้งานจะไม่สามารถใช้โทเค็น API ที่มีอยู่ต่อไปได้
หากบัญชีถูกแบนหรือปิดใช้งาน ให้ใช้
[แบบฟอร์มอุทธรณ์ของ ClawHub](https://appeals.openclaw.ai/) หากเชื่อว่าเกิดจาก
ความผิดพลาด
