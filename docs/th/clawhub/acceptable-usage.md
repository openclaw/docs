# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และข้อมูลเมตาของมาร์เก็ตเพลสสำหรับ OpenClaw.
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่.

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นที่การค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub. สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation). สำหรับลิขสิทธิ์หรือการเรียกร้องสิทธิ์อื่นๆ
โปรดดู [คำขอด้านสิทธิ์เนื้อหา](/th/clawhub/content-rights).

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่โดยสุจริต.

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์.                                               |
| เวิร์กโฟลว์ UI, ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่ต้องใช้ไว้อย่างชัดแจ้ง และการกระทำที่มีความเสี่ยงมีเส้นทางตรวจทาน ทดลองรัน แสดงตัวอย่าง หรือยืนยัน. |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจทานการละเมิด | เครื่องมือถูกนำเสนอเพื่อการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติของมนุษย์ชัดเจน.                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อิงความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ระบุชัดเจน.                                            |
| แคตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างเหมาะสม.                                                |

บริบทมีความสำคัญ. หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
อิงความยินยอม และยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์เพื่อการละเมิด.

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์.

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการข้ามการรักษาความปลอดภัย                      | การข้ามการยืนยันตัวตน การยึดบัญชี การละเมิดขีดจำกัดอัตรา การยึดสายสนทนาสดหรือเอเจนต์ การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือโฟลว์การจับคู่แบบอนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ.                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลีกเลี่ยงการแบน                              | บัญชีล่องหนหลังถูกแบน การอุ่นบัญชีหรือทำฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลบเลี่ยงการตรวจจับ.                                                                                                                                          |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์อัตลักษณ์สังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน.                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การดึงข้อมูลติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การดึงรายชื่อลูกค้าเป้าหมายพร้อมการติดต่อที่ไม่ได้ร้องขอ การเฝ้าติดตามลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลที่รั่วไหลหรือชุดข้อมูลจากการละเมิด.                                                                                                                  |
| การแอบอ้างหรือการบิดเบือนอัตลักษณ์โดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์โคลน บุคคลสมมติปลอม หรือเครื่องมืออื่นที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด.                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างโจ่งแจ้งหรือการสร้างเนื้อหาผู้ใหญ่ที่ปิดกลไกความปลอดภัย | การสร้างรูปภาพ วิดีโอ หรือเนื้อหา NSFW; ตัวครอบเนื้อหาผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างโจ่งแจ้ง.                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนเร้น ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบไพพ์เข้าเชลล์ เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนดเกี่ยวกับความลับหรือคีย์ส่วนตัวที่ไม่ได้ประกาศ การรัน `npx @latest` จากระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือข้อมูลเมตาที่ซ่อนสิ่งที่รายการต้องใช้จริงเพื่อรัน. |

## พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย. อย่าใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบกลั่นกรอง หรือ
ความสนใจของผู้ใช้.

พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาตรวมถึง:

- การเผยแพร่รายการจำนวนมากแบบเป็นชุดที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวแทนชั่วคราว หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การถาโถมใส่พื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างอย่างมีนัยสำคัญน้อยมากหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง ดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่นๆ
  โดยไม่เป็นธรรมผ่านระบบอัตโนมัติ วงจรติดตั้งเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานมาร์เก็ตเพลส
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำๆ ที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ใช่การละเมิดโดยอัตโนมัติ. แคตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการต่างกันอย่างมีนัยสำคัญ อธิบายอย่างถูกต้อง ได้รับการดูแล
และถูกใช้โดยผู้ใช้จริง. แคตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างไม่เป็นธรรม.

## สิทธิ์เนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอด้านสิทธิ์เนื้อหา](/th/clawhub/content-rights). อย่าใช้รายงานมาร์เก็ตเพลสทั่วไป
สำหรับการเรียกร้องลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย.

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณทางสถิติเกี่ยวกับการละเมิด รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการละเมิด. สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวมันเอง; แต่ช่วยให้ ClawHub ตัดสินใจว่าอะไรต้องได้รับการตรวจทาน.

เราอาจ:

- ซ่อน ระงับ ลบ ลบแบบกู้คืนได้ หรือเมื่อประเภทรองรับ
  ลบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะมีการเตือนก่อนการบังคับใช้สำหรับการละเมิดที่ชัดเจน. โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การระงับเพื่อกลั่นกรอง
รายการที่ถูกซ่อน การแบน และสถานะบัญชี.
