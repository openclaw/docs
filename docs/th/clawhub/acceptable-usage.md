# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมตาดาตา marketplace สำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่ขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี ดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับลิขสิทธิ์หรือการอ้างสิทธิ์อื่น ๆ
ดู [คำขอสิทธิ์เนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือปฏิบัติการซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI, ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่จำเป็นอย่างชัดเจน และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับการรีวิว การทดลองรัน การแสดงตัวอย่าง หรือการยืนยัน |
| ความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการรีวิวการละเมิด | เครื่องมือถูกนำเสนอสำหรับการรีวิวที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติของมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อิงตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แคตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในขอบเขตเชิงป้องกันที่แคบหรือ
ในการตั้งค่าที่อิงตามความยินยอม และอาจยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์การละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการหลีกเลี่ยงความปลอดภัย                      | การหลีกเลี่ยงการยืนยันตัวตน การยึดบัญชี การละเมิด rate-limit การยึดสายเรียกสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือโฟลว์การจับคู่แบบอนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลีกเลี่ยงการแบน                              | บัญชีล่องหนหลังถูกแบน การวอร์มหรือฟาร์มบัญชี การสร้าง engagement ปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์การเงินที่ทำให้เข้าใจผิด             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง social proof ปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน                                                                                                                    |
| การเพิ่มข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การดึงข้อมูลผู้ติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การสกัด lead ควบคู่กับการติดต่อที่ไม่ได้ร้องขอ การเฝ้าติดตามแบบปกปิด การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือข้อมูลจากการละเมิดระบบ                                                                                                                  |
| การสวมรอยหรือการจัดการตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคคลปลอม หรือเครื่องมืออื่น ๆ ที่ใช้เพื่อสวมรอยหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |

| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ถูกทำให้อ่านยาก ตัวติดตั้งแบบ pipe-to-shell เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการรีวิวอย่างชัดเจน ข้อกำหนด secret หรือ private key ที่ไม่ประกาศ การดำเนินการ `npx @latest` ระยะไกลโดยไม่มีความสามารถในการรีวิวอย่างชัดเจน หรือเมตาดาตาที่ซ่อนว่ารายการต้องใช้อะไรจริง ๆ เพื่อรัน |

## พฤติกรรม marketplace ที่ไม่อนุญาต

ClawHub ยังรีวิววิธีที่ผู้เผยแพร่ใช้ marketplace ด้วย อย่าใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือ
ความสนใจของผู้ใช้

พฤติกรรม marketplace ที่ไม่อนุญาตรวมถึง:

- การเผยแพร่จำนวนมากของรายการที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็น placeholder หรือ
  สร้างด้วยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแล แหล่งที่มาที่ชัดเจน
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- การเพิ่มจำนวนติดตั้ง ดาวน์โหลด ดาว หรือเมตริก engagement อื่น ๆ
  อย่างไม่เป็นธรรมชาติผ่านระบบอัตโนมัติ วงจรติดตั้งเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  engagement แบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลีกเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การรีวิว marketplace
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ ท่าทีด้านความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำ ๆ ที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ได้เป็นการละเมิดโดยอัตโนมัติ แคตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแล
และถูกใช้งานโดยผู้ใช้จริง แคตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างไม่เป็นธรรมชาติ

## สิทธิ์เนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอสิทธิ์เนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงาน marketplace ปกติ
สำหรับการอ้างสิทธิ์ลิขสิทธิ์หรือสิทธิ์ เว้นแต่รายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การรีวิวและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดเชิงสถิติ รายงานของผู้ใช้ และ
การรีวิวโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการรีวิว

เราอาจ:

- ซ่อน พักไว้ ลบ soft-delete หรือเมื่อประเภทธรรมชาติของทรัพยากรรองรับ
  hard-delete รายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับ release ที่ไม่ปลอดภัย
- เพิกถอน API token
- soft-delete เนื้อหาที่เกี่ยวข้อง
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันการบังคับใช้แบบเตือนก่อนสำหรับการละเมิดที่ชัดเจน ดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อกลั่นกรอง
รายการที่ซ่อน การแบน และสถานะบัญชี
