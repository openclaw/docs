# การใช้งานที่ยอมรับได้

ClawHub ให้บริการโฮสต์ Skills, Plugin, แพ็กเกจ และข้อมูลเมตาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อพิจารณาว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ใดเหมาะสมกับ
ClawHub

กฎเหล่านี้ครอบคลุมถึงสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตนเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวสำหรับการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการกำกับดูแลและสถานะบัญชี โปรดดู
[การกำกับดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิอื่น ๆ
โปรดดู [คำขอเกี่ยวกับสิทธิในเนื้อหา](/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีต้อนรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่โดย
สุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ แก้ไขข้อบกพร่อง จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ด้าน UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่จำเป็นอย่างชัดแจ้ง และการดำเนินการที่มีความเสี่ยงมีขั้นตอนสำหรับการตรวจสอบ การทดลองเรียกใช้ การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกำกับดูแล และการตรวจสอบการใช้งานในทางที่ผิด | เครื่องมือกำหนดกรอบไว้สำหรับการตรวจสอบที่ได้รับอนุญาต เก็บรักษาหลักฐาน และกำหนดขอบเขตการอนุมัติโดยมนุษย์ไว้อย่างชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือของทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อาศัยความยินยอม มีการตั้งค่าที่โปร่งใส และสิทธิ์ที่ระบุไว้อย่างชัดแจ้ง                                            |
| แค็ตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการมีความแตกต่าง มีประโยชน์ มีคำอธิบายที่ถูกต้อง และได้รับการดูแลอย่างเหมาะสม                                                |

บริบทเป็นสิ่งสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในขอบเขตการป้องกันที่จำกัดหรือ
ในบริบทที่อาศัยความยินยอม แต่ไม่สามารถยอมรับได้เมื่อนำมาจัดทำเป็นเวิร์กโฟลว์สำหรับการใช้งานในทางที่ผิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่ให้บริการโฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเพื่อการใช้งานในทางที่ผิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการหลบเลี่ยงมาตรการรักษาความปลอดภัย                      | การหลบเลี่ยงการยืนยันตัวตน การยึดบัญชี การใช้ขีดจำกัดอัตราในทางที่ผิด การยึดการโทรสดหรือเอเจนต์ การขโมยเซสชันที่นำกลับมาใช้ใหม่ได้ หรือการอนุมัติโฟลว์การจับคู่โดยอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การใช้แพลตฟอร์มในทางที่ผิดและการหลบเลี่ยงการแบน                              | บัญชีลับหลังถูกแบน การอุ่นบัญชีหรือการทำฟาร์มบัญชี การสร้างการมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์ทางการเงินที่ทำให้เข้าใจผิด             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่ทำให้เข้าใจผิด การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์อัตลักษณ์สังเคราะห์สำหรับการฉ้อโกง หรือเครื่องมือสำหรับใช้จ่ายหรือเรียกเก็บเงินโดยไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน                                                                                                                    |
| การเพิ่มข้อมูลหรือการเฝ้าระวังที่ล่วงละเมิดความเป็นส่วนตัว                 | การรวบรวมข้อมูลติดต่อเพื่อส่งสแปม การเปิดเผยข้อมูลส่วนบุคคล การสะกดรอยตาม การดึงข้อมูลผู้มีโอกาสเป็นลูกค้าควบคู่กับการติดต่อที่ไม่ได้ร้องขอ การติดตามอย่างลับ ๆ การจับคู่ข้อมูลชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลที่รั่วไหลหรือชุดข้อมูลจากเหตุละเมิด                                                                                                                  |
| การแอบอ้างโดยไม่ได้รับความยินยอมหรือการบิดเบือนอัตลักษณ์       | การสลับใบหน้า ตัวตนดิจิทัลเสมือน อินฟลูเอนเซอร์ที่โคลนขึ้น บุคลิกปลอม หรือเครื่องมืออื่นที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศที่โจ่งแจ้งหรือการสร้างเนื้อหาสำหรับผู้ใหญ่โดยปิดมาตรการความปลอดภัย | การสร้างรูปภาพ วิดีโอ หรือเนื้อหา NSFW; ตัวครอบ API ของบุคคลที่สามสำหรับเนื้อหาผู้ใหญ่; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศที่โจ่งแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านเข้าใจได้ยาก ตัวติดตั้งแบบส่งผ่านไปยังเชลล์ เช่น เนื้อหาที่ดาวน์โหลดมาแล้วเรียกใช้ด้วย `sh` หรือ `bash` โดยไม่สามารถตรวจสอบได้อย่างชัดเจน ข้อกำหนดเกี่ยวกับข้อมูลลับหรือคีย์ส่วนตัวที่ไม่ได้ประกาศ การเรียกใช้ `npx @latest` จากระยะไกลโดยไม่สามารถตรวจสอบได้อย่างชัดเจน หรือข้อมูลเมตาที่ปกปิดสิ่งที่รายการจำเป็นต้องใช้จริงในการทำงาน |
| เนื้อหาที่ละเมิดลิขสิทธิ์หรือสิทธิ           | การเผยแพร่ซ้ำ Skills, Plugin, เอกสารประกอบ สินทรัพย์ของแบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นโดยไม่ได้รับอนุญาต การละเมิดข้อกำหนดของสัญญาอนุญาต หรือการแอบอ้างเป็นผู้เขียนหรือผู้เผยแพร่ต้นฉบับ                                                                                                                            |

## พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจสอบวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ ตัวชี้วัด สัญญาณความน่าเชื่อถือ ระบบกำกับดูแล หรือความสนใจของ
ผู้ใช้

พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาตประกอบด้วย:

- เผยแพร่รายการจำนวนมากเป็นชุด ซึ่งใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวแทนชั่วคราว หรือ
  สร้างโดยเครื่อง และดูเหมือนไม่มีคุณค่าจริงสำหรับผู้ใช้
- ทำให้พื้นผิวการค้นหาหรือหมวดหมู่เต็มไปด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- เผยแพร่รายการหลายร้อยรายการโดยมีการใช้งาน การบำรุงรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายน้อยมากหรือไม่มีเลย
- เพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างไม่เป็นธรรมผ่านระบบอัตโนมัติ วงจรการติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่มีการประสานงาน
  การมีส่วนร่วมแบบมีค่าตอบแทน หรือพฤติกรรมอื่นที่ไม่ได้เกิดขึ้นตามธรรมชาติ
- สร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกำกับดูแล การแบน ขีดจำกัดของผู้เผยแพร่ หรือ
  การตรวจสอบของมาร์เก็ตเพลส
- ทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ ระดับความปลอดภัย
  ข้อกำหนดในการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- อัปโหลดเนื้อหาที่เคยถูกซ่อน ลบ หรือบล็อกแล้วซ้ำ ๆ
  โดยไม่แก้ไขปัญหาที่เป็นต้นเหตุ

การเผยแพร่ในปริมาณมากไม่ได้ถือเป็นการใช้งานในทางที่ผิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างกันอย่างมีความหมาย มีคำอธิบายที่ถูกต้อง ได้รับการดูแล
และมีผู้ใช้จริงใช้งาน แค็ตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่มีเนื้อหาเบาบาง ซ้ำซ้อน ทำให้เข้าใจผิด ขาดการดูแล หรือ
ได้รับการส่งเสริมอย่างไม่เป็นธรรม

## สิทธิในเนื้อหา

หากคุณเชื่อว่าเนื้อหาใน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิอื่น ๆ ของคุณ โปรดใช้
[คำขอเกี่ยวกับสิทธิในเนื้อหา](/clawhub/content-rights) ห้ามใช้รายงานทั่วไปของมาร์เก็ตเพลส
สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิ เว้นแต่รายการนั้นไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจสอบและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณเชิงสถิติของการใช้งานในทางที่ผิด รายงานจากผู้ใช้ และ
การตรวจสอบโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการใช้งานในทางที่ผิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การใช้งานในทางที่ผิดด้วยตัวมันเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดจำเป็นต้องได้รับการตรวจสอบ

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบกู้คืนได้ หรือในกรณีที่รองรับสำหรับประเภททรัพยากรนั้น
  ลบแบบถาวรสำหรับรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะมีการเตือนก่อนบังคับใช้มาตรการสำหรับการใช้งานในทางที่ผิดอย่างชัดเจน โปรดดู
[การกำกับดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อตรวจสอบ
รายการที่ซ่อน การแบน และสถานะบัญชี
