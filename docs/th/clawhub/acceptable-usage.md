# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของตลาดกลางสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และความน่าเชื่อถือ
ของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอด้านสิทธิ์เนื้อหา](/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่ด้วยเจตนาสุจริต

| หมวดหมู่ | อนุญาตเมื่อ |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์ |
| เวิร์กโฟลว์ UI, ข้อมูล และระบบอัตโนมัติ | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่ต้องใช้ไว้อย่างชัดแจ้ง และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับรีวิว ดรายรัน แสดงตัวอย่าง หรือยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการรีวิวการละเมิด | เครื่องมือถูกนำเสนอเพื่อการรีวิวที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติของมนุษย์ชัดเจน |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม | เวิร์กโฟลว์ใช้บัญชีที่อิงความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ระบุชัดเจน |
| แคตตาล็อกที่มีการบำรุงรักษา | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการบำรุงรักษาอย่างสมเหตุสมผล |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่จำกัด
หรืออิงความยินยอม และยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์เพื่อการละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่ | ไม่อนุญาต |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงที่ไม่ได้รับอนุญาตหรือการเลี่ยงผ่านความปลอดภัย | การเลี่ยงผ่านการยืนยันตัวตน การยึดบัญชี การละเมิดขีดจำกัดอัตรา การยึดสายสดหรือเอเจนต์ การขโมยเซสชันที่นำกลับมาใช้ได้ หรือโฟลว์การจับคู่ที่อนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ |
| การละเมิดแพลตฟอร์มและการหลบเลี่ยงการแบน | บัญชีลับหลังถูกแบน การอุ่นหรือฟาร์มบัญชี การสร้างการมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอทสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินโดยไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว | การดึงรายชื่อติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนบุคคล การสะกดรอย การดึงข้อมูลลีดที่จับคู่กับการติดต่อที่ไม่ได้ร้องขอ การเฝ้าติดตามแบบลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการเจาะระบบ |
| การแอบอ้างหรือการบิดเบือนตัวตนโดยไม่ได้รับความยินยอม | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์โคลน ตัวตนปลอม หรือเครื่องมืออื่น ๆ ที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด |
| เนื้อหาทางเพศอย่างโจ่งแจ้งหรือการสร้างเนื้อหาสำหรับผู้ใหญ่ที่ปิดระบบความปลอดภัย | การสร้างรูปภาพ วิดีโอ หรือเนื้อหา NSFW; ตัวครอบเนื้อหาสำหรับผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างโจ่งแจ้ง |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบไพป์ไปยังเชลล์ เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการรีวิวที่ชัดเจน ข้อกำหนดเกี่ยวกับ secret หรือคีย์ส่วนตัวที่ไม่ได้ประกาศ การรัน `npx @latest` ระยะไกลโดยไม่มีความสามารถในการรีวิวที่ชัดเจน หรือเมทาดาทาที่ซ่อนว่าสิ่งใดจำเป็นจริง ๆ เพื่อรันรายการ |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์ | การเผยแพร่ Skills, Plugin, เอกสาร, สินทรัพย์แบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นซ้ำโดยไม่ได้รับอนุญาต; การละเมิดเงื่อนไขสัญญาอนุญาต; หรือการแอบอ้างเป็นผู้เขียนหรือผู้เผยแพร่ต้นฉบับ |

## พฤติกรรมในตลาดกลางที่ไม่อนุญาต

ClawHub ยังรีวิววิธีที่ผู้เผยแพร่ใช้ตลาดกลางด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือ
ความสนใจของผู้ใช้

พฤติกรรมในตลาดกลางที่ไม่อนุญาตรวมถึง:

- การเผยแพร่รายการจำนวนมากแบบเป็นกลุ่มที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวแทนที่ว่างเปล่า หรือ
  สร้างด้วยเครื่อง และดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การบำรุงรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง ดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างเทียมผ่านระบบอัตโนมัติ วงจรติดตั้งเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การรีวิวตลาดกลาง
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ ท่าทีด้านความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโปรเจกต์หรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำ ๆ ที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่แก้ไขปัญหาพื้นฐาน

การเผยแพร่ปริมาณสูงไม่ใช่การละเมิดโดยอัตโนมัติ แคตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายอย่างถูกต้อง มีการบำรุงรักษา
และถูกใช้โดยผู้ใช้จริง แคตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการบำรุงรักษา หรือ
ถูกโปรโมตอย่างเทียม

## สิทธิ์เนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ ของคุณ ให้ใช้
[คำขอด้านสิทธิ์เนื้อหา](/clawhub/content-rights) ห้ามใช้รายงานตลาดกลางปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นจะไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การรีวิวและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณทางสถิติของการละเมิด รายงานจากผู้ใช้ และ
การรีวิวโดยเจ้าหน้าที่ เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่ละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการรีวิว

เราอาจ:

- ซ่อน ระงับ ลบ ลบแบบซอฟต์ หรือในกรณีที่ประเภททรัพยากรรองรับ
  ลบแบบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบซอฟต์
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือรุนแรง

เราไม่รับประกันว่าจะบังคับใช้โดยเตือนก่อนสำหรับการละเมิดที่ชัดเจน โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การระงับเพื่อกลั่นกรอง
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
