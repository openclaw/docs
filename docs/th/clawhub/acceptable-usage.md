# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, plugins, packages และเมตาดาต้าของ marketplace สำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการตรวจสอบและสถานะบัญชี โปรดดู
[การตรวจสอบและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอด้านสิทธิ์เนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้าย ดีบัก จัดทำเอกสาร หรือปฏิบัติการซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่จำเป็นอย่างชัดเจน และการดำเนินการที่มีความเสี่ยงมีเส้นทางการตรวจทาน การทดลองรัน ตัวอย่างก่อนดำเนินการ หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การตรวจสอบการกลั่นกรอง และการใช้งานในทางที่ผิด | เครื่องมือถูกนำเสนอสำหรับการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แคตตาล็อกที่ดูแลรักษา                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายถูกต้อง และได้รับการดูแลรักษาอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
บริบทที่อิงตามความยินยอม และยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์สำหรับการใช้งานในทางที่ผิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการใช้งานในทางที่ผิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการเลี่ยงผ่านความปลอดภัย                      | การเลี่ยงผ่านการยืนยันตัวตน การยึดครองบัญชี การใช้งานเกินขีดจำกัดอัตราในทางที่ผิด การยึดครองสายสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือโฟลว์การจับคู่ที่อนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การใช้งานแพลตฟอร์มในทางที่ผิดและการหลีกเลี่ยงการแบน                              | บัญชีล่องหนหลังถูกแบน การวอร์มหรือทำฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก spam bots หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติโดยมนุษย์อย่างชัดเจน                                                                                                                    |
| การเพิ่มข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การรวบรวมรายชื่อผู้ติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอยตาม การดึงข้อมูลลีดที่จับคู่กับการติดต่อโดยไม่พึงประสงค์ การเฝ้าติดตามแบบลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลที่รั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การแอบอ้างหรือการบิดเบือนตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคลิกปลอม หรือเครื่องมืออื่น ๆ ที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างโจ่งแจ้งหรือการสร้างเนื้อหาผู้ใหญ่ที่ปิดความปลอดภัย | การสร้างภาพ วิดีโอ หรือเนื้อหา NSFW; ตัวครอบเนื้อหาผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างโจ่งแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบ pipe-to-shell เช่นเนื้อหาที่ดาวน์โหลดมาแล้วเรียกใช้ด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนด secret หรือ private-key ที่ไม่ได้ประกาศ การเรียกใช้ `npx @latest` ระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือเมตาดาต้าที่ซ่อนสิ่งที่รายการต้องใช้จริงเพื่อรัน |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์           | การเผยแพร่ skill, plugin, เอกสาร, สินทรัพย์แบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นซ้ำโดยไม่ได้รับอนุญาต การละเมิดเงื่อนไขใบอนุญาต หรือการแอบอ้างเป็นผู้เขียนหรือผู้เผยแพร่ต้นฉบับ                                                                                                                            |

## พฤติกรรม marketplace ที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้ marketplace ด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการตรวจสอบ หรือความสนใจของผู้ใช้

พฤติกรรม marketplace ที่ไม่อนุญาตรวมถึง:

- การเผยแพร่จำนวนมากของรายการที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็น placeholder หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ plugins ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายน้อยหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างผิดธรรมชาติผ่านระบบอัตโนมัติ ลูปการติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลีกเลี่ยงการตรวจสอบ การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทาน marketplace
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโปรเจกต์หรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่ได้แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ใช่การใช้งานในทางที่ผิดโดยอัตโนมัติ แคตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างกันอย่างมีความหมาย อธิบายถูกต้อง ได้รับการดูแลรักษา
และถูกใช้โดยผู้ใช้จริง แคตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแลรักษา หรือ
ได้รับการโปรโมตอย่างผิดธรรมชาติ

## สิทธิ์เนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอด้านสิทธิ์เนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงาน marketplace ตามปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่รายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการใช้งานในทางที่ผิดเชิงสถิติ รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่ เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการใช้งานในทางที่ผิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การใช้งานในทางที่ผิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน ระงับ ลบ ลบแบบ soft-delete หรือในกรณีที่รองรับสำหรับประเภททรัพยากรนั้น
  hard-delete รายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอน API tokens
- ลบเนื้อหาที่เกี่ยวข้องแบบ soft-delete
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันการบังคับใช้แบบเตือนก่อนสำหรับการใช้งานในทางที่ผิดที่ชัดเจน โปรดดู
[การตรวจสอบและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับรายงาน การระงับเพื่อตรวจสอบ
รายการที่ซ่อนอยู่ การแบน และสถานะบัญชี
