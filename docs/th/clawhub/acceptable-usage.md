# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของตลาดกลางสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการดูแลและสถานะบัญชี โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอเกี่ยวกับสิทธิ์ของเนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดูแลการทำงานของซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลรับรองที่จำเป็นอย่างชัดเจน และการดำเนินการที่มีความเสี่ยงมีเส้นทางสำหรับการตรวจทาน การทดลองรันโดยไม่เปลี่ยนแปลงจริง การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การดูแล และการตรวจสอบการใช้งานที่ไม่เหมาะสม | เครื่องมือถูกวางกรอบไว้สำหรับการตรวจสอบที่ได้รับอนุญาต รักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แค็ตตาล็อกที่มีการดูแล                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในสภาพแวดล้อมเชิงป้องกันที่แคบหรือ
สภาพแวดล้อมตามความยินยอม และไม่ยอมรับเมื่อถูกบรรจุเป็นเวิร์กโฟลว์สำหรับการใช้งานที่ไม่เหมาะสม

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการใช้งานที่ไม่เหมาะสม การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการเลี่ยงการรักษาความปลอดภัย                      | การเลี่ยงการยืนยันตัวตน การยึดบัญชี การใช้ขีดจำกัดอัตราในทางที่ผิด การยึดการเรียกแบบสดหรือ agent การขโมยเซสชันที่ใช้ซ้ำได้ หรือโฟลว์จับคู่ที่อนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การใช้งานแพลตฟอร์มในทางที่ผิดและการเลี่ยงการแบน                              | บัญชีลับหลังถูกแบน การอุ่นหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์การเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินโดยไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การดึงข้อมูลผู้ติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การดึงข้อมูลลีดที่จับคู่กับการติดต่อที่ไม่ได้ร้องขอ การตรวจสอบแบบแอบแฝง การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลที่รั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การสวมรอยหรือการจัดการตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่โคลนขึ้นมา บุคคลสมมติปลอม หรือเครื่องมืออื่นที่ใช้เพื่อสวมรอยหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |

## พฤติกรรมตลาดกลางที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้ตลาดกลางด้วย อย่าใช้ ClawHub เพื่อ
จัดการการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการดูแล หรือความสนใจของผู้ใช้

พฤติกรรมตลาดกลางที่ไม่อนุญาตรวมถึง:

- การเผยแพร่จำนวนมากของรายการที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวแทนชั่วคราว หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแล ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- การเพิ่มยอดติดตั้ง ดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ อย่างเทียม
  ผ่านระบบอัตโนมัติ วงจรติดตั้งเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เกิดขึ้นเองตามธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการดูแล การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานตลาดกลาง
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโปรเจกต์หรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาที่เคยถูกซ่อน ลบ หรือบล็อกแล้วซ้ำ ๆ
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ในปริมาณสูงไม่ใช่การใช้งานที่ไม่เหมาะสมโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแล
และถูกใช้งานโดยผู้ใช้จริง แค็ตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างเทียม

## สิทธิ์ของเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอเกี่ยวกับสิทธิ์ของเนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงานตลาดกลางตามปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณเชิงสถิติของการใช้งานที่ไม่เหมาะสม รายงานผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่ เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการใช้งานที่ไม่เหมาะสม สัญญาณหนึ่ง
ไม่ได้พิสูจน์การใช้งานที่ไม่เหมาะสมด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบซ่อน หรือในกรณีที่ประเภทรองรับ
  ลบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบซ่อน
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือผู้กระทำผิดร้ายแรง

เราไม่รับประกันว่าจะเตือนก่อนบังคับใช้สำหรับการใช้งานที่ไม่เหมาะสมที่ชัดเจน โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับรายงาน การพักเพื่อดูแล
รายการที่ซ่อน การแบน และสถานะบัญชี
