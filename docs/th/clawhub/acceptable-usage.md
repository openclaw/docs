# การใช้งานที่ยอมรับได้

ClawHub ให้บริการโฮสต์ Skills, Plugin, แพ็กเกจ และข้อมูลเมตาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อพิจารณาว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ใดเหมาะสำหรับ
ClawHub

กฎเหล่านี้ครอบคลุมถึงสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตนเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นหา การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิอื่น ๆ
โปรดดู [คำขอเกี่ยวกับสิทธิในเนื้อหา](/th/clawhub/content-rights)

ClawHub ยินดีต้อนรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่ด้วย

## เนื้อหาที่ไม่อนุญาต

## พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจสอบวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นหา ตัวชี้วัด สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือ
ความสนใจของผู้ใช้

พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาตประกอบด้วย:

- การเผยแพร่รายการจำนวนมากแบบเป็นชุดซึ่งใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวยึดตำแหน่ง หรือ
  สร้างโดยเครื่อง และดูเหมือนไม่มีคุณค่าที่แท้จริงต่อผู้ใช้
- การทำให้พื้นที่ค้นหาหรือหมวดหมู่เต็มไปด้วย Skills หรือ Plugin ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแล ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายน้อยมากหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างไม่เป็นธรรมผ่านระบบอัตโนมัติ วงจรการติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่
  ประสานกัน การมีส่วนร่วมที่ต้องชำระเงิน หรือพฤติกรรมอื่นที่ไม่ได้เกิดขึ้นตามธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดของผู้เผยแพร่ หรือ
  การตรวจสอบมาร์เก็ตเพลส
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาที่ถูกซ่อน ลบ หรือบล็อกไปแล้วซ้ำ ๆ
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ในปริมาณมากไม่ได้เป็นการใช้งานในทางที่ผิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่เป็นที่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีนัยสำคัญ อธิบายอย่างถูกต้อง ได้รับการดูแล
และใช้งานโดยผู้ใช้จริง แค็ตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่มีเนื้อหาบาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ได้รับการส่งเสริมอย่างไม่เป็นธรรม

## สิทธิในเนื้อหา

หากเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิอื่น ๆ โปรดใช้
[คำขอเกี่ยวกับสิทธิในเนื้อหา](/th/clawhub/content-rights) ห้ามใช้รายงานปกติของมาร์เก็ตเพลส
สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิ เว้นแต่รายการนั้นไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจสอบและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณทางสถิติของการใช้งานในทางที่ผิด รายงานจากผู้ใช้ และ
การตรวจสอบโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการใช้งานในทางที่ผิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การใช้งานในทางที่ผิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดจำเป็นต้องได้รับการตรวจสอบ

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบกู้คืนได้ หรือในกรณีที่รองรับสำหรับประเภททรัพยากรนั้น
  ลบรายการที่ละเมิดอย่างถาวร
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะมีคำเตือนก่อนการบังคับใช้สำหรับการใช้งานในทางที่ผิดอย่างชัดเจน โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/th/clawhub/moderation) สำหรับรายงาน การพักเพื่อตรวจสอบ
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
