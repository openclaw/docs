# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อพิจารณาว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ครอบคลุมถึงสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตนเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการร้องเรียนด้านลิขสิทธิ์หรือสิทธิอื่นๆ
โปรดดู [คำขอเกี่ยวกับสิทธิในเนื้อหา](/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีต้อนรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ แก้ไขข้อบกพร่อง จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ด้าน UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่จำเป็นอย่างชัดแจ้ง และการดำเนินการที่มีความเสี่ยงมีขั้นตอนการตรวจสอบ การทดลองรัน การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจสอบการละเมิด | เครื่องมือถูกกำหนดกรอบไว้สำหรับการตรวจสอบที่ได้รับอนุญาต เก็บรักษาหลักฐาน และกำหนดขอบเขตการอนุมัติโดยมนุษย์ไว้อย่างชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือของทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อาศัยความยินยอม มีการตั้งค่าที่โปร่งใส และมีสิทธิ์ที่ระบุอย่างชัดแจ้ง                                            |
| แค็ตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างเหมาะสม                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่มีขอบเขตจำกัดหรือ
อาศัยความยินยอม แต่ยอมรับไม่ได้เมื่อจัดทำเป็นเวิร์กโฟลว์สำหรับการละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเพื่อการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ

## พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจสอบวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ ตัวชี้วัด สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือความสนใจ
ของผู้ใช้

พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาตประกอบด้วย:

- การเผยแพร่รายการจำนวนมากที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นเพียงตัวยึดตำแหน่ง หรือ
  สร้างโดยเครื่องจักร ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การทำให้พื้นผิวการค้นหาหรือหมวดหมู่ท่วมไปด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่แทบไม่มีหรือไม่มีการใช้งาน การดูแล ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมาย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่นๆ
  อย่างไม่เป็นธรรมผ่านระบบอัตโนมัติ วงจรติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรม
  ที่ประสานงานกัน การมีส่วนร่วมแบบมีค่าใช้จ่าย หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดของผู้เผยแพร่ หรือ
  การตรวจสอบของมาร์เก็ตเพลส
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ ระดับความปลอดภัย
  ข้อกำหนดในการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาที่ถูกซ่อน ลบ หรือบล็อกแล้วซ้ำๆ
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ถือเป็นการละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่เป็นสิ่งที่ยอมรับได้
เมื่อรายการมีความแตกต่างที่มีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแล
และมีผู้ใช้จริงใช้งาน แค็ตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่เนื้อหาบาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ได้รับการส่งเสริมอย่างไม่เป็นธรรมชาติ

## สิทธิในเนื้อหา

หากเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิอื่นๆ ของคุณ ให้ใช้
[คำขอเกี่ยวกับสิทธิในเนื้อหา](/clawhub/content-rights) อย่าใช้รายงานทั่วไปของมาร์เก็ตเพลส
สำหรับการร้องเรียนด้านลิขสิทธิ์หรือสิทธิ เว้นแต่รายการดังกล่าวจะไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจสอบและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณทางสถิติของการละเมิด รายงานจากผู้ใช้ และ
การตรวจสอบโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวมันเอง แต่ช่วยให้ ClawHub ตัดสินใจได้ว่าสิ่งใดต้องได้รับการตรวจสอบ

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบกู้คืนได้ หรือหากประเภททรัพยากรรองรับ
  ลบรายการที่ละเมิดอย่างถาวร
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะเตือนก่อนบังคับใช้กับการละเมิดที่เห็นได้ชัด โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อตรวจสอบ
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
