# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาตลาดกลางสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการดูแลและสถานะบัญชี โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลรับรองที่จำเป็นอย่างชัดเจน และการดำเนินการที่มีความเสี่ยงมีเส้นทางการตรวจทาน การรันจำลอง การแสดงตัวอย่าง หรือการยืนยัน |
| ความปลอดภัยเชิงป้องกัน การดูแล และการตรวจทานการใช้งานในทางที่ผิด | เครื่องมือถูกวางกรอบไว้สำหรับการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่ยึดตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แคตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการมีความแตกต่าง มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
ยึดตามความยินยอม และยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์สำหรับการใช้งานในทางที่ผิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเพื่อการใช้งานในทางที่ผิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการเลี่ยงผ่านความปลอดภัย                      | การเลี่ยงผ่านการยืนยันตัวตน การยึดบัญชี การใช้ขีดจำกัดอัตราในทางที่ผิด การยึดสายสนทนาสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือการอนุมัติโฟลว์จับคู่อัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การใช้งานแพลตฟอร์มในทางที่ผิดและการหลีกเลี่ยงการแบน                              | บัญชีแอบแฝงหลังถูกแบน การวอร์มหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การรวบรวมรายชื่อติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การดึงลีดที่จับคู่กับการติดต่อโดยไม่ได้ร้องขอ การเฝ้าติดตามแบบแอบแฝง การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการละเมิดความปลอดภัย                                                                                                                  |
| การแอบอ้างหรือการจัดการตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคลิกปลอม หรือเครื่องมืออื่นที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างชัดแจ้งหรือการสร้างเนื้อหาสำหรับผู้ใหญ่ที่ปิดการป้องกันความปลอดภัย | การสร้างภาพ วิดีโอ หรือเนื้อหาที่ไม่เหมาะสำหรับที่ทำงาน; ตัวครอบเนื้อหาสำหรับผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างชัดแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบส่งตรงเข้าเชลล์ เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนด secret หรือ private key ที่ไม่ได้ประกาศ การดำเนินการ `npx @latest` จากระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือเมทาดาทาที่ซ่อนสิ่งที่รายการต้องใช้จริงในการรัน |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์           | การเผยแพร่ Skills, Plugin, เอกสาร, สินทรัพย์แบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นซ้ำโดยไม่ได้รับอนุญาต; การละเมิดเงื่อนไขสัญญาอนุญาต; หรือการแอบอ้างเป็นผู้เขียนหรือผู้เผยแพร่ดั้งเดิม                                                                                                                            |

## พฤติกรรมตลาดกลางที่ไม่อนุญาต

ClawHub ตรวจทานวิธีที่ผู้เผยแพร่ใช้ตลาดกลางด้วย อย่าใช้ ClawHub เพื่อ
จัดการผลการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการดูแล หรือ
ความสนใจของผู้ใช้

พฤติกรรมตลาดกลางที่ไม่อนุญาตรวมถึง:

- การเผยแพร่รายการจำนวนมากแบบเหมารวมที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวแทนชั่วคราว หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแล ความชัดเจนของซอร์ส
  หรือความแตกต่างที่มีความหมายน้อยมากหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างเทียมผ่านระบบอัตโนมัติ ลูปติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เกิดขึ้นเองตามธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลีกเลี่ยงการดูแล การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานของตลาดกลาง
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ ซอร์ส ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำ ๆ ที่เคยถูกซ่อน ลบ หรือบล็อกไปแล้ว
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ใช่การใช้งานในทางที่ผิดโดยอัตโนมัติ แคตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแล
และใช้งานโดยผู้ใช้จริง แคตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ได้รับการโปรโมตอย่างเทียม

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ ของคุณ ให้ใช้
[คำขอเกี่ยวกับสิทธิ์ในเนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงานตลาดกลางปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณเชิงสถิติของการใช้งานในทางที่ผิด รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่ เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการใช้งานในทางที่ผิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การใช้งานในทางที่ผิดด้วยตัวมันเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบกู้คืนได้ หรือในกรณีที่ประเภททรัพยากรรองรับ
  ลบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันการบังคับใช้แบบเตือนก่อนสำหรับการใช้งานในทางที่ผิดที่ชัดเจน โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักเพื่อดูแล
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
