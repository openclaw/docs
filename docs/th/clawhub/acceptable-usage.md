# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และข้อมูลเมตาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี ดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิ์อื่น ๆ
ดู [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่ด้วยเจตนาดี

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่ต้องใช้อย่างชัดแจ้ง และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับการตรวจทาน การทดลองรัน การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจทานการละเมิด | เครื่องมือถูกวางกรอบไว้สำหรับการตรวจทานที่ได้รับอนุญาต รักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อิงตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดแจ้ง                                            |
| แค็ตตาล็อกที่ได้รับการดูแลรักษา                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลรักษาอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
อิงตามความยินยอม และยอมรับไม่ได้เมื่อถูกจัดแพ็กเกจเป็นเวิร์กโฟลว์การละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการเลี่ยงมาตรการความปลอดภัย                      | การเลี่ยงการยืนยันตัวตน การยึดบัญชี การละเมิดขีดจำกัดอัตราการใช้งาน การยึดสายสนทนาสดหรือเอเจนต์ การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือโฟลว์จับคู่ที่อนุมัติอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลบเลี่ยงการแบน                              | บัญชีล่องหนหลังถูกแบน การอุ่นเครื่องหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์อัตลักษณ์สังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติโดยมนุษย์อย่างชัดเจน                                                                                                                    |
| การเพิ่มข้อมูลหรือการเฝ้าระวังที่รุกรานความเป็นส่วนตัว                 | การดึงข้อมูลผู้ติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การดึงข้อมูลลูกค้าเป้าหมายควบคู่กับการติดต่อโดยไม่ได้ร้องขอ การติดตามแบบแอบแฝง การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การแอบอ้างหรือการบิดเบือนอัตลักษณ์โดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคคลสมมติปลอม หรือเครื่องมืออื่นที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างชัดแจ้งหรือการสร้างเนื้อหาผู้ใหญ่ที่ปิดการป้องกันด้านความปลอดภัย | การสร้างภาพ วิดีโอ หรือเนื้อหา NSFW; ตัวครอบเนื้อหาผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างชัดแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบส่งต่อเข้าเชลล์ เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนดเกี่ยวกับความลับหรือคีย์ส่วนตัวที่ไม่ได้ประกาศ การดำเนินการ `npx @latest` ระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือข้อมูลเมตาที่ซ่อนสิ่งที่รายการต้องใช้จริงเพื่อให้ทำงาน |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์           | การเผยแพร่ซ้ำ Skills, Plugin, เอกสาร สินทรัพย์แบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นโดยไม่ได้รับอนุญาต การละเมิดเงื่อนไขสัญญาอนุญาต หรือการแอบอ้างเป็นผู้เขียนหรือผู้เผยแพร่ต้นฉบับ                                                                                                                            |

## พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย อย่าใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือ
ความสนใจของผู้ใช้

พฤติกรรมในมาร์เก็ตเพลสที่ไม่อนุญาตรวมถึง:

- เผยแพร่รายการจำนวนมากแบบเป็นชุด ซึ่งใช้ความพยายามต่ำ ซ้ำซ้อน เป็นเพียงตัวแทนที่ยังไม่สมบูรณ์ หรือ
  สร้างโดยเครื่อง และดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- ท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- เผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีนัยสำคัญเพียงเล็กน้อยหรือไม่มีเลย
- เพิ่มจำนวนการติดตั้ง ดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างไม่เป็นธรรมผ่านระบบอัตโนมัติ วงจรติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เกิดขึ้นเองตามธรรมชาติ
- สร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานของมาร์เก็ตเพลส
- ทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- อัปโหลดเนื้อหาซ้ำ ๆ ที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ใช่การละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีนัยสำคัญ อธิบายถูกต้อง ได้รับการดูแลรักษา
และมีผู้ใช้จริงใช้งาน แค็ตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแลรักษา หรือ
ถูกโปรโมตอย่างไม่เป็นธรรม

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ ของคุณ ให้ใช้
[คำขอเกี่ยวกับสิทธิ์ในเนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงานมาร์เก็ตเพลสปกติ
สำหรับการร้องเรียนเรื่องลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดทางสถิติ รายงานจากผู้ใช้ และ
การตรวจทานโดยทีมงานเพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบกู้คืนได้ หรือในกรณีที่ประเภทรองรับ
  ลบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบแบบกู้คืนได้เนื้อหาที่เกี่ยวข้อง
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือรุนแรง

เราไม่รับประกันว่าจะมีการเตือนก่อนบังคับใช้สำหรับการละเมิดที่ชัดเจน ดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อกลั่นกรอง
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
