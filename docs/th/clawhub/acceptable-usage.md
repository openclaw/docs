# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของ marketplace สำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และ
ความน่าเชื่อถือของ ClawHub สำหรับสถานะการดูแลและสถานะบัญชี โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอสิทธิ์ในเนื้อหา](/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI, ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลรับรองที่จำเป็นอย่างชัดเจน และการกระทำที่มีความเสี่ยงมีเส้นทางการตรวจทาน การรันแบบทดลอง การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การดูแล และการตรวจทานการละเมิด | เครื่องมือถูกวางกรอบไว้สำหรับการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อิงตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แค็ตตาล็อกที่มีการดูแลรักษา                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลรักษาอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
อิงตามความยินยอม และยอมรับไม่ได้เมื่อถูกบรรจุเป็นเวิร์กโฟลว์การละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการหลบเลี่ยงความปลอดภัย                      | การหลบเลี่ยงการยืนยันตัวตน การยึดบัญชี การใช้ rate limit ในทางที่ผิด การยึดสายเรียกสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ได้ หรือการอนุมัติโฟลว์จับคู่อัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลบเลี่ยงการแบน                              | บัญชีล่องหนหลังถูกแบน การอุ่นบัญชีหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างมาเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินโดยไม่มีการอนุมัติโดยมนุษย์ที่ชัดเจน                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การขูดรายชื่อติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนบุคคล การสะกดรอย การดึงรายชื่อลีดควบคู่กับการติดต่อที่ไม่ได้ร้องขอ การเฝ้าติดตามแบบลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การแอบอ้างหรือการจัดการตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคลิกปลอม หรือเครื่องมืออื่นที่ใช้เพื่อแอบอ้างหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างโจ่งแจ้งหรือการสร้างเนื้อหาผู้ใหญ่ที่ปิดความปลอดภัยไว้ | การสร้างรูปภาพ วิดีโอ หรือเนื้อหา NSFW; wrapper เนื้อหาผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างโจ่งแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบ pipe-to-shell เช่นเนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนด secret หรือ private key ที่ไม่ได้ประกาศ การรัน `npx @latest` จากระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือเมทาดาทาที่ซ่อนว่าสิ่งใดจำเป็นจริง ๆ เพื่อให้รายการรันได้ |

## พฤติกรรม marketplace ที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้ marketplace ด้วย อย่าใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการดูแล หรือความสนใจของผู้ใช้

พฤติกรรม marketplace ที่ไม่อนุญาต ได้แก่:

- การเผยแพร่รายการจำนวนมากแบบเป็นกลุ่มที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็น placeholder หรือ
  สร้างโดยเครื่อง และดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่เกือบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- การปั่นยอดติดตั้ง ดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  ด้วยระบบอัตโนมัติ ลูปติดตั้งเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการดูแล การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทาน marketplace
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโปรเจกต์หรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำ ๆ ที่เคยถูกซ่อน ลบ หรือบล็อกแล้ว
  โดยไม่ได้แก้ไขปัญหาต้นเหตุ

การเผยแพร่ในปริมาณสูงไม่ใช่การละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแลรักษา
และถูกใช้โดยผู้ใช้จริง แค็ตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแลรักษา หรือ
ถูกโปรโมตอย่างไม่เป็นธรรมชาติ

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอสิทธิ์ในเนื้อหา](/clawhub/content-rights) อย่าใช้รายงาน marketplace ตามปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่ว่ารายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดเชิงสถิติ รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่ละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวมันเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ soft-delete หรือ hard-delete รายการที่ละเมิด
  ในกรณีที่ประเภททรัพยากรรองรับ
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- soft-delete เนื้อหาที่เกี่ยวข้อง
- จำกัดสิทธิ์การเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันการบังคับใช้แบบเตือนก่อนสำหรับการละเมิดที่ชัดเจน โปรดดู
[การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักเพื่อดูแล
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
