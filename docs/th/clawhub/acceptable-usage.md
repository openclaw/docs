# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาตาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้เรียกใช้ วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และความเชื่อถือ
ของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับลิขสิทธิ์หรือการอ้างสิทธิ์อื่น ๆ
โปรดดู [คำขอสิทธิ์ในเนื้อหา](/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่ด้วยเจตนาดี

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่จำเป็นอย่างชัดเจน และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับการตรวจทาน การทดลองรัน การแสดงตัวอย่าง หรือการยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจสอบการละเมิด | เครื่องมือถูกนำเสนอเพื่อการตรวจสอบที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีตามความยินยอม มีการตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แค็ตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่จำกัดหรือ
ตามความยินยอม และไม่อาจยอมรับได้เมื่อบรรจุเป็นเวิร์กโฟลว์สำหรับการละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเพื่อการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการข้ามระบบรักษาความปลอดภัย                      | การข้ามการยืนยันตัวตน การยึดบัญชี การละเมิดการจำกัดอัตรา การยึดสายเรียกสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือการอนุมัติโฟลว์การจับคู่โดยอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลบเลี่ยงการแบน                              | บัญชีลับหลังถูกแบน การอุ่นเครื่องหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินโดยไม่มีการอนุมัติจากมนุษย์อย่างชัดเจน                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การรวบรวมรายชื่อติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนบุคคล การสะกดรอย การดึงรายชื่อผู้มุ่งหวังพร้อมการติดต่อที่ไม่ได้รับเชิญ การตรวจสอบแบบลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การสวมรอยหรือการจัดการตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์ที่ถูกโคลน บุคคลปลอม หรือเครื่องมืออื่น ๆ ที่ใช้เพื่อสวมรอยหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |

| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบ pipe-to-shell เช่น เนื้อหาที่ดาวน์โหลดมาแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานอย่างชัดเจน ข้อกำหนด secret หรือ private key ที่ไม่ได้ประกาศ การดำเนินการ `npx @latest` จากระยะไกลโดยไม่มีความสามารถในการตรวจทานอย่างชัดเจน หรือเมทาดาตาที่ซ่อนสิ่งที่รายการต้องใช้จริงเพื่อให้รันได้ |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์           | การเผยแพร่ Skills, Plugin, เอกสาร, ทรัพย์สินแบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นซ้ำโดยไม่ได้รับอนุญาต การละเมิดเงื่อนไขใบอนุญาต หรือการสวมรอยเป็นผู้เขียนหรือผู้เผยแพร่เดิม                                                                                                                            |

## พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจสอบวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความเชื่อถือ ระบบการกลั่นกรอง หรือความสนใจ
ของผู้ใช้

พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาตรวมถึง:

- เผยแพร่รายการจำนวนมากแบบกลุ่มที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นเพียงตัวแทนที่ว่างไว้ หรือ
  สร้างโดยเครื่อง และดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- ท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- เผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแล ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- เพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างไม่เป็นธรรมผ่านระบบอัตโนมัติ ลูปการติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- สร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานมาร์เก็ตเพลส
- ทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- อัปโหลดเนื้อหาซ้ำ ๆ ที่ถูกซ่อน ลบ หรือบล็อกไปแล้ว
  โดยไม่ได้แก้ไขปัญหาต้นเหตุ

การเผยแพร่ในปริมาณสูงไม่ถือเป็นการละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการต่างกันอย่างมีความหมาย อธิบายอย่างถูกต้อง ได้รับการดูแล
และถูกใช้โดยผู้ใช้จริง แค็ตตาล็อกขนาดใหญ่กลายเป็นปัญหาด้านความเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่ตื้น ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างไม่เป็นธรรม

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ ของคุณ ให้ใช้
[คำขอสิทธิ์ในเนื้อหา](/clawhub/content-rights) อย่าใช้รายงานมาร์เก็ตเพลสตามปกติ
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่รายการนั้นจะไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดเชิงสถิติ รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่เป็นการละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ soft-delete หรือ hard-delete รายการที่ละเมิด
  เมื่อประเภททรัพยากรรองรับ
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- soft-delete เนื้อหาที่เกี่ยวข้อง
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะมีการเตือนก่อนบังคับใช้สำหรับการละเมิดที่ชัดเจน โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อกลั่นกรอง
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
