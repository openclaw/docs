# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของตลาดกลางสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินใจว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และความไว้วางใจ
ของ ClawHub สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับลิขสิทธิ์หรือการอ้างสิทธิ์อื่น ๆ
โปรดดู [คำขอสิทธิ์ในเนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจได้ และเผยแพร่ด้วยเจตนาดี

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                 | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายข้อมูล ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI ข้อมูล และระบบอัตโนมัติ          | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่ต้องใช้อย่างชัดเจน และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับการตรวจทาน dry-run พรีวิว หรือยืนยัน |
| การรักษาความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจทานการละเมิด | เครื่องมือถูกวางกรอบไว้สำหรับการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                    | เวิร์กโฟลว์ใช้บัญชีที่อิงตามความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดเจน                                            |
| แค็ตตาล็อกที่ได้รับการดูแล                     | แต่ละรายการแตกต่างกัน มีประโยชน์ อธิบายอย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันที่แคบหรือ
อิงตามความยินยอม และยอมรับไม่ได้เมื่อถูกจัดแพ็กเป็นเวิร์กโฟลว์การละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการเลี่ยงมาตรการรักษาความปลอดภัย | การเลี่ยงการยืนยันตัวตน การยึดบัญชี การละเมิด rate-limit การยึดสายสนทนาสดหรือ agent การขโมยเซสชันที่นำกลับมาใช้ได้ หรือการอนุมัติโฟลว์จับคู่โดยอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการเลี่ยงการแบน                      | บัญชีลับหลังถูกแบน การวอร์มหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง กลโกง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง        | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อเพื่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินโดยไม่มีการอนุมัติโดยมนุษย์ที่ชัดเจน                                                                                                                    |
| การเพิ่มข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว     | การกวาดข้อมูลติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนตัว การสะกดรอย การดึง lead ที่จับคู่กับการติดต่อโดยไม่ได้รับเชิญ การเฝ้าติดตามแบบลับ การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลที่รั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การสวมรอยหรือการบิดเบือนตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์โคลน บุคคลสมมติปลอม หรือเครื่องมืออื่นที่ใช้เพื่อสวมรอยหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |
| เนื้อหาทางเพศอย่างโจ่งแจ้งหรือการสร้างเนื้อหาผู้ใหญ่ที่ปิดความปลอดภัย | การสร้างภาพ วิดีโอ หรือเนื้อหา NSFW; wrapper เนื้อหาผู้ใหญ่รอบ API ของบุคคลที่สาม; หรือรายการที่มีวัตถุประสงค์หลักเป็นเนื้อหาทางเพศอย่างโจ่งแจ้ง                                                                                                                                                       |
| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบ pipe-to-shell เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนด secret หรือคีย์ส่วนตัวที่ไม่ได้ประกาศ การเรียกใช้ `npx @latest` ระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือเมทาดาทาที่ซ่อนว่ารายการต้องใช้อะไรจริง ๆ เพื่อรัน |
| วัสดุที่ละเมิดลิขสิทธิ์หรือละเมิดสิทธิ์                    | การเผยแพร่ Skill, Plugin, เอกสาร, สินทรัพย์แบรนด์ หรือโค้ดกรรมสิทธิ์ของผู้อื่นซ้ำโดยไม่ได้รับอนุญาต; การละเมิดข้อกำหนดของสัญญาอนุญาต; หรือการสวมรอยเป็นผู้เขียนหรือผู้เผยแพร่ต้นฉบับ                                                                                                                            |

## พฤติกรรมตลาดกลางที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้ตลาดกลางด้วย ห้ามใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความไว้วางใจ ระบบกลั่นกรอง หรือความสนใจของผู้ใช้

พฤติกรรมตลาดกลางที่ไม่อนุญาตรวมถึง:

- การเผยแพร่รายการจำนวนมากแบบเป็นชุดที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็น placeholder หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีนัยสำคัญน้อยมากหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างปลอม ๆ ผ่านระบบอัตโนมัติ วงจรติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานตลาดกลาง
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ ท่าทีด้านความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโปรเจกต์หรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาซ้ำ ๆ ที่ถูกซ่อน ลบ หรือบล็อกไปแล้ว
  โดยไม่ได้แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณมากไม่ได้เป็นการละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการแตกต่างกันอย่างมีนัยสำคัญ อธิบายอย่างถูกต้อง ได้รับการดูแล
และมีผู้ใช้จริงใช้งาน แค็ตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความไว้วางใจและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างปลอม ๆ

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ ให้ใช้
[คำขอสิทธิ์ในเนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงานตลาดกลางปกติ
สำหรับการอ้างสิทธิ์ลิขสิทธิ์หรือสิทธิ์ เว้นแต่รายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดเชิงสถิติ รายงานจากผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่ละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจได้ว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน พักไว้ ลบ ลบแบบ soft-delete หรือเมื่อชนิดทรัพยากรรองรับ
  ลบแบบ hard-delete รายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรีลีสที่ไม่ปลอดภัย
- เพิกถอน API token
- ลบแบบ soft-delete เนื้อหาที่เกี่ยวข้อง
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะบังคับใช้โดยเตือนก่อนสำหรับการละเมิดที่ชัดเจน โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การพักไว้เพื่อกลั่นกรอง
รายการที่ถูกซ่อน การแบน และสถานะบัญชี
