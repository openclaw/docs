# การใช้งานที่ยอมรับได้

ClawHub โฮสต์ Skills, Plugin, แพ็กเกจ และเมทาดาทาของมาร์เก็ตเพลสสำหรับ OpenClaw
ใช้หน้านี้เพื่อตัดสินว่าเนื้อหาหรือพฤติกรรมการเผยแพร่ควรอยู่บน
ClawHub หรือไม่

กฎเหล่านี้ใช้กับสิ่งที่รายการทำ สิ่งที่รายการขอให้ผู้ใช้รัน วิธีที่รายการ
นำเสนอตัวเอง และวิธีที่ผู้เผยแพร่ใช้พื้นผิวการค้นพบ การติดตั้ง และความน่าเชื่อถือของ ClawHub
สำหรับสถานะการกลั่นกรองและสถานะบัญชี โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์อื่น ๆ
โปรดดู [คำขอสิทธิ์ในเนื้อหา](/th/clawhub/content-rights)

## เนื้อหาที่อนุญาต

ClawHub ยินดีรับเนื้อหาที่มีประโยชน์ เข้าใจง่าย และเผยแพร่โดยสุจริต

| หมวดหมู่                                         | อนุญาตเมื่อ                                                                                                                      |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| ประสิทธิภาพการทำงานของนักพัฒนา                           | รายการช่วยให้ผู้ใช้สร้าง ทดสอบ ย้ายระบบ ดีบัก จัดทำเอกสาร หรือดำเนินงานซอฟต์แวร์                                               |
| เวิร์กโฟลว์ UI, ข้อมูล และระบบอัตโนมัติ               | ขอบเขตชัดเจน ระบุข้อมูลประจำตัวที่ต้องใช้ไว้อย่างชัดแจ้ง และการกระทำที่มีความเสี่ยงมีเส้นทางสำหรับการตรวจทาน การทดลองรัน การแสดงตัวอย่าง หรือการยืนยัน |
| ความปลอดภัยเชิงป้องกัน การกลั่นกรอง และการตรวจทานการละเมิด | เครื่องมือถูกนำเสนอเพื่อการตรวจทานที่ได้รับอนุญาต เก็บรักษาหลักฐาน และทำให้ขอบเขตการอนุมัติโดยมนุษย์ชัดเจน                          |
| เวิร์กโฟลว์ส่วนบุคคลหรือทีม                       | เวิร์กโฟลว์ใช้บัญชีที่อิงความยินยอม การตั้งค่าที่โปร่งใส และสิทธิ์ที่ชัดแจ้ง                                            |
| แค็ตตาล็อกที่ได้รับการดูแล                              | แต่ละรายการมีความแตกต่าง มีประโยชน์ อธิบายไว้อย่างถูกต้อง และได้รับการดูแลอย่างสมเหตุสมผล                                                |

บริบทมีความสำคัญ หัวข้อเดียวกันอาจยอมรับได้ในบริบทเชิงป้องกันแบบจำกัดหรือ
บริบทที่อิงความยินยอม และอาจยอมรับไม่ได้เมื่อถูกจัดแพ็กเป็นเวิร์กโฟลว์การละเมิด

## เนื้อหาที่ไม่อนุญาต

ClawHub ไม่โฮสต์เนื้อหาที่มีวัตถุประสงค์หลักเป็นการละเมิด การหลอกลวง การดำเนินการที่ไม่ปลอดภัย
หรือการละเมิดสิทธิ์

| หมวดหมู่                                                    | ไม่อนุญาต                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| การเข้าถึงโดยไม่ได้รับอนุญาตหรือการหลบเลี่ยงความปลอดภัย                      | การข้ามการตรวจสอบสิทธิ์ การยึดบัญชี การละเมิดขีดจำกัดอัตรา การยึดสายเรียกสดหรือเอเจนต์ การขโมยเซสชันที่นำกลับมาใช้ซ้ำได้ หรือการอนุมัติโฟลว์จับคู่โดยอัตโนมัติสำหรับผู้ใช้ที่ไม่ได้รับอนุมัติ                                                                                                                                                   |
| การละเมิดแพลตฟอร์มและการหลบเลี่ยงการแบน                              | บัญชีลับหลังถูกแบน การวอร์มหรือฟาร์มบัญชี การมีส่วนร่วมปลอม ระบบอัตโนมัติหลายบัญชี การโพสต์จำนวนมาก บอตสแปม หรือระบบอัตโนมัติที่สร้างขึ้นเพื่อหลีกเลี่ยงการตรวจจับ                                                                                                                                          |
| การฉ้อโกง การหลอกลวง และเวิร์กโฟลว์ทางการเงินที่หลอกลวง             | ใบรับรองหรือใบแจ้งหนี้ปลอม โฟลว์การชำระเงินที่หลอกลวง การติดต่อหลอกลวง หลักฐานทางสังคมปลอม เวิร์กโฟลว์ตัวตนสังเคราะห์เพื่อการฉ้อโกง หรือเครื่องมือใช้จ่าย/เรียกเก็บเงินที่ไม่มีการอนุมัติโดยมนุษย์อย่างชัดเจน                                                                                                                    |
| การเสริมข้อมูลหรือการเฝ้าระวังที่ละเมิดความเป็นส่วนตัว                 | การดึงข้อมูลผู้ติดต่อเพื่อสแปม การเปิดเผยข้อมูลส่วนบุคคล การสะกดรอยตาม การดึงข้อมูลลีดที่จับคู่กับการติดต่อโดยไม่ได้ร้องขอ การเฝ้าติดตามแบบแอบแฝง การจับคู่ชีวมิติที่ไม่ได้รับความยินยอม หรือการใช้ข้อมูลรั่วไหลหรือชุดข้อมูลจากการละเมิด                                                                                                                  |
| การปลอมแปลงบุคคลหรือการบิดเบือนตัวตนโดยไม่ได้รับความยินยอม       | การสลับใบหน้า ฝาแฝดดิจิทัล อินฟลูเอนเซอร์โคลน บุคลิกปลอม หรือเครื่องมืออื่นที่ใช้เพื่อปลอมแปลงหรือทำให้เข้าใจผิด                                                                                                                                                                                                 |

| ข้อกำหนดการดำเนินการที่ซ่อนอยู่ ไม่ปลอดภัย หรือทำให้เข้าใจผิด        | คำสั่งติดตั้งที่ทำให้อ่านยาก ตัวติดตั้งแบบส่งต่อไปยังเชลล์ เช่น เนื้อหาที่ดาวน์โหลดแล้วรันด้วย `sh` หรือ `bash` โดยไม่มีความสามารถในการตรวจทานที่ชัดเจน ข้อกำหนด secret หรือ private key ที่ไม่ได้ประกาศ การดำเนินการ `npx @latest` ระยะไกลโดยไม่มีความสามารถในการตรวจทานที่ชัดเจน หรือเมทาดาทาที่ซ่อนว่าสิ่งใดจำเป็นจริง ๆ เพื่อให้รายการรัน |

## พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาต

ClawHub ยังตรวจทานวิธีที่ผู้เผยแพร่ใช้มาร์เก็ตเพลสด้วย อย่าใช้ ClawHub เพื่อ
บิดเบือนการค้นพบ เมตริก สัญญาณความน่าเชื่อถือ ระบบการกลั่นกรอง หรือความสนใจของผู้ใช้

พฤติกรรมมาร์เก็ตเพลสที่ไม่อนุญาตประกอบด้วย:

- การเผยแพร่จำนวนมากของรายการที่ใช้ความพยายามต่ำ ซ้ำซ้อน เป็นตัวยึดตำแหน่ง หรือ
  สร้างโดยเครื่อง ซึ่งดูเหมือนไม่มีคุณค่าจริงต่อผู้ใช้
- การท่วมพื้นผิวการค้นหาหรือหมวดหมู่ด้วย Skills หรือ Plugin ที่แทบเหมือนกัน
- การเผยแพร่รายการหลายร้อยรายการที่มีการใช้งาน การดูแลรักษา ความชัดเจนของแหล่งที่มา
  หรือความแตกต่างที่มีความหมายเพียงเล็กน้อยหรือไม่มีเลย
- การเพิ่มจำนวนการติดตั้ง การดาวน์โหลด ดาว หรือเมตริกการมีส่วนร่วมอื่น ๆ
  อย่างไม่เป็นธรรมชาติผ่านระบบอัตโนมัติ ลูปการติดตั้งด้วยตนเอง บัญชีปลอม กิจกรรมที่ประสานกัน
  การมีส่วนร่วมแบบจ่ายเงิน หรือพฤติกรรมอื่นที่ไม่เป็นธรรมชาติ
- การสร้างหรือหมุนเวียนบัญชีเพื่อหลบเลี่ยงการกลั่นกรอง การแบน ขีดจำกัดผู้เผยแพร่ หรือ
  การตรวจทานมาร์เก็ตเพลส
- การทำให้ผู้ใช้เข้าใจผิดเกี่ยวกับความเป็นเจ้าของ แหล่งที่มา ความสามารถ สถานะความปลอดภัย
  ข้อกำหนดการติดตั้ง หรือความเกี่ยวข้องกับโครงการหรือผู้เผยแพร่อื่น
- การอัปโหลดเนื้อหาที่เคยถูกซ่อน ลบ หรือบล็อกแล้วซ้ำ ๆ
  โดยไม่แก้ไขปัญหาต้นเหตุ

การเผยแพร่ปริมาณสูงไม่ใช่การละเมิดโดยอัตโนมัติ แค็ตตาล็อกขนาดใหญ่ยอมรับได้
เมื่อรายการมีความแตกต่างอย่างมีความหมาย อธิบายไว้อย่างถูกต้อง ได้รับการดูแล
และใช้งานโดยผู้ใช้จริง แค็ตตาล็อกขนาดใหญ่จะกลายเป็นปัญหาด้านความน่าเชื่อถือและความปลอดภัยเมื่อ
ปริมาณมาพร้อมกับรายการที่บาง ซ้ำซ้อน ทำให้เข้าใจผิด ไม่ได้รับการดูแล หรือ
ถูกโปรโมตอย่างไม่เป็นธรรมชาติ

## สิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาบน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ ของคุณ ให้ใช้
[คำขอสิทธิ์ในเนื้อหา](/th/clawhub/content-rights) อย่าใช้รายงานมาร์เก็ตเพลสทั่วไป
สำหรับการอ้างสิทธิ์ด้านลิขสิทธิ์หรือสิทธิ์ เว้นแต่รายการนั้นยังไม่ปลอดภัย
เป็นอันตราย หรือทำให้เข้าใจผิดด้วย

## การตรวจทานและการบังคับใช้

ClawHub อาจใช้การตรวจสอบอัตโนมัติ สัญญาณการละเมิดเชิงสถิติ รายงานผู้ใช้ และ
การตรวจทานโดยเจ้าหน้าที่เพื่อระบุเนื้อหาที่ไม่ปลอดภัยหรือพฤติกรรมการเผยแพร่ที่ละเมิด สัญญาณหนึ่ง
ไม่ได้พิสูจน์การละเมิดด้วยตัวเอง แต่ช่วยให้ ClawHub ตัดสินใจว่าสิ่งใดต้องได้รับการตรวจทาน

เราอาจ:

- ซ่อน ระงับ ลบ ลบแบบกู้คืนได้ หรือเมื่อตามชนิดทรัพยากรรองรับ
  ลบถาวรรายการที่ละเมิด
- บล็อกการดาวน์โหลดหรือการติดตั้งสำหรับรุ่นที่ไม่ปลอดภัย
- เพิกถอนโทเค็น API
- ลบเนื้อหาที่เกี่ยวข้องแบบกู้คืนได้
- จำกัดการเข้าถึงการเผยแพร่
- แบนผู้กระทำผิดซ้ำหรือร้ายแรง

เราไม่รับประกันว่าจะบังคับใช้แบบเตือนก่อนสำหรับการละเมิดที่ชัดเจน โปรดดู
[การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation) สำหรับรายงาน การระงับเพื่อกลั่นกรอง
รายการที่ซ่อน การแบน และสถานะบัญชี
