# คำขอด้านสิทธิ์ของเนื้อหา

หากคุณเชื่อว่าเนื้อหาที่เผยแพร่บน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นๆ
ของคุณ ให้ส่ง [คำขอด้านสิทธิ์ของเนื้อหาใน ClawHub](https://forms.openclaw.ai/clawhub-content-rights)

ระบุข้อมูลต่อไปนี้:

- URL `https://clawhub.ai/<owner>/skills/<skill>` ที่ถูกต้องหนึ่งรายการขึ้นไป
- ชื่อ องค์กร และอีเมลติดต่อของคุณ
- คำอธิบายสั้นๆ เกี่ยวกับข้อกังวลด้านสิทธิ์
- หลักฐานประกอบ หากมี

เจ้าหน้าที่ ClawHub ตรวจสอบคำขอด้วยตนเอง และอาจติดต่อผู้ส่งคำขอหรือ
ผู้เผยแพร่เพื่อขอข้อมูลเพิ่มเติม เนื้อหาที่ได้รับผลกระทบอาจถูกซ่อน
กู้คืน หรือคงไว้โดยไม่เปลี่ยนแปลง ทั้งนี้ขึ้นอยู่กับสถานการณ์

สำหรับเนื้อหาในตลาดที่ไม่ปลอดภัยซึ่งไม่ใช่ข้อกังวลด้านสิทธิ์ของเนื้อหา ให้ใช้
ขั้นตอนการรายงานปกติที่อธิบายไว้ใน [การกลั่นกรองและความปลอดภัยของบัญชี](/th/clawhub/moderation)
สำหรับช่องโหว่ในตัว ClawHub เอง โปรดดู [ความปลอดภัย](/th/clawhub/security)
