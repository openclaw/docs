# คำขอเกี่ยวกับสิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาที่เผยแพร่บน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ
ของคุณ ให้ส่ง [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา ClawHub](https://forms.openclaw.ai/clawhub-content-rights)

โปรดระบุ:

- URL `https://clawhub.ai/<owner>/skills/<skill>` ที่ถูกต้องแบบครบถ้วนหนึ่งรายการหรือมากกว่า
- ชื่อ องค์กร และอีเมลติดต่อของคุณ
- คำอธิบายสั้น ๆ เกี่ยวกับข้อกังวลด้านสิทธิ์
- หลักฐานสนับสนุน หากมี

ทีมงาน ClawHub ตรวจสอบคำขอด้วยตนเอง และอาจติดต่อผู้ร้องขอหรือ
ผู้เผยแพร่เพื่อขอข้อมูลเพิ่มเติม ขึ้นอยู่กับสถานการณ์ เนื้อหาที่ได้รับผลกระทบ
อาจถูกซ่อน กู้คืน หรือปล่อยไว้โดยไม่เปลี่ยนแปลง

สำหรับเนื้อหาในมาร์เก็ตเพลสที่ไม่ปลอดภัยซึ่งไม่ใช่ข้อกังวลด้านสิทธิ์ในเนื้อหา ให้ใช้
ขั้นตอนการรายงานตามปกติที่อธิบายไว้ใน [การกลั่นกรองและความปลอดภัยของบัญชี](/clawhub/moderation)
สำหรับช่องโหว่ในตัว ClawHub เอง โปรดดู [ความปลอดภัย](/th/clawhub/security)
