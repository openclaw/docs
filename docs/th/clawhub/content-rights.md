# คำขอเกี่ยวกับสิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาที่เผยแพร่บน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ
ของคุณ ให้ส่ง [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา ClawHub](https://forms.openclaw.ai/clawhub-content-rights)

ระบุข้อมูลต่อไปนี้:

- URL `https://clawhub.ai/<owner>/skills/<skill>` ที่ตรงทั้งหมดอย่างน้อยหนึ่งรายการ
- ชื่อของคุณ องค์กร และอีเมลติดต่อ
- คำอธิบายสั้น ๆ เกี่ยวกับข้อกังวลด้านสิทธิ์
- หลักฐานประกอบ หากมี

เจ้าหน้าที่ ClawHub จะตรวจสอบคำขอด้วยตนเอง และอาจติดต่อผู้ร้องขอหรือ
ผู้เผยแพร่เพื่อขอข้อมูลเพิ่มเติม เนื้อหาที่ได้รับผลกระทบอาจถูกซ่อน กู้คืน
หรือปล่อยไว้โดยไม่มีการเปลี่ยนแปลง ทั้งนี้ขึ้นอยู่กับสถานการณ์

สำหรับเนื้อหาในมาร์เก็ตเพลสที่ไม่ปลอดภัยซึ่งไม่ใช่ข้อกังวลด้านสิทธิ์ในเนื้อหา ให้ใช้
ขั้นตอนการรายงานตามปกติที่อธิบายไว้ใน [การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation)
สำหรับช่องโหว่ใน ClawHub เอง โปรดดู [ความปลอดภัย](/th/clawhub/security)
