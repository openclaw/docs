# คำขอเกี่ยวกับสิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาที่เผยแพร่บน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่น ๆ
ของคุณ โปรดส่ง [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา ClawHub](https://forms.openclaw.ai/clawhub-content-rights)

ระบุข้อมูลต่อไปนี้:

- URL `https://clawhub.ai/<owner>/skills/<skill>` ที่ถูกต้องหนึ่งรายการขึ้นไป
- ชื่อ องค์กร และอีเมลติดต่อของคุณ
- คำอธิบายสั้น ๆ เกี่ยวกับข้อกังวลด้านสิทธิ์
- หลักฐานสนับสนุน หากมี

เจ้าหน้าที่ ClawHub จะตรวจสอบคำขอด้วยตนเอง และอาจติดต่อผู้ร้องขอหรือ
ผู้เผยแพร่เพื่อขอข้อมูลเพิ่มเติม ขึ้นอยู่กับสถานการณ์ เนื้อหาที่ได้รับผลกระทบ
อาจถูกซ่อน กู้คืน หรือคงไว้โดยไม่เปลี่ยนแปลง

สำหรับเนื้อหาในมาร์เก็ตเพลสที่ไม่ปลอดภัยซึ่งไม่ใช่ข้อกังวลด้านสิทธิ์ในเนื้อหา ให้ใช้
ขั้นตอนการรายงานตามปกติที่อธิบายไว้ใน [การดูแลเนื้อหาและความปลอดภัยของบัญชี](/clawhub/moderation)
สำหรับช่องโหว่ใน ClawHub เอง โปรดดู [ความปลอดภัย](/th/clawhub/security)
