# คำขอเกี่ยวกับสิทธิ์ในเนื้อหา

หากคุณเชื่อว่าเนื้อหาที่เผยแพร่บน ClawHub ละเมิดลิขสิทธิ์หรือสิทธิ์อื่นของคุณ
ให้ส่ง [คำขอเกี่ยวกับสิทธิ์ในเนื้อหา ClawHub](https://forms.openclaw.ai/clawhub-content-rights)

โปรดระบุ:

- URL `https://clawhub.ai/<owner>/skills/<skill>` ที่ตรงกันทุกประการอย่างน้อยหนึ่งรายการ
- ชื่อ องค์กร และอีเมลติดต่อของคุณ
- คำอธิบายสั้นๆ เกี่ยวกับข้อกังวลด้านสิทธิ์
- หลักฐานประกอบ หากมี

เจ้าหน้าที่ ClawHub จะตรวจสอบคำขอด้วยตนเอง และอาจติดต่อผู้ร้องขอหรือ
ผู้เผยแพร่เพื่อขอข้อมูลเพิ่มเติม เนื้อหาที่ได้รับผลกระทบอาจถูกซ่อน กู้คืน
หรือปล่อยไว้โดยไม่เปลี่ยนแปลง ขึ้นอยู่กับสถานการณ์

สำหรับเนื้อหาในตลาดกลางที่ไม่ปลอดภัยซึ่งไม่ใช่ข้อกังวลด้านสิทธิ์ในเนื้อหา ให้ใช้
ขั้นตอนการรายงานตามปกติที่อธิบายไว้ใน [การดูแลและความปลอดภัยของบัญชี](/clawhub/moderation)
สำหรับช่องโหว่ใน ClawHub เอง โปรดดู [ความปลอดภัย](/th/clawhub/security)
