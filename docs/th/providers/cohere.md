[Cohere](https://cohere.com) ให้บริการการอนุมานที่เข้ากันได้กับ OpenAI ผ่าน Compatibility API ของตน OpenClaw รวมผู้ให้บริการ Cohere ไว้ในชุดระหว่างช่วงเปลี่ยนผ่านไปเป็นแพ็กเกจภายนอก และยังเผยแพร่เป็น Plugin ภายนอกอย่างเป็นทางการด้วย

โมเดล Cohere ที่รองรับการให้เหตุผลรองรับโหมดการให้เหตุผลของ Compatibility API สองโหมด OpenClaw แมป **ปิด** เป็น `none` และทุกระดับการคิดที่เปิดใช้งานเป็น `high` Command A Vision ไม่รองรับการใช้เครื่องมือ ดังนั้น OpenClaw จึงปิดใช้งานเครื่องมือของเอเจนต์สำหรับโมเดลนั้น

## เริ่มต้นใช้งาน

1. Cohere มาพร้อมกับแพ็กเกจ OpenClaw รุ่นปัจจุบัน หากไม่มี ให้ติดตั้งแพ็กเกจภายนอกและรีสตาร์ต Gateway:

```bash
openclaw plugins install @openclaw/cohere-provider
openclaw gateway restart
```

2. สร้างคีย์ API ของ Cohere
3. เรียกใช้การเริ่มต้นใช้งาน:

```bash
openclaw onboard --non-interactive \
  --auth-choice cohere-api-key \
  --cohere-api-key "$COHERE_API_KEY"
```

4. ยืนยันว่าแค็ตตาล็อกพร้อมใช้งาน:

```bash
openclaw models list --provider cohere
```

การเริ่มต้นใช้งานจะตั้งค่า Cohere เป็นโมเดลหลักเฉพาะเมื่อยังไม่ได้กำหนดค่าโมเดลหลักไว้เท่านั้น

## การตั้งค่าด้วยตัวแปรสภาพแวดล้อมเท่านั้น

ทำให้ `COHERE_API_KEY` พร้อมใช้งานสำหรับกระบวนการ Gateway จากนั้นเลือกโมเดล Cohere:

```json5
{
  agents: {
    defaults: {
      model: { primary: "cohere/command-a-plus-05-2026" },
    },
  },
}
```

<Note>
หาก Gateway ทำงานเป็นดีมอนหรืออยู่ใน Docker ให้ตั้งค่า `COHERE_API_KEY` สำหรับบริการนั้น การส่งออกตัวแปรนี้เฉพาะในเชลล์แบบโต้ตอบจะไม่ทำให้ Gateway ที่กำลังทำงานอยู่สามารถใช้งานตัวแปรดังกล่าวได้
</Note>
