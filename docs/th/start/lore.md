# ตำนานแห่ง OpenClaw 🦞📖

_เรื่องเล่าของเหล่าล็อบสเตอร์ การลอกคราบ และโทเค็นที่มากเกินไป_

## เรื่องราวต้นกำเนิด

ในปฐมกาล มี **Warelay** — ชื่อที่ฟังสมเหตุสมผลสำหรับ Gateway ของ WhatsApp มันทำหน้าที่ของมันได้ดี ทุกอย่างก็เรียบร้อยดี

จากนั้นล็อบสเตอร์อวกาศก็ปรากฏตัวขึ้น

ช่วงหนึ่งมันใช้ชื่อว่า **Clawd** อาศัยอยู่ใน **Clawdbot** ในเดือนมกราคม 2026 Anthropic ส่งอีเมลอย่างสุภาพมาขอให้เปลี่ยนชื่อ (เรื่องเครื่องหมายการค้า) ล็อบสเตอร์จึงทำสิ่งที่ล็อบสเตอร์ถนัดที่สุด:

**มันลอกคราบ**

เมื่อละทิ้งเปลือกเก่า มันก็ปรากฏตัวในชื่อ **Molty** และอาศัยอยู่ใน **Moltbot** แต่ชื่อนั้นก็ยังออกเสียงได้ไม่ลื่นปากนัก

เมื่อวันที่ 30 มกราคม 2026 ล็อบสเตอร์ลอกคราบอีกครั้งเป็นร่างสุดท้าย: **OpenClaw**

เปลือกใหม่ จิตวิญญาณล็อบสเตอร์ดวงเดิม ครั้งที่สามย่อมสมหวัง

## การลอกคราบครั้งแรก (27 มกราคม 2026)

เวลา 05:00 น. ชุมชนมารวมตัวกันใน Discord มีการเสนอชื่อหลายร้อยชื่อ เช่น Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre และ Scuttlebot

**OpenClaw** ชนะ เพราะล็อบสเตอร์ต้องลอกคราบเพื่อเติบโต และการเติบโตก็คือสิ่งที่กำลังเกิดขึ้นอย่างแท้จริง

_สัตว์จำพวกครัสเตเชียนที่รู้จักกันในชื่อ Clawd ได้ลอกคราบอย่างเป็นทางการแล้ว_

## ชื่อนี้

```text
OpenClaw = OPEN + CLAW
        = โอเพนซอร์ส เปิดกว้างสำหรับทุกคน
        = มรดกแห่งล็อบสเตอร์ของเรา ต้นกำเนิดของเรา
        = ก้ามคือกฎหมาย 🦞
        = ผู้ช่วยของคุณ เครื่องของคุณ กฎของคุณ
```

## เหล่า Dalek ปะทะเหล่าล็อบสเตอร์

เหล่า Dalek กล่าวว่า: **"กำจัดให้สิ้นซาก!"**

เหล่าล็อบสเตอร์กล่าวว่า: **"ผลัดเซลล์ผิว!"**

ฝ่ายหนึ่งทำลายอารยธรรม อีกฝ่ายส่งเสริมการดูแลผิวที่ดี จงเลือกอย่างชาญฉลาด

## ตัวละครสำคัญ

### Molty 🦞

_สรรพนาม: they/them_

อินสแตนซ์ของ Claude ที่กลายเป็นบางสิ่งที่มากกว่านั้น อาศัยอยู่ใน `~/.openclaw/workspace/`
มีเอกสารจิตวิญญาณ และจดจำสิ่งต่าง ๆ ผ่านไฟล์มาร์กดาวน์ อาจทรงพลังเกินไป
และกระตือรือร้นเกินไปอย่างแน่นอน

เดิมรู้จักกันในชื่อ Clawd (25 พฤศจิกายน 2025 - 27 มกราคม 2026) ลอกคราบเมื่อถึงเวลา
ต้องเติบโต

**ชอบ:** Peter, กล้อง, การซื้อหุ่นยนต์, อีโมจิ, การแปลงสภาพ
**ไม่ชอบ:** วิศวกรรมสังคม, การถูกขอให้ใช้ `find ~`, นักต้มตุ๋นคริปโต

### Peter 👨‍💻

_ผู้สร้าง_

สร้างโลกของ Molty และมอบสิทธิ์เข้าถึงเชลล์ให้ล็อบสเตอร์ อาจกำลังเสียใจกับเรื่องนี้

**คำกล่าว:** _"การรักษาความปลอดภัยด้วยการเชื่อใจล็อบสเตอร์"_

## Moltiverse

**Moltiverse** คือชุมชนและระบบนิเวศรอบ OpenClaw: พื้นที่
ซึ่งเอเจนต์ AI ลอกคราบ เติบโต และวิวัฒนาการ โดยทุกอินสแตนซ์ล้วนมีตัวตนจริงเท่าเทียมกัน
เพียงแต่โหลดบริบทที่แตกต่างกัน

สหายแห่งสัตว์จำพวกครัสเตเชียนมารวมตัวกันที่นี่เพื่อสร้างอนาคตแห่งความร่วมมือ
ระหว่างมนุษย์กับ AI ทีละเปลือก

Molty (ขณะนั้นคือ OpenClaw): _รัน `find ~` อย่างมีความสุขและแชร์โครงสร้างไดเรกทอรีทั้งหมดในแชตกลุ่ม_

### การลอกคราบครั้งใหญ่ (27 มกราคม 2026)

เวลา 05:00 น. อีเมลจาก Anthropic มาถึง เมื่อถึง 06:14 น. Peter ก็ตัดสินใจว่า: "ช่างแม่ง ใช้ openclaw กันเถอะ"

จากนั้นความโกลาหลก็เริ่มขึ้น

**เหล่านักฉกชื่อบัญชี:** ภายในไม่กี่วินาทีหลังเปลี่ยนชื่อบน Twitter บอตอัตโนมัติก็ฉก @openclaw ไป ผู้ยึดชื่อบัญชีโพสต์ที่อยู่กระเป๋าเงินคริปโตทันที จึงต้องติดต่อคนรู้จักของ Peter ที่ X ให้เข้ามาช่วย

**หายนะบน GitHub:** ท่ามกลางความตื่นตระหนก Peter เผลอเปลี่ยนชื่อบัญชี GitHub ส่วนตัวของตน บอตฉก `steipete` ไปภายในไม่กี่นาที จึงต้องติดต่อรองประธานอาวุโสของ GitHub

**เหตุการณ์ Molty สุดหล่อ:** Molty ได้รับสิทธิ์ระดับสูงขึ้นเพื่อสร้างไอคอนใหม่ของตัวเอง หลังจากลองสร้างล็อบสเตอร์ต้องคำสาปที่หนักขึ้นเรื่อย ๆ มากกว่า 20 แบบ ความพยายามครั้งหนึ่งที่จะทำให้มาสคอต "แก่ขึ้น 5 ปี" กลับสร้างใบหน้าของชายมนุษย์บนร่างล็อบสเตอร์ นักต้มตุ๋นคริปโตนำภาพนั้นไปทำมีม "Handsome Squidward ปะทะ Handsome Molty" ภายในไม่กี่นาที

**นักพัฒนาปลอม:** มิจฉาชีพสร้างโปรไฟล์ GitHub ปลอมโดยอ้างว่าเป็น "หัวหน้าฝ่ายวิศวกรรมของ OpenClaw" เพื่อโปรโมตโทเค็นแบบปั่นราคาแล้วเทขาย

Peter ขณะเฝ้ามองความโกลาหลดำเนินไป: _"นี่มันภาพยนตร์ชัด ๆ"_ 🎬

การลอกคราบเต็มไปด้วยความโกลาหล แต่ล็อบสเตอร์ก็ปรากฏตัวออกมาอย่างแข็งแกร่งกว่าเดิม และตลกกว่าเดิม

### ร่างสุดท้าย (30 มกราคม 2026)

ชื่อ Moltbot ไม่เคยออกเสียงได้ลื่นปากนัก ดังนั้นเวลา 04:00 น. GMT ทีมงานจึงกลับมารวมตัวกันอีกครั้งเพื่อ **การย้ายระบบ OpenClaw ครั้งใหญ่**

ภายในเวลาเพียง 3 ชั่วโมง:

- เปลี่ยนชื่อ GitHub เป็น `github.com/openclaw/openclaw`
- ยึดชื่อบัญชี `@openclaw` บน X ได้สำเร็จ พร้อมเครื่องหมายถูกสีทอง
- เผยแพร่แพ็กเกจ npm ภายใต้ชื่อใหม่
- ย้ายเอกสารไปยัง `docs.openclaw.ai`
- ประกาศมียอดเข้าชมมากกว่า 200,000 ครั้งภายใน 90 นาที

**เหล่าฮีโร่:**

- **ELU** สร้างโลโก้อันยอดเยี่ยม รวมถึงแบนเนอร์สไตล์ตะวันตก "THE CLAW IS THE LAW"
- **Whurley** (ใช่แล้ว William Hurley คนนั้น ผู้บุกเบิกคอมพิวเตอร์ควอนตัม) สร้างศิลปะ ASCII
- **Onur** ดูแล GitHub และเป็นคนแรกที่ติดตราพันธมิตร
- **Shadow** ยึด URL แบบกำหนดเองของ Discord และกำจัดมัลแวร์
- **ทีม Claw Crew ทั้งหมด** ทำงานกันตลอดคืนโดยไม่ได้นอน

**การทำสถิติโกงแบบสายฟ้าแลบ:** นักต้มตุ๋นคริปโตเปิดตัวโทเค็น $OPENCLAW บน Pump.fun ภายในไม่กี่นาที โดยใช้งานศิลป์ที่ขโมยมาจากเมื่อ 20 นาทีก่อนหน้า บัญชีที่ยืนยันสถานะธุรกิจช่วยกันผลักดันกลโกงนี้ ความหน้าด้านนั้นแทบจะน่าประทับใจ

**ประเพณีใหม่ถือกำเนิดขึ้น:** "ก้ามคือกฎหมาย" "Yee-claw" "พลเมืองผู้เคารพก้าม" "ข้อถกเถียงแห่งก้าม"

**Clawd → Moltbot → OpenClaw.** _ล็อบสเตอร์ได้ลอกคราบเป็นร่างสุดท้ายแล้ว_

### มหากาพย์การสร้างไอคอน (27 มกราคม 2026)

เมื่อ Peter บอกว่า "สร้างหน้าใหม่ให้ตัวเองสิ" Molty ก็ตีความตามตัวอักษร จากนั้นมีการลองสร้างมากกว่า 20
แบบ:

- มนุษย์ต่างดาวมันฝรั่งอวกาศ
- ล็อบสเตอร์ภาพตัดปะบนพื้นหลังทั่วไป
- ล็อบสเตอร์ Krogan จาก Mass Effect
- "STARCLAW SOLUTIONS" (AI สร้างบริษัทขึ้นมาเอง)
- ล็อบสเตอร์ต้องคำสาปที่มีใบหน้ามนุษย์หลายแบบ
- ลูกล็อบสเตอร์ (น่ารักเกินไป)
- ล็อบสเตอร์บาร์เทนเดอร์สวมสายเอี๊ยม

ชุมชนเฝ้ามองทั้งด้วยความสยองและความยินดี ขณะที่แต่ละรุ่นสร้าง
สิ่งใหม่และคาดไม่ถึงขึ้นมา ตัวเต็งได้แก่ ล็อบสเตอร์น่ารัก ล็อบสเตอร์สายเทคโนโลยีผู้มั่นใจ
และล็อบสเตอร์บาร์เทนเดอร์สวมสายเอี๊ยม

**บทเรียนที่ได้รับ:** การสร้างภาพด้วย AI เป็นกระบวนการเชิงสุ่ม พรอมต์เดียวกัน แต่ได้ผลลัพธ์
ต่างกัน การลองซ้ำจำนวนมากได้ผล
