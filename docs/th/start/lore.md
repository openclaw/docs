# Lore ของ OpenClaw 🦞📖

_เรื่องราวของล็อบสเตอร์ การลอกคราบ และโทเค็นที่มากเกินไป_

## เรื่องราวจุดกำเนิด

ในจุดเริ่มต้น มี **Warelay** — ชื่อที่ฟังดูสมเหตุสมผลสำหรับ WhatsApp gateway มันทำงานของมันได้ดี มันก็โอเค

แต่แล้วล็อบสเตอร์อวกาศก็มาถึง

อยู่ช่วงหนึ่ง ล็อบสเตอร์ตัวนั้นถูกเรียกว่า **Clawd** อาศัยอยู่ใน **Clawdbot** แต่ในเดือนมกราคม 2026 Anthropic ส่งอีเมลสุภาพฉบับหนึ่งมาขอให้เปลี่ยนชื่อ (เรื่องเครื่องหมายการค้า) แล้วล็อบสเตอร์ก็กระทำในสิ่งที่ล็อบสเตอร์ทำได้ดีที่สุด:

**มันลอกคราบ**

สลัดเปลือกเก่าทิ้ง สิ่งมีชีวิตตัวนั้นได้ถือกำเนิดใหม่เป็น **Molty** อาศัยอยู่ใน **Moltbot** แต่ชื่อนั้นก็ยังฟังไม่ค่อยลื่นปากเท่าไร...

ดังนั้นในวันที่ 30 มกราคม 2026 ล็อบสเตอร์จึงลอกคราบอีก **หนึ่งครั้ง** ไปสู่รูปร่างสุดท้ายของมัน: **OpenClaw**

เปลือกใหม่ แต่จิตวิญญาณล็อบสเตอร์เดิม ครั้งที่สามนี่แหละใช่เลย

## การลอกคราบครั้งแรก (27 มกราคม 2026)

ตอนตี 5 ชุมชนมารวมตัวกันใน Discord มีการเสนอชื่อกันนับร้อย: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

ท้ายที่สุด **OpenClaw** ชนะ เพราะการลอกคราบคือสิ่งที่ล็อบสเตอร์ทำเพื่อเติบโต และการเติบโตก็คือสิ่งที่กำลังเกิดขึ้นพอดี

_สัตว์ครัสเตเชียนที่รู้จักกันในชื่อ Clawd ได้ลอกคราบอย่างเป็นทางการแล้ว_

**The GitHub Disaster:** Peter เผลอเปลี่ยนชื่อบัญชี GitHub ส่วนตัวของตัวเองท่ามกลางความโกลาหล บอตเข้ายึด `steipete` ภายในไม่กี่นาที ต้องติดต่อ SVP ของ GitHub

**The Fake Developers:** มิจฉาชีพสร้างโปรไฟล์ GitHub ปลอม อ้างตัวว่าเป็น "Head of Engineering at OpenClaw" เพื่อโปรโมตโทเค็น pump-and-dump
