# `openclaw docs`

ค้นหาดัชนีเอกสาร OpenClaw ที่ใช้งานอยู่จากเทอร์มินัล

## การใช้งาน

```bash
openclaw docs                       # แสดงจุดเริ่มต้นของเอกสารและตัวอย่างการค้นหา
openclaw docs <query...>            # ค้นหาดัชนีเอกสารที่ใช้งานอยู่
```

| อาร์กิวเมนต์ | คำอธิบาย                                                                                  |
| ------------ | ------------------------------------------------------------------------------------------ |
| `[query...]` | คำค้นหาแบบอิสระ คำค้นหาหลายคำจะถูกเชื่อมด้วยช่องว่างและส่งเป็นคำค้นหาเดียวกัน |

หากไม่มีคำค้นหา `openclaw docs` จะแสดง URL จุดเริ่มต้นของเอกสารและคำสั่งค้นหาตัวอย่างแทนการดำเนินการค้นหา

## ตัวอย่าง

```bash
openclaw docs browser existing-session
openclaw docs sandbox allowHostControl
openclaw docs gateway token secretref
```

## วิธีการทำงาน

`openclaw docs` เรียก `https://docs.openclaw.ai/api/search` และแสดงผลลัพธ์ JSON คำขอค้นหาใช้ระยะหมดเวลาคงที่ 30 วินาที

## ผลลัพธ์

ในเทอร์มินัลแบบสมบูรณ์ (TTY) ผลลัพธ์จะแสดงเป็นหัวข้อตามด้วยรายการหัวข้อย่อย ได้แก่ ชื่อหน้า URL เอกสารที่เป็นลิงก์ และข้อความตัวอย่างสั้น ๆ ในบรรทัดถัดไป หากไม่มีผลลัพธ์ จะแสดงข้อความ "ไม่พบผลลัพธ์"

ในเอาต์พุตแบบไม่สมบูรณ์ (ส่งผ่านไปป์, `--no-color`, สคริปต์) ข้อมูลเดียวกันจะแสดงเป็น Markdown:

```markdown
# ค้นหาเอกสาร: <query>

- [ชื่อ](https://docs.openclaw.ai/...) - ข้อความตัวอย่าง
- [ชื่อ](https://docs.openclaw.ai/...) - ข้อความตัวอย่าง
```

## รหัสการออก

| รหัส | ความหมาย                                                                      |
| ---- | ----------------------------------------------------------------------------- |
| `0`  | การค้นหาสำเร็จ รวมถึงการตอบกลับที่ไม่มีผลลัพธ์                                |
| `1`  | การเรียก API ค้นหาเอกสารที่โฮสต์ไว้ล้มเหลว โดย stderr จะแสดงข้อความข้อผิดพลาด |
