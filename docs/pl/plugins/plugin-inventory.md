# Inventory Plugin

Ta strona jest generowana z `extensions/*/package.json`, `openclaw.plugin.json`,
oraz wykluczeń `files` głównego pakietu npm. Wygeneruj ją ponownie za pomocą:

```bash
pnpm plugins:inventory:gen
```

## Definicje

- **Główny pakiet npm:** wbudowany w pakiet npm `openclaw` i dostępny bez osobnej instalacji Plugin.
- **Oficjalny pakiet zewnętrzny:** Plugin utrzymywany przez OpenClaw, pominięty w głównym pakiecie npm, zachowany w tym oficjalnym inventory i instalowany na żądanie przez ClawHub i/lub npm.
- **Tylko checkout źródłowy:** lokalny dla repozytorium Plugin pominięty w opublikowanych artefaktach npm i niereklamowany jako pakiet możliwy do zainstalowania.

Checkouty źródłowe różnią się od instalacji npm: po `pnpm install` dołączone
Pluginy ładują się z `extensions/<id>`, więc lokalne edycje i lokalne dla pakietu
zależności workspace są dostępne.

## Zainstaluj Plugin

Użyj kolumny **Dystrybucja**, aby zdecydować, czy instalacja jest potrzebna. Pluginy, które
mówią `included in OpenClaw`, są już obecne w głównym pakiecie. Oficjalne
pakiety zewnętrzne wymagają jednej instalacji, a potem restartu Gateway.

Na przykład Discord jest oficjalnym pakietem zewnętrznym:

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

Nagie specyfikacje pakietów próbują najpierw ClawHub, a potem awaryjnie npm. Aby wymusić źródło, użyj
`clawhub:@openclaw/discord` lub `npm:@openclaw/discord`. Po instalacji postępuj zgodnie
z dokumentacją konfiguracji Plugin, taką jak [Discord](/pl/channels/discord), aby dodać dane uwierzytelniające
i konfigurację kanału. Zobacz [Zarządzaj Pluginami](/pl/plugins/manage-plugins), aby poznać polecenia aktualizacji,
odinstalowania i publikowania.
