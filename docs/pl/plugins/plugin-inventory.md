# Inwentarz Plugin

Ta strona jest generowana z `extensions/*/package.json`, `openclaw.plugin.json`
oraz wykluczeń `files` głównego pakietu npm. Wygeneruj ją ponownie za pomocą:

```bash
pnpm plugins:inventory:gen
```

## Definicje

- **Główny pakiet npm:** wbudowany w pakiet npm `openclaw` i dostępny bez osobnej instalacji Plugin.
- **Oficjalny pakiet zewnętrzny:** utrzymywany przez OpenClaw Plugin pominięty w głównym pakiecie npm i instalowany przez ClawHub i/lub npm.
- **Tylko checkout źródeł:** lokalny dla repozytorium Plugin pominięty w opublikowanych artefaktach npm i niereklamowany jako pakiet możliwy do zainstalowania.

Checkouty źródeł różnią się od instalacji npm: po `pnpm install` dołączone
Pluginy ładują się z `extensions/<id>`, więc lokalne edycje i zależności obszaru roboczego
lokalne dla pakietu są dostępne.
