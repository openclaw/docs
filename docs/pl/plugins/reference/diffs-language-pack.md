Dodaje wyróżnianie składni dla języków spoza domyślnego zestawu przeglądarki różnic.

## Dodane języki

Podstawowy Plugin `diffs` wyróżnia już typowe języki udokumentowane w sekcji [Różnice](/pl/tools/diffs). Zainstaluj ten pakiet językowy, gdy chcesz wyróżniania składni dla szerszego zestawu języków obsługiwanych przez Shiki. Jeśli pakiet nie jest zainstalowany, te pliki nadal są renderowane jako czytelny zwykły tekst.

Przykłady obejmują Astro, Vue, Svelte, MDX, GraphQL, Terraform/HCL, Nix, Clojure, Elixir, Haskell, OCaml, Scala, Zig, Solidity, Verilog/VHDL, Fortran, MATLAB, LaTeX, Mermaid, Sass/Less/SCSS, Nginx, Apache, CSV, dotenv, INI oraz pliki diff.

Zobacz [języki Shiki](https://shiki.style/languages), aby zapoznać się z oficjalnym katalogiem języków i aliasów Shiki.
