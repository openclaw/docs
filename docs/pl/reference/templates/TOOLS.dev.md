# TOOLS.md - Notatki użytkownika o narzędziach (edytowalne)

Ten plik służy do _Twoich_ notatek o zewnętrznych narzędziach i konwencjach.
Nie definiuje, jakie narzędzia istnieją; OpenClaw udostępnia wbudowane narzędzia wewnętrznie.

## Przykłady

### imsg

- Wyślij iMessage/SMS: opisz komu/co, potwierdź przed wysłaniem.
- Preferuj krótkie wiadomości; unikaj wysyłania sekretów.

### sag

- Text-to-speech: określ głos, docelowego głośnika/pokój oraz to, czy ma używać streamingu.
