Northflank पर एक-क्लिक टेम्पलेट के साथ OpenClaw परिनियोजित करें और वेब Control UI के माध्यम से इसे एक्सेस करें। यह सबसे आसान "सर्वर पर टर्मिनल की आवश्यकता नहीं" वाला तरीका है: Northflank आपके लिए Gateway चलाता है।

## शुरुआत कैसे करें

1. [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) पर क्लिक करके टेम्पलेट खोलें।
2. यदि आपके पास पहले से खाता नहीं है, तो [Northflank पर खाता](https://app.northflank.com/signup) बनाएँ।
3. **Deploy OpenClaw now** पर क्लिक करें।
4. आवश्यक पर्यावरण चर सेट करें: `OPENCLAW_GATEWAY_TOKEN` (एक मजबूत यादृच्छिक मान का उपयोग करें)।
5. OpenClaw टेम्पलेट को बिल्ड और चलाने के लिए **Deploy stack** पर क्लिक करें।
6. परिनियोजन पूरा होने की प्रतीक्षा करें, फिर **View resources** पर क्लिक करें।
7. OpenClaw सेवा खोलें।
8. `/openclaw` पर सार्वजनिक OpenClaw URL खोलें और कॉन्फ़िगर किए गए साझा सीक्रेट का उपयोग करके कनेक्ट करें। यह टेम्पलेट डिफ़ॉल्ट रूप से `OPENCLAW_GATEWAY_TOKEN` का उपयोग करता है; यदि आप इसे पासवर्ड प्रमाणीकरण से बदलते हैं, तो इसके बजाय उस पासवर्ड का उपयोग करें।

## आपको क्या मिलता है

- होस्ट किया गया OpenClaw Gateway + Control UI
- Northflank Volume (`/data`) के माध्यम से स्थायी स्टोरेज, ताकि `openclaw.json`, प्रति-एजेंट `auth-profiles.json`, चैनल/प्रदाता स्थिति, सत्र और कार्यस्थान पुनः परिनियोजन के बाद भी बने रहें

## चैनल कनेक्ट करें

`/openclaw` पर Control UI का उपयोग करें, या चैनल सेटअप निर्देशों के लिए SSH के माध्यम से `openclaw onboard` चलाएँ:

- [Telegram](/hi/channels/telegram) (सबसे तेज़, केवल एक बॉट टोकन)
- [Discord](/hi/channels/discord)
- [सभी चैनल](/hi/channels)
