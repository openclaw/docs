# Northflank

एक-क्लिक टेम्पलेट के साथ Northflank पर OpenClaw डिप्लॉय करें और वेब Control UI के माध्यम से इसे एक्सेस करें।
यह सबसे आसान "सर्वर पर कोई टर्मिनल नहीं" वाला तरीका है: Northflank आपके लिए Gateway चलाता है।

## शुरुआत कैसे करें

1. टेम्पलेट खोलने के लिए [OpenClaw डिप्लॉय करें](https://northflank.com/stacks/deploy-openclaw) पर क्लिक करें।
2. अगर आपके पास पहले से खाता नहीं है, तो [Northflank पर खाता](https://app.northflank.com/signup) बनाएँ।
3. **अभी OpenClaw डिप्लॉय करें** पर क्लिक करें।
4. आवश्यक environment variable सेट करें: `OPENCLAW_GATEWAY_TOKEN` (एक मजबूत random value का उपयोग करें)।
5. OpenClaw टेम्पलेट को build और run करने के लिए **stack डिप्लॉय करें** पर क्लिक करें।
6. deployment पूरा होने की प्रतीक्षा करें, फिर **resources देखें** पर क्लिक करें।
7. OpenClaw service खोलें।
8. `/openclaw` पर सार्वजनिक OpenClaw URL खोलें और कॉन्फ़िगर किए गए shared secret का उपयोग करके कनेक्ट करें। यह टेम्पलेट डिफ़ॉल्ट रूप से `OPENCLAW_GATEWAY_TOKEN` का उपयोग करता है; अगर आप इसे password auth से बदलते हैं, तो उसके बजाय वही password उपयोग करें।

## आपको क्या मिलता है

- Hosted OpenClaw Gateway + Control UI
- Northflank Volume (`/data`) के माध्यम से persistent storage, ताकि `openclaw.json`,
  प्रति-agent `auth-profiles.json`, channel/provider state, sessions, और
  workspace redeploys के बाद भी बने रहें
