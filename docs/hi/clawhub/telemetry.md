# टेलीमेट्री

ClawHub समेकित इंस्टॉल संख्याएँ निकालने के लिए न्यूनतम CLI टेलीमेट्री का उपयोग करता है।

## टेलीमेट्री कब एकत्र की जाती है

टेलीमेट्री केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हों।
- आप `clawhub install <slug>` चलाएँ।
- टेलीमेट्री **अक्षम नहीं है** (नीचे “अक्षम कैसे करें” देखें)।

यदि आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

प्रत्येक रिपोर्ट किए गए `clawhub install` पर, CLI एक सर्वोत्तम-प्रयास इंस्टॉल इवेंट भेजता है।

इवेंट में शामिल है:

- `slug`: इंस्टॉल किए गए कौशल का स्लग।
- `version`: ज्ञात होने पर, इंस्टॉल किया गया संस्करण।

### हम क्या _नहीं_ एकत्र करते

- कोई फ़ोल्डर पथ या फ़ोल्डर से बने पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- कोई प्रति-रन लॉग, प्रॉम्प्ट, या अन्य CLI आउटपुट नहीं।

## इंस्टॉल संख्याएँ

ClawHub प्रति कौशल समेकित काउंटर बनाए रखता है:

- `installsAllTime`: वे विशिष्ट उपयोगकर्ता जिन्होंने कौशल के लिए कम से कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे विशिष्ट उपयोगकर्ता जिन्होंने कोई इंस्टॉल रिपोर्ट किया है और अपनी
  टेलीमेट्री नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

सभी को केवल **समेकित इंस्टॉल काउंटर** दिखाई देते हैं।

अपना खाता हटाने से आपका टेलीमेट्री डेटा भी हट जाता है।

## टेलीमेट्री अक्षम कैसे करें

पर्यावरण चर सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

यह सेट होने पर, CLI इंस्टॉल टेलीमेट्री नहीं भेजेगा।
