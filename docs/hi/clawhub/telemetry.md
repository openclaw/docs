# टेलीमेट्री

ClawHub कुल इंस्टॉल गणनाएं निकालने के लिए न्यूनतम CLI टेलीमेट्री का उपयोग करता है।

## टेलीमेट्री कब एकत्र की जाती है

टेलीमेट्री केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हों।
- आप `clawhub install <slug>` चलाएं।
- टेलीमेट्री **अक्षम न हो** (नीचे “कैसे अक्षम करें” देखें)।

अगर आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

हर रिपोर्ट किए गए `clawhub install` पर, CLI एक सर्वोत्तम-प्रयास इंस्टॉल इवेंट भेजता है।

इवेंट में शामिल है:

- `slug`: इंस्टॉल किए गए स्किल का slug।
- `version`: इंस्टॉल किया गया संस्करण, जब ज्ञात हो।

### हम क्या एकत्र _नहीं_ करते

- कोई फ़ोल्डर पाथ या फ़ोल्डर से निकाले गए पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- कोई प्रति-रन लॉग, प्रॉम्प्ट, या अन्य CLI आउटपुट नहीं।

## इंस्टॉल गणनाएं

ClawHub प्रत्येक स्किल के लिए कुल काउंटर बनाए रखता है:

- `installsAllTime`: वे विशिष्ट उपयोगकर्ता जिन्होंने स्किल के लिए कम से कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे विशिष्ट उपयोगकर्ता जिन्होंने एक इंस्टॉल रिपोर्ट किया है और अपनी
  टेलीमेट्री नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

सभी को केवल **कुल इंस्टॉल काउंटर** दिखते हैं।

आपका खाता हटाने से आपका टेलीमेट्री डेटा भी हट जाता है।

## टेलीमेट्री कैसे अक्षम करें

पर्यावरण चर सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

यह सेट होने पर, CLI इंस्टॉल टेलीमेट्री नहीं भेजेगा।
