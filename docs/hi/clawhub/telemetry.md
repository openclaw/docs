# टेलीमेट्री

ClawHub समेकित इंस्टॉल संख्याओं की गणना करने के लिए न्यूनतम CLI टेलीमेट्री का उपयोग करता है।

## टेलीमेट्री कब एकत्र की जाती है

टेलीमेट्री केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हों।
- आप `clawhub install <slug>` चलाएँ।
- टेलीमेट्री **अक्षम न हो** (नीचे “अक्षम करने का तरीका” देखें)।

यदि आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

रिपोर्ट किए गए प्रत्येक `clawhub install` पर, CLI सर्वोत्तम प्रयास के आधार पर एक इंस्टॉल इवेंट भेजता है।

इवेंट में शामिल हैं:

- `slug`: इंस्टॉल किए गए skill का स्लग।
- `version`: इंस्टॉल किया गया संस्करण, यदि ज्ञात हो।

### हम क्या एकत्र _नहीं_ करते

- कोई फ़ोल्डर पथ या फ़ोल्डर से व्युत्पन्न पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- प्रति-रन लॉग, प्रॉम्प्ट या अन्य CLI आउटपुट नहीं।

## इंस्टॉल संख्याएँ

ClawHub प्रत्येक skill के लिए समेकित काउंटर बनाए रखता है:

- `installsAllTime`: वे विशिष्ट उपयोगकर्ता जिन्होंने skill के लिए कम-से-कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे विशिष्ट उपयोगकर्ता जिन्होंने इंस्टॉल रिपोर्ट किया है और अपनी
  टेलीमेट्री नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

सभी को केवल **समेकित इंस्टॉल काउंटर** दिखाई देते हैं।

आपका खाता हटाने पर आपका टेलीमेट्री डेटा भी हटा दिया जाता है।

## टेलीमेट्री अक्षम करने का तरीका

पर्यावरण वेरिएबल सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

इसे सेट करने पर CLI इंस्टॉल टेलीमेट्री नहीं भेजेगा।
