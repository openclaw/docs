# Telemetry

ClawHub कुल इंस्टॉल संख्याओं की गणना के लिए न्यूनतम CLI Telemetry का उपयोग करता है।

## Telemetry कब एकत्र की जाती है

Telemetry केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हों।
- आप `clawhub install <slug>` चलाएँ।
- Telemetry **अक्षम न हो** (नीचे “अक्षम कैसे करें” देखें)।

यदि आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

रिपोर्ट किए गए प्रत्येक `clawhub install` पर, CLI एक सर्वोत्तम-प्रयास इंस्टॉल इवेंट भेजता है।

इवेंट में शामिल होता है:

- `slug`: इंस्टॉल की गई skill slug।
- `version`: इंस्टॉल किया गया संस्करण, जब ज्ञात हो।

### हम क्या _नहीं_ एकत्र करते

- कोई फ़ोल्डर पथ या फ़ोल्डर-से-व्युत्पन्न पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- कोई प्रति-रन लॉग, प्रॉम्प्ट, या अन्य CLI आउटपुट नहीं।

## इंस्टॉल संख्याएँ

ClawHub प्रति skill कुल काउंटर बनाए रखता है:

- `installsAllTime`: वे अद्वितीय उपयोगकर्ता जिन्होंने skill के लिए कम से कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे अद्वितीय उपयोगकर्ता जिन्होंने इंस्टॉल रिपोर्ट किया है और अपनी
  Telemetry नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

हर कोई केवल **एकत्रित इंस्टॉल काउंटर** देखता है।

आपका खाता हटाने से आपका Telemetry डेटा भी हट जाता है।

## Telemetry कैसे अक्षम करें

पर्यावरण चर सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

यह सेट होने पर, CLI इंस्टॉल Telemetry नहीं भेजेगा।
