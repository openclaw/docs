# Telemetry

ClawHub समेकित इंस्टॉल गणनाओं की गणना करने के लिए न्यूनतम CLI Telemetry का उपयोग करता है।

## Telemetry कब एकत्र की जाती है

Telemetry केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हों।
- आप `clawhub install <slug>` चलाते हैं।
- Telemetry **अक्षम नहीं है** (नीचे “अक्षम कैसे करें” देखें)।

अगर आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

प्रत्येक रिपोर्ट किए गए `clawhub install` पर, CLI एक सर्वोत्तम-प्रयास इंस्टॉल इवेंट भेजता है।

इवेंट में शामिल है:

- `slug`: इंस्टॉल किए गए skill का slug।
- `version`: इंस्टॉल किया गया संस्करण, जब ज्ञात हो।

### हम क्या एकत्र _नहीं_ करते

- कोई फ़ोल्डर पाथ या फ़ोल्डर से निकले पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- कोई प्रति-रन लॉग, प्रॉम्प्ट, या अन्य CLI आउटपुट नहीं।

## इंस्टॉल गणनाएं

ClawHub प्रति skill समेकित काउंटर बनाए रखता है:

- `installsAllTime`: वे विशिष्ट उपयोगकर्ता जिन्होंने skill के लिए कम से कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे विशिष्ट उपयोगकर्ता जिन्होंने कोई इंस्टॉल रिपोर्ट किया है और अपनी
  Telemetry नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

हर कोई केवल **समेकित इंस्टॉल काउंटर** देखता है।

आपका खाता हटाने से आपका Telemetry डेटा भी हट जाता है।

## Telemetry अक्षम कैसे करें

environment variable सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

इसके सेट होने पर, CLI इंस्टॉल Telemetry नहीं भेजेगा।
