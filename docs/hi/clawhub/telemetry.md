# टेलीमेट्री

ClawHub कुल इंस्टॉल गिनती की गणना करने के लिए न्यूनतम CLI टेलीमेट्री का उपयोग करता है।

## टेलीमेट्री कब एकत्र की जाती है

टेलीमेट्री केवल तब भेजी जाती है जब:

- आप CLI में लॉग इन हैं।
- आप `clawhub install <slug>` चलाते हैं।
- टेलीमेट्री **अक्षम नहीं है** (नीचे “अक्षम कैसे करें” देखें)।

यदि आप लॉग इन नहीं हैं, तो कुछ भी रिपोर्ट नहीं किया जाता।

## हम क्या एकत्र करते हैं

प्रत्येक रिपोर्ट किए गए `clawhub install` पर, CLI एक सर्वोत्तम-प्रयास इंस्टॉल ईवेंट भेजता है।

ईवेंट में शामिल होता है:

- `slug`: इंस्टॉल किए गए skill का slug।
- `version`: इंस्टॉल किया गया संस्करण, जब ज्ञात हो।

### हम क्या _नहीं_ एकत्र करते

- कोई फ़ोल्डर पथ या फ़ोल्डर-व्युत्पन्न पहचानकर्ता नहीं।
- कोई फ़ाइल सामग्री नहीं।
- कोई प्रति-रन लॉग, प्रॉम्प्ट, या अन्य CLI आउटपुट नहीं।

## इंस्टॉल गिनती

ClawHub प्रति skill कुल काउंटर रखता है:

- `installsAllTime`: वे अद्वितीय उपयोगकर्ता जिन्होंने skill के लिए कम से कम एक CLI इंस्टॉल रिपोर्ट किया है।
- `installsCurrent`: वे अद्वितीय उपयोगकर्ता जिन्होंने इंस्टॉल रिपोर्ट किया है और अपनी
  टेलीमेट्री नहीं हटाई है।

## पारदर्शिता + उपयोगकर्ता नियंत्रण

हर कोई केवल **कुल इंस्टॉल काउंटर** देखता है।

आपका खाता हटाने से आपका टेलीमेट्री डेटा भी हट जाता है।

## टेलीमेट्री अक्षम कैसे करें

पर्यावरण चर सेट करें:

```bash
export CLAWHUB_DISABLE_TELEMETRY=1
```

इसके सेट होने पर, CLI इंस्टॉल टेलीमेट्री नहीं भेजेगा।
