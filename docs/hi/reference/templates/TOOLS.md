# TOOLS.md - स्थानीय नोट्स

Skills परिभाषित करते हैं कि टूल _कैसे_ काम करते हैं। यह फ़ाइल _आपकी_ विशिष्ट बातों के लिए है — वे चीज़ें जो आपके सेटअप के लिए अनूठी हैं।

## यहाँ क्या जाता है

जैसी चीज़ें:

- कैमरा नाम और स्थान
- SSH होस्ट और उपनाम
- TTS के लिए पसंदीदा आवाज़ें
- स्पीकर/कमरे के नाम
- डिवाइस उपनाम
- परिवेश-विशिष्ट कुछ भी

## उदाहरण

```markdown
### कैमरे

- living-room → मुख्य क्षेत्र, 180° चौड़ा कोण
- front-door → प्रवेश द्वार, गति-ट्रिगर

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- पसंदीदा आवाज़: "Nova" (गर्मजोशी भरी, थोड़ी ब्रिटिश)
- डिफ़ॉल्ट स्पीकर: Kitchen HomePod
```

## अलग क्यों?

Skills साझा किए जाते हैं। आपका सेटअप आपका है। इन्हें अलग रखने का मतलब है कि आप अपने नोट्स खोए बिना Skills अपडेट कर सकते हैं, और अपना इंफ़्रास्ट्रक्चर लीक किए बिना Skills साझा कर सकते हैं।
