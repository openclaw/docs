# Plugin संदर्भ

यह पृष्ठ `extensions/*/package.json` और
`openclaw.plugin.json` से जनरेट किया गया है। इसे इसके साथ फिर से जनरेट करें:

```bash
pnpm plugins:inventory:gen
```

वितरण, पैकेज, और विवरण के अनुसार सभी 129 जनरेट किए गए plugin संदर्भ पृष्ठों को ब्राउज़ करने के लिए [Plugin इन्वेंट्री](/hi/plugins/plugin-inventory) का उपयोग करें।
