# Plugin संदर्भ

यह पेज `extensions/*/package.json` और
`openclaw.plugin.json` से जनरेट किया गया है। इसे इसके साथ फिर से जनरेट करें:

```bash
pnpm plugins:inventory:gen
```

वितरण, पैकेज, और विवरण के आधार पर सभी 130
जनरेट किए गए Plugin संदर्भ पेज ब्राउज़ करने के लिए [Plugin इन्वेंटरी](/hi/plugins/plugin-inventory) का उपयोग करें।
