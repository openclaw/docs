# Plugin संदर्भ

यह पृष्ठ `extensions/*/package.json` और
`openclaw.plugin.json` से जनरेट किया गया है। इसे इस कमांड से फिर से जनरेट करें:

```bash
pnpm plugins:inventory:gen
```

सभी 143 जनरेट किए गए Plugin संदर्भ पृष्ठों को वितरण, पैकेज और विवरण के अनुसार ब्राउज़ करने के लिए [Plugin इन्वेंटरी](/hi/plugins/plugin-inventory) का उपयोग करें।
