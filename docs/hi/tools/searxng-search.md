OpenClaw [SearXNG](https://docs.searxng.org/) को **स्वयं-होस्टेड,
कुंजी-मुक्त** `web_search` प्रदाता के रूप में समर्थित करता है। SearXNG एक ओपन-सोर्स मेटा-सर्च इंजन है
जो Google, Bing, DuckDuckGo और अन्य स्रोतों से परिणाम एकत्रित करता है।

लाभ:

- **निःशुल्क और असीमित** -- किसी API कुंजी या व्यावसायिक सदस्यता की आवश्यकता नहीं
- **गोपनीयता / एयर-गैप** -- क्वेरी कभी आपके नेटवर्क से बाहर नहीं जातीं
- **हर जगह काम करता है** -- व्यावसायिक खोज API पर कोई क्षेत्रीय प्रतिबंध नहीं

## सेटअप

<Steps>


  </Step>

## कॉन्फ़िगरेशन

```json5
{
  tools: {
    web: {
      search: {
        provider: "searxng",
      },
    },
  },
}
```

SearXNG इंस्टेंस के लिए Plugin-स्तरीय सेटिंग:

```json5
{
  plugins: {
    entries: {
      searxng: {
        config: {
          webSearch: {
            baseUrl: "http://localhost:8888",
            categories: "general,news", // वैकल्पिक
            language: "en", // वैकल्पिक
          },
        },
      },
    },
  },
}
```

`baseUrl` एक SecretRef ऑब्जेक्ट भी स्वीकार करता है (उदाहरण के लिए `{ source: "env", id: "SEARXNG_BASE_URL" }`)।

## एनवायरनमेंट वेरिएबल

कॉन्फ़िगरेशन के विकल्प के रूप में `SEARXNG_BASE_URL` सेट करें:

```bash
export SEARXNG_BASE_URL="http://localhost:8888"
```

समाधान क्रम: कॉन्फ़िगर की गई `baseUrl` स्ट्रिंग, फिर
`baseUrl` पर इनलाइन env SecretRef, और उसके बाद `SEARXNG_BASE_URL`। जब कोई भी कॉन्फ़िगरेशन पथ सेट नहीं होता और
कोई प्रदाता स्पष्ट रूप से चुने बिना `SEARXNG_BASE_URL` मौजूद होता है, तो स्वतः-पहचान
SearXNG को चुनती है।

## Plugin कॉन्फ़िगरेशन संदर्भ

| फ़ील्ड        | विवरण                                                        |
| ------------ | ------------------------------------------------------------------ |
| `baseUrl`    | आपके SearXNG इंस्टेंस का आधार URL (आवश्यक)                       |
| `categories` | कॉमा से अलग की गई श्रेणियाँ, जैसे `general`, `news`, या `science` |
| `language`   | परिणामों के लिए भाषा कोड, जैसे `en`, `de`, या `fr`              |

`web_search` टूल कॉल प्रति-कॉल ओवरराइड के रूप में `count` (1-10 परिणाम), `categories`,
और `language` भी स्वीकार करता है।

## टिप्पणियाँ

- **JSON API** -- HTML स्क्रैपिंग के बजाय SearXNG के मूल `format=json` एंडपॉइंट का उपयोग करता है
- **इमेज परिणाम URL** -- जब SearXNG प्रत्यक्ष इमेज URL लौटाता है, तो इमेज-श्रेणी के परिणामों में
  `img_src` शामिल होता है
- **कोई API कुंजी नहीं** -- किसी भी SearXNG इंस्टेंस के साथ बिना अतिरिक्त सेटअप के काम करता है
- **आधार URL सत्यापन** -- `baseUrl` एक मान्य `http://` या `https://`
  URL होना चाहिए
- **नेटवर्क सुरक्षा** -- `http://` आधार URL को किसी विश्वसनीय निजी या
  लूपबैक होस्ट को लक्षित करना चाहिए (सार्वजनिक होस्ट को `https://` का उपयोग करना होगा); निजी/आंतरिक पते में
  रिज़ॉल्व होने वाले `https://` आधार URL को भी वही स्वयं-होस्टेड अनुमति मिलती है,
  जबकि सार्वजनिक रूप से रिज़ॉल्व होने वाले `https://` आधार URL पर सख्त SSRF सुरक्षा लागू रहती है
- **स्वतः-पहचान क्रम** -- SearXNG के लिए कॉन्फ़िगर किया गया `baseUrl` आवश्यक है (उन प्रदाताओं के बीच क्रम
  200, जिनके पास उनकी आवश्यक क्रेडेंशियल पहले से मौजूद है)। DuckDuckGo या Ollama Web Search जैसे कुंजी-मुक्त
  प्रदाता स्वतः-पहचान में कभी भी स्वतः नहीं चुने जाते;
  वे केवल स्पष्ट `provider` चयन पर सक्रिय होते हैं
- **स्वयं-होस्टेड** -- इंस्टेंस, क्वेरी और अपस्ट्रीम खोज इंजन आपके नियंत्रण में रहते हैं
- **श्रेणियाँ** कॉन्फ़िगर न होने पर डिफ़ॉल्ट रूप से `general` होती हैं
- **श्रेणी फ़ॉलबैक** -- यदि कोई गैर-`general` श्रेणी अनुरोध सफल होता है, लेकिन
  शून्य परिणाम लौटाता है, तो OpenClaw खाली परिणाम सेट लौटाने से पहले
  उसी क्वेरी को `general` के साथ एक बार फिर आज़माता है
- **परिणाम कैशिंग** -- समान क्वेरी (समान क्वेरी, संख्या, श्रेणियाँ,
  भाषा और आधार URL) को छोटी TTL अवधि के लिए प्रक्रिया के भीतर कैश किया जाता है
- **संस्करण आवश्यकता** -- Plugin `minHostVersion: >=2026.6.9` घोषित करता है

<Tip>
  SearXNG JSON API के काम करने के लिए सुनिश्चित करें कि आपके SearXNG इंस्टेंस में `json`
  प्रारूप, `search.formats` के अंतर्गत उसके `settings.yml` में सक्षम हो।
</Tip>
