# TOOLS.md - Notes utilisateur sur les outils (modifiables)

Ce fichier est destiné à _vos_ notes sur les outils externes et les conventions. Il ne définit pas les outils disponibles ; OpenClaw fournit les outils intégrés en interne, et les Skills ajoutent les autres.

## Exemples

### imsg

- Envoyer un iMessage/SMS : décrire le destinataire et le contenu, puis demander confirmation avant l’envoi.
- Privilégier les messages courts ; éviter d’envoyer des secrets.

### sag

- Synthèse vocale : préciser la voix, le haut-parleur ou la pièce cible, et indiquer s’il faut diffuser en continu.
