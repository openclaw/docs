# Notes de version

Ces pages constituent la version destinée aux lecteurs des versions d’OpenClaw. Elles regroupent
le récit principal, l’impact pratique, les liens vers les sources et les crédits des contributeurs,
sans vous obliger à parcourir d’abord le journal des modifications brut.

## Versions

- [v2026.7.1](/releases/2026.7.1) - Refonte de l’interface de contrôle et de l’intégration initiale,
  mises à jour majeures d’iOS et d’Android, prise en charge étendue des modèles et récupération du Gateway.
- [v2026.6.11](/fr/releases/2026.6.11) - Correctifs de fiabilité pour les réponses, les envois,
  les reconnexions, la configuration des modèles, les sessions et des paramètres d’administration par défaut plus sûrs.
