# Inventaire des Plugin

Cette page est générée à partir de `extensions/*/package.json`, `openclaw.plugin.json`,
et des exclusions `files` du paquet npm racine. Régénérez-la avec :

```bash
pnpm plugins:inventory:gen
```

## Définitions

- **Paquet npm principal :** intégré au paquet npm `openclaw` et disponible sans installation séparée de Plugin.
- **Paquet externe officiel :** Plugin maintenu par OpenClaw, omis du paquet npm principal, conservé dans cet inventaire officiel et installé à la demande via ClawHub et/ou npm.
- **Copie de travail source uniquement :** Plugin local au dépôt, omis des artefacts npm publiés et non présenté comme un paquet installable.

Les copies de travail source diffèrent des installations npm : après `pnpm install`, les plugins groupés
se chargent depuis `extensions/<id>` afin que les modifications locales et les dépendances d’espace de travail propres au paquet soient disponibles.
