# Inventaire des Plugins

Cette page est générée à partir de `extensions/*/package.json`, de `openclaw.plugin.json`
et des exclusions `files` du paquet npm racine. Régénérez-la avec :

```bash
pnpm plugins:inventory:gen
```

## Définitions

- **Paquet npm principal :** intégré au paquet npm `openclaw` et disponible sans installation séparée de Plugin.
- **Paquet externe officiel :** Plugin maintenu par OpenClaw, omis du paquet npm principal, conservé dans cet inventaire officiel, et installé à la demande via ClawHub et/ou npm.
- **Extraction source uniquement :** Plugin local au dépôt, omis des artefacts npm publiés et non annoncé comme paquet installable.

Les extractions source diffèrent des installations npm : après `pnpm install`, les
Plugins groupés se chargent depuis `extensions/<id>`, afin que les modifications
locales et les dépendances d’espace de travail propres au paquet soient disponibles.

## Installer un Plugin

Utilisez la colonne **Distribution** pour décider si une installation est nécessaire. Les Plugins qui
indiquent `included in OpenClaw` sont déjà présents dans le paquet principal. Les paquets
externes officiels nécessitent une installation, puis un redémarrage du Gateway.

Par exemple, Discord est un paquet externe officiel :

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

Les spécifications de paquet nues essaient d’abord ClawHub, puis se rabattent sur npm. Pour forcer une source, utilisez
`clawhub:@openclaw/discord` ou `npm:@openclaw/discord`. Après l’installation, suivez
la documentation de configuration du Plugin, comme [Discord](/fr/channels/discord), pour ajouter les identifiants
et la configuration du canal. Consultez [Gérer les Plugins](/fr/plugins/manage-plugins) pour les commandes de mise à jour,
de désinstallation et de publication.
