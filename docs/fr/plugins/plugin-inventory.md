# Inventaire des Plugin

Cette page est générée à partir de `extensions/*/package.json`, `openclaw.plugin.json`,
et des exclusions `files` du package npm racine. Régénérez-la avec :

```bash
pnpm plugins:inventory:gen
```

## Définitions

- **Package npm principal :** intégré au package npm `openclaw` et disponible sans installation de Plugin séparée.
- **Package externe officiel :** Plugin maintenu par OpenClaw, omis du package npm principal et installé via ClawHub et/ou npm.
- **Checkout source uniquement :** Plugin local au dépôt, omis des artefacts npm publiés et non présenté comme package installable.

Les checkouts sources diffèrent des installations npm : après `pnpm install`, les Plugins intégrés
se chargent depuis `extensions/<id>`, ce qui rend disponibles les modifications locales et les dépendances d’espace de travail propres au package.
