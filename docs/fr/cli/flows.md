# `openclaw tasks flow`

Les commandes Flow sont des sous-commandes de `openclaw tasks`, et non une commande autonome `flows`.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Pour la documentation complète, voir [TaskFlow](/fr/automation/taskflow) et la [référence CLI de tasks](/fr/cli/tasks).
