# Plugin de transferencia de archivos

Obtén, enumera y escribe archivos en nodos emparejados mediante comandos de nodo específicos. Evita el truncamiento de la salida estándar de bash mediante el uso de base64 sobre node.invoke para archivos binarios de hasta 16 MB.
