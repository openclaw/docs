# TOOLS.md - Notas de herramientas del usuario (editable)

Este archivo está destinado a _tus_ notas sobre herramientas externas y convenciones. No define qué herramientas existen; OpenClaw proporciona internamente herramientas integradas y las Skills añaden el resto.

## Ejemplos

### imsg

- Enviar un iMessage/SMS: describe a quién y qué se enviará; solicita confirmación antes de enviarlo.
- Prioriza los mensajes breves; evita enviar secretos.

### sag

- Texto a voz: especifica la voz, el altavoz o la sala de destino y si se transmitirá en continuo.

Añade cualquier otra información que quieras que el asistente conozca sobre tu cadena de herramientas local.
