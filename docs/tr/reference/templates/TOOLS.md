# TOOLS.md - Yerel Notlar

Skills, araçların _nasıl_ çalıştığını tanımlar. Bu dosya ise _size_ özgü ayrıntılar içindir — kurulumunuza özgü olan şeyler.

## Buraya Neler Girer

Şunlar gibi şeyler:

- Kamera adları ve konumları
- SSH ana bilgisayarları ve takma adları
- TTS için tercih edilen sesler
- Hoparlör/oda adları
- Cihaz takma adları
- Ortama özgü her şey

## Örnekler

```markdown
### Kameralar

- living-room → Ana alan, 180° geniş açı
- front-door → Giriş, hareket tetiklemeli

### SSH

- home-server → 192.168.1.100, kullanıcı: admin

### TTS

- Tercih edilen ses: "Nova" (sıcak, hafif Britanya aksanlı)
- Varsayılan hoparlör: Kitchen HomePod
```

## Neden Ayrı?

Skills paylaşımlıdır. Kurulumunuz size özeldir. Bunları ayrı tutmak, notlarınızı kaybetmeden Skills'i güncelleyebilmeniz ve altyapınızı sızdırmadan Skills'i paylaşabilmeniz anlamına gelir.
