# `openclaw tasks flow`

Flow komutları bağımsız bir `flows` komutu değil, `openclaw tasks` alt komutlarıdır.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

Tam dokümantasyon için [Task Flow](/tr/automation/taskflow) ve [tasks CLI başvurusu](/cli/index#tasks) sayfalarına bakın.
