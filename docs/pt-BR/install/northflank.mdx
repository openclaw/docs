Implante o OpenClaw na Northflank com um modelo de um clique e acesse-o pela interface web Control UI. Este é o caminho mais fácil, "sem terminal no servidor": a Northflank executa o Gateway para você.

## Como começar

1. Clique em [Implantar o OpenClaw](https://northflank.com/stacks/deploy-openclaw) para abrir o modelo.
2. Crie uma [conta na Northflank](https://app.northflank.com/signup) se ainda não tiver uma.
3. Clique em **Deploy OpenClaw now**.
4. Defina a variável de ambiente obrigatória: `OPENCLAW_GATEWAY_TOKEN` (use um valor aleatório forte).
5. Clique em **Deploy stack** para compilar e executar o modelo do OpenClaw.
6. Aguarde a conclusão da implantação e clique em **View resources**.
7. Abra o serviço do OpenClaw.
8. Abra a URL pública do OpenClaw em `/openclaw` e conecte-se usando o segredo compartilhado configurado. Por padrão, este modelo usa `OPENCLAW_GATEWAY_TOKEN`; se você substituí-lo por autenticação por senha, use essa senha.

## O que você recebe

- Gateway do OpenClaw hospedado + Control UI
- Armazenamento persistente por meio de um volume da Northflank (`/data`), para que `openclaw.json`, os arquivos `auth-profiles.json` de cada agente, o estado de canais/provedores, as sessões e o espaço de trabalho sobrevivam a novas implantações

## Conectar um canal

Use a Control UI em `/openclaw` ou execute `openclaw onboard` via SSH para obter instruções de configuração de canais:

- [Telegram](/pt-BR/channels/telegram) (mais rápido, requer apenas um token de bot)
- [Discord](/pt-BR/channels/discord)
